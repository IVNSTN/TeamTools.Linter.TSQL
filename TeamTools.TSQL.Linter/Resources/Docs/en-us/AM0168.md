

# Non-unique column alias in output

|||
|-|-|
| Id | **AM0168**
| Mnemo | NONUNIQUE_COLUMN_ALIAS
| Severity | ⚠ Warning
| Category | [Ambiguity](./Category_Ambiguity.md)
| Source code | [UniqueColumnAliasRule.cs](../../../Rules/Ambiguity/UniqueColumnAliasRule.cs)

## Cause

<p id="descr">This rule is triggered if the select contains columns with duplicate names or aliases.</p>

## Examples

Bad

```sql
SELECT 
    1 + 1 as c,
    t.c
FROM tbl as t
```

Good

```sql
SELECT
    1 + 1 as c,
    t.c as d
FROM tbl as t
```
