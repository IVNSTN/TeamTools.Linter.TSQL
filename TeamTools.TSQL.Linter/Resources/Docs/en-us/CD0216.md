

# Constraint contains CAST calls instead of CONVERT

|||
|-|-|
| Id | **CD0216**
| Mnemo | CAST_IN_CONSTRAINT
| Severity | ⚠ Warning
| Category | [ContinuousDelivery](./Category_ContinuousDelivery.md)
| Source code | [CastInConstraintRule.cs](../../../Rules/ContinuousDeployment/CastInConstraintRule.cs)

## Cause

This rule is triggered if `CAST` function is used in `CHECK` or `DEFAULT` constraints.
<p id="descr">Use CONVERT function.</p>

## Examples

Bad

```sql
CREATE TABLE foo
(
    bar INT NOT NULL DEFAULT CAST('123' AS INT)
);
```

Good

```sql
CREATE TABLE foo
(
    bar INT NOT NULL DEFAULT CONVERT(INT, '123')
);
```

## Tips

💡 The rule does not apply to temporary tables or table variables.
