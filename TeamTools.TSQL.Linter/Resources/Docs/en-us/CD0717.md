

# CREATE OR ALTER statement is forbidden, use CREATE

|||
|-|-|
| Id | **CD0717**
| Mnemo | CREATE_OR_ALTER
| Severity | ⛔ Error
| Category | [ContinuousDelivery](./Category_ContinuousDelivery.md)
| Source code | [CreateOrAlterRule.cs](../../../Rules/ContinuousDeployment/CreateOrAlterRule.cs)

## Cause

This rule is triggered if the `CREATE OR ALTER` expressions is used in an object creation script.
<p id="descr"> Use CREATE instead of CREATE OR ALTER.</p>

## Examples

Bad

```sql
CREATE OR ALTER PROCEDURE foo
AS
BEGIN
    SET NOCOUNT ON;

    RETURN 0;
END;
GO
```

Good

```sql
CREATE PROCEDURE foo
AS
BEGIN
    SET NOCOUNT ON;

    RETURN 0;
END;
GO
```
