

# ALTER should be rewritten into table definition

|||
|-|-|
| Id | **CD0725**
| Mnemo | ALTER_TO_DEFINITION
| Severity | ⚠ Warning
| Category | [ContinuousDelivery](./Category_ContinuousDelivery.md)
| Source code | [AlterTableToInlineDefinitionRule.cs](../../../Rules/ContinuousDeployment/AlterTableToInlineDefinitionRule.cs)

## Cause

Triggered when using ALTER in the body of the table creation script to add fields/indexes/constraints.
<p id="descr">Move the ALTER-ADD expressions to the table definition.</p>

## Examples

Bad

```sql
CREATE TABLE foo 
(
    bar INT NOT NULL
)
GO

ALTER TABLE foo ADD mar INT NOT NULL
GO

ALTER TABLE foo ADD CONSTRAINT cns DEFAULT 0
```

Good

```sql
CREATE TABLE foo 
(
    bar INT NOT NULL CONSTRAINT cns DEFAULT 0
    , mar INT NOT NULL
)
GO
```
