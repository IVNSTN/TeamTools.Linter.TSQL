

# Argument passed by position, not by name

|||
|-|-|
| Id | **CS0102**
| Mnemo | POSITIONAL_PROC_ARGS
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [PositionalProcArgumentsRule.cs](../../../Rules/CodeSmell/PositionalProcArgumentsRule.cs)

## Cause

This rule is triggered if a stored procedure call is detected with parameters passed by position rather than by name.
<p id="descr">Pass parameters by name.</p>

## Examples

Bad

```sql
EXEC dbo.test
    @var_a
    , @b = @var_b;
```

Good

```sql
EXEC dbo.test
    @a = @var_a
    , @b = @var_b;
```
