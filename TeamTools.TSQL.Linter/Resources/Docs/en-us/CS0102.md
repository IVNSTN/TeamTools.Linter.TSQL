

# Argument passed by position, not by name

|||
|-|-|
| Id | **CS0102**
| Mnemo | POSITIONAL_PROC_ARGS
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [PositionalProcArgumentsRule.cs](../../../Rules/CodeSmell/PositionalProcArgumentsRule.cs)

## Cause

<p id="descr">При вызове хп обязательно нужно передавать параметры по имени, а не по позиции ( exec myproc @x = @y). На системные хп не реагирует</p>
