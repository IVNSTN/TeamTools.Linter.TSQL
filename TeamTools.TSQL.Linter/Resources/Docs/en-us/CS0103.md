

# Insert without column list

|||
|-|-|
| Id | **CS0103**
| Mnemo | INSERT_COLUMN_LIST
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [InsertWithoutColumnsRule.cs](../../../Rules/CodeSmell/InsertWithoutColumnsRule.cs)

## Cause

This rule is triggered if the target table fields are not listed during insertion (`INSERT INTO`, `OUTPUT INTO`).
<p id="descr">List the target table fields.</p>

## Examples

Bad

```sql
INSERT INTO dbo.foo
SELECT a, b, c
FROM tmp.bar
```

Good

```sql
INSERT INTO dbo.foo(a, b, c)
SELECT a, b, c
FROM tmp.bar;
```
