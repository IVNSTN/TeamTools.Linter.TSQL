

# Length or precision and scale should be defined for type

|||
|-|-|
| Id | **CS0111**
| Mnemo | VAR_TYPE_LENGTH
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [VariableSizeTypeUsageRule.cs](../../../Rules/CodeSmell/VariableSizeTypeUsageRule.cs)

## Cause

<p id="descr">В типах допускающих указание размера или точности (строки, нецелые числовые и т.д.) обязательно нужно это указывать.</p>
