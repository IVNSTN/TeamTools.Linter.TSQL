

# Length or precision and scale should be defined for data type

|||
|-|-|
| Id | **CS0111**
| Mnemo | VAR_TYPE_LENGTH
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [VariableSizeTypeUsageRule.cs](../../../Rules/CodeSmell/VariableSizeTypeUsageRule.cs)

## Cause

This rule is triggered if a type that allows specifying a size or precision (strings, non-integer numerics, etc.) is used without specifying this characteristic.
<p id="descr">Specify the data type size or precision.</p>

## Examples

Bad

```sql
DECLARE @foo VARCHAR
    , @dt datetime2
```

Good

```sql
DECLARE @foo VARCHAR(96)
    , @dt datetime2(7)
```
