

# Select * is forbidden

|||
|-|-|
| Id | **CS0112**
| Mnemo | SELECT_STAR
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SelectStarRule.cs](../../../Rules/CodeSmell/SelectStarRule.cs)

## Cause

This rule is triggered if a `SELECT *` query is detected. It is also triggered when used inside the `EXISTS` subquery.
<p id="descr">Replace * with the required columns.</p>

## Examples

Bad

```sql
SELECT a
FROM dbo.foo
WHERE EXISTS(SELECT * FROM bar WHERE b = 10)
```

Good

```sql
SELECT a
FROM dbo.foo
WHERE EXISTS(SELECT 1 FROM bar WHERE b = 10)
```
