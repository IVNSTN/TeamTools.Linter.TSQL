

# Not authorized schema creation

|||
|-|-|
| Id | **CS0118**
| Mnemo | AUTHORIZED_SCHEMA
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SchemaAuthorizedRule.cs](../../../Rules/CodeSmell/SchemaAuthorizedRule.cs)

## Cause

This rule is triggered if the owner is not specified in the schema creation script.
<p id="descr">Specify the schema owner.</p>

## Examples

Bad

```sql
CREATE SCHEMA test
```

Good

```sql
CREATE SCHEMA test
AUTHORIZATION dbo
```
