

# Unspecified grantor

|||
|-|-|
| Id | **CS0125**
| Mnemo | GRANT_GRANTOR
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [GrantorRule.cs](../../../Rules/CodeSmell/GrantorRule.cs)

## Cause

This rule is triggered if the grantor is not specified in the script for issuing rights to the object.
<p id="descr">Specify the grantor.</p>

## Examples

Bad

```sql
GRANT SELECT ON object::foo.bar
TO asdf;
```

Good

```sql
GRANT SELECT ON object::foo.bar
TO asdf
AS [dbo];
```

## Tips

💡 Most often it is dbo.

💡 For schemas, types and services, you do not need to specify grantor ([CD0213](./CD0213.md)).
