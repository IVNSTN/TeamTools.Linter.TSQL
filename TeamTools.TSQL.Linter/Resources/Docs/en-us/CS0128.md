

# Index on different table in create table script

|||
|-|-|
| Id | **CS0128**
| Mnemo | INDEX_MISDIRECTED
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [IndexOnGivenTableOnlyRule.cs](../../../Rules/CodeSmell/IndexOnGivenTableOnlyRule.cs)

## Cause

This rule is triggered if a different table is specified for the index in the table creation script.
<p id="descr">Please specify a valid table name.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
);

CREATE INDEX ix_foo
ON dbo.bar(id)
ON [PRIMARY];
```

Good

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
);

CREATE INDEX ix_foo
ON dbo.foo(id)
ON [PRIMARY];
```
