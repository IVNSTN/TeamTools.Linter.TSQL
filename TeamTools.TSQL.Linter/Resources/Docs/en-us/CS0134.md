

# Forbidden RAISERROR option

|||
|-|-|
| Id | **CS0134**
| Mnemo | RAISERROR_OPTIONS
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [RaisErrorOptionsRule.cs](../../../Rules/CodeSmell/RaisErrorOptionsRule.cs)

## Cause

This rule is triggered if the use of the forbidden `RAISERROR` option is detected.
<p id="descr">Remove the forbidden RAISERROR option.</p>

## Examples

Bad

```sql
RAISERROR ('asdf', 10, 1) WITH LOG;
RAISERROR ('asdf', 10, 1) WITH SETERROR;
```

Good

```sql
RAISERROR ('asdf', 10, 1);
```

## Tips

💡 Forbidden `RAISERROR` options:

- `SETERROR`
- `LOG`
