

# Forbidden hint

|||
|-|-|
| Id | **CS0135**
| Mnemo | BAD_QUERY_HINT
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [QueryHintsRule.cs](../../../Rules/CodeSmell/QueryHintsRule.cs)

## Cause

This rule is triggered when a table-level or query-level forbidden query `HINT` is detected.
<p id="descr">Remove the forbidden hint.</p>

## Examples

Bad

```sql
SELECT bar.aa FROM dbo.bar AS bar WITH (NOLOCK, INDEX = asfd)
    INNER JOIN dbo.foo AS foo WITH (IGNORE_TRIGGERS) ON bar.aa = foo.bb
OPTION (FAST 10, FORCE ORDER);
```

Good

```sql
SELECT bar.aa FROM dbo.bar AS bar WITH (INDEX = asfd)
    INNER JOIN dbo.foo AS foo ON bar.aa = foo.bb;
```

## Tips

💡 Forbidden query hints:

- `NOLOCK`
- `IGNORE_TRIGGERS`
- `FAST <N>`
