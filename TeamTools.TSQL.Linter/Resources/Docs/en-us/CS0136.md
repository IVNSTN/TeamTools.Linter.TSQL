

# READ UNCOMMITTED isolation level used

|||
|-|-|
| Id | **CS0136**
| Mnemo | ISOLATION_UNCOMMITTED
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [IsolationLevelUncommittedRule.cs](../../../Rules/CodeSmell/IsolationLevelUncommittedRule.cs)

## Cause

This rule is triggered if the code tries to use the `READ UNCOMMITTED` transaction isolation level.
<p id="descr">Use a different transaction isolation level.</p>

## Examples

Bad

```sql
SET TRANSACTION ISOLATION LEVEL READ UNCOMMITTED
```

Good

```sql
SET TRANSACTION ISOLATION LEVEL REPEATABLE READ
```
