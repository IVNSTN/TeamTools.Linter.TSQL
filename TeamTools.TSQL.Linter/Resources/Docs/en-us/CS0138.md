

# View with output containing SELECT *

|||
|-|-|
| Id | **CS0138**
| Mnemo | VIEW_SELECT_STAR_OUTPUT
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SelectStarInViewRule.cs](../../../Rules/CodeSmell/SelectStarInViewRule.cs)

## Cause

This rule is triggered if a `SELECT *` query is detected in a view definition.
<p id="descr">Replace * with the required columns.</p>

## Examples

Bad

```sql
CREATE VIEW dbo.bar
AS
    SELECT *, d.a FROM dbo.foo as d;
```

Good

```sql
CREATE VIEW dbo.bar
AS
    SELECT d.a, d.b, d.c FROM dbo.foo as d;
```
