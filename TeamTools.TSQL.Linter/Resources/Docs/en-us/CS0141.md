

# GETDATE() is used for DATETIME2

|||
|-|-|
| Id | **CS0141**
| Mnemo | DATETIME2_SYSDATETIME
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [Datetime2SysdatetimeRule.cs](../../../Rules/CodeSmell/Datetime2SysdatetimeRule.cs)

## Cause

This rule is triggered if an attempt to populate a variable or column of type `DATETIME2` using the `GETDATE()` function is detected.
<p id="descr">Use SYSDATETIME().</p>

## Examples

Bad

```sql
DECLARE
    @dt2 DATETIME2 = GETDATE()
```

Good

```sql
DECLARE
    @dt1   DATETIME  = GETDATE()
    , @dt2 DATETIME2 = SYSDATETIME()
```
