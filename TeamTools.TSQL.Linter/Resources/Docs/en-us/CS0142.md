

# Unspecified RETURN value

|||
|-|-|
| Id | **CS0142**
| Mnemo | PROC_RETURN_VALUE_REQUIRED
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [ReturnValueFromProcRule.cs](../../../Rules/CodeSmell/ReturnValueFromProcRule.cs)

## Cause

This rule is triggered if a use of `RETURN` without an explicit return value is detected.
<p id="descr">Set value for RETURN.</p>

## Examples

Bad

```sql
CREATE PROCEDURE dbo.foo
AS
BEGIN
    SELECT 0;

    RETURN; -- no return value;
END;
```

Good

```sql
CREATE PROCEDURE dbo.foo
AS
BEGIN
    SELECT 0;

    RETURN 1;
END;
GO

ALTER PROCEDURE dbo.foo
AS
BEGIN
    -- return not used;
    SELECT 0;
END;
```
