

# SYSNAME used instead of NVARCHAR with specific length for scalar variable

|||
|-|-|
| Id | **CS0146**
| Mnemo | SYSNAME_FOR_SCALAR_VAR
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SysnameTypeForScalarVariableRule.cs](../../../Rules/CodeSmell/SysnameTypeForScalarVariableRule.cs)

## Cause

<p id="descr">В переменной могут храниться <em>строки необходимой длины и битности.</em> Строковые типы - это VARCHAR, NVARCHAR, CHAR, NCHAR. Выбирайте нужный вам тип, размер и явно его указывайте.</p>
