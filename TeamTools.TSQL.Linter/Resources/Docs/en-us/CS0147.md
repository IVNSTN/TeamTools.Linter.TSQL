

# SYSNAME used instead of NVARCHAR with specific length for table col

|||
|-|-|
| Id | **CS0147**
| Mnemo | SYSNAME_FOR_TABLE_COL
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SysnameTypeForTableColRule.cs](../../../Rules/CodeSmell/SysnameTypeForTableColRule.cs)

## Cause

<p id="descr">В таблице могут храниться <em>строки необходимой длины и битности.</em> Строковые типы - это VARCHAR, NVARCHAR, CHAR, NCHAR. Выбирайте нужный вам тип, размер и явно его указывайте.</p>
