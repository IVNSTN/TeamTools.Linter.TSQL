

# Redundant DISTINCT after GROUP BY

|||
|-|-|
| Id | **CS0152**
| Mnemo | GROUPBY_DISTINCT
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [DistinctAfterGroupByRule.cs](../../../Rules/CodeSmell/DistinctAfterGroupByRule.cs)

## Cause

The rule is triggered if DISTINCT is used on a field that has been grouped by.
<p id="descr">Remove the logically unnecessary DISTINCT.</p>

## Examples

Bad

```sql
SELECT DISTINCT risk_level_id FROM dbo.risk_levels GROUP BY risk_level_id;
```

Good

```sql
SELECT risk_level_id FROM dbo.risk_levels GROUP BY risk_level_id;
```
