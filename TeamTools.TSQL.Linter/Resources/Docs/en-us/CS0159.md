
# Non-ANSI NULL comparison

|||
|-|-|
| Id | **CS0159**
| Mnemo | NON_ANSI_NULL_COMPARISON
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [NonAnsiNullOperationRule.cs](../../../Rules/CodeSmell/NonAnsiNullOperationRule.cs)

## Cause

The rule is triggered if there are non-ANSI comparisons = NULL, <> NULL, which will never work as expected.
<p id="descr">Rewrite using IS (NOT) NULL.</p>

## Examples

Bad

```sql
SELECT IIF('' <> NULL, 1, 2) -- the result is 2, not 1

SELECT IIF(NULL = NULL, 1, 2) -- the result is 2, not 1
```

Good

```sql
SELECT IIF('' IS NOT NULL, 1, 2)

SELECT IIF(NULL IS NULL, 1, 2)
```
