
# Output parameter never assigned

|||
|-|-|
| Id | **CS0160**
| Mnemo | OUTPUT_UNASSIGNED
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [OutputParameterNeverAssignedRule.cs](../../../Rules/CodeSmell/OutputParameterNeverAssignedRule.cs)

## Cause

The rule is triggered if the OUTPUT parameter is never assigned a value.
<p id="descr">Remove the parameter or assign a value to it.</p>

## Examples

Bad

```sql
CREATE PROCEDURE procedure_name
    @x INT OUTPUT
AS
BEGIN
    -- The logic of the procedure, without assigning a value to the variable @x
END
END;
```

Good

```sql
CREATE PROCEDURE procedure_name
    @x INT OUTPUT
AS
BEGIN
    -- The logic of the procedure, without assigning a value to the variable @x

    SET @x = 1;
END

CREATE PROCEDURE procedure_name
AS
BEGIN
    -- The logic of the procedure, without assigning a value to the variable @x
END


CREATE PROCEDURE procedure_name
    @x INT OUTPUT
AS
BEGIN
    SET @x = NULL;
    -- ЛThe logic of the procedure, without assigning a value to the variable @x
END
```
