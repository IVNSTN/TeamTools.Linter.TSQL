
# Unreachable code

|||
|-|-|
| Id | **CS0161**
| Mnemo | UNREACHABLE_CODE
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [UnreachableCodeRule.cs](../../../Rules/CodeSmell/UnreachableCodeRule.cs)

## Cause

The rule is triggered if there is code that will never be executed.
<p id="descr">Delete the code that is never executed.</p>

## Examples

Bad

```sql
CREATE PROCEDURE dbo.proc_name2
AS
BEGIN
--some code
RETURN 0;

EXEC dbo.proc_name1;
END;
```

Good

```sql
CREATE PROCEDURE dbo.proc_name2
AS
BEGIN
--some code
RETURN 0;
END;
```
