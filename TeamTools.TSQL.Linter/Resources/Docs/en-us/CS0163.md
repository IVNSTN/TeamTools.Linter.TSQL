
# Unexpected data fetching from a trigger

|||
|-|-|
| Id | **CS0163**
| Mnemo | OUTPUT_FROM_TRIGGER
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md), [Triggers](./Group_Triggers.md)
| Source code | [FetchingDataFromTriggerRule.cs](../../../Rules/CodeSmell/FetchingDataFromTriggerRule.cs)

## Cause

The rule is triggered if the trigger outputs data using the SELECT or PRINT commands.
<p id="descr">Remove data output from the trigger.</p>

## Examples

Bad

```sql
CREATE TRIGGER dbo.table_name_iu_repl
ON dbo.table_name
FOR INSERT, UPDATE
AS
BEGIN
   --some code
   print 'abc'
   select 'abс'
END;
```

Good

```sql
CREATE TRIGGER dbo.table_name_iu_repl
ON dbo.table_name
FOR INSERT, UPDATE
AS
BEGIN
   --некоторый код
END;
```
