

# Permission misdirected

|||
|-|-|
| Id | **CS0164**
| Mnemo | PERMISSION_MISDIRECTED
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [PermissionMisdirectedRule.cs](../../../Rules/CodeSmell/PermissionMisdirectedRule.cs)

## Cause

The rule is triggered if the grant is issued to a role other than the one it is defined in the script.
<p id="descr">Specify the grant in the file of the role to which it is issued.</p>

## Examples

Bad

```sql
CREATE ROLE Risk AUTHORIZATION dbo;
GO

GRANT SELECT ON SCHEMA::export TO Robots;
GO
```

Good

```sql
CREATE ROLE Robots AUTHORIZATION dbo;
GO

GRANT SELECT ON SCHEMA::export TO Robots;
GO
```
