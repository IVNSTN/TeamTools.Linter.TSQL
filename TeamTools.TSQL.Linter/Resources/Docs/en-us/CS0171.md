
# Fully qualified column instead of two-part links

|||
|-|-|
| Id | **CS0171**
| Mnemo | COLUMN_FULLY_QUALIFIED
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [ColumnFullQualifiedReferenceRule.cs](../../../Rules/CodeSmell/ColumnFullQualifiedReferenceRule.cs)

## Cause

The rule is triggered if the link to the table column contains the name of the database or even the server.
<p id="descr">Specify the column name without the database and server name.</p>

## Examples

Bad

```sql
SELECT bd_name.dbo.table_name.column_name FROM dbo.table_name
```

Good

```sql
SELECT column_name FROM dbo.table_name

SELECT tbl.column_name FROM dbo.table_name AS tbl
```
