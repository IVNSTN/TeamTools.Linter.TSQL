

# This loop can never run more than once

|||
|-|-|
| Id | **CS0172**
| Mnemo | DISPOSABLE_LOOP
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [UnrepeatableLoopRule.cs](../../../Rules/CodeSmell/UnrepeatableLoopRule.cs)

## Cause

The rule is triggered if there is an unconditional exit command in the body of the loop.
<p id="descr">Specify the condition for exiting the loop.</p>

## Examples

Bad

```sql
WHILE(1=1)
BEGIN
exec dbo.proc;

break;
END
```

Good

```sql
WHILE(1=1)
BEGIN
    exec dbo.proc

    if(NOT EXISTS(select 1 from dbo.table))
    BEGIN
        break;
    END;
END
```

## Tips

💡 If only one iteration is truly needed, avoid using a loop. If there is a condition for exiting the loop, specify it.
