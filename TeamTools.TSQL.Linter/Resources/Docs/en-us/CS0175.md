

# Invalid error number value

|||
|-|-|
| Id | **CS0175**
| Mnemo | RAISERROR_VALID_NUMBER
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [RaisErrorNumberValueRule.cs](../../../Rules/CodeSmell/RaisErrorNumberValueRule.cs)

## Cause

The rule is triggered if ERROR_NUMBER is outside the range 50 000 - 9 999 999.
<p id="descr">Specify the ERROR_NUMBER value from the range 50 000 - 9 999 999.</p>

## Examples

Bad

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorNumber INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorState = 16;
    SET @ErrorNumber = 1;

    THROW @ErrorNumber, @ErrorMessage, @ErrorState;
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_NUMBER() AS ErrorNumber;
END CATCH;
```

Good

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorNumber INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorState = 16;
    SET @ErrorNumber = 50001;

    THROW @ErrorNumber, @ErrorMessage, @ErrorState;
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_NUMBER() AS ErrorNumber;
END CATCH;
```
