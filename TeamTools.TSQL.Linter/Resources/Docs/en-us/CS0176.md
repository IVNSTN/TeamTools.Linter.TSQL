
# Invalid error severity value

|||
|-|-|
| Id | **CS0176**
| Mnemo | RAISERROR_VALID_SEVERITY
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [RaisErrorSeverityValueRule.cs](../../../Rules/CodeSmell/RaisErrorSeverityValueRule.cs)

## Cause

The rule is triggered if the ERROR_SEVERITY values are outside the range of 0-17.
<p id="descr">Specify the ERROR_SEVERITY value from the range 0-17.</p>

## Examples

Bad

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorSeverity INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorSeverity = 16;
    SET @ErrorState = 300;

    RAISERROR(@ErrorMessage, @ErrorSeverity, @ErrorState);
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_STATE() AS ErrorState;
END CATCH;
```

Good

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorSeverity INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorSeverity = 16;
    SET @ErrorState = 122;

    RAISERROR(@ErrorMessage, @ErrorSeverity, @ErrorState);
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_STATE() AS ErrorState;
END CATCH;
```
