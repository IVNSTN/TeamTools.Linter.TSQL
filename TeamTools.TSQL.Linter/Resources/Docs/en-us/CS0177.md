
# Invalid error state value

|||
|-|-|
| Id | **CS0177**
| Mnemo | RAISERROR_VALID_STATE
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [RaisErrorStateValueRule.cs](../../../Rules/CodeSmell/RaisErrorStateValueRule.cs)

## Cause

The rule is triggered if the ERROR_STATE values are outside the range of 0-255.
<p id="descr">Specify the ERROR_STATE value from the range 0-255.</p>

## Examples

Bad

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorNumber INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorState = 300;
    SET @ErrorNumber = 51000;

    THROW @ErrorNumber, @ErrorMessage, @ErrorState;
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_STATE() AS ErrorState;
END CATCH;
```

Good

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorNumber INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorState = 300;
    SET @ErrorNumber = 51000;

    THROW @ErrorNumber, @ErrorMessage, @ErrorState;
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_STATE() AS ErrorState;
END CATCH;
```
