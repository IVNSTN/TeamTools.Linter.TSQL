

# @@IDENTITY is used instead of SCOPE_IDENTITY()

|||
|-|-|
| Id | **CS0181**
| Mnemo | IDENTITY_FUNCTION
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [IdentityFunctionRule.cs](../../../Rules/CodeSmell/IdentityFunctionRule.cs)

## Cause

This rule is triggered when @@IDENTITY is used in code.
<p id="descr">Replace @@IDENTITY with SCOPE_IDENTITY().</p>

## Examples

Bad

```sql
SET @id = @@IDENTITY
```

Good

```sql
SET @id = SCOPE_IDENTITY()
```
