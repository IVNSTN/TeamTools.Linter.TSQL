

# Raiserror instead of THROW in trigger

|||
|-|-|
| Id | **CS0186**
| Mnemo | RAISERROR_IN_TRIGGER
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [RaisErrorInTriggerRule.cs](../../../Rules/CodeSmell/RaisErrorInTriggerRule.cs)

## Cause

<p id="descr">Raiserror в триггере без желания откатить транзакцию - это не очень понятный сценарий. А с желанием откатить - придется и делать самому откат, и RETURN явно вызвать. THROW из коробки обеспечивает и выход из триггера, и перевод транзакции в нужный статус.</p>
