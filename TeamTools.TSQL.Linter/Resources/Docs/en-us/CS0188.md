

# Security context control in trigger

|||
|-|-|
| Id | **CS0188**
| Mnemo | USER_CONTEXT_CONTROL_IN_TRIGGER
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md), [Triggers](./Group_Triggers.md)
| Source code | [UserContextControlInTriggerRule.cs](../../../Rules/CodeSmell/UserContextControlInTriggerRule.cs)

## Cause

This rule is triggered when security context control is used in trigger.
<p id="descr">Remove security context control.</p>

## Examples

Bad

```sql
CREATE TRIGGER trg ON dbo.foo
WITH EXECUTE AS 'user'
AFTER UPDATE
AS
BEGIN
    ...
END;
```

Good

```sql
CREATE TRIGGER trg ON dbo.foo
AFTER UPDATE
AS
BEGIN
    ...
END;
```
