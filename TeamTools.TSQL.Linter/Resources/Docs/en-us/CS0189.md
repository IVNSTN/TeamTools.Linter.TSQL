

# WAITFOR in triggers

|||
|-|-|
| Id | **CS0189**
| Mnemo | WAITFOR_IN_TRIGGER
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md), [Triggers](./Group_Triggers.md)
| Source code | [WaitForInTriggerRule.cs](../../../Rules/CodeSmell/WaitForInTriggerRule.cs)

## Cause

This rule is triggered when WAITFOR is used in trigger.
<p id="descr">Remove WAITFOR.</p>

## Examples

Bad

```sql
CREATE TRIGGER trg ON dbo.foo
AFTER UPDATE
AS
BEGIN
    WAITFOR DELAY '00:01';
END;
```

Good

```sql
CREATE TRIGGER trg ON dbo.foo
AFTER UPDATE
AS
BEGIN
    ...
END;
```
