

# Reading from empty table

|||
|-|-|
| Id | **CS0190**
| Mnemo | ALWAYS_EMPTY_SOURCE
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [AlwaysEmptyTableAsSourceRule.cs](../../../Rules/CodeSmell/AlwaysEmptyTableAsSourceRule.cs)

## Cause

This rule is triggered when reading from obviously unfilled table is used in code.
<p id="descr">Fill the table or remove the reading.</p>

## Examples

Bad

```sql
CREATE TABLE #test (name varchar(100));

SELECT name
FROM #test;
```

Good

```sql
CREATE TABLE #test (name varchar(100));

INSERT INTO #test (name)
VALUES ('Name');

SELECT name
FROM #test;
```

## Tips

💡 Triggered when reading from DELETED in an INSERT trigger or from INSERTED in a DELETE trigger is used in code.

💡 The temporary table can be filled outside of the module.
