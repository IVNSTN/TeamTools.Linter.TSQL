

# Source table filled but never used

|||
|-|-|
| Id | **CS0191**
| Mnemo | NEVER_USED_SOURCE
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [TableNeverUsedAsSourceRule.cs](../../../Rules/CodeSmell/TableNeverUsedAsSourceRule.cs)

## Cause

This rule is triggered when temporary table or table variable filled but is never used as a data source.
<p id="descr">Remove an unused temporary table or table variable.</p>

## Examples

Bad

```sql
CREATE TABLE #test (name varchar(100));

INSERT INTO #test (name)
VALUES ('Name');
```

Good

```sql
CREATE TABLE #test (name varchar(100));

INSERT INTO #test (name)
VALUES ('Name');

SELECT TOP(1) @firstName = name
FROM #test;
```

## Tips

💡 The temporary table can be used outside of the module.
