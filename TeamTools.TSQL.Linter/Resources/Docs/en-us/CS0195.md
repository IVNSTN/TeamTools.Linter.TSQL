

# Cursor isn't declared as LOCAL

|||
|-|-|
| Id | **CS0195**
| Mnemo | CURSOR_LOCAL
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md), [Cursors](./Group_Cursors.md)
| Source code | [CursorScopeRule.cs](../../../Rules/CodeSmell/CursorScopeRule.cs)

## Cause

This rule is triggered when cursor isn't declared as LOCAL.
<p id="descr">Set LOCAL option.</p>

## Examples

Bad

```sql
DECLARE cr CURSOR GLOBAL FOR
    SELECT 1;
```

Good

```sql
DECLARE cr CURSOR LOCAL FOR
    SELECT 1;
```
