

# Cursor isn't declared explicitly as READONLY or FOR UPDATE

|||
|-|-|
| Id | **CS0196**
| Mnemo | CURSOR_READONLY
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md), [Cursors](./Group_Cursors.md)
| Source code | [CursorReadonlyRule.cs](../../../Rules/CodeSmell/CursorReadonlyRule.cs)

## Cause

This rule is triggered when cursor isn't declared explicitly as READONLY or FOR UPDATE.
<p id="descr">Set explicitly READONLY or FOR UPDATE.</p>

## Examples

Bad

```sql
DECLARE cr CURSOR FOR
    SELECT 1;
```

Good

```sql
DECLARE cr CURSOR READ_ONLY FOR
    SELECT 1;
```
