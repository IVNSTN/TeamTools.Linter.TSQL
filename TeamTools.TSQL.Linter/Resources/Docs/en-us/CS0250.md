

# Redundant unary operator

|||
|-|-|
| Id | **CS0250**
| Mnemo | REDUNDANT_OPERATOR
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [UnaryOperatorRepeatedRule.cs](../../../Rules/CodeSmell/UnaryOperatorRepeatedRule.cs)

## Cause

<p id="descr">Повторяющиеся унарные операторы - либо результат опечатки и, возможно, приводят к некорректному поведению (двойное отрицание, двойной минус), либо изначально используется не тот знак или логический оператор.</p>
