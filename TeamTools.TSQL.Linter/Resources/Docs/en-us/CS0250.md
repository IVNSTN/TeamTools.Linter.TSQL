

# Redundant unary operator

|||
|-|-|
| Id | **CS0250**
| Mnemo | REDUNDANT_OPERATOR
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [UnaryOperatorRepeatedRule.cs](../../../Rules/CodeSmell/UnaryOperatorRepeatedRule.cs)

## Cause

This rule is triggered when unary operator is redundant.
<p id="descr">Remove redundant unary operator.</p>

## Examples

Bad

```sql
SELECT
    - - SUM(a + b / 3)
WHERE NOT NOT (1=1)
```

Good

```sql
SELECT
    -SUM(a + b / 3)
WHERE NOT (1=1)
```

## Tips

💡 The rule checks repetition of operators in expression.
