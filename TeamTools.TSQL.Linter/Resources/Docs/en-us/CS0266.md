

# Table-level constraint defined not at table level

|||
|-|-|
| Id | **CS0266**
| Mnemo | TABLE_LEVEL_CONSTRAINT_IN_COL
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [TableLevelConstraintInColumnRule.cs](../../../Rules/CodeSmell/TableLevelConstraintInColumnRule.cs)

## Cause

<p id="descr">Констрейнты CHECK, PRIMARY KEY, FOREIGN KEY нужно описывать отдельными строками; выражение должно быть отделено от ближайшей колонки запятой.</p>
