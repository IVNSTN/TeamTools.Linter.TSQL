

# ORDER BY uses column position

|||
|-|-|
| Id | **CS0295**
| Mnemo | ORDER_BY_POSITION
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [OrderByPositionRule.cs](../../../Rules/CodeSmell/OrderByPositionRule.cs)

## Cause

This rule is triggered when ORDER BY uses column position instead of column name.
<p id="descr">Use column name instead of column position.</p>

## Examples

Bad

```sql
SELECT a, b
FROM bar
ORDER BY 1
```

Good

```sql
SELECT a, b
FROM bar
ORDER BY a
```
