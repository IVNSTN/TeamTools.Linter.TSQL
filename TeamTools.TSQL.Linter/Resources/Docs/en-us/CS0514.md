

# Debug construction is used in code

|||
|-|-|
| Id | **CS0514**
| Mnemo | SHOWING_STATS
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [ShowStatisticsFromCodeRule.cs](../../../Rules/CodeSmell/ShowStatisticsFromCodeRule.cs)

## Cause

This rule is triggered when debug construction (SET STATISTICS IO, TIME etc.) is used in code.
<p id="descr">Remove debug construction.</p>

## Examples

Bad

```sql
CREATE PROCEDURE foo
AS
BEGIN
    SET STATISTICS IO ON;
    RETURN 1;
END;
```

Good

```sql
CREATE PROCEDURE foo
AS
BEGIN
    RETURN 1;
END;
```
