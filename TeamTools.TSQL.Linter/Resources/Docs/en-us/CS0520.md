

# App-lock is used in code

|||
|-|-|
| Id | **CS0520**
| Mnemo | APP_LOCK
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [AppLockRule.cs](../../../Rules/CodeSmell/AppLockRule.cs)

## Cause

This rule is triggered when app-lock (sp_getapplock) is used in code.
<p id="descr">Remove app-lock.</p>

## Examples

Bad

```sql
EXEC sp_getapplock;
EXEC foo;
EXEC sp_releaseapplock;
```

Good

```sql
EXEC foo;
```

## Tips

💡 App-locks almost always indicates a poorly chosen approach to solving a problem.
