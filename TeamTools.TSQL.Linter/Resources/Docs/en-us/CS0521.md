

# System procedure return value isn't checked

|||
|-|-|
| Id | **CS0521**
| Mnemo | SYSPROC_RETURN_NOT_CHECKED
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SystemProcedureReturnCodeCheckRule.cs](../../../Rules/CodeSmell/SystemProcedureReturnCodeCheckRule.cs)

## Cause

This rule is triggered when system procedure return value isn't checked.
<p id="descr">Check the return value of the system procedure.</p>

## Examples

Bad

```sql
EXEC sp_oacreate;
```

Good

```sql
EXEC @ret_val = sp_oacreate;
```

## Tips

💡 The rule ignores:

- sp_executesql
- sp_addextendedproperty
- sp_bindrule, sp_unbindrule
- sp_bindefault, sp_unbindefault
