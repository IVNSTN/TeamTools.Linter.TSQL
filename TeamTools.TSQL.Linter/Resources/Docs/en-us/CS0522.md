

# Logic of CONTEXT_INFO()

|||
|-|-|
| Id | **CS0522**
| Mnemo | CONTEXT_INFO
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [ContextInfoRule.cs](../../../Rules/CodeSmell/ContextInfoRule.cs)

## Cause

This rule is triggered when the CONTEXT_INFO() logic is used.
<p id="descr">Remove the logic of CONTEXT_INFO().</p>

## Examples

Bad

```sql
IF CAST(CONTEXT_INFO() AS VARBINARY(128)) = 0x01
    SET @value = 5;
```

Good

```sql
IF (a > b)
    SET @value = 5;
```
