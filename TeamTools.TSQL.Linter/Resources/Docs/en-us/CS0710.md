

# Arithmetics with NULL returns NULL

|||
|-|-|
| Id | **CS0710**
| Mnemo | NULL_ARITHMETICS
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [ArithmeticsWithNullRule.cs](../../../Rules/CodeSmell/ArithmeticsWithNullRule.cs)

## Cause

The rule is triggered if the value NULL is present in arithmetic operations, which leads to a NULL result.
<p id="descr">Check the values that may be NULL.</p>

## Examples

Bad

```sql
DECLARE @str VARCHAR(10)
SELECT 'abc' + @str;
```

Good

```sql
DECLARE @str VARCHAR(10) = 'defg'
SELECT 'abc' + @str;
```

## Tips

💡 This can happen if you forget to initialize a variable or make a copy without considering the specifics of the new code.

💡 The linter assumes that the **CONCAT_WITH_NULL_YIELDS_NULL** option is enabled for the analyzed code.
