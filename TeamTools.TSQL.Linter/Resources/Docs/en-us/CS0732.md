

# Missing GO after procedure body

|||
|-|-|
| Id | **CS0732**
| Mnemo | GO_AFTER_PROC
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [GoAfterProcRule.cs](../../../Rules/CodeSmell/GoAfterProcRule.cs)

## Cause

<p id="descr">Тело хранимой процедуры в скрипте нужно завершать командой GO иначе весь последующий код (например, гранты) тоже будет зачтен как часть тела ХП.</p>
