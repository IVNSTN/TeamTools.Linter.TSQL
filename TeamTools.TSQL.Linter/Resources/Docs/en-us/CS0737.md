

# Enabling the XACT_ABORT option within the trigger body

|||
|-|-|
| Id | **CS0737**
| Mnemo | XACT_ABORT_IN_TRIGGER
| Severity | ⚠ Warning
| Category | [CodeSmell](./Group_Triggers.md)
| Source code | [XactAbortInTriggerRule.cs](../../../Rules/CodeSmell/XactAbortInTriggerRule.cs)

## Cause

The rule is triggered if the setting of the XACT_ABORT option changes to ON in the trigger body.
<p id="descr">Remove the code that changes the setting of the XACT_ABORT option.</p>

## Examples

Bad

```sql
CREATE TRIGGER my_trigger
ON my_table
FOR ...
AS
BEGIN
SET XACT_ABORT ON;
...
END;
```

Good

```sql
CREATE TRIGGER my_trigger
ON my_table
FOR ...
AS
BEGIN
...
END;
```

## Tips

💡 When the trigger is activated, the DBMS automatically sets the XACT_ABORT value to ON. There is no need to try to change this option in the trigger.
