

# Schema needs name

|||
|-|-|
| Id | **CS0739**
| Mnemo | SCHEMA_NEEDS_NAME
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SchemaNeedsNameRule.cs](../../../Rules/CodeSmell/SchemaNeedsNameRule.cs)

## Cause

The rule is triggered if you did not specify the schema name when defining the CREATE SCHEMA command.
<p id="descr">Specify the schema name.</p>

## Examples

Bad

```sql
CREATE SCHEMA  AUTHORIZATION dbo;
```

Good

```sql
CREATE SCHEMA my_schema AUTHORIZATION dbo;
```

## Tips

💡 For backward compatibility, the CREATE SCHEMA expression allows you to omit the schema name. However, this is not recommended.
