

# Unexpected general set option control

|||
|-|-|
| Id | **CS0747**
| Mnemo | GENERAL_SET_CONTROL
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [GeneralSetOptionRule.cs](../../../Rules/CodeSmell/GeneralSetOptionRule.cs)

## Cause

The rule is triggered if there is a control parameter, such as DEADLOCK_PRIORITY, which is not expected in the business logic.
<p id="descr">Check if you need to adjust the SET options.</p>

## Examples

Bad

```sql
SET NOCOUNT ON;
SET DEADLOCK_PRIORITY LOW;
...
```

Good

```sql
SET NOCOUNT ON;
...
```

## Tips

💡 Managing options like DEADLOCK_PRIORITY is not expected in the business logic of a product.
