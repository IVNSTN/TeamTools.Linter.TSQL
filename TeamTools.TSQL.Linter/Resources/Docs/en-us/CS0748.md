

# IDENTITY_INSERT is used

|||
|-|-|
| Id | **CS0748**
| Mnemo | IDENTITY_INSERT
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [IdentityInsertRule.cs](../../../Rules/CodeSmell/IdentityInsertRule.cs)

## Cause

This rule is triggered when IDENTITY_INSERT option is used.
<p id="descr">Remove IDENTITY_INSERT option.</p>

## Examples

Bad

```sql
SET IDENTITY_INSERT dbo.foo ON;
INSERT dbo.foo(id) VALUES(1);
```

Good

```sql
INSERT dbo.foo(id) VALUES(1);
```

## Tips

💡 Insert into an IDENTITY-column doesn't fit well with high-load and concurrency for writes and reads.
