

# Full text search is used

|||
|-|-|
| Id | **CS0751**
| Mnemo | FULLTEXT_SEARCH
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [FullTextSearchRule.cs](../../../Rules/CodeSmell/FullTextSearchRule.cs)

## Cause

This rule is triggered when full text search is used.
<p id="descr">Don't use full text search.</p>

## Examples

Bad

```sql
SELECT TOP 10 *
FROM dbo.foo
WHERE freetext(title, 'asdf')
```

Good

```sql
SELECT TOP 10 *
FROM dbo.foo
WHERE title LIKE 'asdf%'
```
