

# Deadlock priority value is too precise

|||
|-|-|
| Id | **CS0756**
| Mnemo | DEADLOCK_PRIORITY_PRECISE
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [DeadlockPriorityPreciseRule.cs](../../../Rules/CodeSmell/DeadlockPriorityPreciseRule.cs)

## Cause

<p id="descr">Чересчур точная настройка DEADLOCK_PRIORITY конкретными значениями выглядит подозрительно. Конкуренция за ресурсы в высоконагруженной системе так в любом случае не упорядочится. Если нужно повысить или понизить приоритет особенного модуля, воспользуйтесь встроенными значениями LOW, NORMAL, HIGH - этого должно быть достаточно.</p>
