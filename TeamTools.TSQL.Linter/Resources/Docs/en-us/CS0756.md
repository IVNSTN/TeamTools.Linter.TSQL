

# DEADLOCK_PRIORITY is initialized as numeric

|||
|-|-|
| Id | **CS0756**
| Mnemo | DEADLOCK_PRIORITY_PRECISE
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [DeadlockPriorityPreciseRule.cs](../../../Rules/CodeSmell/DeadlockPriorityPreciseRule.cs)

## Cause

This rule is triggered when DEADLOCK_PRIORITY is initialized as numeric.
<p id="descr">Use embedded values LOW/NORMAL/HIGH.</p>

## Examples

Bad

```sql
SET DEADLOCK_PRIORITY -5;
```

Good

```sql
SET DEADLOCK_PRIORITY LOW;
```
