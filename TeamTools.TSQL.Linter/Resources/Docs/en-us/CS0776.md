

# SPARSE columns should not be used in temp tables and table variables

|||
|-|-|
| Id | **CS0776**
| Mnemo | SPARSE_COL_IN_TMP
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SparseColInTempTableRule.cs](../../../Rules/CodeSmell/SparseColInTempTableRule.cs)

## Cause

<p id="descr">Временные таблицы или табличные переменные с неординарно большим числом колонок для хранения NULL-значений - явление редкое и сомнительное. Постарайтесь пересмотреть подход или, например, согласитесь с тем, что экономия места иллюзорна.</p>
