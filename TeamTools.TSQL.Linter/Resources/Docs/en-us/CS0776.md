

# SPARSE columns is used in temp tables or table variables

|||
|-|-|
| Id | **CS0776**
| Mnemo | SPARSE_COL_IN_TMP
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SparseColInTempTableRule.cs](../../../Rules/CodeSmell/SparseColInTempTableRule.cs)

## Cause

This rule is triggered when SPARSE columns is used in temp tables or table variables.
<p id="descr">Don't use SPARSE columns.</p>

## Examples

Bad

```sql
CREATE TABLE #foo (
    id INT NOT NULL
    , calc INT SPARSE NULL
);
```

Good

```sql
CREATE TABLE #foo (
    id INT NOT NULL,
    , calc INT NULL
);
```
