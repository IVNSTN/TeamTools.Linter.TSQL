

# Undocumented LINENO directive

|||
|-|-|
| Id | **CS0786**
| Mnemo | LINE_NO
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [LineNoRule.cs](../../../Rules/CodeSmell/LineNoRule.cs)

## Cause

This rule is triggered when LINENO directive is used in code.
<p id="descr">Remove LINENO directive.</p>

## Examples

Bad

```sql
CREATE PROC dbo.foo
AS
BEGIN
    SET NOCOUNT ON;
    ...
    LINENO 0;
    RAISERROR ('Error', 16, 1);
END;
```

Good

```sql
CREATE PROC dbo.foo
AS
BEGIN
    SET NOCOUNT ON;
    ...
    RAISERROR ('Error', 16, 1);
END;
```
