

# ROWVERSION usage on temporary object

|||
|-|-|
| Id | **CS0787**
| Mnemo | ROWVERSION_MISUSED
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [RowVersionMisusedRule.cs](../../../Rules/CodeSmell/RowVersionMisusedRule.cs)

## Cause

This rule is triggered when ROWVERSION or TIMESTAMP column is used on temporary table or table variable.
<p id="descr">Remove ROWVERSION or TIMESTAMP column.</p>

## Examples

Bad

```sql
CREATE TABLE #foo
(
    id     INT          NOT NULL PRIMARY KEY
    , name VARCHAR(100) NOT NULL
    , rv   ROWVERSION   NOT NULL
);
```

Good

```sql
CREATE TABLE #foo
(
    id     INT          NOT NULL PRIMARY KEY
    , name VARCHAR(100) NOT NULL
);
```
