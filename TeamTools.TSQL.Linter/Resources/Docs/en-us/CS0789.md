

# Approximate dataset usage

|||
|-|-|
| Id | **CS0789**
| Mnemo | TABLESAMPLE_PREDICT
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [TableSamplePredictRule.cs](../../../Rules/CodeSmell/TableSamplePredictRule.cs)

## Cause

This rule is triggered when TABLESAMPLE instruction or PREDICT function is used in code.
<p id="descr">Remove TABLESAMPLE instruction or PREDICT function.</p>

## Examples

Bad

```sql
SELECT *
FROM Sales.Customer TABLESAMPLE SYSTEM(10 PERCENT);

SELECT d.*, p.Score
FROM PREDICT(MODEL = @model,
    DATA = dbo.mytable AS d) WITH (Score FLOAT) AS p;
```
