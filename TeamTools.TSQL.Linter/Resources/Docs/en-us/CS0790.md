

# External dataset access

|||
|-|-|
| Id | **CS0790**
| Mnemo | UNEXPECTED_OPENDATASET
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [OpenDatasetRule.cs](../../../Rules/CodeSmell/OpenDatasetRule.cs)

## Cause

This rule is triggered when OPENDATASOURCE or OPENROWSET is used in code.
<p id="descr">Remove OPENDATASOURCE or OPENROWSET.</p>

## Examples

Bad

```sql
SELECT *
FROM OPENDATASOURCE('SQLNCLI',
    'Data Source=London\Payroll;Integrated Security=SSPI')
    .AdventureWorks2022.HumanResources.Employee;
```
