
# Non-unicode literal contains Unicode symbol

|||
|-|-|
| Id | **CS0792**
| Mnemo | BROKEN_UNICODE_LITERAL
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [LiteralBrokenUnicodeRule.cs](../../../Rules/CodeSmell/LiteralBrokenUnicodeRule.cs)

## Cause

This rule is triggered if the string literal contains unicode characters (or characters from a code page other than 1251), but does not have the `N` prefix.
<p id="descr">Add the prefix 'N' before the string literal.</p>

## Examples

Bad

```sql
SELECT 'adsf' + CAST('☀️' AS NCHAR(10));  -- 1

DECLARE @s NVARCHAR(100) = '☀️☀️☀️';     -- 2
```

Good

```sql
SELECT N'adsf' + N'1 ☀️ 2';

DECLARE @s NVARCHAR(100) = N'☀️☀️☀️';
```
