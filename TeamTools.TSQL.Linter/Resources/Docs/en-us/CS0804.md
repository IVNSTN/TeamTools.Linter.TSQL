

# Per-row deletion via cursor

|||
|-|-|
| Id | **CS0804**
| Mnemo | CURSOR_FOR_DELETE
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md), [Cursors](./Group_Cursors.md)
| Source code | [CursorForDeleteRule.cs](../../../Rules/CodeSmell/CursorForDeleteRule.cs)

## Cause

This rule is triggered when DELETE WHERE CURRENT OF construction is used in code.
<p id="descr">Remove DELETE WHERE CURRENT OF construction.</p>

## Examples

Bad

```sql
DELETE FROM foo.bar
WHERE CURRENT OF @cursor
```
