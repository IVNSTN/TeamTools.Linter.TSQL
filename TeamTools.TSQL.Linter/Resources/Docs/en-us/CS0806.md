

# IDENTITY_INSERT is used on temporary table

|||
|-|-|
| Id | **CS0806**
| Mnemo | IDENTITY_INSERT_TEMP_TABLE
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [IdentityInsertOnTempTableRule.cs](../../../Rules/CodeSmell/IdentityInsertOnTempTableRule.cs)

## Cause

This rule is triggered when IDENTITY_INSERT is used on temporary table.
<p id="descr">Remove IDENTITY_INSERT.</p>

## Examples

Bad

```sql
CREATE TABLE #tmp
(
    id     INT          IDENTITY(1, 1) NOT NULL
    , name VARCHAR(100) NOT NULL
);

SET IDENTITY_INSERT #tmp ON;

INSERT INTO #tmp (id, name)
VALUES (5, 'name');

SET IDENTITY_INSERT #tmp OFF;
```

Good

```sql
CREATE TABLE #tmp
(
    id     INT          NOT NULL
    , name VARCHAR(100) NOT NULL
);

INSERT INTO #tmp (id, name)
VALUES (5, 'name');
```
