

# Global cursor reference

|||
|-|-|
| Id | **CS0807**
| Mnemo | GLOBAL_CURSOR_REF
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md), [Cursors](./Group_Cursors.md)
| Source code | [GlobalCursorReferenceRule.cs](../../../Rules/CodeSmell/GlobalCursorReferenceRule.cs)

## Cause

This rule is triggered when global cursor reference is used in code.
<p id="descr">Remove global cursor reference.</p>

## Examples

Bad

```sql
UPDATE dbo.foo
SET a = b
WHERE CURRENT OF GLOBAL MY_CURSOR
```

Good

```sql
UPDATE dbo.foo
SET a = b
WHERE CURRENT OF @cursor
```
