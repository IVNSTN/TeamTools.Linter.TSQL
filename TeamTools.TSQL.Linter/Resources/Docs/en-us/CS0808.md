

# Redundant DISTINCT for IN predicate source

|||
|-|-|
| Id | **CS0808**
| Mnemo | DISTINCT_FOR_IN
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [DistinctForInSubqueryRule.cs](../../../Rules/CodeSmell/DistinctForInSubqueryRule.cs)

## Cause

This rule is triggered when DISTINCT is specified for IN predicate source.
<p id="descr">Remove DISTINCT.</p>

## Examples

Bad

```sql
SELECT name
FROM dbo.foo
WHERE category_id IN (SELECT DISTINCT category_id FROM dbo.categories)
```

Good

```sql
SELECT name
FROM dbo.foo
WHERE category_id IN (SELECT category_id FROM dbo.categories)
```
