

# DECIMAL without scale

|||
|-|-|
| Id | **CS0821**
| Mnemo | DECIMAL_WITHOUT_DECIMALS
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [DecimalWithNoScaleRule.cs](../../../Rules/CodeSmell/DecimalWithNoScaleRule.cs)

## Cause

This rule is triggered when DECIMAL or NUMERIC type is used without scale.
<p id="descr">Specify scale or use an integer type.</p>

## Examples

Bad

```sql
DECLARE
    @tax DECIMAL
    , @percent NUMERIC(5, 0);
```

Good

```sql
DECLARE
    @tax DECIMAL(18, 3)
    , @percent INT;
```
