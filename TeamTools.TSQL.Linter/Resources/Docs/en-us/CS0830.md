

# Same temporary table names

|||
|-|-|
| Id | **CS0830**
| Mnemo | NAME_REUSED_TEMP_TABLE
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [ReusedTempTableNameRule.cs](../../../Rules/CodeSmell/ReusedTempTableNameRule.cs)

## Cause

This rule is triggered when same names are used for different temporary tables.
<p id="descr">Give different names.</p>

## Examples

Bad

```sql
CREATE TABLE #foo
(
    descendant_id INT NOT NULL
    , parent_id   INT NOT NULL
);

DROP TABLE #foo;

CREATE TABLE #foo
(
    title       VARCHAR(100)
    , lastmod   DATETIME2(7) NOT NULL DEFAULT SYSDATETIME()
);
```

Good

```sql
CREATE TABLE #nodes
(
    descendant_id INT NOT NULL
    , parent_id   INT NOT NULL
);

DROP TABLE #nodes;

CREATE TABLE #titles
(
    title       VARCHAR(100)
    , lastmod   DATETIME2(7) NOT NULL DEFAULT SYSDATETIME()
);
```
