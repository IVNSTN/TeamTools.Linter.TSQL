

# Left part of comparison is similar to the right part

|||
|-|-|
| Id | **CS0853**
| Mnemo | COMPARISON_LEFT_EQUALS_RIGHT
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [ComparedExpressionsEqualRule.cs](../../../Rules/CodeSmell/ComparedExpressionsEqualRule.cs)

## Cause

This rule is triggered when the left part of comparison is similar to the right part.
<p id="descr">Fix the comparison.</p>

## Examples

Bad

```sql
SELECT *
FROM dbo.product_category AS pc
LEFT JOIN dbo.product_item AS pi
    ON pi.category_id = pi.category_id;
```

Good

```sql
SELECT *
FROM dbo.product_category AS pc
LEFT JOIN dbo.product_item AS pi
    ON pi.category_id = pc.category_id;
```

## Tips

💡 The rule ignores 1 = 1 and 0 = 0.
