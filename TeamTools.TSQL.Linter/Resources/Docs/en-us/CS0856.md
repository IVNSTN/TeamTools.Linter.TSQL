

# Programmability invokes itself

|||
|-|-|
| Id | **CS0856**
| Mnemo | SELF_CALL
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [SelfCallRule.cs](../../../Rules/CodeSmell/SelfCallRule.cs)

## Cause

This rule is triggered when programmability invokes itself.
<p id="descr">Remove the recursive call.</p>

## Examples

Bad

```sql
CREATE FUNCTION dbo.inc
(
    @id INT,
    @value INT
)
RETURNS INT
AS
BEGIN
    SET @id = @id + 1;

    IF (@value > 1)
    BEGIN
        RETURN dbo.inc(@id, @value - 1);
    END;

    RETURN @id;
END;
GO
```

Good

```sql
CREATE FUNCTION dbo.inc
(
    @id INT,
    @value INT
)
RETURNS INT
AS
BEGIN
    SET @id = @id + @value;

    RETURN @id;
END;
GO
```
