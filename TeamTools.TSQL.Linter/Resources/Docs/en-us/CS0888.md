

# Extended property duplicate

|||
|-|-|
| Id | **CS0888**
| Mnemo | EXTENDED_PROPERTY_DUP
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [ExtendedPropertyDupRule.cs](../../../Rules/CodeSmell/ExtendedPropertyDupRule.cs)

## Cause

This rule is triggered when the `sp_addextendedproperty` procedure in the object creation script defines extended properties with the same names to the same object.
<p id="descr">Fix sp_addextendedproperty call.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.users
(
    id      INT          NOT NULL
    , login VARCHAR(100) NOT NULL
);
GO

EXEC sp_addextendedproperty
    @name = N'First Description'
    , @value = N'Some description'
    , @level0type = N'SCHEMA'
    , @level0name = N'dbo'
    , @level1type = N'TABLE'
    , @level1name = N'users'
    , @level2type = N'COLUMN'
    , @level2name = N'login';
GO

EXEC sp_addextendedproperty
    @name = N'First Description'
    , @value = N'Some description'
    , @level0type = N'SCHEMA'
    , @level0name = N'dbo'
    , @level1type = N'TABLE'
    , @level1name = N'users'
    , @level2type = N'COLUMN'
    , @level2name = N'login';
GO
```

Good

```sql
CREATE TABLE dbo.users
(
    id      INT          NOT NULL
    , login VARCHAR(100) NOT NULL
);
GO

EXEC sp_addextendedproperty
    @name = N'First Description'
    , @value = N'Some description'
    , @level0type = N'SCHEMA'
    , @level0name = N'dbo'
    , @level1type = N'TABLE'
    , @level1name = N'users'
    , @level2type = N'COLUMN'
    , @level2name = N'login';
GO

EXEC sp_addextendedproperty
    @name = N'Second Description'
    , @value = N'Some description'
    , @level0type = N'SCHEMA'
    , @level0name = N'dbo'
    , @level1type = N'TABLE'
    , @level1name = N'users'
    , @level2type = N'COLUMN'
    , @level2name = N'login';
GO
```
