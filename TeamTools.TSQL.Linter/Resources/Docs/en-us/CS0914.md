

# Literal is constantly different thus statement is redundant or broken

|||
|-|-|
| Id | **CS0914**
| Mnemo | INTERSECT_EXCEPT_BROKEN_BY_LITERAL
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [IntersectExceptOnLiteralRule.cs](../../../Rules/CodeSmell/IntersectExceptOnLiteralRule.cs)

## Cause

<p id="descr">Если в резалтсетах, участвующих в операции INTERSECT или EXCEPT присутствует различающийся литерал, то резалтсеты заведомо различаются. INTERSECT никогда не даст ничего, EXCEPT всегда выдаст всё.</p>
