

# Forbidden INSERT hint is used

|||
|-|-|
| Id | **CS0917**
| Mnemo | FORBIDDEN_INSERT_HINTS
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [InsertHintsRule.cs](../../../Rules/CodeSmell/InsertHintsRule.cs)

## Cause

This rule is triggered when forbidden INSERT hint is used.
<p id="descr">Remove forbidden hint.</p>

## Examples

Bad

```sql
INSERT dbo.Foo WITH (ROWLOCK, FORCESCAN) (id, name)
SELECT 1, '';
```

Good

```sql
INSERT dbo.Foo WITH (TABLOCKX) (id, name)
SELECT 1, '';
```

## Tips

💡 Forbidden:

- FORCESEEK
- FORCESCAN
- READPAST
- READCOMMITTED_LOCK

💡 Rare and questionable:

- IGNORE_TRIGGERS
- IGNORE_CONSTRAINTS

💡 Harmful:

- ROWLOCK
- PAGLOCK
- TABLOCK
