

# Parameter is defined but never used

|||
|-|-|
| Id | **CS0919**
| Mnemo | UNUSED_PARAMETER
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [UnusedParameterRule.cs](../../../Rules/CodeSmell/UnusedParameterRule.cs)

## Cause

This rule is triggered when parameter is defined but never used.
<p id="descr">Remove unused parameter.</p>

## Examples

Bad

```sql
CREATE FUNCTION dbo.foo (@id INT)
RETURNS INT
AS
BEGIN
    RETURN 1;
END;
```

Good

```sql
CREATE FUNCTION dbo.foo (@id INT)
RETURNS INT
AS
BEGIN
    RETURN @id + 1;
END;
```
