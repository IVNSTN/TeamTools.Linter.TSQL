

# Missing pair of XML document state command

|||
|-|-|
| Id | **CS0921**
| Mnemo | UNPAIRED_XMLDOC_STATEMENT
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [UnPairedXmlDocumentInstructionRule.cs](../../../Rules/CodeSmell/UnPairedXmlDocumentInstructionRule.cs)

## Cause

This rule is triggered when one of the paired XML document state commands is missing.
<p id="descr">Specify the pair of XML document state command.</p>

## Examples

Bad

```sql
EXEC sp_xml_preparedocument @handle OUT;
```

Good

```sql
EXEC sp_xml_preparedocument @handle OUT;
EXEC sp_xml_removedocument @handle;
```

## Tips

💡 The rule checks sp_xml_removedocument and sp_xml_preparedocument commands.
