

# Transaction name reused

|||
|-|-|

| Mnemo | NAME_REUSED_TRANSACTION
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)

## Cause

<p id="descr">Если используются именованные транзакции или сейвпойнты, то каждое имя лучше использовать единожды на протяжении всего скрипта для одной конкретной цели.</p>
