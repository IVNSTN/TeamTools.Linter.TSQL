

# Duplicate expression in conditional flow has no effect

|||
|-|-|
| Id | **CS0959**
| Mnemo | DUP_CONDITIONAL_FLOW
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [DuplicateConditionRule.cs](../../../Rules/CodeSmell/DuplicateConditionRule.cs)

## Cause

<p id="descr">Если в коде есть взаимоисключающие условные переходы (CASE, IF-ELSE-IF) и в них встречаются одинаковые условия, то осмысленной является только одна, самая первая проверка. Проверять ее же еще раз ниже по коду не имеет смысла.</p>
