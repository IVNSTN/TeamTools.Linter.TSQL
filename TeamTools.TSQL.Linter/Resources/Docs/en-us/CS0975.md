

# Trigger should be in the same schema as target table

|||
|-|-|
| Id | **CS0975**
| Mnemo | TRIGGER_SCHEMA_TABLE_SCHEMA
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md), [Triggers](./Group_Triggers.md)
| Source code | [TriggerSchemaFromTableRule.cs](../../../Rules/CodeSmell/TriggerSchemaFromTableRule.cs)

## Cause

The rule is triggered if the schema in which the trigger is created does not match the schema of the linked table.
<p id="descr">Specify the correct schema for the trigger.</p>

## Examples

Bad

```sql
CREATE TRIGGER dbo.tr_log ON stage.table
...
```

Good

```sql
CREATE TRIGGER stage.tr_log ON stage.table
...
```

## Tips

💡 The action of DML triggers is limited by the schema of the table or view on which they are created.
