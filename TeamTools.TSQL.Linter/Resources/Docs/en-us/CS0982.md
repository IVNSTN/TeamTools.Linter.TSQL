

# Cursor should be defined as forward only

|||
|-|-|
| Id | **CS0982**
| Mnemo | CURSOR_FORWARD_ONLY
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [CursorForwardOnlyRule.cs](../../../Rules/CodeSmell/CursorForwardOnlyRule.cs)

## Cause

<p id="descr">Если курсор просматривается в одном направлении (FETCH NEXT), то лучше явно это указать в определении курсора с помощью опции FORWARD_ONLY или FAST_FORWARD.</p>
