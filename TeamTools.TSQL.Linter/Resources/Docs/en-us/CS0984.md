
# Variable is self-assigned

|||
|-|-|
| Id | **CS0984**
| Mnemo | VAR_SELF_ASSIGN
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [VariableSelfAssignRule.cs](../../../Rules/CodeSmell/VariableSelfAssignRule.cs)

## Cause

This rule is triggered when a variable is assigned to itself.
<p id="descr">Remove unnecessary assignment.</p>

## Examples

Bad

```sql
SELECT @a = (SELECT (@a))
    , @b = (@b)
    , @c = @c
```

Good

```sql
SELECT @a = 5
    , @b = GETDATE()
    , @c = @d
```
