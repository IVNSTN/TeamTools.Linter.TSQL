
# Variable is self-compared

|||
|-|-|
| Id | **CS0985**
| Mnemo | VAR_SELF_COMPARE
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [VariableSelfCompareRule.cs](../../../Rules/CodeSmell/VariableSelfCompareRule.cs)

## Cause

This rule is triggered when variable is self compared.
<p id="descr">Remove unnecessary comparisons.</p>

## Examples

Bad

```sql
SELECT IIF(@b < (@b), @c, 100500) AS c2
```

Good

```sql
SELECT IIF(@b < 10, @c, 100500) AS c2
```
