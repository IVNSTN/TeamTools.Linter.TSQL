

# WAITFOR DELAY usage is not welcome

|||
|-|-|
| Id | **CS0991**
| Mnemo | WAITFOR_DELAY_USED
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [WaitForDelayUsedRule.cs](../../../Rules/CodeSmell/WaitForDelayUsedRule.cs)

## Cause

<p id="descr">Использование ожиданий в коде механизмов БД очень не рекомендуется. Если в подобных механизмах есть потребность, то лучше их реализацией заниматься на стороне бэкенда или даже фронтенда.</p>
