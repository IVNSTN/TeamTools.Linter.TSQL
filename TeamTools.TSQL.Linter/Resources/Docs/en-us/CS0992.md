

# WAITFOR conversation without timeout limit

|||
|-|-|
| Id | **CS0992**
| Mnemo | WAITFOR_STMT_TIMEOUT
| Severity | ⚠ Warning
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [WaitForStatementWithoutTimeoutRule.cs](../../../Rules/CodeSmell/WaitForStatementWithoutTimeoutRule.cs)

## Cause

This rule is triggered when WAITFOR conversation is used without TIMEOUT.
<p id="descr">Use WAITFOR with TIMEOUT.</p>

## Examples

Bad

```sql
WAITFOR (GET CONVERSATION GROUP @ConversationGroupId FROM backend._TargetQueue_MainEntities);
```

Good

```sql
WAITFOR (GET CONVERSATION GROUP @ConversationGroupId FROM backend._TargetQueue_MainEntities),
TIMEOUT 100;
```
