

# Loss of precision is possible when FLOAT type variables is used

|||
|-|-|
| Id | **CS0993**
| Mnemo | FLOAT_VARIABLE
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [FloatVariableUsedRule.cs](../../../Rules/CodeSmell/FloatVariableUsedRule.cs)

## Cause

This rule is triggered when FLOAT type variables is used in code.
<p id="descr">Use FLOAT type with great care.</p>

## Tips

💡 Specialized modules for floating-point calculations is better to move to separate scheme and add scheme to white list.
