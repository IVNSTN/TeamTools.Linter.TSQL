

# Loss of precision is possible when FLOAT type columns is used

|||
|-|-|
| Id | **CS0994**
| Mnemo | FLOAT_COLUMN
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [FloatColumnStoredRule.cs](../../../Rules/CodeSmell/FloatColumnStoredRule.cs)

## Cause

This rule is triggered when FLOAT type columns is used in code.
<p id="descr">Use FLOAT type with great care.</p>

## Tips

💡 Specialized modules for floating-point calculations is better to move to separate scheme and add scheme to white list.
