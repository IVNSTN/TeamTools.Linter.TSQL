

# Float type is inaccurate

|-|-|

## Cause

<p id="descr">Типом FLOAT нужно пользоваться с большой осторожностью, поскольку вероятна потеря точности вычислений. Правило реагирует на колонки с типом FLOAT и REAL.</p>
