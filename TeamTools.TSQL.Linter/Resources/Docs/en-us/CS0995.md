

# Mixing inaccurate type with accurate

|||
|-|-|
| Id | **CS0995**
| Mnemo | FLOAT_WITH_NON_FLOAT
| Severity | ℹ Hint
| Category | [CodeSmell](./Category_CodeSmell.md)
| Source code | [FloatMixedWithNonFloatRule.cs](../../../Rules/CodeSmell/FloatMixedWithNonFloatRule.cs)

## Cause

<p id="descr">При смешивании в выражениях точных типов с FLOAT или REAL, всегда получается FLOAT. А это - риск потери точности вычислений.</p>
