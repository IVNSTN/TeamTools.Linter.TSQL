
# Unnamed constraint on persistent table

|||
|-|-|
| Id | **CV0117**
| Mnemo | UNNAMED_CONSTRAINT
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [ConstraintNameInPersistentTableRule.cs](../../../Rules/CodingConvention/ConstraintNameInPersistentTableRule.cs)

## Cause

This rule is triggered if the constraint on the persistent table is not named.
<p id="descr">Set a name for the constraint.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.test (
    a INT NOT NULL DEFAULT 0
);
```

Good

```sql
CREATE TABLE dbo.test (
    a INT NOT NULL CONSTRAINT DF_test_a DEFAULT (0)
);
```
