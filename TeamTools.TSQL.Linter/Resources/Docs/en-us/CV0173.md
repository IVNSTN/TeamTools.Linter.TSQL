

# Unexpected non-literal output for EXISTS

|||
|-|-|
| Id | **CV0173**
| Mnemo | EXISTS_NONLITERAL_SELECT
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [ExistsHasNonLiteralSelectElementRule.cs](../../../Rules/CodingConvention/ExistsHasNonLiteralSelectElementRule.cs)

## Cause

<p id="descr">В EXISTS определение SELECT игнорируется. Не нужно там ничего вычислять, перечислять кучи полей - поставьте литерал 1 или 0 или вроде того.</p>
