
# Keyword is not in UPPERCASE

|||
|-|-|
| Id | **CV0201**
| Mnemo | KEYWORD_UPPER
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [KeywordUppercaseRule.cs](../../../Rules/CodingConvention/KeywordUppercaseRule.cs)

## Cause

This rule is triggered if the keyword is not written in UPPERCASE.
<p id="descr">Write keywords in UPPERCASE.</p>

## Examples

Bad

```sql
set nocount on;
declare @a int;
if 1 <> 0
begin
    select top (1) 1 as z from foo
    where bar <> cast(zar as date)
    order by case when upper(a) = a then 1 else 0 end;
end;
```

Good

```sql
SET NOCOUNT ON;
DECLARE @a INT;
IF 1 <> 0
BEGIN
    SELECT TOP (1) 1 AS z FROM dbo.foo
    WHERE bar <> CAST(zar AS DATE)
    ORDER BY CASE WHEN UPPER(a) = a THEN 1 ELSE 0 END;
END;
```
