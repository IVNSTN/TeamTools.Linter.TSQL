
# Built-in function is not in UPPERCASE

|||
|-|-|
| Id | **CV0202**
| Mnemo | SYSTEM_FUNC_UPPER
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [SystemFunctionCallUppercaseRule.cs](../../../Rules/CodingConvention/SystemFunctionCallUppercaseRule.cs)

## Cause

This rule is triggered if the name of the called built-in function is not written in UPPERCASE.
<p id="descr">Write built-in functions name in UPPERCASE.</p>

## Examples

Bad

```sql
SELECT
    try_cast(try_convert(NUMERIC(10, 2), '0') AS NUMERIC(10, 2))
    , iif(0 = 0, 1, 1);
```

Good

```sql
SELECT
    TRY_CAST(TRY_CONVERT(NUMERIC(10, 2), '0') AS NUMERIC(10, 2))
    , IIF(0 = 0, 1, 1);
```
