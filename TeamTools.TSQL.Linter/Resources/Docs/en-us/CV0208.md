
# Join written not in two words

|||
|-|-|
| Id | **CV0208**
| Mnemo | TWO_WORDS_JOIN
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [JoinSpellingRule.cs](../../../Rules/CodingConvention/JoinSpellingRule.cs)

## Cause

This rule is triggered if the type of `JOIN` connection is not specified in two keywords.
<p id="descr">Specify the 'JOIN' type using two keywords.</p>

## Examples

Bad

```sql
SELECT a.z
FROM a
JOIN b
    ON a.a = b.b
LEFT OUTER JOIN c
    ON c.c = b.b
RIGHT OUTER JOIN d
    ON d.d = a.a
FULL OUTER JOIN e
    ON e.e = c.c;
```

Good

```sql
SELECT a.z
FROM a
INNER JOIN b
    ON a.a = b.b
LEFT JOIN c
    ON c.c = b.b
RIGHT JOIN d
    ON d.d = a.a
FULL JOIN e
    ON e.e = c.c;
```

## Tips

💡 Correct `JOIN` definitions:

- `INNER JOIN`
- `LEFT JOIN`
- `RIGHT JOIN`
- `FULL JOIN`
- `CROSS JOIN`
- `CROSS APPLY`
- `OUTER APPLY`
