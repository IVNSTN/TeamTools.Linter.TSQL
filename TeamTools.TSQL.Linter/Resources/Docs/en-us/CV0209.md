
# Non-ISO comparison syntax

|||
|-|-|
| Id | **CV0209**
| Mnemo | C_STYLE_OPERANDS
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [ComparisonSyntaxRule.cs](../../../Rules/CodingConvention/ComparisonSyntaxRule.cs)

## Cause

This rule is triggered if the use of forbidden comparison operators is detected: `!=`, `=*`, `*=`.
<p id="descr">Use '<>' for inequality and the full syntax for 'JOIN' connections.</p>

## Examples

Bad

```sql
IF 1 != 0
BEGIN
    SELECT 1;
END
```

Good

```sql
IF 1 <> 0
BEGIN
    SELECT 1;
END
```
