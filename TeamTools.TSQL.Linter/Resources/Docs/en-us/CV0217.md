

# Constraint defined before NULL/NOT NULL atribute

|||
|-|-|
| Id | **CV0217**
| Mnemo | COLUMN_NULLABILITY_BEFORE_CONSTRAINT
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [ColumnNullableOptionBeforeConstraintRule.cs](../../../Rules/CodingConvention/ColumnNullableOptionBeforeConstraintRule.cs)

## Cause

<p id="descr">В определении таблицы свойства столбца оформляйте в следующем порядке: тип, NULL/NOT NULL, потом все остальное (IDENTITY, CONSTRAINT и т.д.)</p>
