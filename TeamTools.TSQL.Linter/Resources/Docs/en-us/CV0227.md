

# SELECT statement in DECLARE

|||
|-|-|
| Id | **CV0227**
| Mnemo | SELECT_IN_DECLARE
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [SelectInDeclareRule.cs](../../../Rules/CodingConvention/SelectInDeclareRule.cs)

## Cause

<p id="descr">Не надо селектов как и других сложных вычислений в DECLARE. DECLARE должен быть простым для чтения и неспособным породить runtime/compile time ошибки.</p>
