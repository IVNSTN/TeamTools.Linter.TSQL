
# SELECT statement in DECLARE

|||
|-|-|
| Id | **CV0227**
| Mnemo | SELECT_IN_DECLARE
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [SelectInDeclareRule.cs](../../../Rules/CodingConvention/SelectInDeclareRule.cs)

## Cause

This rule is triggered when a value is assigned using the `SELECT` statement inside a `DECLARE` block.
<p id="descr">Simplify the assigned value or move it outside the 'DECLARE' block.</p>

## Examples

Bad

```sql
DECLARE @foo INT = (SELECT a FROM  b);
```

Good

```sql
DECLARE
    @foo   INT         = 0 * 1
    , @bar VARCHAR(10) = 'SELECT'
    , @zar BIT
    , @nar DATETIME    = GETDATE()
    , @trn INT         = @@TRANCOUNT;
```
