
# Using UNION to create a list of scalar values

|||
|-|-|
| Id | **CV0237**
| Mnemo | UNION_SCALAR_TO_VALUES
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [UnionScalarsToValuesRule.cs](../../../Rules/CodingConvention/UnionScalarsToValuesRule.cs)

## Cause

This rule is triggered if `UNION` is used to create a list of scalar values.
<p id="descr">To create a list of scalar values, use 'VALUES'.</p>

## Examples

Bad

```sql
SELECT 1
UNION ALL
SELECT * FROM (VALUES (2), (3)) AS v (id);
```

Good

```sql
SELECT * FROM (VALUES (1), (2), (3)) AS v (id);
```
