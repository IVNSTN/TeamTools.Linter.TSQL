
# Multiple one-line comments in a row

|||
|-|-|
| Id | **CV0245**
| Mnemo | MANY_SL_COMMENTS_TO_ML
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [MultipleSingleLineCommentsToMultilineCommentRule.cs](../../../Rules/CodingConvention/MultipleSingleLineCommentsToMultilineCommentRule.cs)

## Cause

This rule is triggered if the code contains more than two single-line comments in a row.
<p id="descr">Convert to a single multi-line comment.</p>

## Examples

Bad

```sql
-- comment 1
-- comment 2
-- comment 3
SELECT 1;
```

Good

```sql
/*
comment 1
comment 2
comment 3
*/
SELECT 1;
```
