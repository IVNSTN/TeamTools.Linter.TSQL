
# Leading semicolon

|||
|-|-|
| Id | **CV0252**
| Mnemo | LEADING_SEMICOLON
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)

## Cause

This rule is triggered if the leading semicolon `;` is found at the beginning of the code line.
<p id="descr">Move the semicolon ';' to the end of the previous statement.</p>

## Examples

Bad

```sql
BEGIN
    ;THROW 500001, 'test', 1;
END;
```

Good

```sql
BEGIN;
    THROW 500001, 'test', 1;
END;
```
