
# Semicolon after each statement

|||
|-|-|
| Id | **CV0268**
| Mnemo | SEMICOLON_AFTER_STATEMENT
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [SemicolonAfterEachStatementRule.cs](../../../Rules/CodingConvention/SemicolonAfterEachStatementRule.cs)

## Cause

This rule is triggered if the statement is not followed by a semicolon `;`.
<p id="descr">Put a semicolon ';' at the end of the statement.</p>

## Examples

Bad

```sql
IF @a = @b
BEGIN
    SELECT @a;
END -- 1

WHILE @a < @b
BEGIN
    SET @a += 1;
END -- 2
```

Good

```sql
IF @a = @b
BEGIN
    SELECT @a;
END;

WHILE @a < @b
BEGIN
    SET @a += 1;
END;
```
