

# Column alias without 'AS' keyword

|||
|-|-|
| Id | **CV0278**
| Mnemo | ALIAS_WITH_AS
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [AliasWithAsSyntaxRule.cs](../../../Rules/CodingConvention/AliasWithAsSyntaxRule.cs)

## Cause

This rule is triggered when a column alias is used without 'AS' keyword.
<p id="descr">Specify a column alias using 'AS' keyword.</p>

## Examples

Bad

```sql
SELECT
    col_alias1 = a * b
    , c col_alias2
```

Good

```sql
SELECT
    a * b AS col_alias1
    , c AS col_alias2
```
