

# UDF or behavioral function called in DECLARE

|||
|-|-|
| Id | **CV0279**
| Mnemo | UDF_IN_DECLARE
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [UserDefinedFunctionInDeclareRule.cs](../../../Rules/CodingConvention/UserDefinedFunctionInDeclareRule.cs)

## Cause

<p id="descr">Пользовательские функции могут нести в себе всякое, в т.ч. вовсе не примитивные действия. В DECLARE размещать такое крайне нежелательно: код хранимки еще как бы не начался, а мы уже пол-базы просканировали. Это затрудняет понимание кода, отладку, снижает читабельность.</p>
