

# Procedure parameters are enclosed in parentheses

|||
|-|-|
| Id | **CV0285**
| Mnemo | PROC_PARAMS_PARENTHESIS
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [ProcedureParamsParenthesisRule.cs](../../../Rules/CodingConvention/ProcedureParamsParenthesisRule.cs)

## Cause

This rule is triggered when stored procedure parameters are enclosed in parentheses.
<p id="descr">Remove parentheses.</p>

## Examples

Bad

```sql
CREATE PROCEDURE foo(@dt DATETIME2(7))
AS
BEGIN
    SELECT 1;
END;
```

Good

```sql
CREATE PROCEDURE foo
    @dt DATETIME2(7)
AS
BEGIN
    SELECT 1;
END;
```
