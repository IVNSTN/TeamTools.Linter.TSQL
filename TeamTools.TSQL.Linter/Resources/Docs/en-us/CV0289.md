

# ODBC syntax is forbidden

|||
|-|-|
| Id | **CV0289**
| Mnemo | ODBC_SYNTAX
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [OdbcSyntaxRule.cs](../../../Rules/CodingConvention/OdbcSyntaxRule.cs)

## Cause

This rule is triggered when ODBC syntax is used.
<p id="descr">Replace ODBC syntax with TSQL.</p>

## Examples

Bad

```sql
SELECT {fn ABS(EmpNo)}, {fn CONVERT(EmpName,SQL_SMALLINT)}
FROM Employees
WHERE dt > {d '19900101'}
```

Good

```sql
SELECT ABS(EmpNo), CONVERT(SMALLINT, EmpName)
FROM Employees
WHERE dt > '19900101'
```
