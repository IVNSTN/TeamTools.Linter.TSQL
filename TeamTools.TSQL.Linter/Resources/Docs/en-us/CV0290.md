

# The specified spelling of the keyword is violated

|||
|-|-|
| Id | **CV0290**
| Mnemo | KEYWORD_SHORTHAND
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [KeywordFullOrShortVersionRule.cs](../../../Rules/CodingConvention/KeywordFullOrShortVersionRule.cs)

## Cause

This rule is triggered when the specified spelling of the keyword is violated.
<p id="descr">Correct the keyword according to the linter's hint.</p>

## Examples

Bad

```sql
SELECT
    DATEPART(YY, @dt),
    DATENAME(M, @dt);

SET TRAN ISOLATION LEVEL REPEATABLE READ;

EXECUTE dbo.foo;
```

Good

```sql
SELECT
    DATEPART(YEAR, @dt),
    DATENAME(MONTH, @dt);

SET TRANSACTION ISOLATION LEVEL REPEATABLE READ;

EXEC dbo.foo;
```
