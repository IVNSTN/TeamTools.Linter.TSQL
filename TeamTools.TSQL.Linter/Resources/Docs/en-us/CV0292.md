

# Cursor is declared as a variable

|||
|-|-|
| Id | **CV0292**
| Mnemo | CURSOR_NOT_INLINE
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md), [Cursors](./Group_Cursors.md)
| Source code | [CursorDeclareInlineRule.cs](../../../Rules/CodingConvention/CursorDeclareInlineRule.cs)

## Cause

This rule is triggered when cursor is declared as a variable.
<p id="descr">Declare cursor using inline syntax.</p>

## Examples

Bad

```sql
DECLARE
    @foo INT,
    @orders CURSOR

SET @orders = CURSOR FAST_FORWARD FOR
SELECT id
FROM dbo.orders
```

Good

```sql
DECLARE
    @foo INT

DECLARE orders CURSOR FAST_FORWARD FOR
SELECT id
FROM dbo.orders
```
