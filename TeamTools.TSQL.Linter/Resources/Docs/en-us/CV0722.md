

# Redundant parenthesis around standalone statement

|||
|-|-|
| Id | **CV0722**
| Mnemo | STATEMENT_IN_PARENTHESIS
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [StandaloneStatementParenthesisRule.cs](../../../Rules/CodingConvention/StandaloneStatementParenthesisRule.cs)

## Cause

<p id="descr">Синтаксис T-SQL допускает заключение в скобки отдельностоящих выражений, например (SELECT * FROM dbo.foo), но делать так совершенно незачем.</p>
