

# Redundant parentheses around standalone statement

|||
|-|-|
| Id | **CV0722**
| Mnemo | STATEMENT_IN_PARENTHESIS
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [StandaloneStatementParenthesisRule.cs](../../../Rules/CodingConvention/StandaloneStatementParenthesisRule.cs)

## Cause

This rule is triggered when a standalone statement or VIEW's query are enclosed in parentheses.
<p id="descr">Remove redundant parentheses.</p>

## Examples

Bad

```sql
(SELECT a FROM dbo.foo)
```

Good

```sql
SELECT a FROM dbo.foo
```
