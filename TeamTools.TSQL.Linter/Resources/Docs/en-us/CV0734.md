

# Variable declaration isn't combined into a single DECLARE statement

|||
|-|-|
| Id | **CV0734**
| Mnemo | DECLARE_PER_VAR
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [DeclarePerVarRule.cs](../../../Rules/CodingConvention/DeclarePerVarRule.cs)

## Cause

This rule is triggered when variable declaration isn't combined into a single DECLARE statement.
<p id="descr">Combine variable declaration into a single DECLARE statement.</p>

## Examples

Bad

```sql
DECLARE @foo INT;
DECLARE @bar CHAR(1);
```

Good

```sql
DECLARE
    @foo INT
    , @bar CHAR(1);
```
