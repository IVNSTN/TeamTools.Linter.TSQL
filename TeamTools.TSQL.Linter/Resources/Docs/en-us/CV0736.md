

# Double quotes is used

|||
|-|-|
| Id | **CV0736**
| Mnemo | DOUBLE_QUOTES
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [DoubleQuotesRule.cs](../../../Rules/CodingConvention/DoubleQuotesRule.cs)

## Cause

This rule is triggered when string literals or identifiers use double quotes.
<p id="descr">Use single quotes for string literals and square brackets for identifiers.</p>

## Examples

Bad

```sql
SELECT
    "asdf" AS "my col"
FROM tmp."foo bar"
```

Good

```sql
SELECT
    'asdf' AS [my col]
FROM tmp.[foo bar]
```
