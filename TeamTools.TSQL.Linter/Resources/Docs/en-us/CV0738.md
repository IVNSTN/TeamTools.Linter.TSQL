

# Use IS NOT NULL instead of expression negation

|||
|-|-|
| Id | **CV0738**
| Mnemo | NOT_IS_NULL
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [NotIsNullRule.cs](../../../Rules/CodingConvention/NotIsNullRule.cs)

## Cause

<p id="descr">NOT &lt;expression&gt; IS NULL - это отрицание всего выражения, но можно написать более прямолинейно высказанную проверку &lt;expression&gt; IS NOT NULL</p>
