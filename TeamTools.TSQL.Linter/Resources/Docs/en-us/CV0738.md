

# NOT IS NULL is used instead of IS NOT NULL

|||
|-|-|
| Id | **CV0738**
| Mnemo | NOT_IS_NULL
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [NotIsNullRule.cs](../../../Rules/CodingConvention/NotIsNullRule.cs)

## Cause

This rule is triggered when NOT IS NULL is used instead of IS NOT NULL.
<p id="descr">Use IS NOT NULL instead of NOT IS NULL.</p>

## Examples

Bad

```sql
SELECT 1
WHERE NOT @a IS NULL
```

Good

```sql
SELECT 1
WHERE @a IS NOT NULL
```
