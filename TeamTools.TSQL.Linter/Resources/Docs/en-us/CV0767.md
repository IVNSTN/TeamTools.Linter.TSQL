

# Native compilation module parameter nullability isn't explicitly specified

|||
|-|-|
| Id | **CV0767**
| Mnemo | NATIVELY_PARAM_NULLABILITY
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md), [InMemory](./Group_InMemory.md)
| Source code | [NativelyCompiledProcParamNullabilityRule.cs](../../../Rules/CodingConvention/NativelyCompiledProcParamNullabilityRule.cs)

## Cause

This rule is triggered when a native compilation module parameter nullability isn't explicitly specified.
<p id="descr">Specify the parameter nullability.</p>

## Examples

Bad

```sql
CREATE PROCEDURE dbo.foo
    @bar INT = NULL
    , @far CHAR(1)
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```

Good

```sql
CREATE PROCEDURE dbo.foo
    @bar INT NULL
    , @far CHAR(1) NOT NULL
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```
