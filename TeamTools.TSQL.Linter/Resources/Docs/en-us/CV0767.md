

# Parameter nullability should be defined

|||
|-|-|
| Id | **CV0767**
| Mnemo | NATIVELY_PARAM_NULLABILITY
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [NativelyCompiledProcParamNullabilityRule.cs](../../../Rules/CodingConvention/NativelyCompiledProcParamNullabilityRule.cs)

## Cause

<p id="descr">Нативно-компилируемые модули поддерживают возможность указывать для параметров, допустимо ли NULL-значение на входе. Лучше явно указывать, NULL или NOT NULL для каждого параметра.</p>
