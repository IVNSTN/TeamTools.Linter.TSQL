

# Redundant 'AS' when the variable type is specified

|||
|-|-|
| Id | **CV0783**
| Mnemo | DECLARE_AS
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [DeclareAsTypeRule.cs](../../../Rules/CodingConvention/DeclareAsTypeRule.cs)

## Cause

This rule is triggered when the variable type is specified using 'AS'.
<p id="descr">Remove redundant 'AS'.</p>

## Examples

Bad

```sql
DECLARE @i AS INT
```

Good

```sql
DECLARE @i INT
```
