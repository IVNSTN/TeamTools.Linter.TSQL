

# Use conventional type name instead of synonym

|||
|-|-|
| Id | **CV0788**
| Mnemo | TYPE_SYNONYM
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [TypeSynonymRule.cs](../../../Rules/CodingConvention/TypeSynonymRule.cs)

## Cause

This rule is triggered when synonym is used instead of conventional type name.
<p id="descr">Use conventional type name instead of synonym.</p>

## Examples

Bad

```sql
DECLARE @a NATIONAL CHARACTER VARYING (100)
```

Good

```sql
DECLARE @a NVARCHAR(100)
```
