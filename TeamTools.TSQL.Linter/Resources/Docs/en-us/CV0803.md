

# System types should be used without schema

|||
|-|-|
| Id | **CV0803**
| Mnemo | SYSTYPE_HAS_SCHEMA
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [SystemTypeWithSchemaRule.cs](../../../Rules/CodingConvention/SystemTypeWithSchemaRule.cs)

## Cause

This rule is triggered when system type is used with schema name.
<p id="descr">Remove schema name.</p>

## Examples

Bad

```sql
DECLARE @i sys.INT
```

Good

```sql
DECLARE @i INT
```
