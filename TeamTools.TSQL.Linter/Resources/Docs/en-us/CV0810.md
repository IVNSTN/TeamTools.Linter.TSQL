

# Stored procedure call should start with EXEC

|||
|-|-|
| Id | **CV0810**
| Mnemo | EXEC_WITHOUT_EXEC
| Severity | ⚠ Warning
| Category | [CodingConvention](./Category_CodingConvention.md)
| Source code | [ExecWithoutExecRule.cs](../../../Rules/CodingConvention/ExecWithoutExecRule.cs)

## Cause

This rule is triggered when EXEC keyword isn't specified before stored procedure call.
<p id="descr">Add EXEC keyword.</p>

## Examples

Bad

```sql
calculation_procedure @value = 5; 
```

Good

```sql
EXEC calculation_procedure @value = 5;
```
