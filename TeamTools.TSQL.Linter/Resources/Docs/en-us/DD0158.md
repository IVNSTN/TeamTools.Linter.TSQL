

# All table columns can contain a NULL value

|||
|-|-|
| Id | **DD0158**
| Mnemo | TABLE_ALL_COL_NULL
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [TableHasNotNullableColumnsRule.cs](../../../Rules/DatabaseDesign/TableHasNotNullableColumnsRule.cs)

## Cause

This rule is triggered when all table columns can contain a NULL value.
<p id="descr">Define at least one column as NOT NULL.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    id INT NULL,
    name VARCHAR(10) NULL
)
```

Good

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL,
    name VARCHAR(10) NULL
)
```
