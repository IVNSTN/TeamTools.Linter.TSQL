

# The ROWVERSION column must be NOT NULL

|||
|-|-|
| Id | **DD0182**
| Mnemo | ROWVERSION_NOT_NULL
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [RowVersionColumnNullableRule.cs](../../../Rules/DatabaseDesign/RowVersionColumnNullableRule.cs)

## Cause

This rule is triggered when ROWVERSION or TIMESTAMP column is nullable.
<p id="descr">Set NOT NULL option.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    ver ROWVERSION,
    ts TIMESTAMP NULL
)
```

Good

```sql
CREATE TABLE dbo.foo
(
    ver ROWVERSION NOT NULL,
    ts TIMESTAMP NOT NULL
)
```
