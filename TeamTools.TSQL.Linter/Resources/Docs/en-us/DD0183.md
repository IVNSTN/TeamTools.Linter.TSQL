

# The IDENTITY column must be NOT NULL

|||
|-|-|
| Id | **DD0183**
| Mnemo | IDENTITY_NOT_NULL
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [IdentityColumnNullabilityRule.cs](../../../Rules/DatabaseDesign/IdentityColumnNullabilityRule.cs)

## Cause

This rule is triggered when IDENTITY column is nullable.
<p id="descr">Set NOT NULL option.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    id INT NULL IDENTITY(1,1)
)
```

Good

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL IDENTITY(1,1)
)
```
