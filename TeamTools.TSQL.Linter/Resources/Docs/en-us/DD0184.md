

# The primary key column must be NOT NULL

|||
|-|-|
| Id | **DD0184**
| Mnemo | PK_COLUMN_NOT_NULL
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [PrimaryKeyColNullableRule.cs](../../../Rules/DatabaseDesign/PrimaryKeyColNullableRule.cs)

## Cause

This rule is triggered when primary key column is nullable.
<p id="descr">Set NOT NULL option.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    id INT NULL
    , CONSTRAINT PK_foo PRIMARY KEY CLUSTERED (id)
)
```

Good

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
    , CONSTRAINT PK_foo PRIMARY KEY CLUSTERED (id)
)
```
