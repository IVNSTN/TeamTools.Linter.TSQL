

# The unique column is included in unique constraint

|||
|-|-|
| Id | **DD0701**
| Mnemo | UID_UNIQ_QUESTIONED
| Severity | ℹ Hint
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [UidUniquenessQuestionedRule.cs](../../../Rules/DatabaseDesign/UidUniquenessQuestionedRule.cs)

## Cause

This rule is triggered when unique column is included in unique constraint.
<p id="descr">Remove column from unique constraint.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    id UNIQUEIDENTIFIER NOT NULL DEFAULT NEWID()
    , dt DATETIME
    , CONSTRAINT PK PRIMARY KEY (id)
);
GO

CREATE UNIQUE NONCLUSTERED INDEX IX ON dbo.foo (dt, id);
GO
```

Good

```sql
CREATE TABLE dbo.foo
(
    id UNIQUEIDENTIFIER NOT NULL DEFAULT NEWID()
    , dt DATETIME
    , CONSTRAINT PK PRIMARY KEY (id)
);
GO

CREATE UNIQUE NONCLUSTERED INDEX IX ON dbo.foo (dt);
GO
```
