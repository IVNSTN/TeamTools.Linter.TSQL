

# Fixed-length column with big size is not recommended

|||
|-|-|
| Id | **DD0750**
| Mnemo | LONG_FIXED_LENGTH_COL
| Severity | ℹ Hint
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [LongFixedLengthColRule.cs](../../../Rules/DatabaseDesign/LongFixedLengthColRule.cs)

## Cause

<p id="descr">При фиксированном размере строковой колонке, независимо от фактической длины значения, хранится полный указанный в определении колонки размер. Если в колонке встречается много коротких строк и нет потребности хранить "пустое место", то может иметь смысл переделать CHAR в VARCHAR.</p>
