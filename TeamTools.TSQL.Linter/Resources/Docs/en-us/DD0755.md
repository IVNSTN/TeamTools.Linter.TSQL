

# Small size column must be NOT NULL

|||
|-|-|
| Id | **DD0755**
| Mnemo | SMALL_SIZE_COL_NULL
| Severity | ℹ Hint
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [SmallSizeColumnNullableRule.cs](../../../Rules/DatabaseDesign/SmallSizeColumnNullableRule.cs)

## Cause

This rule is triggered when small size column is nullable.
<p id="descr">Set NOT NULL option.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    code CHAR(2) NULL
    , is_enabled BIT
);
```

Good

```sql
CREATE TABLE dbo.foo
(
    code CHAR(2) NOT NULL
    , is_enabled BIT NOT NULL
);
```

## Tips

💡 Small data types are chosen for columns in order to conserve disk space allocated for storage. If a column allows NULL values, then the DBMS allocates extra bits (rounded up to whole bytes) to indicate NULL values.

💡 The rule ignores temporary tables, table variables and SPARSE columns.

💡 The rule checks:

- BIT
- TINYINT
- VARCHAR(N <= 6)
- CHAR(N <= 6)
- NVARCHAR(N <= 6)
- NCHAR(N <= 6)
