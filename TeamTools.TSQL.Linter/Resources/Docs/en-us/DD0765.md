

# Trigger on in-memory table

|||
|-|-|
| Id | **DD0765**
| Mnemo | NATIVELY_COMPILED_TRIGGER
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md), [InMemory](./Group_InMemory.md), [Triggers](./Group_Triggers.md)
| Source code | [NativelyCompiledTriggerRule.cs](../../../Rules/DatabaseDesign/NativelyCompiledTriggerRule.cs)

## Причина

This rule is triggered when trigger is created on in-memory table.
<p id="descr">Remove trigger.</p>

## Examples

Bad

```sql
CREATE TRIGGER dml ON dbo.foo
WITH EXECUTE AS OWNER, NATIVE_COMPILATION, SCHEMABINDING
AFTER INSERT
AS
RETURN 1;
```

## Tips

💡 In-memory tables are used for operational processing and data access, but triggers can significantly impact data processing speed and lead to performance degradation.
