

# SPARSE column of MAX size

|||
|-|-|
| Id | **DD0773**
| Mnemo | MAX_COL_SPARSE
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [MaxColumnSparseRule.cs](../../../Rules/DatabaseDesign/MaxColumnSparseRule.cs)

## Cause

This rule is triggered when SPARSE option is used for column of MAX size.
<p id="descr">Remove SPARSE option.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    data          VARBINARY(MAX) SPARSE NULL
    , description VARCHAR(MAX)   SPARSE NULL
);
```

Good

```sql
CREATE TABLE dbo.foo
(
    data          VARBINARY(MAX) NULL
    , description VARCHAR(MAX)   NULL
);
```
