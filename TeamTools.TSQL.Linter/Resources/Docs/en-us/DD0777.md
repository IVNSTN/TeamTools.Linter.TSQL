

# SPARSE columns should not be used

|||
|-|-|
| Id | **DD0777**
| Mnemo | SPARSE_COL_USED
| Severity | ℹ Hint
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [SparseColumnInGeneralRule.cs](../../../Rules/DatabaseDesign/SparseColumnInGeneralRule.cs)

## Cause

This rule is triggered when SPARSE option is used.
<p id="descr">Remove SPARSE option.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    id     BIGINT   NOT NULL
    , name CHAR(20) SPARSE NULL
)
```

Good

```sql
CREATE TABLE dbo.foo
(
    id     BIGINT   NOT NULL
    , name CHAR(20) NULL
)
```
