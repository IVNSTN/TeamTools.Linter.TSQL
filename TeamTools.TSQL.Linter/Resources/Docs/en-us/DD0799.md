
# Default 'NULL' defined as text

|||
|-|-|
| Id | **DD0799**
| Mnemo | DEFAULT_NULL_WORD
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [DefaultNullAsStringRule.cs](../../../Rules/DatabaseDesign/DefaultNullAsStringRule.cs)

## Cause

This rule is triggered when the DEFAULT value for a string column is defined as 'NULL'.
<p id="descr">Use empty string or NULL.</p>

## Examples

Bad

```sql
CREATE TABLE #foo
(
    bar     VARCHAR(10) NOT NULL DEFAULT '<NULL>'
    , far   NCHAR CONSTRAINT DF_foo_far DEFAULT ('  null  ')
)
```

Good

```sql
CREATE TABLE #foo
(
    bar     VARCHAR(10) NOT NULL DEFAULT ''
    , far   NCHAR       NULL
)
```
