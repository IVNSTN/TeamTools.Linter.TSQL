

# Foreign keys with temporary tables

|||
|-|-|
| Id | **DD0829**
| Mnemo | FK_ON_TMP
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [ForeignKeyOnTempTableRule.cs](../../../Rules/DatabaseDesign/ForeignKeyOnTempTableRule.cs)

## Cause

This rule is triggered when foreign keys point to a temporary table or are created on a temporary table.
<p id="descr">Remove foreign keys.</p>

## Examples

Bad

```sql
ALTER TABLE #calculations
ADD CONSTRAINT fk1 FOREIGN KEY (foo_id) REFERENCES dbo.foo(id);

ALTER TABLE dbo.foo
ADD CONSTRAINT fk2 FOREIGN KEY (type_id) REFERENCES #types(id);
```

Good

```sql
ALTER TABLE dbo.foo
ADD CONSTRAINT fk2 FOREIGN KEY (type_id) REFERENCES foo_types(id);
```
