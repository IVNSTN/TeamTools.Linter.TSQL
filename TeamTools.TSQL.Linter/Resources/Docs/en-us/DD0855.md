

# User-defined scalar type derived from a system type

|||
|-|-|
| Id | **DD0855**
| Mnemo | SCALAR_UDT
| Severity | ℹ Hint
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [ScalarUdtRule.cs](../../../Rules/DatabaseDesign/ScalarUdtRule.cs)

## Cause

This rule is triggered when user-defined scalar type derived from a system type.
<p id="descr">Remove user-defined scalar type derived from a system type.</p>

## Examples

Bad

```sql
CREATE TYPE Id FROM INT;
```
