

# Recursive foreign key

|||
|-|-|
| Id | **DD0906**
| Mnemo | FK_RECURSION
| Severity | ℹ Hint
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [ForeignKeyRecursionRule.cs](../../../Rules/DatabaseDesign/ForeignKeyRecursionRule.cs)

## Cause

This rule is triggered when recursive foreign key is used.
<p id="descr">Remove recursive foreign key.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    id          INT NOT NULL PRIMARY KEY
    , parent_id INT NOT NULL
    , CONSTRAINT FK_parent_id FOREIGN KEY (parent_id) REFERENCES dbo.foo (id)
);
```

Good

```sql
CREATE TABLE dbo.foo
(
    id          INT NOT NULL PRIMARY KEY
    , parent_id INT NOT NULL
);
```
