

# NOCHECK CONSTRAINT is used

|||
|-|-|
| Id | **DD0913**
| Mnemo | NOCHECK_CONSTRAINT
| Severity | ⚠ Warning
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)
| Source code | [NocheckConstraintRule.cs](../../../Rules/DatabaseDesign/NocheckConstraintRule.cs)

## Cause

This rule is triggered when NOCHECK CONSTRAINT is used in code.
<p id="descr">Remove NOCHECK CONSTRAINT or the constraint itself.</p>

## Examples

Bad

```sql
ALTER TABLE dbo.foo
NOCHECK CONSTRAINT CK_code_type;
```
