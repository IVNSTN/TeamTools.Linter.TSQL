

# Type is not recommended for clustered index

|||
|-|-|
| Id | **DD0998**
| Mnemo | BAD_TYPE_FOR_CLUSTERED_IDX
| Severity | ℹ Hint
| Category | [DatabaseDesign](./Category_DatabaseDesign.md)

## Cause

<p id="descr">Не все типы данных одинаково удачны и уместны для использования в составе ключа кластеризованного индекса. Большие строковые типы, числовые типы с десятичной частью - это скорее данные, чем ключи.</p>
