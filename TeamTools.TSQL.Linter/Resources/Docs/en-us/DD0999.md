

# Bloated clustered index

|||
|-|-|
| Id | **DD0999**
| Mnemo | BLOATED_CLUSTERED_IDX
| Severity | ℹ Hint
| Category | [DatabaseDesign](./Category_DatabaseDesign.md), [Indices](./Group_Indices.md)
| Source code | [BloatedClusteredIndexRule.cs](../../../Rules/DatabaseDesign/BloatedClusteredIndexRule.cs)

## Cause

This rule is triggered when the clustered index includes high-dimensional types or a large number of columns.
<p id="descr">Reduce the index size.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    id        BIGINT NOT NULL
    , code    VARCHAR(MAX)  NOT NULL
    , comment VARCHAR(1024) NOT NULL
    , PRIMARY KEY CLUSTERED (id, code)
);
```

Good

```sql
CREATE TABLE dbo.foo
(
    id        BIGINT NOT NULL
    , code    VARCHAR(MAX)  NOT NULL
    , comment VARCHAR(1024) NOT NULL
    , PRIMARY KEY CLUSTERED (id)
);
```
