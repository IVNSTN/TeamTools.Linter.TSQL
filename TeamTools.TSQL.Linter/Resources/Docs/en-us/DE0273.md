

# Deprecated hint syntax

|||
|-|-|
| Id | **DE0273**
| Mnemo | HINT_SYNTAX
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [DeprecatedHintSyntaxRule.cs](../../../Rules/Deprecation/DeprecatedHintSyntaxRule.cs)

## Cause

<p id="descr">For specifying table hints use syntax with key word "WITH" and comma separated hints</p>

## Examples

Bad

```sql
SELECT 1 FROM dbo.bar (TABLOCKX)

SELECT 1 FROM dbo.foo WITH (HOLDLOCK INDEX(my_idx))
```

Good

```sql
SELECT 1 FROM dbo.bar WITH (TABLOCKX)

SELECT 1 FROM dbo.foo WITH (HOLDLOCK, INDEX(my_idx))
```

## Tips

💡 There must be a keyword WITH, hints must be separated by a comma.
