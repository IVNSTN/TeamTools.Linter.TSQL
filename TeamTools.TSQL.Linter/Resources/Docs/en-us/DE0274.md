
# Numbered stored procedure is deprecated naming style

|||
|-|-|
| Id | **DE0274**
| Mnemo | NUMBERED_SP
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [NumberedSpRule.cs](../../../Rules/Deprecation/NumberedSpRule.cs)

## Cause

This rule is triggered if a numbered stored procedure is defined.
<p id="descr">Rename the procedure.</p>

## Examples

Bad

```sql
CREATE PROC dbo.foo;1
AS
BEGIN
    RETURN 1;
END
```

Good

```sql
CREATE PROC dbo.foo
AS
BEGIN
    RETURN 1;
END
```
