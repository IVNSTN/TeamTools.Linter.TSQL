
# Deprecated unit is used

|||
|-|-|
| Id | **DE0401**
| Mnemo | DEPRECATED_UNIT
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [DeprecatedUnitReferenceRule.cs](../../../Rules/Deprecation/DeprecatedUnitReferenceRule.cs)

## Cause

This rule is triggered if deprecated unit (function, stored procedure, table) is used.
<p id="descr">Rewrite your code as suggested in violation message.</p>

## Examples

Bad

```sql
dbo.f_today
```

Good

```sql
CAST(GETDATE() AS DATE)
```

## Tips

💡 For each deprecated unit, the violation message provides advice on what to replace this call with.
