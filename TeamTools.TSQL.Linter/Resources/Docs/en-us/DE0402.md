
# Deprecated type is used

|||
|-|-|
| Id | **DE0402**
| Mnemo | DEPRECATED_TYPE
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [DeprecatedTypeRule.cs](../../../Rules/Deprecation/DeprecatedTypeRule.cs)

## Cause

This rule is triggered if a deprecated datatype is used.
<p id="descr">Replace deprecated datatype.</p>

## Examples

Bad

```sql
DECLARE
    @Comment TEXT
    , @Description NTEXT
    , @DocImage IMAGE
    , @UpdateTime TIMESTAMP
    , @Id dbo.TIndex
    , @Value SQL_VARIANT
```

Good

```sql
DECLARE
    @Comment VARCHAR(MAX)
    , @Description NVARCHAR(MAX)
    , @DocImage VARBINARY
    , @Ver ROWVERSION
    , @Id INT
    -- and do not use SQL_VARIANT
```

## Tips

💡 User-Defined data type may be marked as deprecated in linter configuration file.
