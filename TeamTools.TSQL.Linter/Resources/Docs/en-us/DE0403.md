
# Deprecated TSQL instruction

|||
|-|-|
| Id | **DE0403**
| Mnemo | DEPRECATED_INSTRUCTION
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [DeprecatedInstructionRule.cs](../../../Rules/Deprecation/DeprecatedInstructionRule.cs)

## Cause

This rule is triggered if a deprecated instructions or syntax are used.
<p id="descr">Remove deprecated instructions or syntax.</p>

## Examples

Bad

```sql
DROP INDEX table.idx
GRANT/REVOKE/DENY ALL
SETUSER
SET ROWCOUNT
GROUP BY ALL, COMPUTE BY
```

Good

```sql
DROP INDEX idx ON table
list of particular privileges
EXECUTE AS or REVERT
TOP or WHERE constraint
do not use GROUP BY ALL, COMPUTE BY
```
