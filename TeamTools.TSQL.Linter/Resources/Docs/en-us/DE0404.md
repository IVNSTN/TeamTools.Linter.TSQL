

# Deprecated DBCC command

|||
|-|-|
| Id | **DE0404**
| Mnemo | DEPRECATED_DBCC_COMMAND
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [DeprecatedDbccCommandRule.cs](../../../Rules/Deprecation/DeprecatedDbccCommandRule.cs)

## Cause

<p id="descr">Replace deprecated DBCC commands.</p>

## Examples

Bad

```sql
DBCC DBREINDEX 
DBCC INDEXDEFRAG 
DBCC SHOWCONTIG 
PINTABLE, UNPINTABLE
```

Good

```sql
ALTER INDEX ... REBUILD
ALTER INDEX ... REORGANIZE
use sys.dm_db_index_physical_stats
do not use PINTABLE, UNPINTABLE
```
