
# Deprecated option switch

|||
|-|-|
| Id | **DE0405**
| Mnemo | DEPRECATED_OPTIONS
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [DeprecatedOptionRule.cs](../../../Rules/Deprecation/DeprecatedOptionRule.cs)

## Cause

This rule is triggered if a deprecated or not recommended option is used.
<p id="descr">Remove deprecated (not recommended) options.</p>

## Examples

Bad

```sql
OFF status for ANSI_NULLS, ANSI_PADDING, CONCAT_NULL_YIELDS_NULL options
SET REMOTE_PROC_TRANSACTIONS
ANSI_NULL_DFLT_ON, ANSI_NULL_DFLT_OFF, ANSI_DEFAULTS, FORCE_PLAN, TEXTSIZE in code
FMTONLY, SHOWPLAN_ALL, SHOWPLAN_TEXT, SHOWPLAN_XML in prod
```

Good

```sql
do not use OFF status for ANSI_NULLS, ANSI_PADDING, CONCAT_NULL_YIELDS_NULL options
configure Linked servers
do not use options ANSI_NULL_DFLT_ON, ANSI_NULL_DFLT_OFF, ANSI_DEFAULTS, FORCE_PLAN, TEXTSIZE in code
do not use options FMTONLY, SHOWPLAN_ALL, SHOWPLAN_TEXT, SHOWPLAN_XML in prod
```

## Tips

💡 Some options may be used with particular values.
