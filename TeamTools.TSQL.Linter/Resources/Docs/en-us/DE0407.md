
# Deprecated TOP syntax without parenthesis

|||
|-|-|
| Id | **DE0407**
| Mnemo | DEPRECATED_TOP_SYNTAX
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [DeprecatedTopSyntaxRule.cs](../../../Rules/Deprecation/DeprecatedTopSyntaxRule.cs)

## Cause

This rule is triggered if a deprecated TOP syntax is used.
<p id="descr">Enclose the TOP expression in parentheses.</p>

## Examples

Bad

```sql
SELECT TOP 100 ID FROM dbo.cr_3203 ORDER BY ID
```

Good

```sql
SELECT TOP(100) ID FROM dbo.cr_3203 ORDER BY ID
```

## Tips

💡 Syntax without parentheses exists only for backward compatibility.
