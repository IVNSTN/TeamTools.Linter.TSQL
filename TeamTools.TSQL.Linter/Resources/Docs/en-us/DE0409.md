
# Deprecated RAISERROR syntax

|||
|:-|:-|
| Id | **DE0409**
| Mnemo | DEPRECATED_RAISERROR_SYNTAX
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [DeprecatedRaisErrorSyntaxRule.cs](../../../Rules/Deprecation/DeprecatedRaisErrorSyntaxRule.cs)

## Cause

<p id="descr">Enclose the RAISERROR parameters in parentheses.</p>

## Examples

Bad

```sql
RAISERROR 123123 'Some error message'
```

Good

```sql
RAISERROR(123123, 16, 1)

RAISERROR('Some error message', 16, 2)
```

## Tips

💡 RAISERROR syntax without parenthesis is deprecated and cannot be used in modern versions of SQL SERVER with compatibility level above 100.
