
# Deprecated index option syntax

|||
|-|-|
| Id | **DE0741**
| Mnemo | INDEX_OPTION_SYNTAX_DEPRECATED
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md), [Indices](./Group_Indices.md)
| Source code | [DeprecatedIndexOptionSyntaxRule.cs](../../../Rules/Deprecation/DeprecatedIndexOptionSyntaxRule.cs)

## Cause

This rule is triggered if a deprecated index option syntax is used.
<p id="descr">Use parentheses and equal sign.</p>

## Examples

Bad

```sql
WITH DROP_EXISTING
```

Good

```sql
WITH (DROP_EXISTING = ON)
```
