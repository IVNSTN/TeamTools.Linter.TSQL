

# Data modification with NOLOCK is deprecated

|||
|-|-|
| Id | **DE0819**
| Mnemo | MODIFY_WITH_NOLOCK
| Severity | ⚠ Warning
| Category | [Deprecation](./Category_Deprecation.md)
| Source code | [ModifyWithNolockRule.cs](../../../Rules/Deprecation/ModifyWithNolockRule.cs)

## Cause

This rule is triggered when NOLOCK or READUNCOMMITTED hint is used for DELETE or UPDATE in FROM block.
<p id="descr">Remove NOLOCK or READUNCOMMITTED hint.</p>

## Examples

Bad

```sql
DELETE FROM t
FROM dbo.tbl AS t (READUNCOMMITTED)
WHERE a IS NULL
```

Good

```sql
DELETE FROM t
FROM dbo.tbl AS t
WHERE a IS NULL
```
