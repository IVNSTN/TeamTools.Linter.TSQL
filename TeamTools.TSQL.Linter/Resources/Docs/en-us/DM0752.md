

# Unexpected full text catalog management

|||
|-|-|
| Id | **DM0752**
| Mnemo | FULLTEXT_CATALOG_MANAGEMENT
| Severity | ⚠ Warning
| Category | [DatabaseMaintenance](./Category_DatabaseMaintenance.md)
| Source code | [FullTextCatalogManagementRule.cs](../../../Rules/DatabaseMaintenance/FullTextCatalogManagementRule.cs)

## Cause

This rule is triggered if a DDL statement for full-text catalog management is found.
<p id="descr">Remove DDL for full-text catalog management.</p>

## Examples

Bad

```sql
CREATE FULLTEXT CATALOG foo;

CREATE FULLTEXT STOPLIST bar;

CREATE FULLTEXT INDEX ON dbo.foo(src) KEY INDEX ix;
```

## Tips

💡 Utility and administrative stored procedures should be organized within dedicated schemas and databases.
