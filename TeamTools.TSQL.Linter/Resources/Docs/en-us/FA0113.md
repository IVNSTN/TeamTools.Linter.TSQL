

# Target column count does not match source

|||
|-|-|
| Id | **FA0113**
| Mnemo | INSERT_COL_COUNT
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [InsertColumnCountRule.cs](../../../Rules/Failure/InsertColumnCountRule.cs)

## Cause

<p id="descr">In INSERT statement source column number must be equal to target column number.
If you want to fill a column with NULL - provide this value explicitly as source column value.
Or remove particular column from insert target column list.</p>
