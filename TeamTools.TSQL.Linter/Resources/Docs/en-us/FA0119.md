

# Illegal RETURN <value>

|||
|-|-|
| Id | **FA0119**
| Mnemo | RETURN_VALUE_ILLEGAL
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [ReturnValueIllegalRule.cs](../../../Rules/Failure/ReturnValueIllegalRule.cs)

## Cause

<p id="descr">Invalid use of RETURN with a value. It happens that when returning from, for example, a trigger, 0 or 1 is automatically added as a return code, which is swallowed by the build, however, it is incorrect and will break at the first call.</p>
