
# Unknown identifier

|||
|-|-|
| Id | **FA0127**
| Mnemo | UNKNOWN_IDENTIFIER
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [UnknownIdentifierRule.cs](../../../Rules/Failure/UnknownIdentifierRule.cs)

## Cause

This rule is triggered if the code detects the use of an unknown identifier that cannot be associated with any object (not a variable, not a table column...).
<p id="descr">Add @ to the variable name.</p>

## Examples

Bad

```sql
IF my_value > 0
    RETURN;
```

Good

```sql
IF @my_value > 0
    RETURN;
```
