
# SET \<option> is illegal in current context

|||
|-|-|
| Id | **FA0131**
| Mnemo | SET_OPTION_ILLEGAL
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [SetOptionInFunctionRule.cs](../../../Rules/Failure/SetOptionInFunctionRule.cs)

## Cause

This rule is triggered if an invalid option is used in the function creation script.
<p id="descr">Remove the invalid option.</p>

## Examples

Bad

```sql
CREATE FUNCTION dbo.foo (@id INT)
RETURNS INT
AS
BEGIN
    SET NOCOUNT ON;

    RETURN 1;
END;
GO
```

Good

```sql
CREATE FUNCTION dbo.foo (@id INT)
RETURNS INT
AS
BEGIN
    RETURN 1;
END;
GO
```
