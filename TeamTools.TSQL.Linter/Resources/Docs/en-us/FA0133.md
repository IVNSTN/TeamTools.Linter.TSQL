

# Function argument number mismatch

|||
|-|-|
| Id | **FA0133**
| Mnemo | BUILTINS_ARG_COUNT
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [BuiltInFunctionsArgumentCountRule.cs](../../../Rules/Failure/BuiltInFunctionsArgumentCountRule.cs)

## Cause

<p id="descr">The SSDT compiler accepts erroneous constructions of the form ISNULL(<a href="http://a.name ">a.name </a>), which the T-SQL compiler does not accept. You need to fix the function call and pass the required number of arguments. Checks only simple built-in functions, the set of parameters of which is not variable.</p>
