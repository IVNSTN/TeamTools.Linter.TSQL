
# Value rows have different number of elements

|||
|-|-|
| Id | **FA0199**
| Mnemo | VALUES_ELEMENT_NUMBER
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [ValuesElementCountRule.cs](../../../Rules/Failure/ValuesElementCountRule.cs)

## Cause

This rule is triggered if the sets of values for `VALUES` contain a different number of elements.
<p id="descr">Align the number of elements in value sets.</p>

## Examples

Bad

```sql
SELECT x, y 
FROM
(
    VALUES ('a', 'b', 0)
           , ('c', 'd')
) AS v (x, y);
```

Good

```sql
SELECT x, y 
FROM 
(
    VALUES ('a', 'b')
           , ('c', 'd')
) AS v (x, y);
```
