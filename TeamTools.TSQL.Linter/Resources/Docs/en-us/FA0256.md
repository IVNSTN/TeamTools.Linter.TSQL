
# Missing semicolon before calling THROW

|||
|-|-|
| Id | **FA0256**
| Mnemo | SEMICOLON_BEFORE_THROW
| Severity | ⚠ Warning
| Category | [Failure](./Category_Failure.md)
| Source code | [SemicolonBeforeThrowRule.cs](../../../Rules/Failure/SemicolonBeforeThrowRule.cs)

## Cause

This rule is triggered if there is no semicolon before `THROW`.
<p id="descr">Add a semicolon.</p>

## Examples

Bad

```sql
if @a > @b BEGIN
    THROW 500001, 'test', 1;
END;
```

Good

```sql
if @a > @b BEGIN;
    THROW 500001, 'test', 1;
END
```
