
# Invalid XML literal

|||
|-|-|
| Id | **FA0283**
| Mnemo | INVALID_XML_LITERAL
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [XmlValidSourceRule.cs](../../../Rules/Failure/XmlValidSourceRule.cs)

## Cause

This rule is triggered if literals that are not valid XML structures are used in expressions for creating the `XML SCHEMA COLLECTION` or in assigning XML type variables.
<p id="descr">Correct the text of the XML literal.</p>

## Examples

Bad

```sql
CREATE XML SCHEMA COLLECTION dbo.moex AS
N'<asdf></zzz>';
GO
```

Good

```sql
CREATE XML SCHEMA COLLECTION dbo.moex AS
N'<xsd:schema xmlns:xsd="http://www.w3.org/2001/XMLSchema" elementFormDefault="qualified">
</xsd:schema>';
GO
```
