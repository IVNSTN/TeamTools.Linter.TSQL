

# Invalid number of arguments

|||
|-|-|
| Id | **FA0704**
| Mnemo | INVALID_ARGUMENT_COUNT
| Severity | ⚠ Warning
| Category | [Failure](./Category_Failure.md)
| Source code | [InvalidFunctionArgumentCountRule.cs](../../../Rules/Failure/InvalidFunctionArgumentCountRule.cs)

## Cause

This rule is triggered when an incorrect number of arguments are passed to the function.
<p id="descr">Pass the required number of arguments.</p>

## Examples

Bad

```sql
SET @bar = DATEPART(DAY, @dt, 1, 'M')
```

Good

```sql
SET @bar = DATEPART(DAY, @dt)
```
