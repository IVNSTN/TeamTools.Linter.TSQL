

# Invalid argument value

|||
|-|-|
| Id | **FA0705**
| Mnemo | INVALID_ARGUMENT
| Severity | ⚠ Warning
| Category | [Failure](./Category_Failure.md)
| Source code | [InvalidFunctionArgumentRule.cs](../../../Rules/Failure/InvalidFunctionArgumentRule.cs)

## Cause

This rule is triggered when an invalid argument value is passed to the function.
<p id="descr">Fix argument value.</p>

## Examples

Bad

```sql
SET @bar = DATEPART('DAY', @dt)
```

Good

```sql
SET @bar = DATEPART(DAY, @dt)
```
