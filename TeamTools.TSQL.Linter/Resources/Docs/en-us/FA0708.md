

# Division by zero

|||
|-|-|
| Id | **FA0708**
| Mnemo | ZERO_DIVIDE
| Severity | ⚠ Warning
| Category | [Failure](./Category_Failure.md)
| Source code | [ZeroDivideRule.cs](../../../Rules/Failure/ZeroDivideRule.cs)

## Cause

This rule is triggered when division by zero is occurred in code.
<p id="descr">Remove division by zero.</p>

## Examples

Bad

```sql
DECLARE @div INT = 0
SELECT 10 / @div
```

Good

```sql
DECLARE @div INT = 5
SELECT 10 / @div
```
