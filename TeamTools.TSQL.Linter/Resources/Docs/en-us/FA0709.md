

# Intentional exception

|||
|-|-|

## Cause

This rule is triggered when intentional exception is occurred in code.
<p id="descr">Remove intentional exception.</p>

## Examples

Bad

```sql
SELECT 1 / 0
```

Good

```sql
THROW 58134, 'Divide by zero error encountered.', 1;
```
