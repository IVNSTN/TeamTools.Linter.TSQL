

# DDL trigger cannot have schema name

|||
|-|-|
| Id | **FA0711**
| Mnemo | DDL_TRIGGER_NO_SCHEMA
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md), [Triggers](./Group_Triggers.md)
| Source code | [DdlTriggerSchemaRule.cs](../../../Rules/Failure/DdlTriggerSchemaRule.cs)

## Cause

This rule is triggered when DDL trigger has schema name.
<p id="descr">Remove schema name.</p>

## Examples

Bad

```sql
CREATE TRIGGER ddl.audit_trigger
ON DATABASE
FOR DDL_DATABASE_LEVEL_EVENTS
AS
BEGIN
    ...
END;
```

Good

```sql
CREATE TRIGGER ddl_audit_trigger
ON DATABASE
FOR DDL_DATABASE_LEVEL_EVENTS
AS
BEGIN
    ...
END;
```
