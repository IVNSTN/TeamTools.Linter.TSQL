

# Incompatible index option

|||
|-|-|
| Id | **FA0742**
| Mnemo | INDEX_OPTIONS_INCOMPATIBLE
| Severity | ⚠ Warning
| Category | [Failure](./Category_Failure.md), [Indices](./Group_Indices.md)
| Source code | [IncompatibleIndexOptionRule.cs](../../../Rules/Failure/IncompatibleIndexOptionRule.cs)

## Cause

This rule is triggered when index has incompatible options.
<p id="descr">Remove incompatible options or add missing.</p>

## Examples

Bad

```sql
CREATE INDEX IX
    ON dbo.foo (col)
    WHERE (col2 IS NOT NULL)
    WITH (IGNORE_DUP_KEY = ON);
```

Good

```sql
CREATE INDEX IX
    ON dbo.foo (col)
    WHERE (col2 IS NOT NULL);
```
