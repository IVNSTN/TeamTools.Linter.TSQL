

# Deadlock priority value is out of range

|||
|-|-|
| Id | **FA0757**
| Mnemo | DEADLOCK_PRIORITY_OUT_OF_RANGE
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [DeadlockPriorityValueRangeRule.cs](../../../Rules/Failure/DeadlockPriorityValueRangeRule.cs)

## Cause

This rule is triggered when deadlock priority value is not a number in the range -10 to 10.
<p id="descr">Use the correct value of DEADLOCK_PRIORITY.</p>

## Examples

Bad

```sql
SET DEADLOCK_PRIORITY -100
```

Good

```sql
SET DEADLOCK_PRIORITY -10
```
