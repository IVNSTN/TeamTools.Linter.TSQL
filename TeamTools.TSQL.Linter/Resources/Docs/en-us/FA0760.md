

# Datatype is not supported in natively compiled modules

|||
|-|-|
| Id | **FA0760**
| Mnemo | NATIVELY_UNSUPPORTED_TYPE
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md), [InMemory](./Group_InMemory.md)
| Source code | [NativelyUnsupportedTypeRule.cs](../../../Rules/Failure/NativelyUnsupportedTypeRule.cs)

## Cause

This rule is triggered when datatype is not supported in natively compiled modules.
<p id="descr">Remove unsupported datatype.</p>

## Examples

Bad

```sql
CREATE TABLE foo.bar
(
    sys_name        HIERARCHYID  NOT NULL
    , profile_id    GEOGRAPHY    NOT NULL
    , rv            ROWVERSION   NOT NULL
    , CONSTRAINT PK_foo_bar PRIMARY KEY NONCLUSTERED HASH (sys_name) WITH (BUCKET_COUNT = 8000000)
)
WITH (MEMORY_OPTIMIZED = ON, DURABILITY = SCHEMA_AND_DATA);
```

Good

```sql
CREATE TABLE foo.bar
(
    sys_name        VARCHAR(20)  NOT NULL
    , profile_id    SMALLINT     NOT NULL
    , sign_datetime DATETIME2(0) NOT NULL
    , CONSTRAINT PK_foo_bar PRIMARY KEY NONCLUSTERED HASH (sys_name) WITH (BUCKET_COUNT = 8000000)
)
WITH (MEMORY_OPTIMIZED = ON, DURABILITY = SCHEMA_AND_DATA);
```
