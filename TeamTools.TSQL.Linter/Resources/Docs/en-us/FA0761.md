

# Feature is not supported in natively compiled modules

|||
|-|-|
| Id | **FA0761**
| Mnemo | NATIVELY_UNSUPPORTED_INSTRUCTION
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md), [InMemory](./Group_InMemory.md)
| Source code | [NativelyUnsupportedInstructionRule.cs](../../../Rules/Failure/NativelyUnsupportedInstructionRule.cs)

## Cause

This rule is triggered when feature is not supported in natively compiled modules.
<p id="descr">Remove unsupported feature.</p>

## Examples

Bad

```sql
CREATE PROC #tmp
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```

Good

```sql
CREATE PROC tmp
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```
