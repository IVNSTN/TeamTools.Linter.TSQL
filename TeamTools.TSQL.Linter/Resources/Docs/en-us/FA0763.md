

# SCHEMABINDING isn't specified for natively compiled object

|||
|-|-|
| Id | **FA0763**
| Mnemo | NATIVE_COMPILATION_SCHEMABINDING
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md), [InMemory](./Group_InMemory.md)
| Source code | [NativelyCompiledNeedsSchemabindingRule.cs](../../../Rules/Failure/NativelyCompiledNeedsSchemabindingRule.cs)

## Cause

This rule is triggered when SCHEMABINDING isn't specified for natively compiled object.
<p id="descr">Specify SCHEMABINDING.</p>

## Examples

Bad

```sql
CREATE PROC tmp
WITH NATIVE_COMPILATION, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```

Good

```sql
CREATE PROC tmp
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```
