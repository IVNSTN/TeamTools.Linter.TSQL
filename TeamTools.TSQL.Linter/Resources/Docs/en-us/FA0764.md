

# HASH index does not support sorting

|||
|-|-|
| Id | **FA0764**
| Mnemo | HASH_INDEX_SORTING
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md), [Indices](./Group_Indices.md), [InMemory](./Group_InMemory.md)
| Source code | [HashIndexSortingRule.cs](../../../Rules/Failure/HashIndexSortingRule.cs)

## Cause

This rule is triggered when sorting is specified for HASH index.
<p id="descr">Remove sorting.</p>

## Examples

Bad

```sql
CREATE TABLE foo.bar
(
    sys_name        VARCHAR(20)  NOT NULL
    , sign_datetime DATETIME2(0) NOT NULL
    , CONSTRAINT PK_foo_bar PRIMARY KEY NONCLUSTERED HASH (sys_name ASC) WITH (BUCKET_COUNT = 8000000)
)
WITH (MEMORY_OPTIMIZED = ON, DURABILITY = SCHEMA_AND_DATA);
```

Good

```sql
CREATE TABLE foo.bar
(
    sys_name        VARCHAR(20)  NOT NULL
    , sign_datetime DATETIME2(0) NOT NULL
    , CONSTRAINT PK_foo_bar PRIMARY KEY NONCLUSTERED HASH (sys_name) WITH (BUCKET_COUNT = 8000000)
)
WITH (MEMORY_OPTIMIZED = ON, DURABILITY = SCHEMA_AND_DATA);
```
