

# Illegal datatype is used for SPARSE column

|||
|-|-|
| Id | **FA0771**
| Mnemo | SPARSE_UNSUPPORTED_TYPE
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [SparseUnsupportedTypeRule.cs](../../../Rules/Failure/SparseUnsupportedTypeRule.cs)

## Cause

This rule is triggered when illegal datatype is used for SPARSE column.
<p id="descr">Remove SPARSE option or change datatype.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.foo
(
    cr GEOGRAPHY SPARSE NULL
);
```

Good

```sql
CREATE TABLE dbo.foo
(
    cr INT SPARSE NULL
);
```
