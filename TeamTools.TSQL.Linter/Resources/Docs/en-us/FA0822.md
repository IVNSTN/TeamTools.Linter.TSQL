

# Invalid option for CLR module

|||
|-|-|
| Id | **FA0822**
| Mnemo | INVALID_CLR_OPTION
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [InvalidClrOptionRule.cs](../../../Rules/Failure/InvalidClrOptionRule.cs)

## Cause

This rule is triggered when ENCRYPTION or RECOMPILE option is used for CLR module.
<p id="descr">Remove ENCRYPTION or RECOMPILE option.</p>

## Examples

Bad

```sql
CREATE FUNCTION dbo.foo (@arg VARCHAR(MAX))
RETURNS INT
WITH ENCRYPTION
AS
EXTERNAL NAME proj.dll.func;
GO
```

Good

```sql
CREATE FUNCTION dbo.foo (@arg VARCHAR(MAX))
RETURNS INT
AS
EXTERNAL NAME proj.dll.func;
GO
```
