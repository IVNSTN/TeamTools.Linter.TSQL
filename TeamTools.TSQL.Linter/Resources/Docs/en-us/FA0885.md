

# RAISERROR with severity 19 or higher missing WITH LOG option

|||
|-|-|
| Id | **FA0885**
| Mnemo | RAISERROR_NEEDS_WITH_LOG
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [RaiseErrorNeedsWithLogRule.cs](../../../Rules/Failure/RaiseErrorNeedsWithLogRule.cs)

## Cause

This rule is triggered when `RAISERROR` with severity 19 or higher is specified without `WITH LOG`.
<p id="descr">Specify WITH LOG option.</p>

## Examples

Bad

```sql
RAISERROR ('error', 19, 2);
```

Good

```sql
RAISERROR ('error', 19, 2) WITH LOG;
```
