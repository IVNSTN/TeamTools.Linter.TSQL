

# Column can be modified only once per statement

|||
|-|-|
| Id | **FA0901**
| Mnemo | DUP_COLUMN_MODIFIER
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [UniqueColumnModificatorDefinitionRule.cs](../../../Rules/Failure/UniqueColumnModificatorDefinitionRule.cs)

## Cause

The rule is triggered if the same column is specified more than once in the INSERT, UPDATE, OUTPUT INTO expressions.
<p id="descr">Delete duplicate columns.</p>

## Examples

Bad

```sql
UPDATE mgr
SET
    mgr.name = 'Ivan'
    , mgr.last_name = 'Ivanov'
    , mgr.name = 'Andrey'
FROM dbo.managers AS mgr
WHERE mgr.manager_id = @id;
```

Good

```sql
UPDATE mgr
SET
    mgr.name = 'Ivan'
    , mgr.last_name = 'Ivanov'
FROM dbo.managers AS mgr
WHERE mgr.manager_id = @id;
```
