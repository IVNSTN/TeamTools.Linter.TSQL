

# Index refers to unknown column

|||
|-|-|
| Id | **FA0904**
| Mnemo | INDEX_REFERS_UNKNOWN_COL
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [IndexReferencingUnknownColumnRule.cs](../../../Rules/Failure/IndexReferencingUnknownColumnRule.cs)

## Cause

The rule is triggered if the table definition does not contain a column that is mentioned in the index for that table.
<p id="descr">Specify the columns that are present in the table.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.Example (id INT NOT NULL, value_foo INT NULL, value_bar INT NULL, INDEX ix_dbo_example_id (id, value_foo, dt_create));
```

Good

```sql
CREATE TABLE dbo.Example (id INT NOT NULL, value_foo INT NULL, value_bar INT NULL, dt_create DATE NULL, INDEX ix_dbo_example_id (id, value_foo, dt_create));
```
