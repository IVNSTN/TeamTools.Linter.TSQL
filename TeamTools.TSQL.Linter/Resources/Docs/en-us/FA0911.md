

# Format argument count does not match wildcard count

|||
|-|-|
| Id | **FA0911**
| Mnemo | FORMAT_ARG_COUNT_MISMATCH
| Severity | ⚠ Warning
| Category | [Failure](./Category_Failure.md)
| Source code | [FormatArgumentCountMismatchRule.cs](../../../Rules/Failure/FormatArgumentCountMismatchRule.cs)

## Cause

The rule is triggered if the number of placeholders and values does not match in the FORMATMESSAGE or RAISERROR calls.
<p id="descr">Specify an equal number of placeholders and the values themselves.</p>

## Exmaple

Bad

```sql
SELECT FORMATMESSAGE('Exmaple Message: value1 = %s, value2 = %s', '50');
RAISERROR(N'Exmaple Message: value1 = %s, value2 = %s', 10, 1, '10');
```

Good

```sql
SELECT FORMATMESSAGE('Exmaple Message: value1 = %s, value2 = %s', '50', '100');
RAISERROR(N'Exmaple Message: value1 = %s, value2 = %s', 10, 1, '100', '300');
```
