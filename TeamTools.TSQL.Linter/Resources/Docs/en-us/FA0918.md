

# Invalid 'NOT NULL' constraint for parameter

|||
|-|-|
| Id | **FA0918**
| Mnemo | INVALID_NOT_NULL_PARAM
| Severity | ⛔ Error
| Category | [Failure](./Category_Failure.md)
| Source code | [UnexpectedNotNullParameterRule.cs](../../../Rules/Failure/UnexpectedNotNullParameterRule.cs)

## Cause

The rule is triggered if a NOT NULL constraint is specified for the input parameter. NOT NULL constraint can only be applied to NATIVELY COMPILED modules or inlined TVFs.
<p id="descr">Remove the NOT NULL constraint for the input parameter.</p>

## Examples

Bad

```sql
CREATE PROCEDURE tmp.SPExample
    @id INT NOT NULL
AS
BEGIN
    SELECT @id AS id;
END;
```

Good

```sql
CREATE PROCEDURE tmp.SPExample
    @id INT
AS
BEGIN
    SELECT @id AS id;
END;
```
