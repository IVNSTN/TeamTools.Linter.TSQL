

# Invalid FORMATMESSAGE wildcard

|||
|-|-|
| Id | **FA0986**
| Mnemo | FORMAT_INVALID_WILDCARD
| Severity | ⚠ Warning
| Category | [Failure](./Category_Failure.md)
| Source code | [FormatMessageInvalidWildcardRule.cs](../../../Rules/Failure/FormatMessageInvalidWildcardRule.cs)

## Cause

The rule is triggered if the wildcard is specified incorrectly in the FORMATMESSAGE or RAISERROR call.
<p id="descr">Specify a correct wildcard.</p>

## Examples

Bad

```sql
RAISERROR('The wildcard is specified incorrectly (string) %xs', 10, 1, 'typo');

SELECT FORMATMESSAGE('The wildcard is specified incorrectly (integer) %bi', 10) AS Result;
```

Good

```sql
RAISERROR('The wildcard is specified correctly (string) %s', 10, 1, 'param');

SELECT FORMATMESSAGE('The wildcard is specified correctly (integer) %i', 10) AS Result;
```

## Tips
    
💡 Type specification: Signed integer(d or i), Unsigned octal(o), String(s), Unsigned integer(u), Unsigned hexadecimal(x or X )
