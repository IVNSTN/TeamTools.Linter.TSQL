

# Expected no whitespace characters at the beginning of the file

|||
|-|-|
| Id | **FL0272**
| Mnemo | BOF_REDUNDANT_WHITESPACE
| Severity | ⚠ Warning
| Category | [Files](./Category_Files.md)
| Source code | [NewLineAtBeginOfFileRule.cs](../../../Rules/Files/NewLineAtBeginOfFileRule.cs)

## Cause

This rule is triggered if the file starts with empty lines, spaces or tabs.

<p id="descr">Remove any empty lines, spaces, and tabs.</p>

## Examples

Bad

```sql

      CREATE PROCEDURE dbo.test...
```

Good

```sql
CREATE PROCEDURE dbo.test...
```
