

# Line not end with the CRLF

|||
|-|-|
| Id | **FL0303**
| Mnemo | CRLF
| Severity | ⚠ Warning
| Category | [Files](./Category_Files.md)
| Source code | [CrlfRule.cs](../../../Rules/Files/CrlfRule.cs)

## Cause

This rule is triggered if at least one line does not end with CRLF.

<p id="descr">Replace the end of the line with CRLF.</p>
