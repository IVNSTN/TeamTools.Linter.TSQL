

# Avoid using square brackets in the file name

|||
|-|-|
| Id | **FL0304**
| Mnemo | FILE_NAME_SQUARE_BRACKETS
| Severity | ⚠ Warning
| Category | [Files](./Category_Files.md)
| Source code | [FileNameSquareBracketsRule.cs](../../../Rules/Files/FileNameSquareBracketsRule.cs)

## Cause

This rule is triggered if there are square brackets in the file name.

<p id="descr">Remove the square brackets from the file name.</p>

## Examples

Bad

```
[dbo].[table].sql
```

Good

```
dbo.table.sql
```
