

# The file name does not match the object name

|||
|-|-|
| Id | **FL0305**
| Mnemo | FILE_OBJECT_NAME
| Severity | ⚠ Warning
| Category | [Files](./Category_Files.md)
| Source code | [FileNameMatchesObjectNameRule.cs](../../../Rules/Files/FileNameMatchesObjectNameRule.cs)

## Cause

This rule is triggered when there is a mismatch between the file name and the name of the object found within it.

<p id="descr">Make the file name and object name match.</p>

## Examples

Bad

dbo.**table**.sql content:

```sql
CREATE TABLE dbo.person
...
```

Good

dbo.**person**.sql content:

```sql
CREATE TABLE dbo.person
...
```
