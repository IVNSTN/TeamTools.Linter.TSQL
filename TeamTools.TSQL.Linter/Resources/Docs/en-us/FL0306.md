

# A sngle new line is expected at the end of the file

|||
|-|-|
| Id | **FL0306**
| Mnemo | EOF_NEWLINE
| Severity | ⚠ Warning
| Category | [Files](./Category_Files.md)
| Source code | [NewLineAtEndOfFileRule.cs](../../../Rules/Files/NewLineAtEndOfFileRule.cs)

## Cause

This rule is triggered when there is more or less than one empty line at the end of the file.

<p id="descr">Delete the extra empty lines or add the missing one.</p>

## Examples

Bad

no empty lines
```
...
GO
```

two empty lines
```
...
GO


```

Good

one empty line
```
...
GO

```
