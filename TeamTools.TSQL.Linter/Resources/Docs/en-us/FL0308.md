

# Last command in SQL file must be GO

|||
|-|-|
| Id | **FL0308**
| Mnemo | EOF_GO
| Severity | ⚠ Warning
| Category | [Files](./Category_Files.md)
| Source code | [EndOfFileGoRule.cs](../../../Rules/Files/EndOfFileGoRule.cs)

## Cause

This rule is triggered if the GO command was not found at the end of the script.

<p id="descr">Add GO to the end of the script.</p>

## Examples

Bad

```sql
CREATE PROCEDURE dbo.test
AS
BEGIN
SELECT 1;
END;
```

Good

```sql
CREATE PROCEDURE dbo.test
AS
BEGIN
SELECT 1;
END;
GO
```
