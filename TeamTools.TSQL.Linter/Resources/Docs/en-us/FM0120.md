

# Missing global BEGIN-END

|||
|-|-|
| Id | **FM0120**
| Mnemo | BODY_BEGIN_END
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [BeginEndForBodyRule.cs](../../../Rules/Formatting/BeginEndForBodyRule.cs)

## Cause

The rule is triggered if the body of a stored procedure or trigger is not enclosed in BEGIN-END.
<p id="descr">Enclose the body of the stored procedure or trigger within a surrounding BEGIN-END block.</p>

## Examples

Bad

```sql
CREATE PROCEDURE SampleProcedure
AS
SELECT * FROM TableA;
END;
```

Good

```sql
CREATE PROCEDURE SampleProcedure
AS
BEGIN
    SELECT * FROM TableA;
END;
```
