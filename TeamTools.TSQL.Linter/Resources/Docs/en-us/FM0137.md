

# Missing BEGIN-END after IF/ELSE

|||
|-|-|
| Id | **FM0137**
| Mnemo | IF_BEGIN_END
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [BeginEndForIfRule.cs](../../../Rules/Formatting/BeginEndForIfRule.cs)

## Cause

The rule is triggered if the IF-ELSE does not have a BEGIN-END block.
<p id="descr">Add the BEGIN-END blocks for IF/ELSE.</p>

## Examples

Bad

```sql
IF @a = 1
    SET @a = 0;
ELSE
    SET @a = 1;
```

Good

```sql
IF @a = 1
BEGIN
    SET @a = 0;
END;
ELSE
BEGIN
    SET @a = 1;
END;
```
