

# Missing BEGIN-END after WHILE

|||
|-|-|
| Id | **FM0150**
| Mnemo | WHILE_BEGIN_END
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [BeginEndForWhileRule.cs](../../../Rules/Formatting/BeginEndForWhileRule.cs)

## Cause

The rule is triggered if the body of the WHILE loop is not enclosed in the BEGIN-END.
<p id="descr">Enclose the body of the loop in the BEGIN-END.</p>

## Examples

Bad

```sql
WHILE(1 = 1)
    EXEC some_procedure
```

Good

```sql
WHILE(1 = 1)
BEGIN
    EXEC some_procedure
END;
```
