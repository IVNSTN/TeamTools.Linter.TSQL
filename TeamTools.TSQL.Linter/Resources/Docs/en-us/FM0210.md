

# Declaration is not formatted as table

|||
|-|-|
| Id | **FM0210**
| Mnemo | DECLARE_FORMAT
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [DeclareFormatAsTableRule.cs](../../../Rules/Formatting/DeclareFormatAsTableRule.cs)

## Cause

The rule is triggered if the variable declaration block is not tabularly aligned.
<p id="descr">Align the names and types tabularly.</p>

## Examples

Bad

```sql
CREATE PROCEDURE InvalidDeclareFormat
AS
BEGIN
    DECLARE @Var1 INT, @Var2 VARCHAR(100)
END
```

Good

```sql
CREATE PROCEDURE ValidDeclareFormat
AS
BEGIN
    DECLARE
        @Var1       INT
        , @Var2     VARCHAR(100);
END
```
