

# Parameter list is not formatted as table

|||
|-|-|
| Id | **FM0211**
| Mnemo | EXEC_FORMAT
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [ExecParamsFormatAsTableRule.cs](../../../Rules/Formatting/ExecParamsFormatAsTableRule.cs)

## Cause

The rule is triggered if the parameters of the called stored procedure are not formatted as a table.
<p id="descr">When making an stored procedure call, the parameters should be tabulated.</p>

## Examples

Bad

```sql
CREATE PROCEDURE dbo.SaveData
    @EventID UNIQUEIDENTIFIER
    , @MessageTypeID UNIQUEIDENTIFIER
AS
BEGIN
    RETURN 1
END
```

Good

```sql
CREATE PROCEDURE dbo.SaveData
    @EventID         UNIQUEIDENTIFIER
    , @MessageTypeID UNIQUEIDENTIFIER
AS
BEGIN
    RETURN 1
END
```
