
# END not aligned with its own BEGIN

|||
|-|-|
| Id | **FM0218**
| Mnemo | BEGIN_END_ALIGN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [BeginEndAlignedRule.cs](../../../Rules/Formatting/BeginEndAlignedRule.cs)

## Cause

The rule is triggered if the BEGIN and the corresponding END are not aligned the same way.
<p id="descr">Set the same indentation on the left for BEGIN and the corresponding END.</p>

## Examples

Bad

```sql
IF(@i = 1)
BEGIN
    RETURN 1
    END
```

Good

```sql
IF(@i = 1)
BEGIN
    RETURN 1
END
```
