

# END not aligned with it's own BEGIN

|||
|-|-|
| Id | **FM0218**
| Mnemo | BEGIN_END_ALIGN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [BeginEndAlignedRule.cs](../../../Rules/Formatting/BeginEndAlignedRule.cs)

## Cause

<p id="descr">BEGIN и соответствующий ему END должны быть выровнены одинаково, иметь одинаковый отступ слева.</p>
