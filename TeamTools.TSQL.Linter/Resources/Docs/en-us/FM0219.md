

# DML statement key elements not aligned similarly

|||
|-|-|
| Id | **FM0219**
| Mnemo | STATEMENT_ALIGN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [StatementElementsAlignedRule.cs](../../../Rules/Formatting/StatementElementsAlignedRule.cs)

## Cause

<p id="descr">Ключевые элементы DML выражений должны быть выровнены одинаково, иметь одинаковый отступ слева. На однострочные выражения не реагирует.</p>
