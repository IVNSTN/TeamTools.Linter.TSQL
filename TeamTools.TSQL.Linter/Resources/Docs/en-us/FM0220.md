

# JOIN/APPLY clauses not aligned with FROM clause

|||
|-|-|
| Id | **FM0220**
| Mnemo | JOINS_ALIGN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [JoinsAlignedToFromRule.cs](../../../Rules/Formatting/JoinsAlignedToFromRule.cs)

## Cause

The rule is triggered if the join operators are not formatted in a tabular manner.
<p id="descr">Format the join operators in a tabular manner.</p>

## Examples

Bad

```sql
SELECT * FROM
TableA AS A
 INNER JOIN  TableB AS B
    on A.ID = B.ID
  INNER JOIN  TableC AS C
    on A.ID = C.ID
```

Good

```sql
SELECT * FROM
TableA AS A
INNER JOIN  TableB AS B
    on A.ID = B.ID
INNER JOIN  TableC AS C
    on A.ID = C.ID
```
