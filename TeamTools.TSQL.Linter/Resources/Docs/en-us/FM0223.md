

# Selected items not aligned similarly

|||
|-|-|
| Id | **FM0223**
| Mnemo | SELECTED_ITEMS_ALIGN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [SelectedItemsAlignedRule.cs](../../../Rules/Formatting/SelectedItemsAlignedRule.cs)

## Cause

The rule is triggered if the list of selected columns is not formatted in a tabular manner and the columns have different left indentations.
<p id="descr">Format the list of selected columns in a tabular manner and set the same left indentation.</p>

## Examples

Bad

```sql
SELECT
TradeDate, ISIN, ISIN144A, ShortNamer
, Country, Status, DtInsert, DtUpdate
FROM dbo.Securities;
```

Good

```sql
SELECT
TradeDate
, ISIN
, ISIN144A
, ShortNamer
, Country
, Status
, DtInsert
, DtUpdate
FROM dbo.Securities;
```

## Tips

💡 Single-line expressions are not affected.
