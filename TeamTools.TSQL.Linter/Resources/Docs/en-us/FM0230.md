

# Single whitespace around operator expected

|||
|-|-|
| Id | **FM0230**
| Mnemo | WHITESPACE_AROUND_OPERATOR
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [WhitespaceAroundOperatorRule.cs](../../../Rules/Formatting/WhitespaceAroundOperatorRule.cs)

## Cause

The rule is triggered if there are no spaces or too many spaces around simple operators.
<p id="descr">Make sure there is exactly one space around the operators.</p>

## Examples

Bad

```sql
IF(@i >  1)
BEGIN
    SET @i = 0;
END
```

Good

```sql
IF(@i > 1)
BEGIN
    SET @i = 0;
END
```
