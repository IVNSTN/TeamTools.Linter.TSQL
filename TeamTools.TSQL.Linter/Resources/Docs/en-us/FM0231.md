

# Single whitespace after comma expected

|||
|-|-|
| Id | **FM0231**
| Mnemo | WHITESPACE_AFTER_COMMA
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [WhitespaceAfterCommaRule.cs](../../../Rules/Formatting/WhitespaceAfterCommaRule.cs)

## Cause

The rule is triggered if there is no space or too many spaces after the comma in lists.
<p id="descr">Make sure there is exactly one space after each comma in lists before the next literal.</p>

## Examples

Bad

```sql
DECLARE
    @a INT
    ,  @b INT
    ,@c DECIMAL(18,   2);
```

Good

```sql
DECLARE
    @a  INT
    , @b INT
    , @c DECIMAL(18, 2);
```
