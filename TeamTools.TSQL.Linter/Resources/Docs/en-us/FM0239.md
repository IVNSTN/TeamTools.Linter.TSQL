

# Single space after single-line comment start expected

|||
|-|-|
| Id | **FM0239**
| Mnemo | SL_COMMENT_LEADING_SPACE
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [SingleSpaceAfterSingleLineCommentBeginRule.cs](../../../Rules/Formatting/SingleSpaceAfterSingleLineCommentBeginRule.cs)

## Cause

The rule is triggered if a single-line comment does not start with two dashes and a space.
<p id="descr">Make sure that there are two dashes and one space before the text of a single-line comment.</p>

## Examples

Bad

```sql
/* comment text */
```

Good

```sql
-- comment text
```
