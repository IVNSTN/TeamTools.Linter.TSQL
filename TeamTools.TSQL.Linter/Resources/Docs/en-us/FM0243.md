

# Multiline comment alignment

|||
|-|-|
| Id | **FM0243**
| Mnemo | ML_COMMENT_CONTENT_ALIGN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [MultilineCommentOpenCloseTagAlignRule.cs](../../../Rules/Formatting/MultilineCommentOpenCloseTagAlignRule.cs)

## Cause

This rule is triggered if the indentation of а multiline comment is less than that of the opening and closing tags.
<p id="descr">Increase the indentation of the comment text.</p>

## Examples

Bad

```sql
    /*
This is a
multiline comment
    */

    /* This is a
multiline comment */
```

Good

```sql
    /*
    This is a
    multiline comment
    */

    /* This is a
    multiline comment */
```
