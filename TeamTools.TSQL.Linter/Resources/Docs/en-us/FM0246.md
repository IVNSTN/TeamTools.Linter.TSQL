

# Multiple single-line comments alignment

|||
|-|-|
| Id | **FM0246**
| Mnemo | MANY_SL_COMMENTS_ALIGNED
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [MultipleSingleLineCommentsAlignRule.cs](../../../Rules/Formatting/MultipleSingleLineCommentsAlignRule.cs)

## Cause

This rule is triggered if multiple single-line comments following one another have different indentations.
<p id="descr">Align the indentation of the comments.</p>

## Examples

Bad

```sql
-- Comment
    -- Yet another comment
```

Good

```sql
-- Comment
-- Yet another comment
```
