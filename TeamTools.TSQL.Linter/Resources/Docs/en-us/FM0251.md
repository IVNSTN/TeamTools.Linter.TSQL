

# Dangling semicolon

|||
|-|-|
| Id | **FM0251**
| Mnemo | DANGLING_SEMICOLON
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [DanglingSemicolonRule.cs](../../../Rules/Formatting/DanglingSemicolonRule.cs)

## Cause

This rule is triggered if there is a single semicolon in the line.
<p id="descr">Delete the single semicolon or the entire line.</p>

## Examples

Bad

```sql
SELECT Id FROM SomeTable;
;
```

Good

```sql
SELECT Id FROM SomeTable;
```

## Tips

💡 This can happen as result of applying the code formatting if semicolon was poorly placed.
