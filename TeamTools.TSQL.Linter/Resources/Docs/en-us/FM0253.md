

# Open-close parenthesis not aligned

|||
|-|-|
| Id | **FM0253**
| Mnemo | OPEN_CLOSE_PARENTHESIS_ALIGNED
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [OpenCloseParenthesisAlignRule.cs](../../../Rules/Formatting/OpenCloseParenthesisAlignRule.cs)

## Cause

<p id="descr">Если конструкция в скобках оформлена как блок, где ( и ) находятся на отдельной строке, то ( и ) должны иметь одинаковый отступ слева как BEGIN и END</p>
