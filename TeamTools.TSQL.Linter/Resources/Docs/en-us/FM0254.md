

# Nested BEGIN-END left margin alignment

|||
|-|-|
| Id | **FM0254**
| Mnemo | NESTED_BEGIN_END_LEFT_MARGIN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [NestedBeginEndAlignLeftMarginRule.cs](../../../Rules/Formatting/NestedBeginEndAlignLeftMarginRule.cs)

## Cause

This rule is triggered if a nested BEGIN-END blocks indentation is less than or equals to the outer blocks indentation.
<p id="descr">Increase the indentation of the nested BEGIN-END blocks.</p>

## Examples

Bad

```sql
BEGIN -- parent block
BEGIN -- nested block
END;
END;
```

Good

```sql
BEGIN -- parent block
    BEGIN -- nested block
    END;
END;
```
