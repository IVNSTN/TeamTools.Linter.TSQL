

# Nested parenthesis left margin alignment

|||
|-|-|
| Id | **FM0255**
| Mnemo | NESTED_PARENTHESIS_LEFT_MARGIN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [NestedParenthesisAlignLeftMarginRule.cs](../../../Rules/Formatting/NestedParenthesisAlignLeftMarginRule.cs)

## Cause

This rule is triggered if a nested parentheses indentation is less than or equals to the outer parentheses indentation.
<p id="descr">Increase the indentation of the nested parentheses.</p>

## Examples

Bad

```sql
SELECT Field
FROM SomeTable
WHERE
(
    Field1 = 'value1' AND
(
    Field2 = 'value2' OR Field3 = 'value2'
)
);
```

Good

```sql
SELECT Field
FROM SomeTable
WHERE
(
    Field1 = 'value1' AND
    (
        Field2 = 'value2' OR Field3 = 'value2'
    )
);
```
