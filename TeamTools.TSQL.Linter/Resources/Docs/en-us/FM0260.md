

# JOIN parenthesis alignment

|||
|-|-|
| Id | **FM0260**
| Mnemo | JOIN_PARENTHESIS_ALIGN
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [JoinParenthesisAlignedToFromRule.cs](../../../Rules/Formatting/JoinParenthesisAlignedToFromRule.cs)

## Cause

This rule is triggered if the opening and closing parentheses in a join subquery have different indentation.
<p id="descr">Align the open-close parentheses of the join subquery.</p>

## Examples

Bad

```sql
SELECT
    tbl.Id
FROM SomeTable AS tbl
INNER JOIN (
    SELECT Id FROM SomeTable2
) AS joinTbl ON tbl.Id = joinTbl.Id;
```

Good

```sql
SELECT
    tbl.Id
FROM SomeTable AS tbl
INNER JOIN
(
    SELECT Id FROM SomeTable2
) AS joinTbl ON tbl.Id = joinTbl.Id;
```
