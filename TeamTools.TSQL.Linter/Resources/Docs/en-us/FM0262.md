

# Single space around multiline comment tags

|||
|-|-|
| Id | **FM0262**
| Mnemo | ML_COMMENT_OPEN_CLOSE_WHITESPACE
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [MultilineCommentOpenCloseWhitespaceRule.cs](../../../Rules/Formatting/MultilineCommentOpenCloseWhitespaceRule.cs)

## Cause

This rule is triggered if there is more than one space right after the opening or before the closing tags in a multiline comment formatted on lines with text.
<p id="descr">Delete the extra spaces around multiline comment tags.</p>

## Examples

Bad

```sql
/*   This is a
multiline comment   */
```

Good

```sql
/* This is a
multiline comment */
```
