

# Tab symbol in comments

|||
|-|-|
| Id | **FM0267**
| Mnemo | TAB_IN_COMMENTS
| Severity | ℹ Hint
| Category | [Formatting](./Category_Formatting.md)
| Source code | [TabInCommentsRule.cs](../../../Rules/Formatting/TabInCommentsRule.cs)

## Cause

This rule is triggered if there is a tab symbol in comments.
<p id="descr">Replace the tab symbols with spaces.</p>

## Examples
<!-- markdownlint-disable no-hard-tabs -->
Bad

```sql
-- Comment	with	tabs
```
<!-- markdownlint-enable no-hard-tabs -->

Good

```sql
-- Comment with spaces
```
