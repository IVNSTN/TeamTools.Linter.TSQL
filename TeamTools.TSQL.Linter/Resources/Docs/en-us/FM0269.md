

# Column types alignment

|||
|-|-|
| Id | **FM0269**
| Mnemo | COLUMN_TYPE_ALIGNED
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [ColumnTypeAlignedRule.cs](../../../Rules/Formatting/ColumnTypeAlignedRule.cs)

## Cause

This rule is triggered if column types in table or table variable definitions are not aligned.
<p id="descr">Align column types.</p>

## Examples

Bad

```sql
DECLARE @Table TABLE
(
    Id INT NOT NULL
    , Name VARHCAR(50) NOT NULL
);
```

Good

```sql
DECLARE @Table TABLE
(
    Id      INT NOT NULL
    , Name  VARHCAR(50) NOT NULL
);
```

## Tips

💡 Single-line expressions are ignored.
