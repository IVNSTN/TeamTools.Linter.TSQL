

# Variable types alignment

|||
|-|-|
| Id | **FM0270**
| Mnemo | VAR_TYPE_ALIGNED
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [VariableTypeAlignedRule.cs](../../../Rules/Formatting/VariableTypeAlignedRule.cs)

## Cause

This rule is triggered if types in variable or parameter definitions are not aligned.
<p id="descr">Align types.</p>

## Examples

Bad

```sql
DECLARE
    @Id INT
    , @Name VARHCAR(50);

CREATE PROCEDURE dbo.procName
    @Id INT
    , @Name VARHCAR(50)
AS
```

Good

```sql
DECLARE
    @Id     INT
    , @Name VARHCAR(50);

CREATE PROCEDURE dbo.procName
    @Id     INT
    , @Name VARHCAR(50)
AS
```

## Tips

💡 Single-line expressions are ignored.
