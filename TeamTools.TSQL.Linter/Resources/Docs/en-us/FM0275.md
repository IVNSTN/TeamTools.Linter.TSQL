

# Single space around AS keyword

|||
|-|-|
| Id | **FM0275**
| Mnemo | AS_WITH_SPACES
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [SingleSpaceAroundAsRule.cs](../../../Rules/Formatting/SingleSpaceAroundAsRule.cs)

## Cause

This rule is triggered if there is more than one space around `AS` keyword when defining a table or a column alias.
<p id="descr">Remove extra spaces around AS.</p>

## Examples

Bad

```sql
SELECT Id FROM SomeTable   AS   tbl
```

Good

```sql
SELECT Id FROM SomeTable AS tbl
```
