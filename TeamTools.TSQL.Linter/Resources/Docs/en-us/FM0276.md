

# Single space between words

|||
|-|-|
| Id | **FM0276**
| Mnemo | TWO_WORD_WITH_SINGLE_SPACE
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [SingleSpaceInTwoWordInstructionRule.cs](../../../Rules/Formatting/SingleSpaceInTwoWordInstructionRule.cs)

## Cause

This rule is triggered if there is more than one space between the words in the multiword TSQL instructions.
<p id="descr">Remove extra spaces.</p>

## Examples

Bad

```sql
INNER  JOIN

GROUP   BY

BEGIN     TRAN
```

Good

```sql
INNER JOIN

GROUP BY

BEGIN TRAN
```
