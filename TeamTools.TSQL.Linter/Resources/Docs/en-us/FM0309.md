

# Use of tab symbol

|||
|-|-|
| Id | **FM0309**
| Mnemo | TAB_SYMBOL
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [TabSymbolRule.cs](../../../Rules/Formatting/TabSymbolRule.cs)

## Cause

This rule is triggered if a SQL source file contains a tab symbol.
<p id="descr">Use spaces.</p>

## Examples

Bad
<!-- markdownlint-disable no-hard-tabs -->
```sql
	SELECT Id FROM SomeTable; -- The line contains tab symbol

    --	The comment contains	tab	symbols
```
<!-- markdownlint-enable no-hard-tabs -->

Good

```sql
    SELECT Id FROM SomeTable; -- The line contains spaces only

    -- The comment contains spaces only
```

## Tips

💡 If you need to make a line/text in which the tab symbol should be used then create a variable, assign it a value of type `CHAR(9)` and use this variable to generate the required line/text.
