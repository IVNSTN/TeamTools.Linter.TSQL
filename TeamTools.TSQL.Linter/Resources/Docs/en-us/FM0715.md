

# Semicolon at the end of a statement

|||
|-|-|
| Id | **FM0715**
| Mnemo | SEMICOLON_IS_THE_END
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [GarbageBeforeSemicolonRule.cs](../../../Rules/Formatting/GarbageBeforeSemicolonRule.cs)

## Cause

This rule is triggered if there are extra characters before the last semicolon at the end of the expression.
<p id="descr">Delete the extra characters before the semicolon.</p>

## Examples

Bad

```sql
SELECT Id FROM SomeTable ;

SELECT Id FROM SomeTable /* comment */;
```

Good

```sql
SELECT Id FROM SomeTable;
```
