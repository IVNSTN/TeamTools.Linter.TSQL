

# Composite identifier isn't written as one word

|||
|-|-|
| Id | **FM0809**
| Mnemo | SPACE_IN_NAME_REF
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [SpaceInMultipartReferenceRule.cs](../../../Rules/Formatting/SpaceInMultipartReferenceRule.cs)

## Cause

This rule is triggered when the composite identifier isn't written as one word.
<p id="descr">Write the identifier as one word.</p>

## Examples

Bad

```sql
SELECT t.    col FROM dbo    .tbl AS t;
```

Good

```sql
SELECT t.col FROM dbo.tbl AS t;
```
