

# Dimension of a data type is indented from the data type name

|||
|-|-|
| Id | **FM0815**
| Mnemo | SPACE_BEFORE_SIZE
| Severity | ⚠ Warning
| Category | [Formatting](./Category_Formatting.md)
| Source code | [SpaceBeforeTypeSizeRule.cs](../../../Rules/Formatting/SpaceBeforeTypeSizeRule.cs)

## Cause

This rule is triggered when the dimension of a data type is indented from the data type name.
<p id="descr">Write the dimension of the data type together with the data type name.</p>

## Examples

Bad

```sql
DECLARE
    @d DECIMAL  (18,3)
    , @s VARCHAR/**/(MAX);
```

Good

```sql
DECLARE
    @d DECIMAL(18,3)
    , @s VARCHAR(MAX);
```
