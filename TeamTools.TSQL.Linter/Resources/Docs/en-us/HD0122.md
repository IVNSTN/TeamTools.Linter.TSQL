

# Cross-database reference not via synonym

|||
|-|-|
| Id | **HD0122**
| Mnemo | CROSS_DB_REF_HARDCODED
| Severity | ⚠ Warning
| Category | [Hardcode](./Category_Hardcode.md)
| Source code | [CrossDatabaseRefsHardcodedRule.cs](../../../Rules/Hardcode/CrossDatabaseRefsHardcodedRule.cs)

## Cause

This rule is triggered if cross-database reference not via synonyms detected.
<p id="descr">Use synonyms for cross-database reference.</p>

## Examples

Bad

```sql
SELECT id FROM other_db.dbo.some_table;
```

Good

```sql
CREATE SYNONYM other_db_synonym.some_table
FOR [$(other_db)].dbo.some_table;

SELECT id FROM other_db_synonym.some_table;
```
