

# Filegroup explicitly defined for temporary object

|||
|-|-|
| Id | **HD0180**
| Mnemo | TEMP_TABLE_FILEGROUP
| Severity | ⚠ Warning
| Category | [Hardcode](./Category_Hardcode.md)
| Source code | [TempTableFileGroupRule.cs](../../../Rules/Hardcode/TempTableFileGroupRule.cs)

## Cause

This rule is triggered if a filegroup is specified for a temporary table or its index.
<p id="descr">Remove the filegroup definition.</p>

## Examples

Bad

```sql
CREATE TABLE #far (id int) ON [PRIMARY]
CREATE INDEX #idx ON #far (id) ON [PRIMARY]
```

Good

```sql
CREATE TABLE #far (id int)
CREATE INDEX #idx ON #far (id)
```
