

# Exception swallowed

|||
|-|-|
| Id | **MA0116**
| Mnemo | ERROR_SWALLOW
| Severity | ⚠ Warning
| Category | [Maintainablity](./Category_Maintainablity.md)
| Source code | [ErrorSwallowingRule.cs](../../../Rules/Maintainability/ErrorSwallowingRule.cs)

## Cause

<p id="descr">This rule is triggered if an empty CATCH block is used in a TRY-CATCH.</p>

## Examples

Bad

```sql
BEGIN TRY
    SELECT 1 / 0;
END TRY
BEGIN CATCH
--PRINT ""error"";
END CATCH;
```

Good

```sql
BEGIN TRY
    SELECT 1 / 0;
END TRY
BEGIN CATCH
    PRINT 'error';
END CATCH;
```
