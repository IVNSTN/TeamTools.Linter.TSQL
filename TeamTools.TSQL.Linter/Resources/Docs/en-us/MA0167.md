
# Computed column missing alias

|||
|-|-|
| Id | **MA0167**
| Mnemo | COMPUTED_OUTPUT_NO_ALIAS
| Severity | ⚠ Warning
| Category | [Maintainablity](./Category_Maintainablity.md)
| Source code | [ComputedColumnOutputAliasRule.cs](../../../Rules/Maintainability/ComputedColumnOutputAliasRule.cs)

## Cause

This rule is triggered if the computed column does not have an alias assigned.
<p id="descr">Give the column an alias.</p>

## Examples

Bad

```sql
SELECT
    (SELECT a FROM bar)
    , foo.b
FROM foo
```

Good

```sql
SELECT
    (SELECT a FROM bar) AS a
    , foo.c
FROM foo
```
