

# INSERT-EXEC is hard to debug and maintain

|||
|-|-|
| Id | **MA0812**
| Mnemo | INSERT_EXEC
| Severity | ℹ Hint
| Category | [Maintainablity](./Category_Maintainablity.md)
| Source code | [InsertExecRule.cs](../../../Rules/Maintainability/InsertExecRule.cs)

## Cause

This rule is triggered when INSERT-EXEC is used in code.
<p id="descr">Remove INSERT-EXEC.</p>

## Examples

Bad

```sql
INSERT dbo.foo
EXEC dbo.bar
```

## Tips

💡 INSERT-EXEC is acceptable in tests.
