

# Function or stored procedure has no arguments

|||
|-|-|
| Id | **MA0817**
| Mnemo | ZERO_ARGS
| Severity | ℹ Hint
| Category | [Maintainablity](./Category_Maintainablity.md)
| Source code | [ProgrammabilityHasZeroArgsRule.cs](../../../Rules/Maintainability/ProgrammabilityHasZeroArgsRule.cs)

## Причина

This rule is triggered when the function or stored procedure has no arguments.
<p id="descr">Add arguments.</p>

## Examples

Bad

```sql
CREATE PROC dbo.foo
AS
BEGIN
    PRINT 'no ok'
END
```

Good

```sql
CREATE PROC dbo.foo
    @message VARCHAR(100)
AS
BEGIN
    PRINT message
END
```
