

# Function or stored procedure has too many arguments

|||
|-|-|
| Id | **MA0818**
| Mnemo | TOO_MANY_ARGS
| Severity | ℹ Hint
| Category | [Maintainablity](./Category_Maintainablity.md)
| Source code | [ProgrammabilityHasManyArgsRule.cs](../../../Rules/Maintainability/ProgrammabilityHasManyArgsRule.cs)

## Cause

This rule is triggered when the function or stored procedure has more than 12 arguments.
<p id="descr">Use table types or data exchange formats (JSON, XML).</p>

## Examples

Bad

```sql
CREATE PROC dbo.foo
    @a INT
    , @b INT
    , @c INT
    , @d INT
    , @e INT
    , @f INT
    , @g INT
    , @h INT
    , @i INT
    , @j INT
    , @k INT
    , @l INT
    , @m INT
AS;
```

Good

```sql
CREATE PROC dbo.foo
    @json_request NVARCHAR(MAX)
AS;
```
