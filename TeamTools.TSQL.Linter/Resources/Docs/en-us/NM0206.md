

# System-like object name

|||
|-|-|
| Id | **NM0206**
| Mnemo | SYS_LIKE_NAME
| Severity | ⚠ Warning
| Category | [Naming](./Category_Naming.md)
| Source code | [SystemLikeObjectNamesRule.cs](../../../Rules/Naming/SystemLikeObjectNamesRule.cs)

## Cause

This rule is triggered if a system prefix (sp_, sys_ etc.) for object name is found in the code.
<p id="descr">Remove the system prefix.</p>

## Examples

Bad

```sql
CREATE TABLE sys_test (a INT);
CREATE VIEW xp_foo AS SELECT * FROM bar;
```

Good

```sql
CREATE TABLE test (a INT);
CREATE VIEW foo AS SELECT * FROM bar;
```
