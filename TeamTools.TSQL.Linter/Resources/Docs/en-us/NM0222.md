

# ID suffix/preffix applied to non-integer column

|||
|-|-|
| Id | **NM0222**
| Mnemo | ID_FOR_INT
| Severity | ℹ Hint
| Category | [Naming](./Category_Naming.md)
| Source code | [IdSuffixForIntegerTypesOnlyRule.cs](../../../Rules/Naming/IdSuffixForIntegerTypesOnlyRule.cs)

## Cause

This rule is triggered when ID suffix/preffix is applied to non-integer column.
<p id="descr">Use ID suffix/preffix only with integer columns.</p>

## Examples

Bad

```sql
act_code    INT
place_id    VARCHAR
```

Good

```sql
act_id        INT
place_code    VARCHAR
```
