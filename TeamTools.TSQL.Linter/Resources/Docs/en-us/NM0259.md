

# Alphabet mix in identifier

|||
|-|-|
| Id | **NM0259**
| Mnemo | ALPHABET_MIX_IDENTIFIER
| Severity | ⚠ Warning
| Category | [Naming](./Category_Naming.md)
| Source code | [AlphabetMixInIdentifierRule.cs](../../../Rules/Naming/AlphabetMixInIdentifierRule.cs)

## Cause

This rule is triggered if alphabet mix in identifier is found in the code.
<p id="descr">Use one alphabet in identifier.</p>

## Examples

Bad

```sql
DECLARE @var_число INT
```

Good

```sql
DECLARE @var_number INT
```
