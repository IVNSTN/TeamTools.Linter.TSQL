

# Reserved keyword used for identifier

|||
|-|-|
| Id | **NM0263**
| Mnemo | KEYWORD_IDENTIFIER
| Severity | ℹ Hint
| Category | [Naming](./Category_Naming.md)
| Source code | [KeywordIdentifierRule.cs](../../../Rules/Naming/KeywordIdentifierRule.cs)

## Cause

This rule is triggered when reserved keyword used for identifier.
<p id="descr">Don't use reserved keyword for identifier.</p>

## Exmaples

Bad

```sql
DECLARE @[return] INT
```

Good

```sql
DECLARE @start INT
```
