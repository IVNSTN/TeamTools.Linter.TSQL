
# CAPS-LOCK identifier

|||
|-|-|
| Id | **NM0265**
| Mnemo | CAPS_IDENTIFIER
| Severity | ℹ Hint
| Category | [Naming](./Category_Naming.md)
| Source code | [CapsIdentifierRule.cs](../../../Rules/Naming/CapsIdentifierRule.cs)

## Cause

This rule is triggered if CAPS-LOCK identifier is found in the code.
<p id="descr">Rename the CAPS-LOCK identifiers using the project naming convention.</p>

## Examples

Bad

```sql
DECLARE @VAR INT
CREATE TABLE MY_TABLE (COL_1 INT)
```

Good

```sql
DECLARE @var INT;
CREATE TABLE my_table (col_1 INT)
```

## Tips

💡 In the translation or ETL code you have to give up the external names as early as possible.
