

# Forbidden name @@, ## or similar

|||
|-|-|
| Id | **NM0271**
| Mnemo | MAGIC_@@_##_NAME
| Severity | ⚠ Warning
| Category | [Naming](./Category_Naming.md)
| Source code | [MagicNameRule.cs](../../../Rules/Naming/MagicNameRule.cs)

## Cause

This rule is triggered if unreadable name (@@, ## or similar) is found in the code.
<p id="descr">Use readable identifier.</p>

## Examples

Bad

```sql
DECLARE @@ INT
```

Good

```sql
DECLARE @var INT
```
