

# Object of this type cannot be temporary

|||
|-|-|
| Id | **NM0712**
| Mnemo | NON_TEMP_OBJECT_LIKE_TEMP
| Severity | ⚠ Warning
| Category | [Naming](./Category_Naming.md)
| Source code | [NonTempObjectLikeTempRule.cs](../../../Rules/Naming/NonTempObjectLikeTempRule.cs)

## Cause

This rule is triggered if prefix # is used for object that is not table or stored procedure.
<p id="descr">Remove prefix #.</p>

## Examples

Bad

```sql
CREATE SCHEMA #tmp
```

Good

```sql
CREATE SCHEMA tmp
```
