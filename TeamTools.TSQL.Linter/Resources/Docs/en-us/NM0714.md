

# Keyword is used for alias

|||
|-|-|
| Id | **NM0714**
| Mnemo | ALIAS_IS_KEYWORD
| Severity | ℹ Hint
| Category | [Naming](./Category_Naming.md)
| Source code | [KeywordAliasRule.cs](../../../Rules/Naming/KeywordAliasRule.cs)

## Cause

This rule is triggered when keyword is used as an alias.
<p id="descr">Replace the keyword in the alias.</p>

## Examples

Bad

```sql
SELECT * FROM dbo.temp AS INT
```

Good

```sql
SELECT * FROM dbo.temp AS tmp
```
