

# Index name violates naming pattern

|||
|-|-|
| Id | **NM0961**
| Mnemo | INDEX_NAME_PATTERN
| Severity | ⚠ Warning
| Category | [Naming](./Category_Naming.md), [Indiсes](./Group_Indices.md)
| Source code | [IndexNamePatternRule.cs](../../../Rules/Naming/IndexNamePatternRule.cs)

## Cause

This rule is triggered when index name violates naming pattern.
<p id="descr">Make the index name match the naming pattern.</p>

## Examples

Bad

```sql
CREATE INDEX IXI_zoo_foo_open_date ON zoo.foo (open_date);
CREATE UNIQUE INDEX IX_zoo_foo_title_open_date ON zoo.foo (title, open_date);
```

Good

```sql
CREATE INDEX IX_zoo_foo_open_date ON zoo.foo (open_date);
CREATE UNIQUE INDEX IU_zoo_foo_title_open_date ON zoo.foo (title, open_date);
```
