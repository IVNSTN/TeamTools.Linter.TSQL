

# Trigger name violates naming pattern

|||
|-|-|
| Id | **NM0962**
| Mnemo | TRIGGER_NAME_PATTERN
| Severity | ⚠ Warning
| Category | [Naming](./Category_Naming.md), [Triggers](./Group_Triggers.md)
| Source code | [TriggerNamePatternRule.cs](../../../Rules/Naming/TriggerNamePatternRule.cs)

## Cause

This rule is triggered when trigger name violates naming pattern.
<p id="descr">Make the trigger name match the naming pattern.</p>

## Examples

Bad

```sql
CREATE TRIGGER zoo.instead_upd
ON zoo.foo
INSTEAD OF INSERT, UPDATE
AS
    RETURN;
```

Good

```sql
CREATE TRIGGER zoo.foo_instead_upd
ON zoo.foo
INSTEAD OF INSERT, UPDATE
AS
    RETURN;
```
