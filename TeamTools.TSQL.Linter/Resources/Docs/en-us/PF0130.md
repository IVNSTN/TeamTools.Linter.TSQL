

# ROWVERSION / TIMESTAMP index not marked as unique

|||
|-|-|
| Id | **PF0130**
| Mnemo | ROWVERSION_UNIQUE
| Severity | ⚠ Warning
| Category | [Performance](./Category_Performance.md), [Indices](./Group_Indices.md)
| Source code | [RowVersionIndexUniqueRule.cs](../../../Rules/Performance/RowVersionIndexUniqueRule.cs)

## Cause

<p id="descr">ROWVERSION / TIMESTAMP is unique within the database. Use a unique index for this column.</p>

## Examples

Bad

```sql
CREATE TABLE dbo.acme (id INT NOT NULL, rn ROWVERSION NOT NULL);

CREATE INDEX ix_bar ON dbo.acme (rn, id);
```

Good

```sql
CREATE TABLE dbo.acme (id INT NOT NULL, rn ROWVERSION NOT NULL);

CREATE UNIQUE INDEX ix_foo ON dbo.acme (rn);
CREATE INDEX ix_bar ON dbo.acme (id);
CREATE UNIQUE INDEX ix_zar ON dbo.acme (rn, id);
```
