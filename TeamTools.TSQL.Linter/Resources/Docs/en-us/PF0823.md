
# Double recompile requested

|||
|-|-|
| Id | **PF0823**
| Mnemo | RECOMPILE_RECOMPILE
| Severity | ℹ Hint
| Category | [Performance](./Category_Performance.md)
| Source code | [DoubleRecompileRule.cs](../../../Rules/Performance/DoubleRecompileRule.cs)

## Cause

This rule is triggered when `RECOMPILE` option is specified for module whose queries already contain `RECOMPILE`.
<p id="descr">Remove RECOMPILE option for module.</p>

## Examples

Bad

```sql
CREATE PROC dbo.foo
    @category_id INT
WITH RECOMPILE
AS
BEGIN
    SELECT *
    FROM dbo.bar
    WHERE category_id = @category_id
    OPTION (RECOMPILE)
END
```

Good

```sql
CREATE PROC dbo.foo
    @category_id INT
AS
BEGIN
    SELECT *
    FROM dbo.bar
    WHERE category_id = @category_id
    OPTION (RECOMPILE)
END
```
