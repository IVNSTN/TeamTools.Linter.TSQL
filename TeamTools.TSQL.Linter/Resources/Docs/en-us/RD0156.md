

# Redundant option switch

|||
|-|-|
| Id | **RD0156**
| Mnemo | REDUNDANT_OPTION_SWITCH
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantOptionSwitchRule.cs](../../../Rules/Redundancy/RedundantOptionSwitchRule.cs)

## Cause

<p id="descr">Unnecessary switching of options without expressions between them should be removed.</p>

## Examples

Bad

```sql
SET XACT_ABORT ON;

SET XACT_ABORT OFF;

SELECT id FROM dbo.employee
```

Good

```sql
SET XACT_ABORT OFF;

SELECT id FROM dbo.employee
```
