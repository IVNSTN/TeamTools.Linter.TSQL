
# Redundant option switch

|||
|-|-|
| Id | **RD0156**
| Mnemo | REDUNDANT_OPTION_SWITCH
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantOptionSwitchRule.cs](../../../Rules/Redundancy/RedundantOptionSwitchRule.cs)

## Cause

This rule is triggered if consecutive `SET` statements for the same option are detected without any expression between them.
<p id="descr">Remove unnecessary option set.</p>

## Examples

Bad

```sql
SET XACT_ABORT ON;

SET XACT_ABORT OFF;

SELECT id FROM dbo.employee
```

Good

```sql
SET XACT_ABORT OFF;

SELECT id FROM dbo.employee
```
