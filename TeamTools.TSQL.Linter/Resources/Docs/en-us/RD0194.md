

# Redundant COALESCE argument

|||
|-|-|
| Id | **RD0194**
| Mnemo | REDUNDANT_COALESCE_ARGUMENT
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantCoalesceArgumentRule.cs](../../../Rules/Redundancy/RedundantCoalesceArgumentRule.cs)

## Cause

<p id="descr">An unnecessary argument in COALESCE: an explicit NULL is simply not needed and does not affect anything, just like an argument after an explicit non-NULL (literal) - everything will end on the literal.</p>

## Examples

Bad

```sql
SELECT COALESCE(@var1, @var2, NULL)
```

Good

```sql
SELECT COALESCE(@var1, @var2)
```
