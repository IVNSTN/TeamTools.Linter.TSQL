
# Redundant COALESCE argument

|||
|-|-|
| Id | **RD0194**
| Mnemo | REDUNDANT_COALESCE_ARGUMENT
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantCoalesceArgumentRule.cs](../../../Rules/Redundancy/RedundantCoalesceArgumentRule.cs)

## Cause

This rule is triggered if the `COALESCE` function contains unnecessary arguments: an explicit `NULL` or arguments after a literal that is not `NULL`.
<p id="descr">Remove unnecessary arguments from the COALESCE function.</p>

## Examples

Bad

```sql
SELECT COALESCE(@var1, @var2, NULL)
```

Good

```sql
SELECT COALESCE(@var1, @var2)
```
