

# Unnecessary nested BEGIN-END

|||
|-|-|
| Id | **RD0221**
| Mnemo | BEGIN_BEGIN_END_END
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [BeginBeginEndEndRule.cs](../../../Rules/Redundancy/BeginBeginEndEndRule.cs)

## Cause

<p id="descr">Pointless BEGIN-ENDS nested inside each other, having no expressions between each other, are completely unnecessary</p>

## Examples

Bad

```sql
BEGIN
    BEGIN
        SELECT TOP(10) salary FROM dbo.employee ORDER BY salary
    END
END
```

Good

```sql
BEGIN
    SELECT TOP(10) salary FROM dbo.employee ORDER BY salary
END
```
