
# Unnecessary nested BEGIN-END

|||
|-|-|
| Id | **RD0221**
| Mnemo | BEGIN_BEGIN_END_END
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [BeginBeginEndEndRule.cs](../../../Rules/Redundancy/BeginBeginEndEndRule.cs)

## Cause

This rule is triggered if there are nested `BEGIN-END` constructions, between which no expressions are present.
<p id="descr">Delete unnecessary nested BEGIN-END blocks.</p>

## Examples

Bad

```sql
BEGIN
    BEGIN
        SELECT TOP(10) salary FROM dbo.employee ORDER BY salary
    END
END
```

Good

```sql
BEGIN
    SELECT TOP(10) salary FROM dbo.employee ORDER BY salary
END
```
