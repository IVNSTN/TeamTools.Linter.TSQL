
# Unnecessary multiple GO one after another

|||
|-|-|
| Id | **RD0228**
| Mnemo | GO_GO
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [GoGoRule.cs](../../../Rules/Redundancy/GoGoRule.cs)

## Cause

This rule is triggered if the code contains multiple consecutive `GO` statements that have no functional meaning.
<p id="descr">Remove unnecessary GO statements.</p>

## Examples

Bad

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END
GO
GO
GO
```

Good

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END
GO
```

## Tips

💡 Please take a closer look at the results of resolving the conflict in the version control system or importing code from the server.
