

# Unnecessary multiple GO one after another

|||
|-|-|
| Id | **RD0228**
| Mnemo | GO_GO
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [GoGoRule.cs](../../../Rules/Redundancy/GoGoRule.cs)

## Cause

<p id="descr">Several meaningless GO's in a row are not needed in the code.</p>

## Examples

Bad

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END
GO
GO
GO
```

Good

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END
GO
```
