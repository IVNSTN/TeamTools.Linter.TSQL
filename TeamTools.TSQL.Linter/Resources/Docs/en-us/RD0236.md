
# Redundant empty line

|||
|-|-|
| Id | **RD0236**
| Mnemo | REDUNDANT_NEWLINE
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantNewLineBetweenStatementsRule.cs](../../../Rules/Redundancy/RedundantNewLineBetweenStatementsRule.cs)

## Cause

This rule is triggered if the expressions are separated from each other by more than two blank lines.
<p id="descr">Remove extra blank lines between the expressions.</p>

## Examples

Bad

```sql
SELECT TOP (10) id from dbo.employee ORDER BY salary



UPDATE dbo.employee SET salary += 1000
```

Good

```sql
SELECT TOP (10) id from dbo.employee ORDER BY salary

UPDATE dbo.employee SET salary += 1000
```

## Tips

💡 Make sure that you have applied the formatter and it is configured correctly.
