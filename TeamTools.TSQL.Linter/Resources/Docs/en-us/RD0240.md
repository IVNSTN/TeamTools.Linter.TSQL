
# Two dashes expected for starting single-line comment

|||
|-|-|
| Id | **RD0240**
| Mnemo | SL_COMMENT_LEADING_DASHES
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantDashBeforeSingleLineCommentRule.cs](../../../Rules/Redundancy/RedundantDashBeforeSingleLineCommentRule.cs)

## Cause

This rule is triggered if more than two dash characters are used to define a single-line comment.
<p id="descr">Remove the extra dashes from the definition of the single-line comment.</p>

## Examples

Bad

```sql
------- one-line comment
```

Good

```sql
-- one-line comment
```
