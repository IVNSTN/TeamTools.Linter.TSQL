
# No trailing dashes expected for single-line comment

|||
|-|-|
| Id | **RD0241**
| Mnemo | SL_COMMENT_TRAILING_DASHES
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantDashAfterSingleLineCommentRule.cs](../../../Rules/Redundancy/RedundantDashAfterSingleLineCommentRule.cs)

## Cause

This rule is triggered if a single-line comment ends with a sequence of trailing dashes.
<p id="descr">Remove the trailing dashes from the comment.</p>

## Examples

Bad

```sql
-- one-line comment --
```

Good

```sql
-- one-line comment
```
