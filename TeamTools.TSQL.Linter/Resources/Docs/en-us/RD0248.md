

# Redundant semicolon

|||
|-|-|
| Id | **RD0248**
| Mnemo | REDUNDANT_SEMICOLON
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantSemicolonRule.cs](../../../Rules/Redundancy/RedundantSemicolonRule.cs)

## Cause

<p id="descr">One semicolon is enough to complete the expression. You do not need to put several in a row.</p>

## Examples

Bad

```sql
SELECT id FROM dbo.employee;;;
```

Good

```sql
SELECT id FROM dbo.employee;
```
