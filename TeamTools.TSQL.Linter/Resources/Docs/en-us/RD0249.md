

# Redundant brackets

|||
|-|-|
| Id | **RD0249**
| Mnemo | REDUNDANT_BRACKETS
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantSquareBracketsRule.cs](../../../Rules/Redundancy/RedundantSquareBracketsRule.cs)

## Cause

<p id="descr">Valid identifiers do not need to be quoted.</p>

## Examples

Bad

```sql
SELECT id FROM [dbo].[employee];
```

Good

```sql
SELECT id FROM dbo.employee;
```
