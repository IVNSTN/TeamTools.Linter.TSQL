

# Redundant new line in multiline comment

|||
|-|-|
| Id | **RD0261**
| Mnemo | ML_COMMENT_REDUNDANT_NEWLINE
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantNewLineInMultilineCommentRule.cs](../../../Rules/Redundancy/RedundantNewLineInMultilineCommentRule.cs)

## Cause

<p id="descr">You do not need to use more than two empty lines to separate the text in a multiline comment.</p>

## Examples

Bad

```sql
/*
multi



line



comment
*/
```

Good

```sql
/*
multi

line

comment
*/
```
