
# Redundant function call

|||
|-|-|
| Id | **RD0284**
| Mnemo | REDUNDANT_FUNCTION_CALL
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantFunctionCallRule.cs](../../../Rules/Redundancy/RedundantFunctionCallRule.cs)

## Cause

This rule is triggered if a function call is made that has no effect and leads to no result.
<p id="descr">Delete the function calls that do not affect the outcome.</p>

## Examples

Bad

```sql
SELECT DATEADD(YEAR, DATEDIFF(YEAR, 0, GETDATE()), 0);
```

Good

```sql
SELECT DATEFROMPARTS(YEAR(GETDATE()), 1, 1);
```
