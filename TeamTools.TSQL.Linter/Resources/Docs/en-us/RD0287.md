
# Redundant case nesting. Combine with the outer one

|||
|-|-|
| Id | **RD0287**
| Mnemo | REDUNDANT_NESTED_CASE
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantNestedCaseRule.cs](../../../Rules/Redundancy/RedundantNestedCaseRule.cs)

## Cause

This rule is triggered if in the `CASE` block, the next branch can be continued without additional `CASE` nesting.
<p id="descr">Remove the extra nesting and continue with the CASE at the same level.</p>

## Examples

Bad

```sql
CASE
    WHEN tbl.col = 1 THEN 'a'
    ELSE
        CASE WHEN tbl.col = 2 THEN 'b' ELSE 'x' END
END
```

Good

```sql
CASE
    WHEN tbl.col = 1 THEN 'a'
    WHEN tbl.col = 2 THEN 'b'
    ELSE 'x'
END
```
