

# Redundant ordering by constant value attempt

|||
|-|-|
| Id | **RD0296**
| Mnemo | REDUNDANT_ORDER_BY_CONST
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantOrderByConstantRule.cs](../../../Rules/Redundancy/RedundantOrderByConstantRule.cs)

## Cause

<p id="descr">The presence of a literal or scalar variable in ORDER BY does not affect anything.</p>

## Examples

Bad

```sql
SELECT treaty FROM dbo.accounts ORDER BY treaty, @a
```

Good

```sql
SELECT treaty FROM dbo.accounts ORDER BY treaty
```
