
# Redundant argument

|||
|-|-|
| Id | **RD0707**
| Mnemo | REDUNDANT_ARGUMENT
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantFunctionArgumentRule.cs](../../../Rules/Redundancy/RedundantFunctionArgumentRule.cs)

## Cause

This rule is triggered if an optional argument is passed to a function, the value of which has no effect on the result of execution.
<p id="descr">Remove the unnecessary and meaningless argument.</p>

## Examples

Bad

```sql
SELECT CONCAT(@a, @b, '')
```

Good

```sql
SELECT CONCAT(@a, @b)
```
