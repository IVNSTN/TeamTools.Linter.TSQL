

# Redundant argument

|||
|-|-|
| Id | **RD0707**
| Mnemo | REDUNDANT_ARGUMENT
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantFunctionArgumentRule.cs](../../../Rules/Redundancy/RedundantFunctionArgumentRule.cs)

## Cause

<p id="descr">An argument is passed to the function, which is optional and contains a value that does not affect the result in any way.</p>

## Examples

Bad

```sql
SELECT CONCAT(@a, @b, '')
```

Good

```sql
SELECT CONCAT(@a, @b)
```
