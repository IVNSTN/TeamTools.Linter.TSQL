
# Alias equals column name and can be omitted

|||
|-|-|
| Id | **RD0713**
| Mnemo | COLUMN_ALIAS_IS_THE_SAME
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [ColumnAliasSameAsSourceRule.cs](../../../Rules/Redundancy/ColumnAliasSameAsSourceRule.cs)

## Cause

This rule is triggered if the column alias matches its name.
<p id="descr">Delete the redundant column alias.</p>

## Examples

Bad

```sql
SELECT treaty AS treaty FROM dbo.accounts
```

Good

```sql
SELECT treaty FROM dbo.accounts
```
