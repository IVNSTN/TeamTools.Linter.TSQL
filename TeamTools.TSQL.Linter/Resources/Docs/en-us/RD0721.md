

# Redundant NULL handling for concatenation function

|||
|-|-|
| Id | **RD0721**
| Mnemo | NULL_HANDLING_FOR_CONCAT
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [NullHandlingForConcatRule.cs](../../../Rules/Redundancy/NullHandlingForConcatRule.cs)

## Cause

This rule is triggered if NULL values ​​being converted to empty strings inside of the CONCAT function arguments.
<p id="descr">Remove the ISNULL call with conversion to an empty string.</p>

## Examples

Bad

```sql
SELECT CONCAT('asdf', @var, ISNULL(@bar, ''))
```

Good

```sql
SELECT CONCAT('asdf', @var, @bar)
```
