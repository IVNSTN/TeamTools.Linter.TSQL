

# Redundant sign for zero

|||
|-|-|
| Id | **RD0731**
| Mnemo | SIGNED_ZERO
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [SignedZeroRule.cs](../../../Rules/Redundancy/SignedZeroRule.cs)

## Cause

This rule is triggered if an attempt to assign a plus (+) or minus (-) sign to zero (0) or NULL is detected.
<p id="descr">Remove the sign assigned to zero or NULL.</p>

## Examples

Bad

```sql
SET @foo = +NULL
SET @foo = +0
SELECT 3 * (-(0.0)) AS result
```

Good

```sql
SET @foo = 1 + 0
SELECT -3 * 0 AS result
```
