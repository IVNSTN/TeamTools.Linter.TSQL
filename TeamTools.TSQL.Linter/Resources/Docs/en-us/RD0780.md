

# Redundant parenthesis around join predicate

|||
|-|-|
| Id | **RD0780**
| Mnemo | JOIN_PREDICATE_PARENTHETHIS
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [JoinWithParenthethisRule.cs](../../../Rules/Redundancy/JoinWithParenthethisRule.cs)

## Cause

This rule is triggered if join predicates are enclosed in parentheses.
<p id="descr">Remove parentheses around join predicate.</p>

## Examples

Bad

```sql
SELECT *
FROM dbo.foo f
INNER JOIN dbo.bar b
    ON (f.id = b.id)

SELECT *
FROM dbo.foo f
LEFT JOIN (SELECT id FROM dbo.far) a
INNER JOIN dbo.bar b
    ON a.id = b.parent_id
    ON (f.id = b.id)
```

Good

```sql
SELECT *
FROM dbo.foo f
INNER JOIN dbo.bar b
    ON f.id = b.id

SELECT *
FROM dbo.foo f
LEFT JOIN (SELECT id FROM dbo.far) a
INNER JOIN dbo.bar b
    ON a.id = b.parent_id
    ON f.id = b.id
```
