

# Redundant variable initialization with NULL

|||
|-|-|
| Id | **RD0798**
| Mnemo | REDUNDANT_INIT_NULL
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [RedundantInitWithNullRule.cs](../../../Rules/Redundancy/RedundantInitWithNullRule.cs)

## Cause

This rule is triggered when variable is initialized with NULL.
<p id="descr">Remove initialization with NULL.</p>

## Examples

Bad

```sql
DECLARE @percent INT = NULL;
```

Good

```sql
DECLARE @percent INT;
```
