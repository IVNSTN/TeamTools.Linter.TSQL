

# Redundant EXECUTE AS CALLER directive

|||
|-|-|
| Id | **RD0811**
| Mnemo | EXECUTE_AS_CALLER
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [ExecuteAsCallerRule.cs](../../../Rules/Redundancy/ExecuteAsCallerRule.cs)

## Cause

This rule is triggered when EXECUTE AS CALLER directive is added to module.
<p id="descr">Remove EXECUTE AS CALLER.</p>

## Examples

Bad

```sql
CREATE PROC dbo.bar
WITH EXECUTE AS CALLER
AS;
```

Good

```sql
CREATE PROC dbo.bar
AS;
```
