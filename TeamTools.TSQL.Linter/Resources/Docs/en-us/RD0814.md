

# Variable is specified more than once for IN predicate

|||
|-|-|
| Id | **RD0814**
| Mnemo | IN_DUP_VAR
| Severity | ⚠ Warning
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [DuplicateInVariableRule.cs](../../../Rules/Redundancy/DuplicateInVariableRule.cs)

## Cause

This rule is triggered when variable is specified more than once for IN predicate.
<p id="descr">Remove duplicates.</p>

## Examples

Bad

```sql
SELECT name
FROM dbo.foo
WHERE name IN (@first_name, @second_name, @first_name)
```

Good

```sql
SELECT name
FROM dbo.foo
WHERE name IN (@first_name, @second_name)
```
