

# CTE declared but never used

|||
|-|-|
| Id | **RD0934**
| Mnemo | CTE_UNUSED
| Severity | ℹ Hint
| Category | [Redundancy](./Category_Redundancy.md)
| Source code | [UnusedCteRule.cs](../../../Rules/Redundancy/UnusedCteRule.cs)

## Cause

This rule is triggered if a CTE is declared but never used in the code.
<p id="descr">Remove the unused CTE.</p>

## Examples

Bad

```sql
WITH cte AS (SELECT id FROM dbo.bar)
DELETE f
FROM dbo.foo AS f;
```

Good

```sql
WITH cte AS (SELECT id FROM dbo.bar)
SELECT d, b
FROM dbo.foo AS f
WHERE EXISTS(SELECT 1 FROM cte)
```
