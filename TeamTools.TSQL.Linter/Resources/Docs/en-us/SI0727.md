

# Negated comparison can be simplified

|||
|-|-|
| Id | **SI0727**
| Mnemo | NOT_FOR_SIMPLE_PREDICATE
| Severity | ⚠ Warning
| Category | [Simplification](./Category_Simplification.md)
| Source code | [SimplePredicateNegationRule.cs](../../../Rules/Simplification/SimplePredicateNegationRule.cs)

## Cause

This rule is triggered if a negation of a simple comparison condition is detected.
<p id="descr">Get rid of negation by reversing the comparison condition.</p>

## Examples

Bad

```sql
IF NOT (@a = @b)
    SELECT CASE WHEN (NOT (@c > @d)) THEN 1 ELSE 0 END AS res
```

Good

```sql
IF @a <> @b
    SELECT CASE WHEN @c <= @d THEN 1 ELSE 0 END AS res
```
