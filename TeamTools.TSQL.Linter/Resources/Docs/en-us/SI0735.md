

# Variable assignment can be simplified - set value in DECLARE

|||
|-|-|
| Id | **SI0735**
| Mnemo | SET_TO_DECLARE
| Severity | ℹ Hint
| Category | [Simplification](./Category_Simplification.md)
| Source code | [SetToDeclareRule.cs](../../../Rules/Simplification/SetToDeclareRule.cs)

## Cause

This rule is triggered if it detects an assignment of an initial value to a variable that can be specified in a DECLARE statement.
<p id="descr">Assign a value in a DECLARE statement.</p>

## Examples

Bad

```sql
DECLARE @a   INT
        , @b DATETIME

SET @a = 2
SELECT @b = GETDATE()
```

Good

```sql
DECLARE @a   INT      = 2
        , @b DATETIME = GETDATE()
```
