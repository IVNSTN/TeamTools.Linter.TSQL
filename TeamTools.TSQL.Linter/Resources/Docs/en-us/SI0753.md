

# Multiple DROP statements can be collapsed into one

|||
|-|-|
| Id | **SI0753**
| Mnemo | DROP_STATEMENTS_INTO_ONE
| Severity | ℹ Hint
| Category | [Simplification](./Category_Simplification.md)
| Source code | [DropStatementsIntoOneRule.cs](../../../Rules/Simplification/DropStatementsIntoOneRule.cs)

## Cause

This rule is triggered if multiple DROP statements for objects of the same type are detected.
<p id="descr">Collapse multiple DROP statements into one.</p>

## Examples

Bad

```sql
DROP TABLE dbo.foo
DROP TABLE dbo.bar

DROP PROC dbo.far
DROP PROC dbo.jar
```

Good

```sql
DROP TABLE dbo.foo, dbo.bar

DROP PROC dbo.far, dbo.jar
```
