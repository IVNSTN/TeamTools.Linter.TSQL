

# Multiple ALTER statements can be collapsed into one

|||
|-|-|
| Id | **SI0754**
| Mnemo | ALTER_STATEMENTS_INTO_ONE
| Severity | ℹ Hint
| Category | [Simplification](./Category_Simplification.md)
| Source code | [AlterStatementsIntoOneRule.cs](../../../Rules/Simplification/AlterStatementsIntoOneRule.cs)

## Cause

This rule is triggered if multiple ALTER TABLE statements are detected on the same table with objects of the same type (columns or constraints).
<p id="descr">Collapse multiple ALTER TABLE statements into one.</p>

## Examples

Bad

```sql
ALTER TABLE dbo.foo ADD bar INT
ALTER TABLE dbo.foo ADD far BIT
ALTER TABLE dbo.foo ADD jar VARCHAR(10);

ALTER TABLE dbo.foo DROP CONSTRAINT zar
ALTER TABLE dbo.foo DROP CONSTRAINT car
ALTER TABLE dbo.foo DROP CONSTRAINT tar
```

Good

```sql
ALTER TABLE dbo.foo ADD bar INT, far BIT, jar VARCHAR(10);

ALTER TABLE dbo.foo DROP CONSTRAINT zar, car, tar
```
