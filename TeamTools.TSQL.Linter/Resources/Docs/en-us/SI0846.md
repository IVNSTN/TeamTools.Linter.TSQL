
# Multiple inequality checks can be combined into single NOT IN predicate

|||
|-|-|
| Id | **SI0846**

## Cause

This rule is triggered when multiple inequality checks can be combined into single `NOT IN` predicate.
<p id="descr">Combine the checks into single NOT IN predicate.</p>

## Examples

Bad

```sql
SELECT *
FROM dbo.foo
WHERE category_id <> 1 
    AND category_id <> 2;
```

Good

```sql
SELECT *
FROM dbo.foo
WHERE category_id NOT IN (1, 2);
```
