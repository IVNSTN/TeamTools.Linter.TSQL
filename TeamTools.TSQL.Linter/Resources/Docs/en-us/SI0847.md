
# Multiple similar IN predicates can be combined into single one

|||
|-|-|
| Id | **SI0847**
| Mnemo | MULTIPLE_IN_TO_SINGLE
| Severity | ℹ Hint
| Category | [Simplification](./Category_Simplification.md)
| Source code | [MultipleInPredicateIntoOneRule.cs](../../../Rules/Simplification/MultipleInPredicateIntoOneRule.cs)

## Cause

This rule is triggered when multiple similar `IN` predicates can be combined into single one.
<p id="descr">Combine IN predicates.</p>

## Examples

Bad

```sql
SELECT *
FROM dbo.foo
WHERE category_id IN (1, 2, 3) 
    OR category_id IN (4, 5);
```

Good

```sql
SELECT *
FROM dbo.foo
WHERE category_id IN (1, 2, 3, 4, 5);
```
