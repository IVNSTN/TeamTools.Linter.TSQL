

# Unexpected database context change within code block

|||
|-|-|
| Id | **VU0503**

| Severity | ⚠ Warning
| Category | [Vulnerability](./Category_Vulnerability.md)
| Source code | [DatabaseContextChangeRule.cs](../../../Rules/Vulnerability/DatabaseContextChangeRule.cs)

## Cause

The rule is triggered when code contains USE command.
<p id="descr">Remove USE command.</p>

## Examples

Bad

```sql
USE master;
```
