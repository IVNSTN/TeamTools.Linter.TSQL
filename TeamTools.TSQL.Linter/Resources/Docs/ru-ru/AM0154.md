

# Неоднозначность сортировки результата запроса

|||
|-|-|
| Id | **AM0154**
| Мнемо | ORDERBY_DIRECTION_AMBIGUOUS
| Серьёзность | ⚠ Предупреждение
| Категория | [Неоднозначность](./Категория_Неоднозначность.md)
| Исходный код | [OrderByDirectionAmbiguityRule.cs](../../../Rules/Ambiguity/OrderByDirectionAmbiguityRule.cs)

## Причина

Правило срабатывает, если ORDER BY имеет хотя бы одно выражение с DESC и хотя бы одно выражение без указания направления сортировки.
<p id="descr">Укажите направление сортировки.</p>

## Примеры

Некорректно

```sql

SELECT name, hired_at, fired_at FROM dbo.hiredate_view ORDER BY hired_at, fired_at DESC
```

Корректно

```sql
SELECT name, hired_at, fired_at FROM dbo.hiredate_view ORDER BY hired_at ASC, fired_at DESC
```

## Подсказки

💡 Не следует вводить в заблуждение читателя. Направление сортировки неочевидно для читателя, если не указано явно. Может быть воспринято неправильно, что DESC действует для всех выражений инструкции ORDER BY, если указан в конце.
