

# Неуникальный алиас или имя столбца

|||
|-|-|
| Id | **AM0168**
| Мнемо | NONUNIQUE_COLUMN_ALIAS
| Серьёзность | ⚠ Предупреждение
| Категория | [Неоднозначность](./Категория_Неоднозначность.md)
| Исходный код | [UniqueColumnAliasRule.cs](../../../Rules/Ambiguity/UniqueColumnAliasRule.cs)

## Причина

Правило срабатывает, если в выборке есть столбцы с дублирующимися именами или алиасами.
<p id="descr">Задайте уникальные алиасы столбцов.</p>

## Примеры

Некорректно

```sql
SELECT
    1 + 1 as c,
    t.c
FROM tbl as t
```

Корректно

```sql
SELECT
    1 + 1 as c,
    t.c as d
FROM tbl as t
```
