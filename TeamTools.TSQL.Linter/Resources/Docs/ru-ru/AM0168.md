

# Неуникальный алиас или имя колонки

|||
|-|-|
| Id | **AM0168**
| Мнемо | NONUNIQUE_COLUMN_ALIAS
| Серьёзность | ⚠ Предупреждение
| Категория | [Неоднозначность](./Категория_Неоднозначность.md)
| Исходный код | [UniqueColumnAliasRule.cs](../../../Rules/Ambiguity/UniqueColumnAliasRule.cs)

## Причина

<p id="descr">Правило срабатывает, если в селекте есть колонки с дублирующимися именами или алиасами.</p>

## Примеры

Некорректно

```sql
SELECT 
    1 + 1 as c,
    t.c
FROM tbl as t
```

Корректно

```sql
SELECT
    1 + 1 as c,
    t.c as d
FROM tbl as t
```
