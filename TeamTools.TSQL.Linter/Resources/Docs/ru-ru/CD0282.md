

# Используйте SUBSTRING вместо LEFT, RIGHT

|||
|-|-|
| Id | **CD0282**
| Мнемо | STRING_FN_IN_CONSTRAINT
| Серьёзность | ⚠ Предупреждение
| Категория | [НепрерывнаяПоставка](./Категория_НепрерывнаяПоставка.md)
| Исходный код | [StringFunctionInConstraintRule.cs](../../../Rules/ContinuousDeployment/StringFunctionInConstraintRule.cs)

## Причина

Правило срабатывает, если в ограничениях используются функции `LEFT`, `RIGHT`.
<p id="descr">Замените на SUBSTRING.</p>

## Примеры

Некорректно

```sql
CREATE TABLE foo
(
    bar VARCHAR(20) NOT NULL
    , mar AS LEFT(bar, 20)
);
```

Корректно

```sql
CREATE TABLE foo
(
    bar VARCHAR(20) NOT NULL
    , mar AS SUBSTRING(bar, 1, 20)
);
```

## Подсказки

💡 На временные таблицы (#) и табличные переменные (@) не срабатывает.

💡 DacFx при публикации конвертирует эти функции в `SUBSTRING` и так получается "вечный дифф".
