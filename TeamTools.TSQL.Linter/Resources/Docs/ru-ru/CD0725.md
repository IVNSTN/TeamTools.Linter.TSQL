

# ALTER должен быть переписан в определение таблицы

|||
|-|-|
| Id | **CD0725**
| Мнемо | ALTER_TO_DEFINITION
| Серьёзность | ⚠ Предупреждение
| Категория | [НепрерывнаяПоставка](./Категория_НепрерывнаяПоставка.md)
| Исходный код | [AlterTableToInlineDefinitionRule.cs](../../../Rules/ContinuousDeployment/AlterTableToInlineDefinitionRule.cs)

## Причина

Правило срабатывает при использовании `ALTER` в теле скрипта создания таблицы для добавления столбцов/индексов/ограничений.
<p id="descr">Перенесите ALTER-ADD выражения в определение таблицы.</p>

## Примеры

Некорректно

```sql
CREATE TABLE foo
(
    bar INT NOT NULL
)
GO

ALTER TABLE foo ADD mar INT NOT NULL
GO

ALTER TABLE foo ADD CONSTRAINT cns DEFAULT 0
```

Корректно

```sql
CREATE TABLE foo
(
    bar INT NOT NULL CONSTRAINT cns DEFAULT 0
    , mar INT NOT NULL
)
GO
```
