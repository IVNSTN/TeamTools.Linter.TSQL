

# Запрещенная конструкция SELECT-INTO

|||
|-|-|
| Id | **CS0101**
| Мнемо | SELECT_INTO
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SelectIntoRule.cs](../../../Rules/CodeSmell/SelectIntoRule.cs)

## Причина

Правило срабатывает, если обнаружена конструкция `SELECT`-`INTO`.
<p id="descr">Создайте таблицу и заполните её с помощью INSERT-SELECT.</p>

## Примеры

Некорректно

```sql
SELECT a
INTO #bar
FROM dbo.foo;
```

Корректно

```sql
CREATE TABLE #bar (a VARCHAR(255));

INSERT INTO #bar
SELECT a
FROM dbo.foo;
```

## Подсказки

💡 В процессе доработки нужно проанализировать код, использующий собранные данные, вполне может оказаться, что действительно нужны всего несколько столбцов.
