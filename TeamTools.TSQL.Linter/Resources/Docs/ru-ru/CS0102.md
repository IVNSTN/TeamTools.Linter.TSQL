

# Argument passed by position, not by name

|||
|-|-|
| Id | **CS0102**
| Мнемо | POSITIONAL_PROC_ARGS
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)

## Причина

<p id="descr">При вызове хп обязательно нужно передавать параметры по имени, а не по позиции ( exec myproc @x = @y). На системные хп не реагирует</p>
