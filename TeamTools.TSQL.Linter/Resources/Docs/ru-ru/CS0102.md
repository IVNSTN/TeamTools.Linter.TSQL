

# Аргумент передан по позиции, а не по имени

|||
|-|-|
| Id | **CS0102**
| Мнемо | POSITIONAL_PROC_ARGS
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [PositionalProcArgumentsRule.cs](../../../Rules/CodeSmell/PositionalProcArgumentsRule.cs)

## Причина

Правило срабатывает, если обнаружен вызов хранимой процедуры с передачей параметров по позиции, а не по имени.
<p id="descr">Передайте параметры по имени.</p>

## Примеры

Некорректно

```sql
EXEC dbo.test
    @var_a
    , @b = @var_b;
```

Корректно

```sql
EXEC dbo.test
    @a = @var_a
    , @b = @var_b;
```
