

# Вставка без указания столбцов целевой таблицы

|||
|-|-|
| Id | **CS0103**
| Мнемо | INSERT_COLUMN_LIST
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [InsertWithoutColumnsRule.cs](../../../Rules/CodeSmell/InsertWithoutColumnsRule.cs)

## Причина

Правило срабатывает, если при вставке (`INSERT INTO`, `OUTPUT INTO`) не перечислены столбцы целевой таблицы.
<p id="descr">Перечислите столбцы целевой таблицы.</p>

## Примеры

Некорректно

```sql
INSERT INTO dbo.foo
SELECT a, b, c
FROM tmp.bar
```

Корректно

```sql
INSERT INTO dbo.foo(a, b, c)
SELECT a, b, c
FROM tmp.bar;
```
