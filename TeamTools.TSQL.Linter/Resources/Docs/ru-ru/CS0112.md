

# Использование SELECT * запрещено

|||
|-|-|
| Id | **CS0112**
| Мнемо | SELECT_STAR
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SelectStarRule.cs](../../../Rules/CodeSmell/SelectStarRule.cs)

## Причина

Правило срабатывает, если обнаружен запрос вида `SELECT *`. Срабатывает, в том числе, и при использовании внутри блока `EXISTS`.
<p id="descr">Замените * на список необходимых столбцов.</p>

## Примеры

Некорректно

```sql
SELECT a
FROM dbo.foo
WHERE EXISTS(SELECT * FROM bar WHERE b = 10)
```

Корректно

```sql
SELECT a
FROM dbo.foo
WHERE EXISTS(SELECT 1 FROM bar WHERE b = 10)
```
