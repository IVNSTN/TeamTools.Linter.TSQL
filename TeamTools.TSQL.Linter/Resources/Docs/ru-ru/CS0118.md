

# Не указан владелец схемы

|||
|-|-|
| Id | **CS0118**
| Мнемо | AUTHORIZED_SCHEMA
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SchemaAuthorizedRule.cs](../../../Rules/CodeSmell/SchemaAuthorizedRule.cs)

## Причина

Правило срабатывает, если в скрипте создания схемы не указан её владелец.
<p id="descr">Укажите владельца схемы.</p>

## Примеры

Некорректно

```sql
CREATE SCHEMA test
```

Корректно

```sql
CREATE SCHEMA test
AUTHORIZATION dbo
```
