

# Не указан грантор

|||
|-|-|
| Id | **CS0125**
| Мнемо | GRANT_GRANTOR
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [GrantorRule.cs](../../../Rules/CodeSmell/GrantorRule.cs)

## Причина

Правило срабатывает, если в скрипте выдачи прав на объект не указан грантор.
<p id="descr">Укажите грантора.</p>

## Примеры

Некорректно

```sql
GRANT SELECT ON object::foo.bar
TO asdf;
```

Корректно

```sql
GRANT SELECT ON object::foo.bar
TO asdf
AS [dbo];
```

## Подсказки

💡 Чаще всего это dbo.

💡 Для схем, типов и сервисов грантора указывать не нужно ([CD0213](./CD0213.md)).
