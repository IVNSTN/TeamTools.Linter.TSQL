

# Index on different table in create table script

|||
|-|-|
| Id | **CS0128**
| Мнемо | INDEX_MISDIRECTED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [IndexOnGivenTableOnlyRule.cs](../../../Rules/CodeSmell/IndexOnGivenTableOnlyRule.cs)

## Причина

<p id="descr">В результате копипастинга <em>подходящего</em> скрипта создания индекса из одной таблицы в другую он может так и остаться нацелен на исходную таблицу. Правило проверяет что в скрипте создания таблицы все индексы смотрят именно на нее.</p>
