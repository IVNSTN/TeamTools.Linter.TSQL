

# Индекс создается для другой таблицы

|||
|-|-|
| Id | **CS0128**
| Мнемо | INDEX_MISDIRECTED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [IndexOnGivenTableOnlyRule.cs](../../../Rules/CodeSmell/IndexOnGivenTableOnlyRule.cs)

## Причина

Правило срабатывает, если в файле скрипта создания таблицы для индекса указана другая таблица.
<p id="descr">Укажите правильное имя таблицы.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
);

CREATE INDEX ix_foo
ON dbo.bar(id)
ON [PRIMARY];
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
);

CREATE INDEX ix_foo
ON dbo.foo(id)
ON [PRIMARY];
```
