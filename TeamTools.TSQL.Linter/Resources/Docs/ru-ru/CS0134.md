

# Запрещенная опция RAISERROR

|||
|-|-|
| Id | **CS0134**
| Мнемо | RAISERROR_OPTIONS
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [RaisErrorOptionsRule.cs](../../../Rules/CodeSmell/RaisErrorOptionsRule.cs)

## Причина

Правило срабатывает, если обнаружено использование запрещенной опции `RAISERROR`.
<p id="descr">Удалите запрещенную опцию RAISERROR.</p>

## Примеры

Некорректно

```sql
RAISERROR ('asdf', 10, 1) WITH LOG;
RAISERROR ('asdf', 10, 1) WITH SETERROR;
```

Корректно

```sql
RAISERROR ('asdf', 10, 1);
```

## Подсказки

💡 К запрещенным относятся следующие опции:

- `SETERROR`
- `LOG`
