

# Использование уровня изоляции READ UNCOMMITTED

|||
|-|-|
| Id | **CS0136**
| Мнемо | ISOLATION_UNCOMMITTED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [IsolationLevelUncommittedRule.cs](../../../Rules/CodeSmell/IsolationLevelUncommittedRule.cs)

## Причина

Правило срабатывает, если обнаружено использование уровня изоляции транзакций `READ UNCOMMITTED`.
<p id="descr">Используйте другой уровень изоляции транзакций.</p>

## Примеры

Некорректно

```sql
SET TRANSACTION ISOLATION LEVEL READ UNCOMMITTED
```

Корректно

```sql
SET TRANSACTION ISOLATION LEVEL REPEATABLE READ
```
