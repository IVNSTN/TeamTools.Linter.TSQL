

# Использование SELECT * в представлениях

|||
|-|-|
| Id | **CS0138**
| Мнемо | VIEW_SELECT_STAR_OUTPUT
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SelectStarInViewRule.cs](../../../Rules/CodeSmell/SelectStarInViewRule.cs)

## Причина

Правило срабатывает, если в итоговой выборке представления используется запрос вида `SELECT *`.
<p id="descr">Замените * на список необходимых столбцов.</p>

## Примеры

Некорректно

```sql
CREATE VIEW dbo.bar
AS
    SELECT *, d.a FROM d;
```

Корректно

```sql
CREATE VIEW dbo.bar
AS
    SELECT d.a, d.b, d.c FROM d;
```
