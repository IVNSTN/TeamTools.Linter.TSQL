

# For DATETIME2 type use SYSDATETIME() instead of GETDATE()

|||
|-|-|
| Id | **CS0141**
| Мнемо | DATETIME2_SYSDATETIME
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [Datetime2SysdatetimeRule.cs](../../../Rules/CodeSmell/Datetime2SysdatetimeRule.cs)

## Причина

<p id="descr">Для наполнения переменных и колонок типа DATETIME2 корректнее использовать SYSDATETIME() чем GETDATE().</p>
