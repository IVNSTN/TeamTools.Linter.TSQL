

# Заполнение DATETIME2 с помощью GETDATE()

|||
|-|-|
| Id | **CS0141**
| Мнемо | DATETIME2_SYSDATETIME
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [Datetime2SysdatetimeRule.cs](../../../Rules/CodeSmell/Datetime2SysdatetimeRule.cs)

## Причина

Правило срабатывает, если обнаружена попытка заполнения переменной или столбца типа `DATETIME2` при помощи функции `GETDATE()`.
<p id="descr">Используйте SYSDATETIME().</p>

## Примеры

Некорректно

```sql
DECLARE
    @dt2 DATETIME2 = GETDATE()
```

Корректно

```sql
DECLARE
    @dt1   DATETIME  = GETDATE()
    , @dt2 DATETIME2 = SYSDATETIME()
```
