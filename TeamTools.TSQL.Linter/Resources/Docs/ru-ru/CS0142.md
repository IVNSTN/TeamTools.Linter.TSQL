

# Не задан код возврата для RETURN

|||
|-|-|
| Id | **CS0142**
| Мнемо | PROC_RETURN_VALUE_REQUIRED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [ReturnValueFromProcRule.cs](../../../Rules/CodeSmell/ReturnValueFromProcRule.cs)

## Причина

Правило срабатывает, если обнаружено использование `RETURN` без явного указания значения для возврата.
<p id="descr">Укажите значение для RETURN.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.foo
AS
BEGIN
    SELECT 0;

    RETURN; -- no return value;
END;
```

Корректно

```sql
CREATE PROCEDURE dbo.foo
AS
BEGIN
    SELECT 0;

    RETURN 1;
END;
GO

ALTER PROCEDURE dbo.foo
AS
BEGIN
    -- return not used;
    SELECT 0;
END;
```
