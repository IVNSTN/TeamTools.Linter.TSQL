

# RETURN value not redefined explicitly after CATCH

|||
|-|-|
| Id | **CS0143**
| Мнемо | PROC_RETURN_REQUIRED_AFTER_CATCH
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [ReturnValueFromProcIfTryCatchUsedRule.cs](../../../Rules/CodeSmell/ReturnValueFromProcIfTryCatchUsedRule.cs)

## Причина

<p id="descr">Если используете CATCH - или пробрасывайте ошибку дальше, или - если заглушили ошибку - явно указывайте код возврата. Т.е. если RETURN не было, то надо дописать и указать конкретное значение.</p>
