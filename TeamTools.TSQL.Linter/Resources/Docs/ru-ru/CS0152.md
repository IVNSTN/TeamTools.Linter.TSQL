
# Избыточный DISTINCT после GROUP BY

|||
|-|-|
| Id | **CS0152**
| Мнемо | GROUPBY_DISTINCT
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [DistinctAfterGroupByRule.cs](../../../Rules/CodeSmell/DistinctAfterGroupByRule.cs)

## Причина

Правило срабатывает, если DISTINCT и GROUP BY выполняется на одном и том же столбце.
<p id="descr">Уберите логически ненужный DISTINCT.</p>

## Примеры

Некорректно

```sql
SELECT DISTINCT risk_level_id FROM dbo.risk_levels GROUP BY risk_level_id;
```

Корректно

```sql
SELECT risk_level_id FROM dbo.risk_levels GROUP BY risk_level_id;
```
