
# Не ANSI-сравнение с NULL

|||
|-|-|
| Id | **CS0159**
| Мнемо | NON_ANSI_NULL_COMPARISON
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [NonAnsiNullOperationRule.cs](../../../Rules/CodeSmell/NonAnsiNullOperationRule.cs)

## Причина

Правило срабатывает, если есть не ANSI сравнения (= NULL), (<> NULL), которые никогда не сработают корректно.
<p id="descr">Перепишите используя IS (NOT) NULL.</p>

## Примеры

Некорректно

```sql
IIF('' <> NULL, 1, 2) -- результат равен 2, а не 1

IIF(NULL = NULL, 1, 2) -- результат равен 2, а не 1
```

Корректно

```sql
IIF('' IS NOT NULL, 1, 2)

IIF(NULL IS NULL, 1, 2)
```
