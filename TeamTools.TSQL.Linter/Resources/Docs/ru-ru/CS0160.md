
# Output параметру не присвоено значение

|||
|-|-|
| Id | **CS0160**
| Мнемо | OUTPUT_UNASSIGNED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [OutputParameterNeverAssignedRule.cs](../../../Rules/CodeSmell/OutputParameterNeverAssignedRule.cs)

## Причина

Правило срабатывает, если параметру OUTPUT никогда не присваивается значение.
<p id="descr">Удалите параметр или присвойте ему значение.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE procedure_name
    @x INT OUTPUT
AS
BEGIN
    -- Логика процедуры, без присовения значения переменной @x
END
END;
```

Корректно

```sql
CREATE PROCEDURE procedure_name
    @x INT OUTPUT
AS
BEGIN
    -- Логика процедуры, без присовения значения переменной @x

     SET @x = 1;
END

CREATE PROCEDURE procedure_name
AS
BEGIN
    -- Логика процедуры, без присовения значения переменной @x
END


CREATE PROCEDURE procedure_name
    @x INT OUTPUT
AS
BEGIN
    SET @x = NULL;
    -- Логика процедуры, без присовения значения переменной @x
END
```
