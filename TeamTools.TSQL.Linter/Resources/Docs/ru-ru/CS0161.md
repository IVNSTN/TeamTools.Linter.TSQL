
# Недостижимый код

|||
|-|-|
| Id | **CS0161**
| Мнемо | UNREACHABLE_CODE
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [UnreachableCodeRule.cs](../../../Rules/CodeSmell/UnreachableCodeRule.cs)

## Причина

Правило срабатывает, если есть код, который никогда не выполнится.
<p id="descr">Удалите код, который никогда не выполняется.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.proc_name2
AS
BEGIN
--некоторый код
RETURN 0;

EXEC dbo.proc_name1;
END;
```

Корректно

```sql
CREATE PROCEDURE dbo.proc_name2
AS
BEGIN
--некоторый код
RETURN 0;
END;
```
