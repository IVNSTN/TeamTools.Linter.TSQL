
# Неожиданный возврат данных из триггера

|||
|-|-|
| Id | **CS0163**
| Мнемо | OUTPUT_FROM_TRIGGER
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Триггеры](./Группа_Триггеры.md)
| Исходный код | [FetchingDataFromTriggerRule.cs](../../../Rules/CodeSmell/FetchingDataFromTriggerRule.cs)

## Причина

Правило срабатывает, если тригер выводит данные с помощью команд SELECT или PRINT.
<p id="descr">Уберите из тригера вывод данных.</p>

## Примеры

Некорректно

```sql
CREATE TRIGGER dbo.table_name_iu_repl
ON dbo.table_name
FOR INSERT, UPDATE
AS
BEGIN
   --некоторый код
   print 'abc'
   select 'abс'
END;
```

Корректно

```sql
CREATE TRIGGER dbo.table_name_iu_repl
ON dbo.table_name
FOR INSERT, UPDATE
AS
BEGIN
   --некоторый код
END;
```
