
# Полное имя столбца вместо ссылки, состоящей из двух частей

|||
|-|-|
| Id | **CS0171**
| Мнемо | COLUMN_FULLY_QUALIFIED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [ColumnFullQualifiedReferenceRule.cs](../../../Rules/CodeSmell/ColumnFullQualifiedReferenceRule.cs)

## Причина

Правило срабатывает, если ссылка на столбец таблицы содержит имя базы данных или даже сервера.
<p id="descr">Укажите имя столбца без имени базы данных и сервера.</p>

## Примеры

Некорректно

```sql
SELECT bd_name.dbo.table_name.column_name FROM dbo.table_name
```

Корректно

```sql
SELECT column_name FROM dbo.table_name

SELECT tbl.column_name FROM dbo.table_name AS tbl
```
