
# Цикл не сможет выполниться более одного раза

|||
|-|-|
| Id | **CS0172**
| Мнемо | DISPOSABLE_LOOP
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [UnrepeatableLoopRule.cs](../../../Rules/CodeSmell/UnrepeatableLoopRule.cs)

## Причина

Правило срабатывает, если в теле цикла стоит команда безусловного выхода.
<p id="descr">Укажите условие выхода из цикла.</p>

## Примеры

Некорректно

```sql
WHILE(1=1)
BEGIN
exec dbo.proc;

break;
END
```

Корректно

```sql
WHILE(1=1)
BEGIN
    exec dbo.proc

    if(NOT EXISTS(select 1 from dbo.table))
    BEGIN
        break;
    END;
END
```

## Подсказки

💡 Если итерация действительно нужна одна - откажитесь от цикла. Если выход из цикла имеет условие укажите его.
