
# Недопустимое значение номера ошибки

|||
|-|-|
| Id | **CS0175**
| Мнемо | RAISERROR_VALID_NUMBER
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [RaisErrorNumberValueRule.cs](../../../Rules/CodeSmell/RaisErrorNumberValueRule.cs)

Правило срабатывает, если ERROR_NUMBER находятся вне диапазона 50 000 - 9 999 999.
<p id="descr">Укажите значение ERROR_NUMBER из диапазона 50 000 - 9 999 999.</p>

## Примеры

Некорректно

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorNumber INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorState = 16;
    SET @ErrorNumber = 1;

    THROW @ErrorNumber, @ErrorMessage, @ErrorState;
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_NUMBER() AS ErrorNumber;
END CATCH;
```

Корректно

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorNumber INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorState = 16;
    SET @ErrorNumber = 50001;

    THROW @ErrorNumber, @ErrorMessage, @ErrorState;
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_NUMBER() AS ErrorNumber;
END CATCH;
```
