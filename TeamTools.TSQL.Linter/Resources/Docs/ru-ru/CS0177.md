
# Недопустимое значение состояния ошибки

|||
|-|-|
| Id | **CS0177**
| Мнемо | RAISERROR_VALID_STATE
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [RaisErrorStateValueRule.cs](../../../Rules/CodeSmell/RaisErrorStateValueRule.cs)

## Причина

Правило срабатывает, если значения ERROR_STATE находятся вне диапазона 0-255.
<p id="descr">Укажите значение ERROR_STATE из диапазона 0-255.</p>

## Примеры

Некорректно

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorNumber INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorState = 300;
    SET @ErrorNumber = 51000;

    THROW @ErrorNumber, @ErrorMessage, @ErrorState;
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_STATE() AS ErrorState;
END CATCH;
```

Корректно

```sql
BEGIN TRY
    DECLARE @ErrorMessage NVARCHAR(4000);
    DECLARE @ErrorNumber INT;
    DECLARE @ErrorState INT;

    SET @ErrorMessage = 'This is a test error';
    SET @ErrorState = 100;
    SET @ErrorNumber = 51000;

    THROW @ErrorNumber, @ErrorMessage, @ErrorState;
END TRY
BEGIN CATCH
    SELECT
        ERROR_MESSAGE() AS ErrorMessage,
        ERROR_SEVERITY() AS ErrorSeverity,
        ERROR_STATE() AS ErrorState;
END CATCH;
```
