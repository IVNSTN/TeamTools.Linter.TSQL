

# Использование @@IDENTITY вместо SCOPE_IDENTITY()

|||
|-|-|
| Id | **CS0181**
| Мнемо | IDENTITY_FUNCTION
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [IdentityFunctionRule.cs](../../../Rules/CodeSmell/IdentityFunctionRule.cs)

## Причина

Правило срабатывает, если в коде используется @@IDENTITY.
<p id="descr">Замените @@IDENTITY на SCOPE_IDENTITY().</p>

## Примеры

Некорректно

```sql
SET @id = @@IDENTITY
```

Корректно

```sql
SET @id = SCOPE_IDENTITY()
```
