

# SCOPE_IDENTITY() should be used instead of @@IDENTITY

|||
|-|-|
| Id | **CS0181**
| Мнемо | IDENTITY_FUNCTION
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [IdentityFunctionRule.cs](../../../Rules/CodeSmell/IdentityFunctionRule.cs)

## Причина

<p id="descr">Последний вставленный в текущем батче ID возвращает функция SCOPE_IDENTITY(). Практическое применение функции IDENTITY() в "бизнес-логике" продукта придумать сложно. Не нужно ей пользоваться - она возвращает не то.</p>
