

# Использование RAISERROR вместо THROW в триггере

|||
|-|-|
| Id | **CS0186**
| Мнемо | RAISERROR_IN_TRIGGER
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Триггеры](./Группа_Триггеры.md)
| Исходный код | [RaisErrorInTriggerRule.cs](../../../Rules/CodeSmell/RaisErrorInTriggerRule.cs)

## Причина

Правило срабатывает, если в коде триггера используется RAISERROR.
<p id="descr">Замените RAISERROR на THROW.</p>

## Примеры

Некорректно

```sql
CREATE TRIGGER trg ON dbo.foo
AFTER UPDATE
AS
BEGIN
    RAISERROR('test', 16, 1);
END;
```

Корректно

```sql
CREATE TRIGGER trg ON dbo.foo
AFTER UPDATE
AS
BEGIN
    THROW 50000, 'test', 1;
END;
```
