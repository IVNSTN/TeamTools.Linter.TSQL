

# WAITFOR в триггере

|||
|-|-|
| Id | **CS0189**
| Мнемо | WAITFOR_IN_TRIGGER
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Триггеры](./Группа_Триггеры.md)
| Исходный код | [WaitForInTriggerRule.cs](../../../Rules/CodeSmell/WaitForInTriggerRule.cs)

## Причина

Правило срабатывает, если в триггере используется WAITFOR.
<p id="descr">Удалите WAITFOR.</p>

## Примеры

Некорректно

```sql
CREATE TRIGGER trg ON dbo.foo
AFTER UPDATE
AS
BEGIN
    WAITFOR DELAY '00:01';
END;
```

Корректно

```sql
CREATE TRIGGER trg ON dbo.foo
AFTER UPDATE
AS
BEGIN
    ...
END;
```
