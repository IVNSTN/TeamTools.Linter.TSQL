

# Чтение из пустой таблицы

|||
|-|-|
| Id | **CS0190**
| Мнемо | ALWAYS_EMPTY_SOURCE
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [AlwaysEmptyTableAsSourceRule.cs](../../../Rules/CodeSmell/AlwaysEmptyTableAsSourceRule.cs)

## Причина

Правило срабатывает, если происходит чтение из незаполненной таблицы.
<p id="descr">Заполните таблицу, или удалите чтение.</p>

## Примеры

Некорректно

```sql
CREATE TABLE #test (name varchar(100));

SELECT name
FROM #test;
```

Корректно

```sql
CREATE TABLE #test (name varchar(100));

INSERT INTO #test (name)
VALUES ('Name');

SELECT name
FROM #test;
```

## Подсказки

💡 Реагирует на попытки читать из DELETED в триггере на INSERT, и из INSERTED в триггере на DELETE.

💡 Временная таблица может заполняться за пределами модуля.
