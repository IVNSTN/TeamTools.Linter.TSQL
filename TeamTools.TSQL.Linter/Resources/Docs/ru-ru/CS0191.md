

# Таблица заполняется, но не используется

|||
|-|-|
| Id | **CS0191**
| Мнемо | NEVER_USED_SOURCE
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [TableNeverUsedAsSourceRule.cs](../../../Rules/CodeSmell/TableNeverUsedAsSourceRule.cs)

## Причина

Правило срабатывает, если временная таблица или табличная переменная заполняется, но не используется как источник данных.
<p id="descr">Удалите неиспользуемую временную таблицу или табличную переменную.</p>

## Примеры

Некорректно

```sql
CREATE TABLE #test (name varchar(100));

INSERT INTO #test (name)
VALUES ('Name');
```

Корректно

```sql
CREATE TABLE #test (name varchar(100));

INSERT INTO #test (name)
VALUES ('Name');

SELECT TOP(1) @firstName = name
FROM #test;
```

## Подсказки

💡 Временная таблица может использоваться за пределами модуля.
