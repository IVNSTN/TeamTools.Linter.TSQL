

# Курсор не объявлен как LOCAL

|||
|-|-|
| Id | **CS0195**
| Мнемо | CURSOR_LOCAL
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Курсоры](./Группа_Курсоры.md)
| Исходный код | [CursorScopeRule.cs](../../../Rules/CodeSmell/CursorScopeRule.cs)

## Причина

Правило срабатывает, если курсор не объявлен как LOCAL.
<p id="descr">Установите свойство LOCAL.</p>

## Примеры

Некорректно

```sql
DECLARE cr CURSOR GLOBAL FOR
    SELECT 1;
```

Корректно

```sql
DECLARE cr CURSOR LOCAL FOR
    SELECT 1;
```
