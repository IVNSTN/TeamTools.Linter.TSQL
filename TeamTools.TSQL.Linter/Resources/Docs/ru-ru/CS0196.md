

# Курсор не объявлен явно как READONLY, или FOR UPDATE

|||
|-|-|
| Id | **CS0196**
| Мнемо | CURSOR_READONLY
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Курсоры](./Группа_Курсоры.md)
| Исходный код | [CursorReadonlyRule.cs](../../../Rules/CodeSmell/CursorReadonlyRule.cs)

## Причина

Правило срабатывает, если курсор не объявлен явно как READONLY, или FOR UPDATE.
<p id="descr">Установите явно READONLY, или FOR UPDATE.</p>

## Примеры

Некорректно

```sql
DECLARE cr CURSOR FOR
    SELECT 1;
```

Корректно

```sql
DECLARE cr CURSOR READ_ONLY FOR
    SELECT 1;
```
