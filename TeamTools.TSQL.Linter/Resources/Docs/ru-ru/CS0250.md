

# Избыточный унарный оператор

|||
|-|-|
| Id | **CS0250**
| Мнемо | REDUNDANT_OPERATOR
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [UnaryOperatorRepeatedRule.cs](../../../Rules/CodeSmell/UnaryOperatorRepeatedRule.cs)

## Причина

Правило срабатывает, если унарный оператор избыточен.
<p id="descr">Удалите избыточный унарный оператор.</p>

## Примеры

Некорректно

```sql
SELECT
    - - SUM(a + b / 3)
WHERE NOT NOT (1=1)
```

Корректно

```sql
SELECT
    -SUM(a + b / 3)
WHERE NOT (1=1)
```

## Подсказки

💡 Правило проверяет повторение операторов в выражении.
