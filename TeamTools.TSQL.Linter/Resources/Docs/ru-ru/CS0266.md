

# Ограничение уровня таблицы задано не на уровне таблицы

|||
|-|-|
| Id | **CS0266**
| Мнемо | TABLE_LEVEL_CONSTRAINT_IN_COL
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [TableLevelConstraintInColumnRule.cs](../../../Rules/CodeSmell/TableLevelConstraintInColumnRule.cs)

## Причина

Правило срабатывает, если ограничение уровня таблицы не отделено запятой от ближайшего столбца.
<p id="descr">Отделите ограничение от ближайшего столбца запятой.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    group_id INT NULL FOREIGN KEY (group_id) REFERENCES dbo.foo_group(id)
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    group_id INT NULL,
    CONSTRAINT FK_GROUP FOREIGN KEY (group_id) REFERENCES dbo.foo_group(id)
)
```

## Подсказки

💡 Правило не проверяет временные таблицы и табличные переменные.

💡 Правило проверяет:

- `PRIMARY KEY`
- `FOREIGN KEY`
- `CHECK`
- `UNIQUE`
