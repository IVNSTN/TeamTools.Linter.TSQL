

# ORDER BY ссылается на позицию

|||
|-|-|
| Id | **CS0295**
| Мнемо | ORDER_BY_POSITION
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [OrderByPositionRule.cs](../../../Rules/CodeSmell/OrderByPositionRule.cs)

## Причина

Правило срабатывает, если ORDER BY ссылается на столбец по позиции, а не по имени.
<p id="descr">Замените позиции столбцов на имена.</p>

## Примеры

Некорректно

```sql
SELECT a, b
FROM bar
ORDER BY 1
```

Корректно

```sql
SELECT a, b
FROM bar
ORDER BY a
```
