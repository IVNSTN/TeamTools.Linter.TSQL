

# Отладочные конструкции в коде

|||
|-|-|
| Id | **CS0514**
| Мнемо | SHOWING_STATS
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [ShowStatisticsFromCodeRule.cs](../../../Rules/CodeSmell/ShowStatisticsFromCodeRule.cs)

## Причина

Правило срабатывает, если в коде встречаются отладочные конструкции (SET STATISTICS IO, TIME и т.п.).
<p id="descr">Удалите отладочные конструкции.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE foo
AS
BEGIN
    SET STATISTICS IO ON;
    RETURN 1;
END;
```

Корректно

```sql
CREATE PROCEDURE foo
AS
BEGIN
    RETURN 1;
END;
```
