

# Блокировки уровня приложения в коде

|||
|-|-|
| Id | **CS0520**
| Мнемо | APP_LOCK
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [AppLockRule.cs](../../../Rules/CodeSmell/AppLockRule.cs)

## Причина

Правило срабатывает, если в коде встречаются блокировки уровня приложения (sp_getapplock).
<p id="descr">Удалите блокировки уровня приложения.</p>

## Примеры

Некорректно

```sql
EXEC sp_getapplock;
EXEC foo;
EXEC sp_releaseapplock;
```

Корректно

```sql
EXEC foo;
```

## Подсказки

💡 Блокировки уровня приложения почти всегда свидетельствует о неудачно выбранном подходе к решению задачи.
