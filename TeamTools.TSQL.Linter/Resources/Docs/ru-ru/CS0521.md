

# Не проверяется возвращаемое значение системной процедуры

|||
|-|-|
| Id | **CS0521**
| Мнемо | SYSPROC_RETURN_NOT_CHECKED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SystemProcedureReturnCodeCheckRule.cs](../../../Rules/CodeSmell/SystemProcedureReturnCodeCheckRule.cs)

## Причина

Правило срабатывает, если не проверяется возвращаемое значение системной процедуры.
<p id="descr">Добавьте проверку возвращаемого значения системной процедуры.</p>

## Примеры

Некорректно

```sql
EXEC sp_oacreate;
```

Корректно

```sql
EXEC @ret_val = sp_oacreate;
```

## Подсказки

💡 Правило игнорирует:

- sp_executesql
- sp_addextendedproperty
- sp_bindrule, sp_unbindrule
- sp_bindefault, sp_unbindefault
