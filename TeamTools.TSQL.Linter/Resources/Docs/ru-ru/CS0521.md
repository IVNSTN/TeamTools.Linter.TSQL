

# System proc return value not checked

|||
|-|-|
| Id | **CS0521**
| Мнемо | SYSPROC_RETURN_NOT_CHECKED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SystemProcedureReturnCodeCheckRule.cs](../../../Rules/CodeSmell/SystemProcedureReturnCodeCheckRule.cs)

## Причина

<p id="descr">После вызова большинства системных хп необходимо узнавать об успешности вызова через проверку RETURN значения. Если это значение не анализировать, то дальнейшие действия, рассчитывающие на успешность вызова системной хп до них, может быть недопустимо или нежелательно вызывать.</p>
