

# Логика на основе CONTEXT_INFO()

|||
|-|-|
| Id | **CS0522**
| Мнемо | CONTEXT_INFO
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [ContextInfoRule.cs](../../../Rules/CodeSmell/ContextInfoRule.cs)

## Причина

Правило срабатывает, если в коде встречается логика на основе CONTEXT_INFO().
<p id="descr">Удалите логику на основе CONTEXT_INFO().</p>

## Примеры

Некорректно

```sql
IF CAST(CONTEXT_INFO() AS VARBINARY(128)) = 0x01
    SET @value = 5;
```

Корректно

```sql
IF (a > b)
    SET @value = 5;
```
