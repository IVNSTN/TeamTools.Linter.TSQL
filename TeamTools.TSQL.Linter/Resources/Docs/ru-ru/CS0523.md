## Причина

<p id="descr">Исходники лежат в репозитории и у каждого разработчика локально в открытом виде, шифрование SQL SERVER не сказать чтобы надежное. Шифрование кода нередко усложняет жизнь лишь при обслуживании БД и отладке кода.</p>
