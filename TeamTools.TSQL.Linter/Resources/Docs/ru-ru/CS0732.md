

# Missing GO after procedure body

|||
|-|-|
| Id | **CS0732**
| Мнемо | GO_AFTER_PROC
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [GoAfterProcRule.cs](../../../Rules/CodeSmell/GoAfterProcRule.cs)

## Причина

<p id="descr">Тело хранимой процедуры в скрипте нужно завершать командой GO иначе весь последующий код (например, гранты) тоже будет зачтен как часть тела ХП.</p>
