

# Не указано имя схемы

|||
|-|-|
| Id | **CS0739**
| Мнемо | SCHEMA_NEEDS_NAME
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SchemaNeedsNameRule.cs](../../../Rules/CodeSmell/SchemaNeedsNameRule.cs)

## Причина

Правило срабатывает, если при определении команды CREATE SCHEMA не указано имя схемы.
<p id="descr">Укажите имя схемы.</p>

## Примеры

Некорректно

```sql
CREATE SCHEMA  AUTHORIZATION dbo;
```

Корректно

```sql
CREATE SCHEMA my_schema AUTHORIZATION dbo;
```

## Подсказки

💡 В целях обратной совместимости выражение CREATE SCHEMA позволяет не указывать имя схемы. Однако делать это не рекомендуется.
