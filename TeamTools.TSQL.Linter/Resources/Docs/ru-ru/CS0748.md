

# Использование IDENTITY_INSERT

|||
|-|-|
| Id | **CS0748**
| Мнемо | IDENTITY_INSERT
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [IdentityInsertRule.cs](../../../Rules/CodeSmell/IdentityInsertRule.cs)

## Причина

Правило срабатывает, если используется опция IDENTITY_INSERT.
<p id="descr">Удалите опцию IDENTITY_INSERT.</p>

## Примеры

Некорректно

```sql
SET IDENTITY_INSERT dbo.foo ON;
INSERT dbo.foo(id) VALUES(1);
```

Корректно

```sql
INSERT dbo.foo(id) VALUES(1);
```

## Подсказки

💡 Вставка в IDENTITY-столбец плохо совмещается с высокой нагрузкой и конкуренцией за вставку и чтение.
