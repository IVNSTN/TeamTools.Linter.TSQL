

# Использование полнотекстового поиска

|||
|-|-|
| Id | **CS0751**
| Мнемо | FULLTEXT_SEARCH
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [FullTextSearchRule.cs](../../../Rules/CodeSmell/FullTextSearchRule.cs)

## Причина

Правило срабатывает, если используется полнотекстовый поиск.
<p id="descr">Не используйте полнотекстовый поиск.</p>

## Примеры

Некорректно

```sql
SELECT TOP 10 *
FROM dbo.foo
WHERE freetext(title, 'asdf')
```

Корректно

```sql
SELECT TOP 10 *
FROM dbo.foo
WHERE title LIKE 'asdf%'
```
