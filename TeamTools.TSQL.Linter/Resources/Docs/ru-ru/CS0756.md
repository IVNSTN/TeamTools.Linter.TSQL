

# DEADLOCK_PRIORITY задан числовым значением

|||
|-|-|
| Id | **CS0756**
| Мнемо | DEADLOCK_PRIORITY_PRECISE
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [DeadlockPriorityPreciseRule.cs](../../../Rules/CodeSmell/DeadlockPriorityPreciseRule.cs)

## Причина

Правило срабатывает, если DEADLOCK_PRIORITY задан числовым значением.
<p id="descr">Используйте встроенные значения LOW/NORMAL/HIGH.</p>

## Примеры

Некорректно

```sql
SET DEADLOCK_PRIORITY -5;
```

Корректно

```sql
SET DEADLOCK_PRIORITY LOW;
```
