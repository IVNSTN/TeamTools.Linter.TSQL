

# Использование SPARSE столбцов во временных таблицах или табличных переменных

|||
|-|-|
| Id | **CS0776**
| Мнемо | SPARSE_COL_IN_TMP
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SparseColInTempTableRule.cs](../../../Rules/CodeSmell/SparseColInTempTableRule.cs)

## Причина

Правило срабатывает, если во временных таблицах или табличных переменных используются SPARSE столбцы.
<p id="descr">Не используйте SPARSE столбцы.</p>

## Примеры

Некорректно

```sql
CREATE TABLE #foo (
    id INT NOT NULL
    , calc INT SPARSE NULL
);
```

Корректно

```sql
CREATE TABLE #foo (
    id INT NOT NULL,
    , calc INT NULL
);
```
