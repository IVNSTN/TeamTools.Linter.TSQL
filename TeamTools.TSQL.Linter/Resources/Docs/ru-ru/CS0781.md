

# INFORMATION_SCHEMA views mixed with SQL SERVER sys-views

|||
|-|-|
| Id | **CS0781**
| Мнемо | INFORMATION_SCHEMA_SYS_MIX
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [InformationSchemaSysTablesMixRule.cs](../../../Rules/CodeSmell/InformationSchemaSysTablesMixRule.cs)

## Причина

<p id="descr">Набор представлений INFORMATION_SCHEMA нужен для совместимости со стандартом ANSI. Если целью обращения к этим представлениям была именно совместимость, то странно вплетать в эти же запросы системные представления SQL SEVER, которые со стандартом ANSI никак не совместимы. Если INFORMATION_SCHEMA недостаточно, то лучше полностью переписать запрос на sys-представления.</p>
