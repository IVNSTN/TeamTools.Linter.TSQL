

# Незадокументированная инструкция LINENO

|||
|-|-|
| Id | **CS0786**
| Мнемо | LINE_NO
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [LineNoRule.cs](../../../Rules/CodeSmell/LineNoRule.cs)

## Причина

Правило срабатывает, если в коде используется инструкция LINENO.
<p id="descr">Удалите инструкцию LINENO.</p>

## Примеры

Некорректно

```sql
CREATE PROC dbo.foo
AS
BEGIN
    SET NOCOUNT ON;
    ...
    LINENO 0;
    RAISERROR ('Error', 16, 1);
END;
```

Корректно

```sql
CREATE PROC dbo.foo
AS
BEGIN
    SET NOCOUNT ON;
    ...
    RAISERROR ('Error', 16, 1);
END;
```
