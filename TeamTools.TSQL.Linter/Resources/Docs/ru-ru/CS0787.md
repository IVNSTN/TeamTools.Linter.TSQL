

# Использование ROWVERSION на временном объекте

|||
|-|-|
| Id | **CS0787**
| Мнемо | ROWVERSION_MISUSED
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [RowVersionMisusedRule.cs](../../../Rules/CodeSmell/RowVersionMisusedRule.cs)

## Причина

Правило срабатывает, если во временной таблице или табличной переменной используется столбец `ROWVERSION` или `TIMESTAMP`.
<p id="descr">Удалите столбец ROWVERSION или TIMESTAMP.</p>

## Примеры

Некорректно

```sql
CREATE TABLE #foo
(
    id     INT          NOT NULL PRIMARY KEY
    , name VARCHAR(100) NOT NULL
    , rv   ROWVERSION   NOT NULL
);
```

Корректно

```sql
CREATE TABLE #foo
(
    id     INT          NOT NULL PRIMARY KEY
    , name VARCHAR(100) NOT NULL
);
```
