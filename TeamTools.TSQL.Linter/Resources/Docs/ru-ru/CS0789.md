

# Приблизительный набор данных

|||
|-|-|
| Id | **CS0789**
| Мнемо | TABLESAMPLE_PREDICT
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [TableSamplePredictRule.cs](../../../Rules/CodeSmell/TableSamplePredictRule.cs)

## Причина

Правило срабатывает, если в коде используется инструкция TABLESAMPLE или функция PREDICT.
<p id="descr">Удалите инструкцию TABLESAMPLE или функцию PREDICT.</p>

## Примеры

Некорректно

```sql
SELECT *
FROM Sales.Customer TABLESAMPLE SYSTEM(10 PERCENT);

SELECT d.*, p.Score
FROM PREDICT(MODEL = @model,
    DATA = dbo.mytable AS d) WITH (Score FLOAT) AS p;
```
