
# Строковый литерал содержит символы юникода

|||
|-|-|
| Id | **CS0792**
| Мнемо | BROKEN_UNICODE_LITERAL
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [LiteralBrokenUnicodeRule.cs](../../../Rules/CodeSmell/LiteralBrokenUnicodeRule.cs)

## Причина

Правило срабатывает, если строковый литерал содержит символы юникода (или из кодовой страницы, отличной от 1251), но не имеет префикса `N`.
<p id="descr">Добавьте префикс 'N' перед строковым литералом.</p>

## Примеры

Некорректно

```sql
SELECT 'adsf' + CAST('☀️' AS NCHAR(10));  -- 1

DECLARE @s NVARCHAR(100) = '☀️☀️☀️';     -- 2
```

Корректно

```sql
SELECT N'adsf' + N'1 ☀️ 2';

DECLARE @s NVARCHAR(100) = N'☀️☀️☀️';
```
