

# Ссылка на глобальный курсор

|||
|-|-|
| Id | **CS0807**
| Мнемо | GLOBAL_CURSOR_REF
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Курсоры](./Группа_Курсоры.md)
| Исходный код | [GlobalCursorReferenceRule.cs](../../../Rules/CodeSmell/GlobalCursorReferenceRule.cs)

## Причина

Правило срабатывает, если в коде используется ссылка на глобальный курсор.
<p id="descr">Удалите ссылку на глобальный курсор.</p>

## Примеры

Некорректно

```sql
UPDATE dbo.foo
SET a = b
WHERE CURRENT OF GLOBAL MY_CURSOR
```

Корректно

```sql
UPDATE dbo.foo
SET a = b
WHERE CURRENT OF @cursor
```
