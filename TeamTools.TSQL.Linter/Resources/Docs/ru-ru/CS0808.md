

# Излишний DISTINCT для источника предиката IN

|||
|-|-|
| Id | **CS0808**
| Мнемо | DISTINCT_FOR_IN
| Серьёзность | ℹ Подсказка
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [DistinctForInSubqueryRule.cs](../../../Rules/CodeSmell/DistinctForInSubqueryRule.cs)

## Причина

Правило срабатывает, если для источника предиката IN указан DISTINCT.
<p id="descr">Удалите DISTINCT.</p>

## Примеры

Некорректно

```sql
SELECT name
FROM dbo.foo
WHERE category_id IN (SELECT DISTINCT category_id FROM dbo.categories)
```

Корректно

```sql
SELECT name
FROM dbo.foo
WHERE category_id IN (SELECT category_id FROM dbo.categories)
```
