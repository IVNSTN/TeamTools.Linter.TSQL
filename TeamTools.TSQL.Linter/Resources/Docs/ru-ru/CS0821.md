

# DECIMAL без дробной части

|||
|-|-|
| Id | **CS0821**
| Мнемо | DECIMAL_WITHOUT_DECIMALS
| Серьёзность | ℹ Подсказка
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [DecimalWithNoScaleRule.cs](../../../Rules/CodeSmell/DecimalWithNoScaleRule.cs)

## Причина

Правило срабатывает, если типы DECIMAL или NUMERIC используются без дробной части.
<p id="descr">Укажите дробную часть, или используйте целочисленный тип.</p>

## Примеры

Некорректно

```sql
DECLARE
    @tax DECIMAL
    , @percent NUMERIC(5, 0);
```

Корректно

```sql
DECLARE
    @tax DECIMAL(18, 3)
    , @percent INT;
```
