

# Одинаковые имена временных таблиц

|||
|-|-|
| Id | **CS0830**
| Мнемо | NAME_REUSED_TEMP_TABLE
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [ReusedTempTableNameRule.cs](../../../Rules/CodeSmell/ReusedTempTableNameRule.cs)

## Причина

Правило срабатывает, если используются одинаковые имена для разных временных таблиц.
<p id="descr">Укажите различные имена.</p>

## Примеры

Некорректно

```sql
CREATE TABLE #foo
(
    descendant_id INT NOT NULL
    , parent_id   INT NOT NULL
);

DROP TABLE #foo;

CREATE TABLE #foo
(
    title       VARCHAR(100)
    , lastmod   DATETIME2(7) NOT NULL DEFAULT SYSDATETIME()
);
```

Корректно

```sql
CREATE TABLE #nodes
(
    descendant_id INT NOT NULL
    , parent_id   INT NOT NULL
);

DROP TABLE #nodes;

CREATE TABLE #titles
(
    title       VARCHAR(100)
    , lastmod   DATETIME2(7) NOT NULL DEFAULT SYSDATETIME()
);
```
