
# Символ другого алфавита в слове текста комментария

|||
|-|-|
| Id | **CS0835**
| Мнемо | COMMENT_LOOK_ALIKE_CHAR
| Серьёзность | ℹ Подсказка
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [CommentContainsLookAlikeCharRule.cs](../../../Rules/CodeSmell/CommentContainsLookAlikeCharRule.cs)

## Причина

Правило срабатывает, если в слове в тексте комментария, набранном преимущественно символами одного алфавита, обнаружен символ из другого алфавита, имеющий визуально близкий аналог в первом алфавите.
<p id="descr">Перепишите используя символы только одного алфавита.</p>

## Примеры

Некорректно

```sql
-- сomment contents аre supposed to be handled by this rule (cyrillic 'с' and 'а' here)
/*
    coМment contents are supposed to be handled by Тhis rule
    (cyrillic 'М' and 'Т' in the line above)
*/
```

Корректно

```sql
-- good comment

/* ничего 123 интересного */
```
