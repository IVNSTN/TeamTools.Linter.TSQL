

# Недопустимый hint в INSERT

|||
|-|-|
| Id | **CS0917**
| Мнемо | FORBIDDEN_INSERT_HINTS
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [InsertHintsRule.cs](../../../Rules/CodeSmell/InsertHintsRule.cs)

## Причина

Правило срабатывает, если в INSERT использован недопустимый hint.
<p id="descr">Удалите недопустимый hint.</p>

## Примеры

Некорректно

```sql
INSERT dbo.Foo WITH (ROWLOCK, FORCESCAN) (id, name)
SELECT 1, '';
```

Корректно

```sql
INSERT dbo.Foo WITH (TABLOCKX) (id, name)
SELECT 1, '';
```

## Подсказки

💡 Недопустимы:

- FORCESEEK
- FORCESCAN
- READPAST
- READCOMMITTED_LOCK

💡 Редки и сомнительны:

- IGNORE_TRIGGERS
- IGNORE_CONSTRAINTS

💡 Вредны:

- ROWLOCK
- PAGLOCK
- TABLOCK
