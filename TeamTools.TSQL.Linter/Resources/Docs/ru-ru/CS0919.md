

# Параметр объявлен, но ни разу не использован

|||
|-|-|
| Id | **CS0919**
| Мнемо | UNUSED_PARAMETER
| Серьёзность | ℹ Подсказка
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [UnusedParameterRule.cs](../../../Rules/CodeSmell/UnusedParameterRule.cs)

## Причина

Правило срабатывает, если параметр хранимой процедуры или функции объявлен, но ни разу не использован.
<p id="descr">Удалите неиспользуемый параметр.</p>

## Примеры

Некорректно

```sql
CREATE FUNCTION dbo.foo (@id INT)
RETURNS INT
AS
BEGIN
    RETURN 1;
END;
```

Корректно

```sql
CREATE FUNCTION dbo.foo (@id INT)
RETURNS INT
AS
BEGIN
    RETURN @id + 1;
END;
```
