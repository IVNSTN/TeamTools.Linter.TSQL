

# Пропущена парная конструкция управления XML документом

|||
|-|-|
| Id | **CS0921**
| Мнемо | UNPAIRED_XMLDOC_STATEMENT
| Серьёзность | ℹ Подсказка
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [UnPairedXmlDocumentInstructionRule.cs](../../../Rules/CodeSmell/UnPairedXmlDocumentInstructionRule.cs)

## Причина

Правило срабатывает, если пропущена парная конструкция управления XML документом.
<p id="descr">Укажите парную конструкцию управления XML документом.</p>

## Примеры

Некорректно

```sql
EXEC sp_xml_preparedocument @handle OUT;
```

Корректно

```sql
EXEC sp_xml_preparedocument @handle OUT;
EXEC sp_xml_removedocument @handle;
```

## Подсказки

💡 Правило проверяет sp_xml_removedocument sp_xml_preparedocument.
