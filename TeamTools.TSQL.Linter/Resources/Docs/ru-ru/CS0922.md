

# Parameter is expected to be OUTPUT

|||
|-|-|
| Id | **CS0922**
| Мнемо | PARAM_EXPECTED_AS_OUTPUT
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [BuiltInProcsArgumentOutputRule.cs](../../../Rules/CodeSmell/BuiltInProcsArgumentOutputRule.cs)

## Причина

<p id="descr">Параметр служебной хранимой процедуры задуман как OUTPUT и передавать его в вызове без указания OUTPUT не имеет смысла.</p>
