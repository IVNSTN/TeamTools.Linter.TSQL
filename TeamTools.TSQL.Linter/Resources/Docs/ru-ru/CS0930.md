

# Повторное использование имени курсора

|||
|-|-|
| Id | **CS0930**
| Мнемо | NAME_REUSED_CURSOR
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Курсоры](./Группа_Курсоры.md)
| Исходный код | [ReusedCursorNameRule.cs](../../../Rules/CodeSmell/ReusedCursorNameRule.cs)

## Причина

Правило срабатывает, если имя курсора в скрипте повторно используется для другого курсора.
<p id="descr">Замените повторяющиеся имена курсора.</p>

## Примеры

Некорректно

```sql
DECLARE base_cursor CURSOR
    FOR SELECT * FROM dbo.table_1
...

DECLARE base_cursor CURSOR
    FOR SELECT * FROM dbo.table_2
...
```

Корректно

```sql
DECLARE first_cursor CURSOR
    FOR SELECT * FROM dbo.table_1
...

DECLARE second_cursor CURSOR
    FOR SELECT * FROM dbo.table_2
...
```

## Подсказки

💡 Повторное использование имени курсора может вызвать путаницу и привести к неожиданным результатам в сложном коде. В результате код может успешно скомпилироваться и даже работать, но с использованием другого курсора, не того, который ожидался.
