

# Infinite loop has no exit

|||
|-|-|
| Id | **CS0955**
| Мнемо | LOOP_INFINITE_MISSING_BREAK
| Серьёзность | ⚠ Предупреждение

## Причина

<p id="descr">Если цикл WHILE объявлен как бесконечный (always true predicate), то в теле цикла ожидается наличие команды выхода иначе цикл никогда не закончится.</p>
