

# Повторяющееся выражение в логике ветвления

|||
|-|-|
| Id | **CS0959**
| Мнемо | DUP_CONDITIONAL_FLOW
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [DuplicateConditionRule.cs](../../../Rules/CodeSmell/DuplicateConditionRule.cs)

## Причина

Правило срабатывает, если в коде есть взаимоисключающие условные переходы (CASE, IF-ELSE-IF) и в них встречаются одинаковые условия.
<p id="descr">Проверьте логику оператора или выражения ветвления.</p>

## Примеры

Некорректно

```sql
SELECT
    CASE
        WHEN @a > @b THEN 1
        WHEN @a > @b THEN 2
        WHEN @b < @a THEN 4
        ...
    END
...
```

Корректно

```sql
SELECT
    CASE
        WHEN @a > @b THEN 1
        ...
    END
...
```
