

# Триггер должен находиться в той же схеме, что и целевая таблица

|||
|-|-|
| Id | **CS0975**
| Мнемо | TRIGGER_SCHEMA_TABLE_SCHEMA
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md), [Триггеры](./Группа_Триггеры.md)
| Исходный код | [TriggerSchemaFromTableRule.cs](../../../Rules/CodeSmell/TriggerSchemaFromTableRule.cs)

## Причина

Правило срабатывает, если схема, в которой создается триггер, не совпадает со схемой связанной таблицы.
<p id="descr">Укажите правильную схему для триггера.</p>

## Примеры

Некорректно

```sql
CREATE TRIGGER dbo.tr_log ON stage.table
...
```

Корректно

```sql
CREATE TRIGGER stage.tr_log ON stage.table
...
```

## Подсказки

💡 Действие триггеров DML ограничивается схемой той таблицы или того представления, для которых они созданы.
