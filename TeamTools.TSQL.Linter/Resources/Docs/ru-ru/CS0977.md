

# Dropping without existance check is not error prone

|||
|-|-|
| Id | **CS0977**
| Мнемо | DROPPING_WITH_NO_CHECK
| Серьёзность | ℹ Подсказка
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)

## Причина

<p id="descr">Дропы лучше предварять проверкой наличия объекта через OBJECT_ID(...) IS NOT NULL или применять конструкцию DROP IF EXISTS.</p>
