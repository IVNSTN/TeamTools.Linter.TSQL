

# Cursor should be defined as forward only

|||
|-|-|
| Id | **CS0982**
| Мнемо | CURSOR_FORWARD_ONLY
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [CursorForwardOnlyRule.cs](../../../Rules/CodeSmell/CursorForwardOnlyRule.cs)

## Причина

<p id="descr">Если курсор просматривается в одном направлении (FETCH NEXT), то лучше явно это указать в определении курсора с помощью опции FORWARD_ONLY или FAST_FORWARD.</p>
