

# Переменная присваивается сама себе

|||
|-|-|
| Id | **CS0984**
| Мнемо | VAR_SELF_ASSIGN
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [VariableSelfAssignRule.cs](../../../Rules/CodeSmell/VariableSelfAssignRule.cs)

## Причина

Правило срабатывает, если переменная присваивается сама себе.
<p id="descr">Удалите ненужное присваивание.</p>

## Примеры

Некорректно

```sql
SELECT @a = (SELECT (@a))
    , @b = (@b)
    , @c = @c
```

Корректно

```sql
SELECT @a = 5
    , @b = GETDATE()
    , @c = @d
```
