

# Переменная сравнивается сама с собой

|||
|-|-|
| Id | **CS0985**
| Мнемо | VAR_SELF_COMPARE
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [VariableSelfCompareRule.cs](../../../Rules/CodeSmell/VariableSelfCompareRule.cs)

## Причина

Правило срабатывает, если переменная сравнивается сама с собой.
<p id="descr">Удалите ненужное сравнение.</p>

## Примеры

Некорректно

```sql
SELECT IIF(@b < (@b), @c, 100500) AS c2
```

Корректно

```sql
SELECT IIF(@b < 10, @c, 100500) AS c2
```
