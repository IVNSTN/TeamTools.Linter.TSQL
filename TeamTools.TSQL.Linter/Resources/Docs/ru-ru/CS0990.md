

# Suspicious keyword used for tran name

|||
|-|-|
| Id | **CS0990**
| Мнемо | SUSPICIOUS_TRAN_NAME
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [SuspiciousTranNameRule.cs](../../../Rules/CodeSmell/SuspiciousTranNameRule.cs)

## Причина

<p id="descr">Если в качестве имени транзакции использовано ключевое слово больше похожее на отдельную команду, то возможно в коде потерялся разделитель команд ";". При срабатывании правила нужно подобрать другое имя для транзакции или поставить точку с запятой сразу после выражения, управляющего транзакцией.</p>
