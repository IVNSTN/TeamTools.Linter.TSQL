

# WAITFOR используется без TIMEOUT

|||
|-|-|
| Id | **CS0992**
| Мнемо | WAITFOR_STMT_TIMEOUT
| Серьёзность | ⚠ Предупреждение
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [WaitForStatementWithoutTimeoutRule.cs](../../../Rules/CodeSmell/WaitForStatementWithoutTimeoutRule.cs)

## Причина

Правило срабатывает, если инструкция WAITFOR используется без TIMEOUT.
<p id="descr">Задайте TIMEOUT в инструкции WAITFOR.</p>

## Примеры

Некорректно

```sql
WAITFOR (GET CONVERSATION GROUP @ConversationGroupId FROM backend._TargetQueue_MainEntities);
```

Корректно

```sql
WAITFOR (GET CONVERSATION GROUP @ConversationGroupId FROM backend._TargetQueue_MainEntities),
TIMEOUT 100;
```
