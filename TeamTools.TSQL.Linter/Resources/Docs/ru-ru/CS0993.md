

# Возможна потеря точности при использовании переменных типа FLOAT

|||
|-|-|
| Id | **CS0993**
| Мнемо | FLOAT_VARIABLE
| Серьёзность | ℹ Подсказка
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [FloatVariableUsedRule.cs](../../../Rules/CodeSmell/FloatVariableUsedRule.cs)

## Причина

Правило срабатывает, если в коде используются переменные с типом FLOAT.
<p id="descr">Используйте тип FLOAT с большой осторожностью.</p>

## Подсказки

💡 Специализированные модули для вычислений с плавающей точкой лучше поместить в отдельную схему и добавить её в исключения.
