

# Float type is inaccurate

|||
|-|-|
| Id | **CS0993**
| Мнемо | FLOAT_VARIABLE
| Серьёзность | ℹ Подсказка
| Категория | [СомнительноеКодирование](./Категория_СомнительноеКодирование.md)
| Исходный код | [FloatVariableUsedRule.cs](../../../Rules/CodeSmell/FloatVariableUsedRule.cs)

## Причина

<p id="descr">Типом FLOAT нужно пользоваться с большой осторожностью, поскольку вероятна потеря точности вычислений.</p>


## Подсказки

💡 Правило реагирует на _переменные_ с типом FLOAT и REAL
