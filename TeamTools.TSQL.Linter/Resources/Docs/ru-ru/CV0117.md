
# Имя ограничения на персистентной таблице

|||
|-|-|
| Id | **CV0117**
| Мнемо | UNNAMED_CONSTRAINT
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [ConstraintNameInPersistentTableRule.cs](../../../Rules/CodingConvention/ConstraintNameInPersistentTableRule.cs)

## Причина

Правило срабатывает, если ограничению на персистентной таблице не присвоено имя.
<p id="descr">Задайте имя для ограничения.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.test (
    a INT NOT NULL DEFAULT 0
);
```

Корректно

```sql
CREATE TABLE dbo.test (
    a INT NOT NULL CONSTRAINT DF_test_a DEFAULT (0)
);
```
