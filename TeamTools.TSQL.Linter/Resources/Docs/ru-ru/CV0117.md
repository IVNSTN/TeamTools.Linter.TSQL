

# Unnamed constraint in persistent table

|||
|-|-|
| Id | **CV0117**
| Мнемо | UNNAMED_CONSTRAINT
| Серьёзность | ⚠ Предупреждение

## Причина

<p id="descr">В персистентных таблицах констрейнтам должны быть присвоены имена. Чаще всего неименованными остаются дефолты. Такие констрейнты потом сложнее обслуживать.</p>
