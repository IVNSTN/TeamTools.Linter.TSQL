
# Предоставление разрешений в файле определения объекта

|||
|-|-|
| Id | **CV0165**
| Мнемо | PERMISSION_IN_DEFINITION
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [PermissionsInDefinitionFileRule.cs](../../../Rules/CodingConvention/PermissionsInDefinitionFileRule.cs)

## Причина

Правило срабатывает, если в файле определения объекта предоставляются привилегии доступа.
<p id="descr">Перенесите предоставление привилегий в файл соответствующей роли.</p>

## Примеры

Некорректно

```sql
/* dbo.Bar.sql */
CREATE SYNONYM Bar
FOR FAR;
GO

GRANT SELECT ON OBJECT::dbo.Bar TO Foo;
GO
```

Корректно

```sql
/* dbo.Bar.sql */
CREATE SYNONYM Bar
FOR FAR;
GO

/* Foo.sql */
CREATE ROLE Foo ...
...

GRANT SELECT ON OBJECT::dbo.Bar TO Foo;
GO
```
