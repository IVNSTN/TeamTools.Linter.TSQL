
# Регистр при написании ключевых слов

|||
|-|-|
| Id | **CV0201**
| Мнемо | KEYWORD_UPPER
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [KeywordUppercaseRule.cs](../../../Rules/CodingConvention/KeywordUppercaseRule.cs)

## Причина

Правило срабатывает, если ключевое слово написано не в верхнем регистре.
<p id="descr">Пишите ключевые слова в верхнем регистре.</p>

## Примеры

Некорректно

```sql
set nocount on;
declare @a int;
if 1 <> 0
begin
    select top (1) 1 as z from foo
    where bar <> cast(zar as date)
    order by case when upper(a) = a then 1 else 0 end;
end;
```

Корректно

```sql
SET NOCOUNT ON;
DECLARE @a INT;
IF 1 <> 0
BEGIN
    SELECT TOP (1) 1 AS z FROM dbo.foo
    WHERE bar <> CAST(zar AS DATE)
    ORDER BY CASE WHEN UPPER(a) = a THEN 1 ELSE 0 END;
END;
```
