
# Регистр при написании имен системных функций

|||
|-|-|
| Id | **CV0202**
| Мнемо | SYSTEM_FUNC_UPPER
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [SystemFunctionCallUppercaseRule.cs](../../../Rules/CodingConvention/SystemFunctionCallUppercaseRule.cs)

## Причина

Правило срабатывает, если имя вызываемой системной функции написано не в верхнем регистре.
<p id="descr">Пишите имена вызываемых системных функций в верхнем регистре.</p>

## Примеры

Некорректно

```sql
SELECT
    try_cast(try_convert(NUMERIC(10, 2), '0') AS NUMERIC(10, 2))
    , iif(0 = 0, 1, 1);
```

Корректно

```sql
SELECT
    TRY_CAST(TRY_CONVERT(NUMERIC(10, 2), '0') AS NUMERIC(10, 2))
    , IIF(0 = 0, 1, 1);
```
