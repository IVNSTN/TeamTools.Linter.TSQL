

# Built-in function is not in UPPERCASE

|||
|-|-|
| Id | **CV0202**
| Мнемо | SYSTEM_FUNC_UPPER
| Серьёзность | ⚠ Предупреждение

## Причина

<p id="descr">Вызовы системных функций, функций встроенных в SQL SERVER должны быть в верхнем регистре.</p>
