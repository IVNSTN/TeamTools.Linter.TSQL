
# Указание вида соединения в два ключевых слова

|||
|-|-|
| Id | **CV0208**
| Мнемо | TWO_WORDS_JOIN
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [JoinSpellingRule.cs](../../../Rules/CodingConvention/JoinSpellingRule.cs)

## Причина

Правило срабатывает, если вид соединения (`JOIN`) указан не в два ключевых слова.
<p id="descr">Укажите вид соединения используя два ключевых слова.</p>

## Примеры

Некорректно

```sql
SELECT a.z
FROM a
JOIN b
    ON a.a = b.b
LEFT OUTER JOIN c
    ON c.c = b.b
RIGHT OUTER JOIN d
    ON d.d = a.a
FULL OUTER JOIN e
    ON e.e = c.c;
```

Корректно

```sql
SELECT a.z
FROM a
INNER JOIN b
    ON a.a = b.b
LEFT JOIN c
    ON c.c = b.b
RIGHT JOIN d
    ON d.d = a.a
FULL JOIN e
    ON e.e = c.c;
```

## Подсказки

💡 Правильное указание вида соединения:

- `INNER JOIN`
- `LEFT JOIN`
- `RIGHT JOIN`
- `FULL JOIN`
- `CROSS JOIN`
- `CROSS APPLY`
- `OUTER APPLY`
