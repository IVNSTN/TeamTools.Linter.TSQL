
# Использование операторов сравнения, не соответствующих стандарту ISO

|||
|-|-|
| Id | **CV0209**
| Мнемо | C_STYLE_OPERANDS
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [ComparisonSyntaxRule.cs](../../../Rules/CodingConvention/ComparisonSyntaxRule.cs)

## Причина

Правило срабатывает, если обнаружено использование запрещенных операторов сравнения: `!=`, `=*`, `*=`.
<p id="descr">Для неравенства используйте '<>', для внешних соединений используйте полноценный синтаксис.</p>

## Примеры

Некорректно

```sql
IF 1 != 0
BEGIN
    SELECT 1;
END
```

Корректно

```sql
IF 1 <> 0
BEGIN
    SELECT 1;
END
```
