

# Constraint defined before NULL/NOT NULL atribute

|||
|-|-|
| Id | **CV0217**
| Мнемо | COLUMN_NULLABILITY_BEFORE_CONSTRAINT
| Серьёзность | ⚠ Предупреждение

## Причина

<p id="descr">В определении таблицы свойства столбца оформляйте в следующем порядке: тип, NULL/NOT NULL, потом все остальное (IDENTITY, CONSTRAINT и т.д.)</p>
