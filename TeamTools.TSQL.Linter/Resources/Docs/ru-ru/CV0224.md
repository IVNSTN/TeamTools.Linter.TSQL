
# Написание команд управления транзакциями

|||
|-|-|
| Id | **CV0224**
| Мнемо | TRAN_CONTROL_TWO_WORDS
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [TransactionControlInTwoWordsRule.cs](../../../Rules/CodingConvention/TransactionControlInTwoWordsRule.cs)

## Причина

Правило срабатывает, если управление транзакциями не использует однообразное написание (или полное или сокращенное), состоящее из двух ключевых слов.
<p id="descr">Пишите команды управления транзакциями в два ключевых слова, используя либо сокращенное, либо полное написание на протяжении всего кода.</p>

## Примеры

Некорректно

```sql
BEGIN TRAN;
COMMIT TRANSACTION;
ROLLBACK;
```

Корректно

```sql
BEGIN TRAN;
COMMIT TRAN;
ROLLBACK TRAN;

-- or
BEGIN TRANSACTION;
COMMIT TRANSACTION;
ROLLBACK TRANSACTION;
```
