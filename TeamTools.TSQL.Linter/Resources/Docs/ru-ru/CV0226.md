
# Громоздкие конструкции в блоке DECLARE

|||
|-|-|
| Id | **CV0226**
| Мнемо | MULTILINE_DECLARE_ELEMENT
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [SingleLineDeclareElementRule.cs](../../../Rules/CodingConvention/SingleLineDeclareElementRule.cs)

## Причина

Правило срабатывает, если в блоке `DECLARE` обнаружено присвоение многострочных значений переменным.
<p id="descr">Упростите присваиваемое значение или перенесите его за блок 'DECLARE'.</p>

## Примеры

Некорректно

```sql
DECLARE
    @foo INT = (1 - 2
    * 3)
    , @bar VARCHAR(10) = 'aasdf
                            asdf
                                xcvzxv'
    , @dt DATETIME = (
        SELECT next_date
        FROM calendar
        WHERE dt = DATEADD(DD, -1, GETDATE())
    );
```

Корректно

```sql
DECLARE 
    @foo INT = (1 - 2 * 3)
    , @bar VARCHAR(10) = ''
    , @dt DATETIME = DATEADD(DD, -1, GETDATE());
```
