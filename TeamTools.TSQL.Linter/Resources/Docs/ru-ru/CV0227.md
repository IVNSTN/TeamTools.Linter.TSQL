
# Использование SELECT в блоке DECLARE

|||
|-|-|
| Id | **CV0227**
| Мнемо | SELECT_IN_DECLARE
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [SelectInDeclareRule.cs](../../../Rules/CodingConvention/SelectInDeclareRule.cs)

## Причина

Правило срабатывает, если в блоке `DECLARE` обнаружено присвоение значения переменным с использованием конструкций типа `SELECT`.
<p id="descr">Упростите присваиваемое значение или перенесите его за блок 'DECLARE'.</p>

## Примеры

Некорректно

```sql
DECLARE @foo INT = (SELECT a FROM  b);
```

Корректно

```sql
DECLARE
    @foo   INT         = 0 * 1
    , @bar VARCHAR(10) = 'SELECT'
    , @zar BIT
    , @nar DATETIME    = GETDATE()
    , @trn INT         = @@TRANCOUNT;
```
