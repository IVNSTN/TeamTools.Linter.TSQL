

# SELECT statement in DECLARE

|||
|-|-|
| Id | **CV0227**
| Мнемо | SELECT_IN_DECLARE
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [SelectInDeclareRule.cs](../../../Rules/CodingConvention/SelectInDeclareRule.cs)

## Причина

<p id="descr">Не надо селектов как и других сложных вычислений в DECLARE. DECLARE должен быть простым для чтения и неспособным породить runtime/compile time ошибки.</p>
