
# Использование UNION для формирования списка скалярных значений

|||
|-|-|

| Серьёзность | ⚠ Предупреждение

## Причина

Правило срабатывает, если обнаружено формирование списка скалярных значений при помощи объединения (`UNION`).
<p id="descr">Для формирования списка скалярных значений используйте 'VALUES'.</p>

## Примеры

Некорректно

```sql
SELECT 1
UNION ALL
SELECT * FROM (VALUES (2), (3)) AS v (id);
```

Корректно

```sql
SELECT * FROM (VALUES (1), (2), (3)) AS v (id);
```
