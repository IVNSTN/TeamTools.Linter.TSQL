
# Несколько однострочных комментариев подряд

|||
|-|-|
| Id | **CV0245**
| Мнемо | MANY_SL_COMMENTS_TO_ML
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [MultipleSingleLineCommentsToMultilineCommentRule.cs](../../../Rules/CodingConvention/MultipleSingleLineCommentsToMultilineCommentRule.cs)

## Причина

Правило срабатывает, если в коде найдено более двух однострочных комментариев подряд.
<p id="descr">Оформите в виде одного многострочного комментария.</p>

## Примеры

Некорректно

```sql
-- comment 1
-- comment 2
-- comment 3
SELECT 1;
```

Корректно

```sql
/*
comment 1
comment 2
comment 3
*/
SELECT 1;
```
