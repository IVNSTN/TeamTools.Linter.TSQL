## Причина

Правило срабатывает, если в начале строки обнаружена лидирующая точка с запятой `;`.
<p id="descr">Перенесите точку с запятой ';' в конец предыдущей конструкции/выражения.</p>

## Примеры

Некорректно

```sql
BEGIN
    ;THROW 500001, 'test', 1;
END;
```

Корректно

```sql
BEGIN;
    THROW 500001, 'test', 1;
END;
```
