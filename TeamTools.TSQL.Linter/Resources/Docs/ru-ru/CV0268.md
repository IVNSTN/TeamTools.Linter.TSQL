
# Точка с запятой в конце выражения

|||
|-|-|
| Id | **CV0268**
| Мнемо | SEMICOLON_AFTER_STATEMENT
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [SemicolonAfterEachStatementRule.cs](../../../Rules/CodingConvention/SemicolonAfterEachStatementRule.cs)

## Причина

Правило срабатывает, если после выражения не поставлена точка с запятой `;`.
<p id="descr">Поставьте точку с запятой ';' в конце выражения.</p>

## Примеры

Некорректно

```sql
IF @a = @b
BEGIN
    SELECT @a;
END -- 1

WHILE @a < @b
BEGIN
    SET @a += 1;
END -- 2
```

Корректно

```sql
IF @a = @b
BEGIN
    SELECT @a;
END;

WHILE @a < @b
BEGIN
    SET @a += 1;
END;
```
