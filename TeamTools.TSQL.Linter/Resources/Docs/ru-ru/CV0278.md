

# Алиас столбца без ключевого слова 'AS'

|||
|-|-|
| Id | **CV0278**
| Мнемо | ALIAS_WITH_AS
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [AliasWithAsSyntaxRule.cs](../../../Rules/CodingConvention/AliasWithAsSyntaxRule.cs)

## Причина

Правило срабатывает, если алиас столбца указан без ключевого слова `AS`.
<p id="descr">Укажите алиас столбца с использованием ключевого слова 'AS'.</p>

## Примеры

Некорректно

```sql
SELECT
    col_alias1 = a * b
    , c col_alias2
```

Корректно

```sql
SELECT
    a * b AS col_alias1
    , c AS col_alias2
```
