

# Пользовательские функции в блоке DECLARE

|||
|-|-|
| Id | **CV0279**
| Мнемо | UDF_IN_DECLARE
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [UserDefinedFunctionInDeclareRule.cs](../../../Rules/CodingConvention/UserDefinedFunctionInDeclareRule.cs)

## Причина

Правило срабатывает, если в блоке DECLARE вызываются пользовательские функции.
<p id="descr">Вынесете вызов в SET или SELECT после блока DECLARE.</p>

## Примеры

Некорректно

```sql
DECLARE
    @var INT = (SELECT dbo.max_var(1))
    , @dt DATETIME = dbo.f_today()
    , @int INT = my.f_last_update()
```

Корректно

```sql
DECLARE
    @var INT
    , @dt DATETIME
    , @int INT

SELECT
    @var = dbo.max_var(1)
    , @dt = dbo.f_today()
    , @int = my.f_last_update()
```
