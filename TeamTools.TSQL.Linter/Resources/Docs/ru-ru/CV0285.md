

# Параметры процедуры заключены в скобки

|||
|-|-|
| Id | **CV0285**
| Мнемо | PROC_PARAMS_PARENTHESIS
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [ProcedureParamsParenthesisRule.cs](../../../Rules/CodingConvention/ProcedureParamsParenthesisRule.cs)

## Причина

Правило срабатывает, если параметры хранимой процедуры заключены в скобки.
<p id="descr">Удалите скобки.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE foo(@dt DATETIME2(7))
AS
BEGIN
    SELECT 1;
END;
```

Корректно

```sql
CREATE PROCEDURE foo
    @dt DATETIME2(7)
AS
BEGIN
    SELECT 1;
END;
```
