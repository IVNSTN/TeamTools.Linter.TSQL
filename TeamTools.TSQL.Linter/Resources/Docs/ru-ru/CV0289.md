

# ODBC синтаксис запрещён

|||
|-|-|
| Id | **CV0289**
| Мнемо | ODBC_SYNTAX
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [OdbcSyntaxRule.cs](../../../Rules/CodingConvention/OdbcSyntaxRule.cs)

## Причина

Правило срабатывает, если используется ODBC синтаксис.
<p id="descr">Замените ODBC синтаксис на TSQL.</p>

## Примеры

Некорректно

```sql
SELECT {fn ABS(EmpNo)}, {fn CONVERT(EmpName,SQL_SMALLINT)}
FROM Employees
WHERE dt > {d '19900101'}
```

Корректно

```sql
SELECT ABS(EmpNo), CONVERT(SMALLINT, EmpName)
FROM Employees
WHERE dt > '19900101'
```
