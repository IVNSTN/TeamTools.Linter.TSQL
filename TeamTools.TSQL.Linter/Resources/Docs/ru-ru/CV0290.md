

# Нарушена орфография ключевого слова

|||
|-|-|
| Id | **CV0290**
| Мнемо | KEYWORD_SHORTHAND
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [KeywordFullOrShortVersionRule.cs](../../../Rules/CodingConvention/KeywordFullOrShortVersionRule.cs)

## Причина

Правило срабатывает, если нарушена орфография ключевого слова.
<p id="descr">Исправьте ключевое слово согласно подсказке линтера.</p>

## Примеры

Некорректно

```sql
SELECT
    DATEPART(YY, @dt),
    DATENAME(M, @dt);

SET TRAN ISOLATION LEVEL REPEATABLE READ;

EXECUTE dbo.foo;
```

Корректно

```sql
SELECT
    DATEPART(YEAR, @dt),
    DATENAME(MONTH, @dt);

SET TRANSACTION ISOLATION LEVEL REPEATABLE READ;

EXEC dbo.foo;
```
