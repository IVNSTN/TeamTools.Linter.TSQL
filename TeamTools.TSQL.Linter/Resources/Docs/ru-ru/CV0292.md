

# Курсор объявлен как переменная

|||
|-|-|
| Id | **CV0292**
| Мнемо | CURSOR_NOT_INLINE
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md), [Курсоры](./Группа_Курсоры.md)
| Исходный код | [CursorDeclareInlineRule.cs](../../../Rules/CodingConvention/CursorDeclareInlineRule.cs)

## Причина

Правило срабатывает, если курсор объявлен как переменная.
<p id="descr">Объявите курсор с использованием inline синтаксиса.</p>

## Примеры

Некорректно

```sql
DECLARE
    @foo INT,
    @orders CURSOR

SET @orders = CURSOR FAST_FORWARD FOR
SELECT id
FROM dbo.orders
```

Корректно

```sql
DECLARE
    @foo INT

DECLARE orders CURSOR FAST_FORWARD FOR
SELECT id
FROM dbo.orders
```
