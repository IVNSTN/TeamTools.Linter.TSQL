

# Ожидается целочисленный литерал '1'

|||
|-|-|
| Id | **CV0294**
| Мнемо | FIXED_TRIVIAL_SCALAR
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [FixedTrivialScalarExpressionRule.cs](../../../Rules/CodingConvention/FixedTrivialScalarExpressionRule.cs)

## Причина

Правило срабатывает, если подсчёт строк не использует COUNT(1), или EXISTS не использует SELECT 1.
<p id="descr">Используйте целочисленный литерал '1' для COUNT, или EXISTS.</p>

## Примеры

Некорректно

```sql
SELECT COUNT(*) FROM foo;
SELECT 1 WHERE EXISTS(SELECT a FROM dbo.bar);
```

Корректно

```sql
SELECT COUNT(1) FROM foo;
SELECT 1 WHERE EXISTS(SELECT 1 FROM dbo.bar);
```
