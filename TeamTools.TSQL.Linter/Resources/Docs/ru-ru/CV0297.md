

# Ожидается оговорённое имя алиаса

|||
|-|-|
| Id | **CV0297**
| Мнемо | FIXED_MERGE_ALIASES
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [FixedMergeParticipantsAliasesRule.cs](../../../Rules/CodingConvention/FixedMergeParticipantsAliasesRule.cs)

## Причина

Правило срабатывает, если в MERGE для целевой таблицы не используется алиас 'trg', или для источника не используется алиас 'src'.
<p id="descr">Исправьте имена алиасов.</p>

## Примеры

Некорректно

```sql
MERGE dbo.foo AS dest
USING (SELECT a, b, c FROM dbo.bar AS br) AS source
ON dest.id = source.id
WHEN MATCHED THEN
    DELETE;
```

Корректно

```sql
MERGE dbo.foo AS trg
USING (SELECT a, b, c FROM dbo.bar AS br) AS src
ON trg.id = src.id
WHEN MATCHED THEN
    DELETE;
```
