

# Избыточные скобки вокруг отдельного выражения

|||
|-|-|
| Id | **CV0722**
| Мнемо | STATEMENT_IN_PARENTHESIS
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [StandaloneStatementParenthesisRule.cs](../../../Rules/CodingConvention/StandaloneStatementParenthesisRule.cs)

## Причина

Правило срабатывает, если отдельностоящие выражения или запрос VIEW заключены в скобки.
<p id="descr">Удалите избыточные скобки.</p>

## Примеры

Некорректно

```sql
(SELECT a FROM dbo.foo)
```

Корректно

```sql
SELECT a FROM dbo.foo
```
