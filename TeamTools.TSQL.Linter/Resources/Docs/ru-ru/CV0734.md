

# Объявление переменных не сгруппировано в одну конструкцию DECLARE

|||
|-|-|
| Id | **CV0734**
| Мнемо | DECLARE_PER_VAR
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [DeclarePerVarRule.cs](../../../Rules/CodingConvention/DeclarePerVarRule.cs)

## Причина

Правило срабатывает, если объявление переменных не сгруппировано в одну конструкцию DECLARE.
<p id="descr">Сгруппируйте объявление переменных в одну конструкцию DECLARE.</p>

## Примеры

Некорректно

```sql
DECLARE @foo INT;
DECLARE @bar CHAR(1);
```

Корректно

```sql
DECLARE
    @foo INT
    , @bar CHAR(1);
```
