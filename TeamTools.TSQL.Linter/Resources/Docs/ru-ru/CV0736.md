

# Квотирование двойными кавычками

|||
|-|-|
| Id | **CV0736**
| Мнемо | DOUBLE_QUOTES
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [DoubleQuotesRule.cs](../../../Rules/CodingConvention/DoubleQuotesRule.cs)

## Причина

Правило срабатывает, если строковые литералы или идентификаторы квотированы двойными кавычками.
<p id="descr">Квотируйте строковые литералы одинарными кавычками, а идентификаторы - квадратными скобками.</p>

## Примеры

Некорректно

```sql
SELECT
    "asdf" AS "my col"
FROM tmp."foo bar"
```

Корректно

```sql
SELECT
    'asdf' AS [my col]
FROM tmp.[foo bar]
```
