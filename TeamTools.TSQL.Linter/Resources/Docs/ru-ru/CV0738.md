

# Использование NOT IS NULL вместо IS NOT NULL

|||
|-|-|
| Id | **CV0738**
| Мнемо | NOT_IS_NULL
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [NotIsNullRule.cs](../../../Rules/CodingConvention/NotIsNullRule.cs)

## Причина

Правило срабатывает, если вместо IS NOT NULL используется NOT IS NULL.
<p id="descr">Используйте IS NOT NULL вместо NOT IS NULL.</p>

## Примеры

Некорректно

```sql
SELECT 1
WHERE NOT @a IS NULL
```

Корректно

```sql
SELECT 1
WHERE @a IS NOT NULL
```
