

# Parameter nullability should be defined

|||
|-|-|
| Id | **CV0767**
| Мнемо | NATIVELY_PARAM_NULLABILITY
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [NativelyCompiledProcParamNullabilityRule.cs](../../../Rules/CodingConvention/NativelyCompiledProcParamNullabilityRule.cs)

## Причина

<p id="descr">Нативно-компилируемые модули поддерживают возможность указывать для параметров, допустимо ли NULL-значение на входе. Лучше явно указывать, NULL или NOT NULL для каждого параметра.</p>
