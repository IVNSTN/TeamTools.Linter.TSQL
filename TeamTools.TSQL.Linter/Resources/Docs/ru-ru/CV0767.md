

# Явно не указана допустимость NULL значения для параметра нативно-компилируемого модуля

|||
|-|-|
| Id | **CV0767**
| Мнемо | NATIVELY_PARAM_NULLABILITY
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md), [ИнМемори](./Группа_ИнМемори.md)
| Исходный код | [NativelyCompiledProcParamNullabilityRule.cs](../../../Rules/CodingConvention/NativelyCompiledProcParamNullabilityRule.cs)

## Причина

Правило срабатывает, если явно не указана допустимость NULL значения для параметра нативно-компилируемого модуля.
<p id="descr">Укажите допустимость NULL значения для параметра.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.foo
    @bar INT = NULL
    , @far CHAR(1)
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```

Корректно

```sql
CREATE PROCEDURE dbo.foo
    @bar INT NULL
    , @far CHAR(1) NOT NULL
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```
