

# Явно не указан порядок сортировки столбца индекса

|||
|-|-|
| Id | **CV0768**
| Мнемо | INMEMORY_INDEX_SORTING
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md), [Индексы](./Группа_Индексы.md), [ИнМемори](./Группа_ИнМемори.md)
| Исходный код | [InMemoryIndexSortingRule.cs](../../../Rules/CodingConvention/InMemoryIndexSortingRule.cs)

## Причина

Правило срабатывает, если в определении индекса для таблицы, хранимой в памяти, явно не указан порядок сортировки столбца.
<p id="descr">Укажите порядок сортировки столбца индекса.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    major_id        BIGINT          NOT NULL
    , secondary_id  INT             NOT NULL
    , INDEX ix_secondary_id NONCLUSTERED (secondary_id DESC, major_id) WITH (BUCKET_COUNT = 5000000)
)
WITH (MEMORY_OPTIMIZED = ON);
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    major_id        BIGINT          NOT NULL
    , secondary_id  INT             NOT NULL
    , INDEX ix_secondary_id NONCLUSTERED (secondary_id DESC, major_id ASC) WITH (BUCKET_COUNT = 5000000)
)
WITH (MEMORY_OPTIMIZED = ON);
```
