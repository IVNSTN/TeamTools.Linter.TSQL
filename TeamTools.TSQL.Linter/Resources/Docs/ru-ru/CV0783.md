

# Избыточный 'AS' при указании типа переменной

|||
|-|-|
| Id | **CV0783**
| Мнемо | DECLARE_AS
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [DeclareAsTypeRule.cs](../../../Rules/CodingConvention/DeclareAsTypeRule.cs)

## Причина

Правило срабатывает, если тип переменной указывается через 'AS'.
<p id="descr">Удалите избыточный 'AS'.</p>

## Примеры

Некорректно

```sql
DECLARE @i AS INT
```

Корректно

```sql
DECLARE @i INT
```
