

# Вместо синонима используйте основное имя типа

|||
|-|-|
| Id | **CV0788**
| Мнемо | TYPE_SYNONYM
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [TypeSynonymRule.cs](../../../Rules/CodingConvention/TypeSynonymRule.cs)

## Причина

Правило срабатывает, если вместо основного имени типа используется синоним.
<p id="descr">Используйте основное имя вместо синонима.</p>

## Примеры

Некорректно

```sql
DECLARE @a NATIONAL CHARACTER VARYING (100)
```

Корректно

```sql
DECLARE @a NVARCHAR(100)
```
