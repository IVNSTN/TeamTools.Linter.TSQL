

# Системные типы нужно использовать без схемы

|||
|-|-|
| Id | **CV0803**
| Мнемо | SYSTYPE_HAS_SCHEMA
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [SystemTypeWithSchemaRule.cs](../../../Rules/CodingConvention/SystemTypeWithSchemaRule.cs)

## Причина

Правило срабатывает, если имя системного типа содержит схему
<p id="descr">Удалите имя схемы.</p>

## Примеры

Некорректно

```sql
DECLARE @i sys.INT
```

Корректно

```sql
DECLARE @i INT
```
