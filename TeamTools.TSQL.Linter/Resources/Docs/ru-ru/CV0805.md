

# PRINT в бизнес-логике

|||
|-|-|
| Id | **CV0805**
| Мнемо | PRINTING
| Серьёзность | ℹ Подсказка
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [PrintRule.cs](../../../Rules/CodingConvention/PrintRule.cs)

## Причина

Правило срабатывает, если в бизнес-логике используется команда PRINT.
<p id="descr">Удалите PRINT.</p>

## Примеры

Некорректно

```sql
DECLARE @i INT = 1;
SELECT @i = @i + 5;

PRINT 'i = ' + CAST(@i AS VARCHAR(11));
```

Корректно

```sql
DECLARE @i INT = 1;
SELECT @i = @i + 5;
```

## Подсказки

💡 В тестах применение PRINT допустимо.
