

# Вызов хранимой процедуры нужно предварять словом EXEC

|||
|-|-|
| Id | **CV0810**
| Мнемо | EXEC_WITHOUT_EXEC
| Серьёзность | ⚠ Предупреждение
| Категория | [СоглашенияПоКодированию](./Категория_СоглашенияПоКодированию.md)
| Исходный код | [ExecWithoutExecRule.cs](../../../Rules/CodingConvention/ExecWithoutExecRule.cs)

## Причина

Правило срабатывает, если перед вызовом хранимой процедуры не указано ключевое слово EXEC.
<p id="descr">Добавьте ключевое слово EXEC.</p>

## Примеры

Некорректно

```sql
calculation_procedure @value = 5; 
```

Корректно

```sql
EXEC calculation_procedure @value = 5;
```
