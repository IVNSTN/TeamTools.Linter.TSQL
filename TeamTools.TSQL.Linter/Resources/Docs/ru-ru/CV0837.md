

# Рассмотрите переход на XQuery

|||
|-|-|

| Серьёзность | ℹ Подсказка

## Причина

Правило срабатывает, если для парсинга XML используется sp_xml_preparedocument и OPENXML().
<p id="descr">Рассмотрите переход на XQuery.</p>

## Примеры

Некорректно

```sql
DECLARE
    @idoc INT
    , @dx VARCHAR(1000) = '<ROOT><a>111</a></ROOT>';

EXEC sp_xml_preparedocument @idoc OUTPUT, @doc;

SELECT *
FROM
    OPENXML(@idoc, '/ROOT/a', 1)
        WITH (CustomerID VARCHAR(10), ContactName VARCHAR(20));
```

Корректно

```sql
DECLARE @x XML;

SET @x = '<ROOT><a>111</a></ROOT>';

SELECT @x.query('/ROOT/a');
```

## Подсказки

💡 В большинстве случаев работа с XML через XQuery поможет создать более простой код.
