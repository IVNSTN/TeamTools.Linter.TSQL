

# Все столбцы таблицы допускают NULL

|||
|-|-|
| Id | **DD0158**
| Мнемо | TABLE_ALL_COL_NULL
| Серьёзность | ⚠ Предупреждение
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [TableHasNotNullableColumnsRule.cs](../../../Rules/DatabaseDesign/TableHasNotNullableColumnsRule.cs)

## Причина

Правило срабатывает, если все столбцы таблицы допускают NULL.
<p id="descr">Объявите хотя бы один столбец как NOT NULL.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id INT NULL,
    name VARCHAR(10) NULL
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL,
    name VARCHAR(10) NULL
)
```
