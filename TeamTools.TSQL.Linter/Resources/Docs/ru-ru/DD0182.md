

# Столбец ROWVERSION должен быть NOT NULL

|||
|-|-|
| Id | **DD0182**
| Мнемо | ROWVERSION_NOT_NULL
| Серьёзность | ⚠ Предупреждение
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [RowVersionColumnNullableRule.cs](../../../Rules/DatabaseDesign/RowVersionColumnNullableRule.cs)

## Причина

Правило срабатывает, если столбец ROWVERSION или TIMESTAMP допускает NULL.
<p id="descr">Укажите свойство NOT NULL.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    ver ROWVERSION,
    ts TIMESTAMP NULL
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    ver ROWVERSION NOT NULL,
    ts TIMESTAMP NOT NULL
)
```
