

# Столбец IDENTITY должен быть NOT NULL

|||
|-|-|
| Id | **DD0183**
| Мнемо | IDENTITY_NOT_NULL
| Серьёзность | ⚠ Предупреждение
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [IdentityColumnNullabilityRule.cs](../../../Rules/DatabaseDesign/IdentityColumnNullabilityRule.cs)

## Причина

Правило срабатывает, если столбец IDENTITY допускает NULL.
<p id="descr">Укажите свойство NOT NULL.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id INT NULL IDENTITY(1,1)
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL IDENTITY(1,1)
)
```
