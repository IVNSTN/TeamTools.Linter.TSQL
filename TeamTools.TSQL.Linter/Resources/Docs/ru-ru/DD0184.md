

# Столбец первичного ключа должен быть NOT NULL

|||
|-|-|
| Id | **DD0184**
| Мнемо | PK_COLUMN_NOT_NULL
| Серьёзность | ⚠ Предупреждение
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [PrimaryKeyColNullableRule.cs](../../../Rules/DatabaseDesign/PrimaryKeyColNullableRule.cs)

## Причина

Правило срабатывает, если столбец первичного ключа допускает NULL.
<p id="descr">Укажите свойство NOT NULL.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id INT NULL
    , CONSTRAINT PK_foo PRIMARY KEY CLUSTERED (id)
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
    , CONSTRAINT PK_foo PRIMARY KEY CLUSTERED (id)
)
```
