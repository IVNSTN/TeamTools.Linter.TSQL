
# Столбец фиксированной длины большого размера

|||
|-|-|
| Id | **DD0750**
| Мнемо | LONG_FIXED_LENGTH_COL
| Серьёзность | ℹ Подсказка
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [LongFixedLengthColRule.cs](../../../Rules/DatabaseDesign/LongFixedLengthColRule.cs)

## Причина

Правило срабатывает, если столбец фиксированной длины имеет большой размер.
<p id="descr">Используйте столбец переменной длины.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
    , description CHAR(4000) NOT NULL
);
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
    , description VARCHAR(4000) NOT NULL
);
```

## Подсказки

💡 В столбцах фиксированного размера, независимо от фактической длины строки, хранится полный указанный в определении столбца размер. Если в столбце встречается много коротких строк или размер строк значительно изменяется, то рекомендуется изменить тип с CHAR на VARCHAR.
