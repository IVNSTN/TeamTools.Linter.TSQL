

# Триггер для in-memory таблицы

|||
|-|-|

| Серьёзность | ⚠ Предупреждение

| Исходный код | [NativelyCompiledTriggerRule.cs](../../../Rules/DatabaseDesign/NativelyCompiledTriggerRule.cs)

## Причина

Правило срабатывает, если триггер создаётся для in-memory таблицы.
<p id="descr">Удалите триггер.</p>

## Примеры

Некорректно

```sql
CREATE TRIGGER dml ON dbo.foo
WITH EXECUTE AS OWNER, NATIVE_COMPILATION, SCHEMABINDING
AFTER INSERT
AS
RETURN 1;
```

## Подсказки

💡 In-memory таблицы используются для оперативной обработки и доступа к данным, однако триггеры могут оказывать значительное влияние на скорость обработки данных и привести к деградации производительности.
