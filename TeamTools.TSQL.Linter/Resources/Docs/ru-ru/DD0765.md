

# Triggers are not welcome for in-memory tables

|||
|-|-|
| Id | **DD0765**
| Мнемо | NATIVELY_COMPILED_TRIGGER
| Серьёзность | ⚠ Предупреждение
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [NativelyCompiledTriggerRule.cs](../../../Rules/DatabaseDesign/NativelyCompiledTriggerRule.cs)

## Причина

<p id="descr">Применение триггеров на хранимых в памяти таблицах выглядит довольно странно, неоднозначно. Лучше оставить триггеры для обычным базам данных и обычным таблицам, а в IN-MEMORY их избегать.</p>
