

# SPARSE столбец размерности MAX

|||
|-|-|
| Id | **DD0773**
| Мнемо | MAX_COL_SPARSE
| Серьёзность | ⚠ Предупреждение
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [MaxColumnSparseRule.cs](../../../Rules/DatabaseDesign/MaxColumnSparseRule.cs)

## Причина

Правило срабатывает, если опция SPARSE указана для столбца с размеростью MAX.
<p id="descr">Удалите опцию SPARSE.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    data          VARBINARY(MAX) SPARSE NULL
    , description VARCHAR(MAX)   SPARSE NULL
);
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    data          VARBINARY(MAX) NULL
    , description VARCHAR(MAX)   NULL
);
```
