

# SPARSE столбцы не должны использоваться

|||
|-|-|
| Id | **DD0777**
| Мнемо | SPARSE_COL_USED
| Серьёзность | ℹ Подсказка
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [SparseColumnInGeneralRule.cs](../../../Rules/DatabaseDesign/SparseColumnInGeneralRule.cs)

## Причина

Правило срабатывает, если указана опция SPARSE.
<p id="descr">Удалите опцию SPARSE.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id     BIGINT   NOT NULL
    , name CHAR(20) SPARSE NULL
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id     BIGINT   NOT NULL
    , name CHAR(20) NULL
)
```
