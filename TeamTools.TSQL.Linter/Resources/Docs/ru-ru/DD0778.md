

# SPARSE столбцы не преобладают в таблице

|||
|-|-|
| Id | **DD0778**
| Мнемо | SPARSE_AND_NOT_SPARSE
| Серьёзность | ℹ Подсказка
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [SparseColumnsAsSecondaryRule.cs](../../../Rules/DatabaseDesign/SparseColumnsAsSecondaryRule.cs)

## Причина

Правило срабатывает, если число SPARSE столбцов меньше обычных.
<p id="descr">Удалите опции SPARSE или вынесите SPARSE столбцы в отдельную таблицу.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id           INT  NOT NULL
    , date       DATE NOT NULL DEFAULT GETDATE()
    , code       INT  NULL
    , is_enabled BIT  SPARSE NULL
);
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id           INT  NOT NULL
    , date       DATE NOT NULL DEFAULT GETDATE()
    , code       INT  NULL
    , is_enabled BIT  NULL
);
```
