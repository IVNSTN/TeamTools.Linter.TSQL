
# Строка 'NULL' указана в качестве DEFAULT

|||
|-|-|
| Id | **DD0799**
| Мнемо | DEFAULT_NULL_WORD
| Серьёзность | ⚠ Предупреждение
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [DefaultNullAsStringRule.cs](../../../Rules/DatabaseDesign/DefaultNullAsStringRule.cs)

## Причина

Правило срабатывает, если в качестве DEFAULT для строкового столбца задано значение 'NULL'.
<p id="descr">Используйте пустую строку или NULL.</p>

## Примеры

Некорректно

```sql
CREATE TABLE #foo
(
    bar     VARCHAR(10) NOT NULL DEFAULT '<NULL>'
    , far   NCHAR CONSTRAINT DF_foo_far DEFAULT ('  null  ')
)
```

Корректно

```sql
CREATE TABLE #foo
(
    bar     VARCHAR(10) NOT NULL DEFAULT ''
    , far   NCHAR       NULL
)
```
