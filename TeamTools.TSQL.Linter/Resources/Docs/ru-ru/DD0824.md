
# Короткий строковый столбец допускает юникод

|||
|-|-|
| Id | **DD0824**
| Мнемо | SMALL_SIZE_UNICODE
| Серьёзность | ℹ Подсказка
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [ForeignKeyRecursionRule.cs](../../../Rules/DatabaseDesign/SmallSizeColumnUnicodeRule.cs)

## Причина

Правило срабатывает, если обнаружен столбец с типом данных `NVARCHAR` или `NCHAR` длиной до 12 символов.
<p id="descr">Увеличьте длину столбца или откажитесь от поддержки юникода.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id        INT         NOT NULL
    , title   NVARCHAR(8) NOT NULL
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id        INT            NOT NULL
    , title   NVARCHAR(100)  NOT NULL
)

-- or
CREATE TABLE dbo.foo
(
    id        INT         NOT NULL
    , title   VARCHAR(8)  NOT NULL
)
```

## Подсказки

💡 Строковый столбец с поддержкой юникода и при этом способный хранить только очень короткие строки - это подозрительно. Такие столбцы обычно нужны для хранения мнемокодов, аббревиатур и enum-подобных значений, которым юникод не нужен.
