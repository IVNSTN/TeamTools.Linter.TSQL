
# Таблица из одного столбца

|||
|-|-|

## Причина

Правило срабатывает, если обнаружено объявление таблицы состоящей из одного столбца.
<p id="descr">Добавьте дополнительные столбцы.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id INT NOT NULL
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id      INT          NOT NULL
    , title VARCHAR(255) NOT NULL
);
```

## Подсказки

💡 Таблица, состоящая из одного единственного столбца, - очень спорный паттерн проектирования баз данных. Подобное может встречаться в табличных переменных или табличных типах. Почти в любых ситуациях, даже для хранения единственного натурального значения, могут быть полезными: суррогатный ключ, дата вставки, дата последней модификации, признак вкл/выкл, ROWVERSION. Без таких столбцов работать с таблицей становится довольно непросто, хоть она и предельно проста.
