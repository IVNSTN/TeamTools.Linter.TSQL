
# Значение вычисляемого столбца - константа

|||
|-|-|
| Id | **DD0827**
| Мнемо | COMPUTED_COL_CONST
| Серьёзность | ℹ Подсказка
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [ForeignKeyRecursionRule.cs](../../../Rules/DatabaseDesign/ComputedColumnIsConstantRule.cs)

## Причина

Правило срабатывает, если вычисляемый столбец определен литералом (его значение ни при каких обстоятельствах не меняется).
<p id="descr">Замените константу выражением или удалите столбец.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    num     DECIMAL NOT NULL
    , calc  AS 1
);
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    num     DECIMAL NOT NULL
    , calc  AS num + 1
);
```

## Подсказки

💡 Возможно, столбец нужен как промежуточный этап, для наглядности или с целью обеспечения обратной совместимости, но в целом выглядит подозрительно. Вероятно, столбец просто не нужен и может быть удален.
