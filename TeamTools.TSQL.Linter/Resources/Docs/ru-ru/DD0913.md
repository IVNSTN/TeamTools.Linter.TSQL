

# Использование NOCHECK CONSTRAINT

|||
|-|-|
| Id | **DD0913**
| Мнемо | NOCHECK_CONSTRAINT
| Серьёзность | ⚠ Предупреждение
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [NocheckConstraintRule.cs](../../../Rules/DatabaseDesign/NocheckConstraintRule.cs)

## Причина

Правило срабатывает, если в коде используется NOCHECK CONSTRAINT.
<p id="descr">Удалите NOCHECK CONSTRAINT или само ограничение.</p>

## Примеры

Некорректно

```sql
ALTER TABLE dbo.foo
NOCHECK CONSTRAINT CK_code_type;
```
