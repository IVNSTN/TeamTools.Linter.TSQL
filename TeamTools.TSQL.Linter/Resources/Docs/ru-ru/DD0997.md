

# Тип данных не рекомендован для использования в составе ключа

|||
|-|-|
| Id | **DD0997**
| Мнемо | BAD_TYPE_FOR_KEY
| Серьёзность | ℹ Подсказка
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [BadTypeForKeyRule.cs](../../../Rules/DatabaseDesign/BadTypeForKeyRule.cs)

## Причина

Правило срабатывает, если в состав ключа входят столбцы строковых типов большой размерности или числовых типов с десятичной частью.
<p id="descr">Используйте столбцы иных типов в составе ключа.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id        VARCHAR(MAX) PRIMARY KEY 
    , type_id FLOAT FOREIGN KEY REFERENCES dbo.foo_types(type_id)
)
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id        VARCHAR(20) PRIMARY KEY 
    , type_id INT FOREIGN KEY REFERENCES dbo.foo_types(type_id)
)
```

## Подсказки

💡 Правило игнорирует COLUMNSTORE индексы.
