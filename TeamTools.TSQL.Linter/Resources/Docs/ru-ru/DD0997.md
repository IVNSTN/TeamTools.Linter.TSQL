

# Type is not recommended for keys

|||
|-|-|
| Id | **DD0997**
| Мнемо | BAD_TYPE_FOR_KEY
| Серьёзность | ℹ Подсказка
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md)
| Исходный код | [BadTypeForKeyRule.cs](../../../Rules/DatabaseDesign/BadTypeForKeyRule.cs)

## Причина

<p id="descr">Не все типы данных одинаково удачны и уместны для использования в качестве ключа. Большие строковые типы, числовые типы с десятичной частью - это скорее данные, чем ключи.</p>
