

# Раздутый кластерный индекс

|||
|-|-|
| Id | **DD0999**
| Мнемо | BLOATED_CLUSTERED_IDX
| Серьёзность | ℹ Подсказка
| Категория | [ДизайнБазДанных](./Категория_ДизайнБазДанных.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [BloatedClusteredIndexRule.cs](../../../Rules/DatabaseDesign/BloatedClusteredIndexRule.cs)

## Причина

Правило срабатывает, если в кластерный индекс включаются столбцы большой размерности или большое количество столбцов.
<p id="descr">Уменьшите размер индекса.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    id        BIGINT NOT NULL
    , code    VARCHAR(MAX)  NOT NULL
    , comment VARCHAR(1024) NOT NULL
    , PRIMARY KEY CLUSTERED (id, code)
);
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    id        BIGINT NOT NULL
    , code    VARCHAR(MAX)  NOT NULL
    , comment VARCHAR(1024) NOT NULL
    , PRIMARY KEY CLUSTERED (id)
);
```
