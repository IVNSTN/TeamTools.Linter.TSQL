
# Устаревший синтаксис подсказок оптимизатору

|||
|-|-|
| Id | **DE0273**
| Мнемо | HINT_SYNTAX
| Серьёзность | ⚠ Предупреждение
| Категория | [Устаревание](./Категория_Устаревание.md)
| Исходный код | [DeprecatedHintSyntaxRule.cs](../../../Rules/Deprecation/DeprecatedHintSyntaxRule.cs)

## Причина

Правило срабатывает, если подсказки оптимизатору заданы без указания ключевого слова WITH или они перечислены через пробел.
<p id="descr">Используйте WITH с перечислением подсказок через запятую.</p>

## Примеры

Некорректно

```sql
SELECT 1 FROM dbo.bar (TABLOCKX)

SELECT 1 FROM dbo.foo WITH (HOLDLOCK INDEX(my_idx))
```

Корректно

```sql
SELECT 1 FROM dbo.bar WITH (TABLOCKX)

SELECT 1 FROM dbo.foo WITH (HOLDLOCK, INDEX(my_idx))
```

## Подсказки

💡 Обязательно должно быть ключевое слово WITH, подсказки оптимизатору обязательно должны быть разделены запятой.
