
# Нумерованые хранимые процедуры - это устаревший стиль именования

|||
|-|-|
| Id | **DE0274**
| Мнемо | NUMBERED_SP
| Серьёзность | ⚠ Предупреждение
| Категория | [Устаревание](./Категория_Устаревание.md)
| Исходный код | [NumberedSpRule.cs](../../../Rules/Deprecation/NumberedSpRule.cs)

## Причина

Правило срабатывает, если обнаружено создание нумерованной хранимой процедуры.
<p id="descr">Переименуйте процедуру.</p>

## Примеры

Некорректно

```sql
CREATE PROC dbo.foo;1
AS
BEGIN
    RETURN 1;
END
```

Корректно

```sql
CREATE PROC dbo.foo
AS
BEGIN
    RETURN 1;
END
```
