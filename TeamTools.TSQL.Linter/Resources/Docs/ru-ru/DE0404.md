

# Устаревшая команда DBCC

|||
|-|-|
| Id | **DE0404**
| Мнемо | DEPRECATED_DBCC_COMMAND
| Серьёзность | ⚠ Предупреждение
| Категория | [Устаревание](./Категория_Устаревание.md)
| Исходный код | [DeprecatedDbccCommandRule.cs](../../../Rules/Deprecation/DeprecatedDbccCommandRule.cs)

## Причина

<p id="descr">Замените устаревшие команды DBCC.</p>

## Примеры

Некорректно

```sql
DBCC DBREINDEX
DBCC INDEXDEFRAG
DBCC SHOWCONTIG
PINTABLE, UNPINTABLE
```

Корректно

```sql
ALTER INDEX ... REBUILD
ALTER INDEX ... REORGANIZE
используйте sys.dm_db_index_physical_stats
не используйте PINTABLE, UNPINTABLE
```
