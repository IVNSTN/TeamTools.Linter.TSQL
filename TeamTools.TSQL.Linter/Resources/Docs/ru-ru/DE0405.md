

# Установка устаревшей опции

|||
|-|-|
| Id | **DE0405**
| Мнемо | DEPRECATED_OPTIONS
| Серьёзность | ⚠ Предупреждение
| Категория | [Устаревание](./Категория_Устаревание.md)
| Исходный код | [DeprecatedOptionRule.cs](../../../Rules/Deprecation/DeprecatedOptionRule.cs)

## Причина

Перечень устаревших фич для 
[2019](https://docs.microsoft.com/en-us/sql/database-engine/deprecated-database-engine-features-in-sql-server-version-15?view=sql-server-ver15)
[2017](https://docs.microsoft.com/en-us/sql/database-engine/deprecated-database-engine-features-in-sql-server-2017?view=sql-server-ver15)
[2016](https://docs.microsoft.com/en-us/sql/database-engine/deprecated-database-engine-features-in-sql-server-2016?view=sql-server-ver15)

<p id="descr">Не используйте устаревшие (не рекомендованные) опции.</p>

## Примеры

Некорректно

```sql
значение OFF для опций ANSI_NULLS, ANSI_PADDING, CONCAT_NULL_YIELDS_NULL
SET REMOTE_PROC_TRANSACTIONS
ANSI_NULL_DFLT_ON, ANSI_NULL_DFLT_OFF, ANSI_DEFAULTS, FORCE_PLAN, TEXTSIZE
FMTONLY, SHOWPLAN_ALL, SHOWPLAN_TEXT, SHOWPLAN_XML 
```

Корректно

```sql
не используйте значение OFF для опций ANSI_NULLS, ANSI_PADDING, CONCAT_NULL_YIELDS_NULL
настраивайте Linked servers
не используйте в коде опции ANSI_NULL_DFLT_ON, ANSI_NULL_DFLT_OFF, ANSI_DEFAULTS, FORCE_PLAN, TEXTSIZE
не используйте в продакшене опции FMTONLY, SHOWPLAN_ALL, SHOWPLAN_TEXT, SHOWPLAN_XML 
```

## Подсказки

💡 Некоторые опции могут быть использованы с конкретными значениями.
