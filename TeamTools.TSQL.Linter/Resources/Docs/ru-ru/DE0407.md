

# Устаревший синтаксис TOP без скобок

|||
|-|-|
| Id | **DE0407**
| Мнемо | DEPRECATED_TOP_SYNTAX
| Серьёзность | ⚠ Предупреждение
| Категория | [Устаревание](./Категория_Устаревание.md)
| Исходный код | [DeprecatedTopSyntaxRule.cs](../../../Rules/Deprecation/DeprecatedTopSyntaxRule.cs)

## Причина

<p id="descr">Заключите выражение TOP в круглые скобки.</p>

## Примеры

Некорректно

```sql
SELECT TOP 100 ID FROM dbo.cr_3203 ORDER BY ID
```

Корректно

```sql
SELECT TOP(100) ID FROM dbo.cr_3203 ORDER BY ID
```

## Подсказки

💡 Синтаксис без круглых скобок существует только для обратной совместимости.
