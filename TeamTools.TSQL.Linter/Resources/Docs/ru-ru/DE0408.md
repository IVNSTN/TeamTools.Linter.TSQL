
# Ссылка на устаревшее системное представление

|||
|-|-|
| Id | **DE0408**
| Мнемо | DEPRECATED_SYS_VIEW
| Серьёзность | ⚠ Предупреждение
| Категория | [Устаревание](./Категория_Устаревание.md)
| Исходный код | [DeprecatedSystemViewRule.cs](../../../Rules/Deprecation/DeprecatedSystemViewRule.cs)

## Причина

Правило срабатывает, если обнаружено использование устаревших системных представлений.
<p id="descr">Используйте новые системные представления.</p>

## Примеры

Некорректно

```sql
SELECT name FROM syscolumns
```

Корректно

```sql
SELECT name FROM sys.columns
```
