
# Устаревший синтаксис RAISERROR

|||
|:-|:-|
| Id | **DE0409**
| Mnemo | DEPRECATED_RAISERROR_SYNTAX
| Severity | ⚠ Предупреждение
| Category | [Устаревание](./Категория_Устаревание.md)
| Source code | [DeprecatedRaisErrorSyntaxRule.cs](../../../Rules/Deprecation/DeprecatedRaisErrorSyntaxRule.cs)

## Причина

Правило срабатывает, если в коде обнаружено использование устаревшего синтаксиса RAISERROR.
<p id="descr">Заключите параметры RAISERROR в круглые скобки.</p>

## Примеры

Некорректно

```sql
RAISERROR 123123 'Сообщение об ошибке'
```

Корректно

```sql
RAISERROR(123123, 16, 1)

RAISERROR('Сообщение об ошибке', 16, 2)
```

## Подсказки

💡 Синтаксис RAISERROR без скобок устарел, его невозможно применять в более новых версиях SQL SERVER с установленным уровнем совместимости свыше 100.
