
# Устаревший синтаксис указания свойств индекса

|||
|-|-|
| Id | **DE0741**
| Мнемо | INDEX_OPTION_SYNTAX_DEPRECATED
| Серьёзность | ⚠ Предупреждение
| Категория | [Устаревание](./Категория_Устаревание.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [DeprecatedIndexOptionSyntaxRule.cs](../../../Rules/Deprecation/DeprecatedIndexOptionSyntaxRule.cs)

## Причина

Правило срабатывает, если обнаружено использование устаревшего синтаксиса указания свойств индекса.
<p id="descr">Используйте круглые скобки и знак равно.</p>

## Примеры

Некорректно

```sql
WITH DROP_EXISTING
```

Корректно

```sql
WITH (DROP_EXISTING = ON)
```
