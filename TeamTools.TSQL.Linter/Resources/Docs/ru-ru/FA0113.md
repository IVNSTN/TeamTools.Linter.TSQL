

# Количество полей в источнике не совпадает с количеством полей в получателе

|||
|-|-|
| Id | **FA0113**
| Мнемо | INSERT_COL_COUNT
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [InsertColumnCountRule.cs](../../../Rules/Failure/InsertColumnCountRule.cs)

## Причина

<p id="descr">Количество колонок в источнике для вставки должно совпадать с количеством колонок в получателе. Проверяются как INSERT SELECT, так и OUTPUT INTO. Билд проекта такое не выявляет, без линтинга всплыть может только на компиляции SQL.</p>
