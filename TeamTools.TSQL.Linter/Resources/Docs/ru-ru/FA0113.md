
# Количество столбцов в источнике не совпадает с количеством столбцов в получателе

|||
|-|-|
| Id | **FA0113**
| Мнемо | INSERT_COL_COUNT
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [InsertColumnCountRule.cs](../../../Rules/Failure/InsertColumnCountRule.cs)

## Причина

Правило срабатывает, если для операции  `MERGE`, `INSERT SELECT` или `OUTPUT INTO`, количество столбцов в источнике не совпадает с количеством столбцов в получателе.
<p id="descr">Выровняйте количество столбцов в источнике и получателе.</p>

## Примеры

Некорректно

```sql
MERGE dbo.foo AS dst
USING dbo.bar AS src
ON src.id = dst.id
WHEN NOT MATCHED BY TARGET THEN
    INSERT (aa, bb, cc)
    VALUES (a, b);
```

Корректно

```sql
MERGE dbo.foo AS dst
USING dbo.bar AS src
ON src.id = dst.id
WHEN NOT MATCHED BY TARGET THEN
    INSERT (aa, bb, cc)
    VALUES (a, b, c);
```
