
# Недопустимое использование RETURN со значением

|||
|-|-|
| Id | **FA0119**
| Мнемо | RETURN_VALUE_ILLEGAL
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [ReturnValueIllegalRule.cs](../../../Rules/Failure/ReturnValueIllegalRule.cs)

## Причина

Правило срабатывает, если в коде используется `RETURN` со значением, но для создаваемого типа объектов это недопустимо (триггер или табличная функция).
<p id="descr">Удалите возвращаемое через RETURN значение.</p>

## Примеры

Некорректно

```sql
CREATE TRIGGER dbo.bar
ON dbo.zar
AFTER DELETE
AS
BEGIN
    RETURN 1;
END;
```

Корректно

```sql
CREATE TRIGGER dbo.bar
ON dbo.zar
AFTER DELETE
AS
BEGIN
    RETURN;
END;
```
