

# Недопустимое использование RETURN со значением

|||
|-|-|
| Id | **FA0119**
| Мнемо | RETURN_VALUE_ILLEGAL
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [ReturnValueIllegalRule.cs](../../../Rules/Failure/ReturnValueIllegalRule.cs)

## Причина

<p id="descr">Недопустимое использование RETURN со значением. Бывает что при ретурне из, например, триггера, машинально добавляют и 0 или 1 в качестве кода возврата, что проглатывается билдом, однако является некорректным и сломается при первом же вызове.</p>
