
# Неизвестный идентификатор

|||
|-|-|
| Id | **FA0127**
| Мнемо | UNKNOWN_IDENTIFIER
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [UnknownIdentifierRule.cs](../../../Rules/Failure/UnknownIdentifierRule.cs)

## Причина

Правило срабатывает, если в коде обнаружено использование неизвестного идентификатора, который не может быть сопоставлен ни с каким объектом (ни с переменной, ни со столбцом таблицы, ни с чем).
<p id="descr">Добавьте @ к имени переменной.</p>

## Примеры

Некорректно

```sql
IF my_value > 0
    RETURN;
```

Корректно

```sql
IF @my_value > 0
    RETURN;
```
