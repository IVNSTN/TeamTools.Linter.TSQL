
# SET \<option> недопустима в текущем контексте

|||
|-|-|
| Id | **FA0131**
| Мнемо | SET_OPTION_ILLEGAL
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [SetOptionInFunctionRule.cs](../../../Rules/Failure/SetOptionInFunctionRule.cs)

## Причина

Правило срабатывает, если в скрипте создания функции используется недопустимая опция.
<p id="descr">Удалите недопустимую опцию.</p>

## Примеры

Некорректно

```sql
CREATE FUNCTION dbo.foo (@id INT)
RETURNS INT
AS
BEGIN
    SET NOCOUNT ON;

    RETURN 1;
END;
GO
```

Корректно

```sql
CREATE FUNCTION dbo.foo (@id INT)
RETURNS INT
AS
BEGIN
    RETURN 1;
END;
GO
```
