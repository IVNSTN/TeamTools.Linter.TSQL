

# Число колонок различается в подзапросах UNION/EXCEPT/INTERSECT

|||
|-|-|
| Id | **FA0132**
| Мнемо | UNION_COL_COUNT
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [UnionColumnCountRule.cs](../../../Rules/Failure/UnionColumnCountRule.cs)

## Причина

<p id="descr">Количество колонок должно быть одинаковым во всех подзапросах выражений UNION, EXCEPT, INTERSECT.
Если для какой-то из колонок подзапроса нет данных - колонка со значением NULL все равно должна быть явно объявлена в выборке.</p>
