
# Разное число столбцов в подзапросах UNION/EXCEPT/INTERSECT

|||
|-|-|
| Id | **FA0132**
| Мнемо | UNION_COL_COUNT
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [UnionColumnCountRule.cs](../../../Rules/Failure/UnionColumnCountRule.cs)

## Причина

Правило срабатывает, если в подзапросах `UNION`, `EXCEPT` или `INTERSECT` отличается число столбцов.
<p id="descr">Выровняйте число возвращаемых подзапросами столбцов.</p>

## Примеры

Некорректно

```sql
SELECT a, b, c FROM dbo.table
INTERSECT
SELECT b, c, d, e FROM dbo.another_table;
```

Корректно

```sql
SELECT a, b, c, d FROM dbo.table
INTERSECT
SELECT b, c, d, e FROM dbo.another_table;
```

## Подсказки

💡 Количество столбцов должно быть одинаковым во всех подзапросах выражений `UNION`, `EXCEPT`, `INTERSECT`. Если для столбца подзапроса нет данных - столбец со значением `NULL` всё равно должен быть явно объявлен в выборке.
