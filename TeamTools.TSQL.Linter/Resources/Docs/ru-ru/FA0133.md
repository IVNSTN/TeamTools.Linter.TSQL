

# Несоответствие количества аргументов функции

|||
|-|-|
| Id | **FA0133**
| Мнемо | BUILTINS_ARG_COUNT
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [BuiltInFunctionsArgumentCountRule.cs](../../../Rules/Failure/BuiltInFunctionsArgumentCountRule.cs)

## Причина

<p id="descr">SSDT компилятор пропускает ошибочные конструкции вида ISNULL(<a href="http://a.name">a.name</a>), что не пропускает компилятор T-SQL. Нужно исправить вызов функции и передать нужное кол-во аргументов. Проверяет только простые built-in функции у которых нет вариативности в наборе параметров.</p>
