
# Строки значений содержат разное количество элементов

|||
|-|-|
| Id | **FA0199**
| Мнемо | VALUES_ELEMENT_NUMBER
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [ValuesElementCountRule.cs](../../../Rules/Failure/ValuesElementCountRule.cs)

## Причина

Правило срабатывает, если наборы значений для `VALUES` содержат разное количество элементов.
<p id="descr">Выровняйте количество элементов в наборах значений.</p>

## Примеры

Некорректно

```sql
SELECT x, y 
FROM
(
    VALUES ('a', 'b', 0)
           , ('c', 'd')
) AS v (x, y);
```

Корректно

```sql
SELECT x, y 
FROM 
(
    VALUES ('a', 'b')
           , ('c', 'd')
) AS v (x, y);
```
