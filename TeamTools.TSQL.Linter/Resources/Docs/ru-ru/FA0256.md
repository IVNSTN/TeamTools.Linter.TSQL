
# Отсутствует точка с запятой перед THROW

|||
|-|-|
| Id | **FA0256**
| Мнемо | SEMICOLON_BEFORE_THROW
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [SemicolonBeforeThrowRule.cs](../../../Rules/Failure/SemicolonBeforeThrowRule.cs)

## Причина

Правило срабатывает, если перед `THROW` отсутствует точка с запятой.
<p id="descr">Поставьте точку с запятой.</p>

## Примеры

Некорректно

```sql
if @a > @b BEGIN
    THROW 500001, 'test', 1;
END;
```

Корректно

```sql
if @a > @b BEGIN;
    THROW 500001, 'test', 1;
END;
```
