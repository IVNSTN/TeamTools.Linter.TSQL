
# Определение подзапроса содержит повторяющиеся имена столбцов

|||
|-|-|
| Id | **FA0281**
| Мнемо | UNIQUE_DERIVED_DEF_COL_NAME
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [UniqueDerivedColumnAliasRule.cs](../../../Rules/Failure/UniqueDerivedColumnAliasRule.cs)

## Причина

Правило срабатывает, если определение подзапроса или CTE содержит повторяющиеся имена столбцов.
<p id="descr">Укажите уникальные имена столбцов.</p>

## Примеры

Некорректно

```sql
WITH cte (a, b, b) AS (SELECT 1, 2, 3)
SELECT * FROM cte
```

Корректно

```sql
WITH cte (a, b, c) AS (SELECT 1, 2, 3)
SELECT * FROM cte
```
