
# Не валидный XML-литерал

|||
|-|-|
| Id | **FA0283**
| Мнемо | INVALID_XML_LITERAL
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [XmlValidSourceRule.cs](../../../Rules/Failure/XmlValidSourceRule.cs)

## Причина

Правило срабатывает, если в выражениях создания `XML SCHEMA COLLECTION` или в присвоении переменным типа XML используются литералы, которые не являются валидными XML-структурами.
<p id="descr">Исправьте текст XML-литерала.</p>

## Примеры

Некорректно

```sql
CREATE XML SCHEMA COLLECTION dbo.moex AS
N'<asdf></zzz>';
GO
```

Корректно

```sql
CREATE XML SCHEMA COLLECTION dbo.moex AS
N'<xsd:schema xmlns:xsd="http://www.w3.org/2001/XMLSchema" elementFormDefault="qualified">
</xsd:schema>';
GO
```
