

# Columnstore index cannot contain columns of this type

|||
|-|-|
| Id | **FA0703**
| Мнемо | BAD_TYPE_FOR_COLUMNSTORE
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [BadTypeForColumnstoreIndexRule.cs](../../../Rules/Failure/BadTypeForColumnstoreIndexRule.cs)

## Причина

<p id="descr">Некоторые типы данных не поддерживаются колоночными индексами. Для таблицы с такими колонками создать COLUMNSTORE индекс не получится.</p>
