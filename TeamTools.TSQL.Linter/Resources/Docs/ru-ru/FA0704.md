

# Недопустимое число аргументов

|||
|-|-|
| Id | **FA0704**
| Мнемо | INVALID_ARGUMENT_COUNT
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [InvalidFunctionArgumentCountRule.cs](../../../Rules/Failure/InvalidFunctionArgumentCountRule.cs)

## Причина

Правило срабатывает, если в функцию передаётся недопустимое число аргументов.
<p id="descr">Передайте необходимое число аргументов.</p>

## Примеры

Некорректно

```sql
SET @bar = DATEPART(DAY, @dt, 1, 'M')
```

Корректно

```sql
SET @bar = DATEPART(DAY, @dt)
```
