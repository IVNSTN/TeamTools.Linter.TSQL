

# Недопустимое значение аргумента

|||
|-|-|
| Id | **FA0705**
| Мнемо | INVALID_ARGUMENT
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [InvalidFunctionArgumentRule.cs](../../../Rules/Failure/InvalidFunctionArgumentRule.cs)

## Причина

Правило срабатывает, если в функцию передаётся недопустимое значение аргумента.
<p id="descr">Исправьте значение аргумента.</p>

## Примеры

Некорректно

```sql
SET @bar = DATEPART('DAY', @dt)
```

Корректно

```sql
SET @bar = DATEPART(DAY, @dt)
```
