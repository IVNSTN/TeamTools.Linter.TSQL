

# Деление на ноль

|||
|-|-|
| Id | **FA0708**
| Мнемо | ZERO_DIVIDE
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [ZeroDivideRule.cs](../../../Rules/Failure/ZeroDivideRule.cs)

## Причина

Правило срабатывает, если в коде встречается деление на ноль.
<p id="descr">Удалите деление на ноль.</p>

## Примеры

Некорректно

```sql
DECLARE @div INT = 0
SELECT 10 / @div
```

Корректно

```sql
DECLARE @div INT = 5
SELECT 10 / @div
```
