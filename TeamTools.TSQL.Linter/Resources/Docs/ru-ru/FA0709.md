

# Преднамеренное исключение

|||
|-|-|
| Id | **FA0709**
| Мнемо | INTENTIONAL_EXCEPTION
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [IntentionalExceptionRule.cs](../../../Rules/Failure/IntentionalExceptionRule.cs)

## Причина

Правило срабатывает, если в коде встречается преднамеренное исключение.
<p id="descr">Удалите преднамеренное исключение.</p>

## Примеры

Некорректно

```sql
SELECT 1 / 0
```

Корректно

```sql
THROW 58134, 'Divide by zero error encountered.', 1;
```
