

# Указана схема для DDL-триггера

|||
|-|-|
| Id | **FA0711**
| Мнемо | DDL_TRIGGER_NO_SCHEMA
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md), [Триггеры](./Группа_Триггеры.md)
| Исходный код | [DdlTriggerSchemaRule.cs](../../../Rules/Failure/DdlTriggerSchemaRule.cs)

## Причина

Правило срабатывает, если у DDL-триггера указана схема.
<p id="descr">Удалите имя схемы.</p>

## Примеры

Некорректно

```sql
CREATE TRIGGER ddl.audit_trigger
ON DATABASE
FOR DDL_DATABASE_LEVEL_EVENTS
AS
BEGIN
    ...
END;
```

Корректно

```sql
CREATE TRIGGER ddl_audit_trigger
ON DATABASE
FOR DDL_DATABASE_LEVEL_EVENTS
AS
BEGIN
    ...
END;
```
