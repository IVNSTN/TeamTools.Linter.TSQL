

# Incompatible index option

|||
|-|-|
| Id | **FA0742**
| Мнемо | INDEX_OPTIONS_INCOMPATIBLE
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [IncompatibleIndexOptionRule.cs](../../../Rules/Failure/IncompatibleIndexOptionRule.cs)

## Причина

<p id="descr">Некоторые опции не совместимы с другими атрибутами индекса, с другими опциями. А некоторые не имеют смысла без включения взаимосвязанных опций. Например, IGNORE_DUP_KEY может быть установлено только на уникальном, нефильтрованном индексе. А PAD_INDEX не имеет смысла включать, если не указан FILLFACTOR.</p>
