

# Несовместимые свойства индекса

|||
|-|-|
| Id | **FA0742**
| Мнемо | INDEX_OPTIONS_INCOMPATIBLE
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [IncompatibleIndexOptionRule.cs](../../../Rules/Failure/IncompatibleIndexOptionRule.cs)

## Причина

Правило срабатывает, если у индекса указаны несовместимые свойства.
<p id="descr">Удалите несовместимые свойства, или добавьте недостающие.</p>

## Примеры

Некорректно

```sql
CREATE INDEX IX
    ON dbo.foo (col)
    WHERE (col2 IS NOT NULL)
    WITH (IGNORE_DUP_KEY = ON);
```

Корректно

```sql
CREATE INDEX IX
    ON dbo.foo (col)
    WHERE (col2 IS NOT NULL);
```
