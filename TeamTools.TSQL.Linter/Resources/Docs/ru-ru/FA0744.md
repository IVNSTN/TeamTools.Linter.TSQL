

# Unable to modify a cursor not marked for update

|||
|-|-|
| Id | **FA0744**
| Мнемо | CURSOR_NOT_MODIFIABLE
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md)

## Причина

<p id="descr">Если определение курсора не подразумевает обновление записей, то DML-выражения с инструкцией WHERE CURRENT OF &lt;cursor&gt; завершатся ошибкой.</p>
