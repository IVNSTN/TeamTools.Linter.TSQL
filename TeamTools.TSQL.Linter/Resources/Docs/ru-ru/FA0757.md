

# Значение DEADLOCK_PRIORITY не принадлежит допустимому диапазону

|||
|-|-|
| Id | **FA0757**
| Мнемо | DEADLOCK_PRIORITY_OUT_OF_RANGE
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [DeadlockPriorityValueRangeRule.cs](../../../Rules/Failure/DeadlockPriorityValueRangeRule.cs)

## Причина

Правило срабатывает, если значение DEADLOCK_PRIORITY не является числом в диапазоне от -10 до 10.
<p id="descr">Укажите корректное значение DEADLOCK_PRIORITY.</p>

## Примеры

Некорректно

```sql
SET DEADLOCK_PRIORITY -100
```

Корректно

```sql
SET DEADLOCK_PRIORITY -10
```
