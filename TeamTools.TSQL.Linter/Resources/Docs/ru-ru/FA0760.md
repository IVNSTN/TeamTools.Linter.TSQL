

# Тип данных не поддерживается в natively compiled модуле

|||
|-|-|
| Id | **FA0760**
| Мнемо | NATIVELY_UNSUPPORTED_TYPE
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md), [ИнМемори](./Группа_ИнМемори.md)
| Исходный код | [NativelyUnsupportedTypeRule.cs](../../../Rules/Failure/NativelyUnsupportedTypeRule.cs)

## Причина

Правило срабатывает, если тип данных не поддерживается в natively compiled модуле.
<p id="descr">Удалите неподдерживаемый тип данных.</p>

## Примеры

Некорректно

```sql
CREATE TABLE foo.bar
(
    sys_name        HIERARCHYID  NOT NULL
    , profile_id    GEOGRAPHY    NOT NULL
    , rv            ROWVERSION   NOT NULL
    , CONSTRAINT PK_foo_bar PRIMARY KEY NONCLUSTERED HASH (sys_name) WITH (BUCKET_COUNT = 8000000)
)
WITH (MEMORY_OPTIMIZED = ON, DURABILITY = SCHEMA_AND_DATA);
```

Корректно

```sql
CREATE TABLE foo.bar
(
    sys_name        VARCHAR(20)  NOT NULL
    , profile_id    SMALLINT     NOT NULL
    , sign_datetime DATETIME2(0) NOT NULL
    , CONSTRAINT PK_foo_bar PRIMARY KEY NONCLUSTERED HASH (sys_name) WITH (BUCKET_COUNT = 8000000)
)
WITH (MEMORY_OPTIMIZED = ON, DURABILITY = SCHEMA_AND_DATA);
```
