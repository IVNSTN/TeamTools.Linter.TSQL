

# Конструкция не поддерживается в natively compiled модуле

|||
|-|-|
| Id | **FA0761**
| Мнемо | NATIVELY_UNSUPPORTED_INSTRUCTION
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md), [ИнМемори](./Группа_ИнМемори.md)
| Исходный код | [NativelyUnsupportedInstructionRule.cs](../../../Rules/Failure/NativelyUnsupportedInstructionRule.cs)

## Причина

Правило срабатывает, если конструкция не поддерживается в natively compiled модуле.
<p id="descr">Удалите неподдерживаемую языковую конструкцию.</p>

## Примеры

Некорректно

```sql
CREATE PROC #tmp
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```

Корректно

```sql
CREATE PROC tmp
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```
