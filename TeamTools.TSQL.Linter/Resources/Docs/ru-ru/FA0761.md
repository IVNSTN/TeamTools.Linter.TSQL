

# Feature is not supported in natively compiled modules

|||
|-|-|
| Id | **FA0761**
| Мнемо | NATIVELY_UNSUPPORTED_INSTRUCTION
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [NativelyUnsupportedInstructionRule.cs](../../../Rules/Failure/NativelyUnsupportedInstructionRule.cs)

## Причина

<p id="descr">Некоторые функции, конструкции T-SQL не поддерживаются в нативно-компилируемых модулях. Например, в выражениях UPDATE недопустимо указывать соединения и сам блок FROM.</p>
