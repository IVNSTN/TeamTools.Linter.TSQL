

# Option is not allowed for in-memory table

|||
|-|-|
| Id | **FA0762**
| Мнемо | INMEMORY_RESTRICTED_TABLE_OPTIONS
| Серьёзность | ⛔ Ошибка

## Причина

<p id="descr">Некоторые опции создания таблиц не поддерживаются для таблиц, хранимых в памяти. Например, кластеризованные индексы или сжатие.</p>
