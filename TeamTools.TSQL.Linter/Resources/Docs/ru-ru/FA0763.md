

# Не указан SCHEMABINDING для natively compiled модуля

|||
|-|-|
| Id | **FA0763**
| Мнемо | NATIVE_COMPILATION_SCHEMABINDING
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md), [ИнМемори](./Группа_ИнМемори.md)
| Исходный код | [NativelyCompiledNeedsSchemabindingRule.cs](../../../Rules/Failure/NativelyCompiledNeedsSchemabindingRule.cs)

## Причина

Правило срабатывает, если не указан SCHEMABINDING для natively compiled модуля.
<p id="descr">Укажите SCHEMABINDING.</p>

## Примеры

Некорректно

```sql
CREATE PROC tmp
WITH NATIVE_COMPILATION, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```

Корректно

```sql
CREATE PROC tmp
WITH NATIVE_COMPILATION, SCHEMABINDING, EXECUTE AS OWNER
AS
BEGIN ATOMIC WITH (TRANSACTION ISOLATION LEVEL = SNAPSHOT, LANGUAGE = N'English')
    RETURN 1
END
```
