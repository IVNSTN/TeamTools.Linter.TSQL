

# HASH-индекс не поддерживает сортировку

|||
|-|-|
| Id | **FA0764**
| Мнемо | HASH_INDEX_SORTING
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md), [Индексы](./Группа_Индексы.md), [ИнМемори](./Группа_ИнМемори.md)
| Исходный код | [HashIndexSortingRule.cs](../../../Rules/Failure/HashIndexSortingRule.cs)

## Причина

Правило срабатывает, если для HASH-индекса указана сортировка.
<p id="descr">Удалите сортировку.</p>

## Примеры

Некорректно

```sql
CREATE TABLE foo.bar
(
    sys_name        VARCHAR(20)  NOT NULL
    , sign_datetime DATETIME2(0) NOT NULL
    , CONSTRAINT PK_foo_bar PRIMARY KEY NONCLUSTERED HASH (sys_name ASC) WITH (BUCKET_COUNT = 8000000)
)
WITH (MEMORY_OPTIMIZED = ON, DURABILITY = SCHEMA_AND_DATA);
```

Корректно

```sql
CREATE TABLE foo.bar
(
    sys_name        VARCHAR(20)  NOT NULL
    , sign_datetime DATETIME2(0) NOT NULL
    , CONSTRAINT PK_foo_bar PRIMARY KEY NONCLUSTERED HASH (sys_name) WITH (BUCKET_COUNT = 8000000)
)
WITH (MEMORY_OPTIMIZED = ON, DURABILITY = SCHEMA_AND_DATA);
```
