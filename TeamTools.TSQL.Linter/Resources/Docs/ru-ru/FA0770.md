
# SPARSE столбцы не поддерживают указанные опции или недопустимы в данной конструкции

|||
|-|-|
| Id | **FA0770**
| Мнемо | SPARSE_UNSUPPORTED_FEATURE
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [SparseUnsupportedFeatureRule.cs](../../../Rules/Failure/SparseUnsupportedFeatureRule.cs)

## Причина

Правило срабатывает, если SPARSE столбцы не поддерживают указанные опции или недопустимы в данной конструкции.
<p id="descr">Удалите SPARSE столбцы, или неподдерживаемые опции.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    bar    TINYINT SPARSE NULL DEFAULT 0
    , id   INT     SPARSE NULL IDENTITY(1, 1)
    , far  CHAR(1) SPARSE NOT NULL
);
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    bar    TINYINT NULL DEFAULT 0
    , id   INT     NULL IDENTITY(1, 1)
    , far  CHAR(1) NOT NULL
);
```
