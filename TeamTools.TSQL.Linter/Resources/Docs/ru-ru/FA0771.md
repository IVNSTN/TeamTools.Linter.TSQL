
# Недопустимый тип данных для SPARSE столбца

|||
|-|-|
| Id | **FA0771**
| Мнемо | SPARSE_UNSUPPORTED_TYPE
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [SparseUnsupportedTypeRule.cs](../../../Rules/Failure/SparseUnsupportedTypeRule.cs)

## Причина

Правило срабатывает, если для SPARSE столбца указан недопустимый тип данных.
<p id="descr">Удалите опцию SPARSE, или измените тип данных.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.foo
(
    cr GEOGRAPHY SPARSE NULL
);
```

Корректно

```sql
CREATE TABLE dbo.foo
(
    cr INT SPARSE NULL
);
```
