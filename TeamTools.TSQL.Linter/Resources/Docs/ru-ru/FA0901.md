

# Колонка может быть изменена только один раз в одной инструкции

|||
|-|-|
| Id | **FA0901**
| Мнемо | DUP_COLUMN_MODIFIER
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [UniqueColumnModificatorDefinitionRule.cs](../../../Rules/Failure/UniqueColumnModificatorDefinitionRule.cs)

## Причина

Правило срабатывает, если в выражениях INSERT, UPDATE, OUTPUT INTO одну и ту же колонку указали больше одного раза.
<p id="descr">Удалите повторяющиеся колонки.</p>

## Примеры

Некорректно

```sql
UPDATE mgr
SET
    mgr.name = 'Ivan'
    , mgr.last_name = 'Ivanov'
    , mgr.name = 'Andrey'
FROM dbo.managers AS mgr
WHERE mgr.manager_id = @id;
```

Корректно

```sql
UPDATE mgr
SET
    mgr.name = 'Ivan'
    , mgr.last_name = 'Ivanov'
FROM dbo.managers AS mgr
WHERE mgr.manager_id = @id;
```
