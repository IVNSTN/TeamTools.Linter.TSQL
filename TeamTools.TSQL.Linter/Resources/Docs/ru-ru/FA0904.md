

# Индекс ссылается на неизвестную колонку

|||
|-|-|
| Id | **FA0904**
| Мнемо | INDEX_REFERS_UNKNOWN_COL
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [IndexReferencingUnknownColumnRule.cs](../../../Rules/Failure/IndexReferencingUnknownColumnRule.cs)

## Причина

Правило срабатывает, если в определении таблицы нет колонки, упомянутой в индексе на эту таблицу.
<p id="descr">Укажите колонки которые присутствуют в таблице.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.Example (id INT NOT NULL, value_foo INT NULL, value_bar INT NULL, INDEX ix_dbo_example_id (id, value_foo, dt_create));
```

Корректно

```sql
CREATE TABLE dbo.Example (id INT NOT NULL, value_foo INT NULL, value_bar INT NULL, dt_create DATE NULL, INDEX ix_dbo_example_id (id, value_foo, dt_create));
```
