

# Количество аргументов FORMATMESSAGE, RAISERROR не соответствует количеству подстановочных знаков

|||
|-|-|
| Id | **FA0911**
| Мнемо | FORMAT_ARG_COUNT_MISMATCH
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [FormatArgumentCountMismatchRule.cs](../../../Rules/Failure/FormatArgumentCountMismatchRule.cs)

## Причина

Правило срабатывает, если в вызовах FORMATMESSAGE, RAISERROR не совпадает количество параметров подстановки для значений и самих значений.
<p id="descr">Укажите равное количество параметров подстановки и значений.</p>

## Примеры

Некорректно

```sql
SELECT FORMATMESSAGE('Exmaple Message: value1 = %s, value2 = %s', '50');
RAISERROR(N'Exmaple Message: value1 = %s, value2 = %s', 10, 1, '10');
```

Корректно

```sql
SELECT FORMATMESSAGE('Exmaple Message: value1 = %s, value2 = %s', '50', '100');
RAISERROR(N'Exmaple Message: value1 = %s, value2 = %s', 10, 1, '100', '300');
```
