

# Недопустимое ограничение 'NOT NULL' для параметра

|||
|-|-|
| Id | **FA0918**
| Мнемо | INVALID_NOT_NULL_PARAM
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [UnexpectedNotNullParameterRule.cs](../../../Rules/Failure/UnexpectedNotNullParameterRule.cs)

## Причина

Правило срабатывает, если указанно NOT NULL ограничение для входного параметра. NOT NULL ограничение может быть только у NATIVELY COMPILED модулей или встроенных функций с табличным значением.
<p id="descr">Удалите NOT NULL ограничение для входного параметра.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE tmp.SPExample
    @id INT NOT NULL
AS
BEGIN
    SELECT @id AS id;
END;
```

Корректно

```sql
CREATE PROCEDURE tmp.SPExample
    @id INT
AS
BEGIN
    SELECT @id AS id;
END;
```
