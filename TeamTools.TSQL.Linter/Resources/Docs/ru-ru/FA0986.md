

# Недопустимый подстановочный знак в FORMATMESSAGE

|||
|-|-|
| Id | **FA0986**
| Мнемо | FORMAT_INVALID_WILDCARD
| Серьёзность | ⚠ Предупреждение
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [FormatMessageInvalidWildcardRule.cs](../../../Rules/Failure/FormatMessageInvalidWildcardRule.cs)

## Причина

Правило срабатывает, если параметр подстановки указан неверно в вызове FORMATMESSAGE или RAISERROR.
<p id="descr">Укажите правильный параметр подстановки.</p>

## Примеры

Некорректно

```sql
RAISERROR('Параметр подстановки указан неверно (string) %xs', 10, 1, 'опечатка');

SELECT FORMATMESSAGE('Параметр подстановки указан неверно (integer) %bi', 10) AS Result; 
```

Корректно

```sql
RAISERROR('Параметр подстановки указан верно (string) %s', 10, 1, 'параметр');

SELECT FORMATMESSAGE('Параметр подстановки указан верно (integer) %i', 10) AS Result; 
```

## Подсказки
    
💡 Спецификация типа: Signed integer(d or i), Unsigned octal(o), String(s), Unsigned integer(u), Unsigned hexadecimal(x or X)
