
# Повторяющееся имя столбца недопустимо в определении таблицы

|||
|-|-|
| Id | **FA0987**
| Мнемо | DUP_COL_IN_TABLE
| Серьёзность | ⛔ Ошибка
| Категория | [Сбой](./Категория_Сбой.md)
| Исходный код | [DuplicateColInTableDefinitionRule.cs](../../../Rules/Failure/DuplicateColInTableDefinitionRule.cs)

## Причина

Правило срабатывает, если в определении таблицы или табличной переменной одно и то же имя столбца использовано более одного раза.
<p id="descr">Придумайте уникальное имя столбца.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.Product
(
    ProductId     INT          IDENTITY(1, 1) NOT NULL
    , ProductName VARCHAR(100) NOT NULL
    , ProductType VARCHAR(100) NOT NULL
    , ProductName VARCHAR(100) NOT NULL
);
```

Корректно

```sql
CREATE TABLE dbo.Product
(
    ProductId     INT          IDENTITY(1, 1) NOT NULL
    , ProductName VARCHAR(100) NOT NULL
    , ProductType VARCHAR(100) NOT NULL
);
```
