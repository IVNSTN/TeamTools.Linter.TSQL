

# В начале файла не допускаются пустые строки и пробелы

|||
|-|-|
| Id | **FL0272**
| Мнемо | BOF_REDUNDANT_WHITESPACE
| Серьёзность | ⚠ Предупреждение
| Категория | [Файлы](./Категория_Файлы.md)
| Исходный код | [NewLineAtBeginOfFileRule.cs](../../../Rules/Files/NewLineAtBeginOfFileRule.cs)

## Причина

Правило срабатывает, если в начале файла найдены пустые строки, пробелы или знаки табуляции.

<p id="descr">Удалите пустые строки, пробелы и знаки табуляции.</p>

## Примеры

Некорректно

```sql

      CREATE PROCEDURE dbo.test...
```

Корректно

```sql
CREATE PROCEDURE dbo.test...
```
