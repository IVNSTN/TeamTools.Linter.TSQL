

# Избегайте квадратных скобок в имени файла

|||
|-|-|
| Id | **FL0304**
| Мнемо | FILE_NAME_SQUARE_BRACKETS
| Серьёзность | ⚠ Предупреждение
| Категория | [Файлы](./Категория_Файлы.md)
| Исходный код | [FileNameSquareBracketsRule.cs](../../../Rules/Files/FileNameSquareBracketsRule.cs)

## Причина

Правило срабатывает, если в имени файла найдены квадратные скобоки.

<p id="descr">Удалите квадратные скобки из имени файла.</p>

## Примеры

Некорректно

```sql
[dbo].[table].sql
```

Корректно

```sql
dbo.table.sql
```
