

# Имя файла не соответствует имени объекта

|||
|-|-|
| Id | **FL0305**
| Мнемо | FILE_OBJECT_NAME
| Серьёзность | ⚠ Предупреждение
| Категория | [Файлы](./Категория_Файлы.md)
| Исходный код | [FileNameMatchesObjectNameRule.cs](../../../Rules/Files/FileNameMatchesObjectNameRule.cs)

## Причина

Правило срабатывает, если имя файла не соответствует имени объекта, определённого в нём.

<p id="descr">Приведите имя файла и имя объекта в соответствие.</p>

## Примеры

Некорректно

содержимое dbo.**table**.sql:

```sql
CREATE TABLE dbo.person
...
```

Корректно

содержимое dbo.**person**.sql:

```sql
CREATE TABLE dbo.person
...
```
