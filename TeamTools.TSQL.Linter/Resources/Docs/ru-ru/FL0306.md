

# В конце файла ожидается ровно одна пустая строка

|||
|-|-|
| Id | **FL0306**
| Мнемо | EOF_NEWLINE
| Серьёзность | ⚠ Предупреждение
| Категория | [Файлы](./Категория_Файлы.md)
| Исходный код | [NewLineAtEndOfFileRule.cs](../../../Rules/Files/NewLineAtEndOfFileRule.cs)

## Причина

Правило срабатывает, если найдено больше или меньше одной пустой строки в конце файла.

<p id="descr">Удалите лишние пустые строки или добавьте недостающую.</p>

## Примеры

Некорректно

нет пустых скрок

```sql
...
GO
```

две пустые строки

```sql
...
GO


```

Корректно

одна пустая строка

```sql
...
GO

```
