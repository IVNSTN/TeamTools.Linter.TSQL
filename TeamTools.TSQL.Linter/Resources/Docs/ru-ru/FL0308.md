

# Последней командой в SQL-файле должна быть GO

|||
|-|-|
| Id | **FL0308**
| Мнемо | EOF_GO
| Серьёзность | ⚠ Предупреждение
| Категория | [Файлы](./Категория_Файлы.md)
| Исходный код | [EndOfFileGoRule.cs](../../../Rules/Files/EndOfFileGoRule.cs)

## Причина

Правило срабатывает, если не найдена команда GO в конце скрипта.

<p id="descr">Добавьте GO в конец скрипта.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.test
AS 
BEGIN
    SELECT 1;
END;
```

Корректно

```sql
CREATE PROCEDURE dbo.test
AS 
BEGIN
    SELECT 1;
END;
GO
```
