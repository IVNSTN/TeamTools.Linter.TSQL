

# Отсутствует обрамляющий BEGIN-END

|||
|-|-|
| Id | **FM0120**
| Мнемо | BODY_BEGIN_END
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [BeginEndForBodyRule.cs](../../../Rules/Formatting/BeginEndForBodyRule.cs)

## Причина

Правило срабатывает, если тело хранимой процедуры или триггера не заключено в обрамляющий BEGIN-END.
<p id="descr">Заключите тело хранимой процедуры или триггера в обрамляющий BEGIN-END.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE SampleProcedure
AS
SELECT * FROM TableA;
END;
```

Корректно

```sql
CREATE PROCEDURE SampleProcedure
AS
BEGIN
    SELECT * FROM TableA;
END;
```
