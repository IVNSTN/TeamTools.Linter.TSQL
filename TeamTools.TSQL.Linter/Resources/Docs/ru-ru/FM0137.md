

# Отсутствует BEGIN-END после IF/ELSE

|||
|-|-|
| Id | **FM0137**
| Мнемо | IF_BEGIN_END
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [BeginEndForIfRule.cs](../../../Rules/Formatting/BeginEndForIfRule.cs)

## Причина

Правило срабатывает, если у IF-ELSE нет блока BEGIN-END.
<p id="descr">У IF/ELSE добавьте блоки BEGIN-END.</p>

## Примеры

Некорректно

```sql
IF @a = 1
    SET @a = 0;
ELSE
    SET @a = 1;
```

Корректно

```sql
IF @a = 1
BEGIN
    SET @a = 0;
END;
ELSE
BEGIN
    SET @a = 1;
END;
```
