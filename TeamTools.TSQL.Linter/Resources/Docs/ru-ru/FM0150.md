

# Отсутствует BEGIN-END после WHILE

|||
|-|-|
| Id | **FM0150**
| Мнемо | WHILE_BEGIN_END
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [BeginEndForWhileRule.cs](../../../Rules/Formatting/BeginEndForWhileRule.cs)

## Причина

Правило срабатывает, если тело цикла WHILE не заключено в BEGIN-END.
<p id="descr">Заключите тело цикла в BEGIN-END.</p>

## Примеры

Некорректно

```sql
WHILE(1 = 1)
    EXEC some_procedure
```

Корректно

```sql
WHILE(1 = 1)
BEGIN
    EXEC some_procedure
END;
```
