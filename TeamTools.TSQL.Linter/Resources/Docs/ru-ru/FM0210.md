

# Объявление не отформатировано в виде таблицы

|||
|-|-|
| Id | **FM0210**
| Мнемо | DECLARE_FORMAT
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [DeclareFormatAsTableRule.cs](../../../Rules/Formatting/DeclareFormatAsTableRule.cs)

## Причина

Правило срабатывает, если блок объявления переменных не выровнен таблично.
<p id="descr">Выровняйте имена и типы таблично.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE InvalidDeclareFormat
AS
BEGIN
    DECLARE @Var1 INT, @Var2 VARCHAR(100)
END
```

Корректно

```sql
CREATE PROCEDURE ValidDeclareFormat
AS
BEGIN
    DECLARE
        @Var1       INT
        , @Var2     VARCHAR(100);
END
```
