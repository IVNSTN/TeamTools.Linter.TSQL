

# Список параметров не отформатирован в виде таблицы

|||
|-|-|
| Id | **FM0211**
| Мнемо | EXEC_FORMAT
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [ExecParamsFormatAsTableRule.cs](../../../Rules/Formatting/ExecParamsFormatAsTableRule.cs)

## Причина

Правило срабатывает, если параметры хранимой процедуры не отформатированы в виде таблицы.
<p id="descr">При оформлении вызова хранимой процедуры параметры должны быть оформлены таблично.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.SaveData
    @EventID UNIQUEIDENTIFIER
    , @MessageTypeID UNIQUEIDENTIFIER
AS
BEGIN
    RETURN 1
END
```

Корректно

```sql
CREATE PROCEDURE dbo.SaveData
    @EventID         UNIQUEIDENTIFIER
    , @MessageTypeID UNIQUEIDENTIFIER
AS
BEGIN
    RETURN 1
END
```
