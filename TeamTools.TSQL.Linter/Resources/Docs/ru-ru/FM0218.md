

# Отступ пары BEGIN-END не совпадает

|||
|-|-|
| Id | **FM0218**
| Мнемо | BEGIN_END_ALIGN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [BeginEndAlignedRule.cs](../../../Rules/Formatting/BeginEndAlignedRule.cs)

## Причина

Правило срабатывает, если BEGIN и соответствующий ему END не выровнены одинаково.
<p id="descr">Установите одинаковый отступ слева у BEGIN и соответствующего ему END.</p>

## Примеры

Некорректно

```sql
IF(@i = 1)
BEGIN
    RETURN 1
    END
```

Корректно

```sql
IF(@i = 1)
BEGIN
    RETURN 1
END
```
