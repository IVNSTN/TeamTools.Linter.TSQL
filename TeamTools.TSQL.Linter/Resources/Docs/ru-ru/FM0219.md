

# Ключевые элементы инструкции DML не выровнены одинаково

|||
|-|-|
| Id | **FM0219**
| Мнемо | STATEMENT_ALIGN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [StatementElementsAlignedRule.cs](../../../Rules/Formatting/StatementElementsAlignedRule.cs)

## Причина

Правило срабатывает, если ключевые элементы DML выражений не выровнены одинаково и не имеют одинакового отступа слева.
<p id="descr">Выровняйте ключевые элементы DML выражений.</p>

## Примеры

Некорректно

```sql
INSERT INTO Employees (FirstName, LastName, Age)
    VALUES ('John', 'Doe', 30)
```

Корректно

```sql
INSERT INTO Employees (FirstName, LastName, Age)
VALUES ('John', 'Doe', 30)
```
