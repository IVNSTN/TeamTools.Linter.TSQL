

# DML statement key elements not aligned similarly

|||
|-|-|
| Id | **FM0219**
| Мнемо | STATEMENT_ALIGN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [StatementElementsAlignedRule.cs](../../../Rules/Formatting/StatementElementsAlignedRule.cs)

## Причина

<p id="descr">Ключевые элементы DML выражений должны быть выровнены одинаково, иметь одинаковый отступ слева. На однострочные выражения не реагирует.</p>
