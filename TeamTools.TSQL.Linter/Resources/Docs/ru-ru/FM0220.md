

# JOIN/APPLY не выровнены в соотвествии с FROM

|||
|-|-|
| Id | **FM0220**
| Мнемо | JOINS_ALIGN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [JoinsAlignedToFromRule.cs](../../../Rules/Formatting/JoinsAlignedToFromRule.cs)

## Причина

Правило срабатывает, если операторы соеднений не отформатированы таблично.
<p id="descr">Отформатируйте операторы соеднений таблично.</p>

## Примеры

Некорректно

```sql
SELECT * FROM
TableA AS A
 INNER JOIN  TableB AS B
    on A.ID = B.ID
  INNER JOIN  TableC AS C
    on A.ID = C.ID
```

Корректно

```sql
SELECT * FROM
TableA AS A
INNER JOIN  TableB AS B
    on A.ID = B.ID
INNER JOIN  TableC AS C
    on A.ID = C.ID
```
