
# Отсутствуют одиночные пробелы вокруг оператора

|||
|-|-|
| Id | **FM0230**
| Мнемо | WHITESPACE_AROUND_OPERATOR
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [WhitespaceAroundOperatorRule.cs](../../../Rules/Formatting/WhitespaceAroundOperatorRule.cs)

## Причина

Правило срабатывает, если вокруг простых операторов отсутствуют пробелы или их слишком много.
<p id="descr">Убедитесь, что вокруг операторов установлено ровно по одному пробелу.</p>

## Примеры

Некорректно

```sql
IF(@i >  1)
BEGIN
    SET @i = 0;
END
```

Корректно

```sql
IF(@i > 1)
BEGIN
    SET @i = 0;
END
```
