

# Отсутствует одиночный пробел после запятой

|||
|-|-|
| Id | **FM0231**
| Мнемо | WHITESPACE_AFTER_COMMA
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [WhitespaceAfterCommaRule.cs](../../../Rules/Formatting/WhitespaceAfterCommaRule.cs)

## Причина

Правило срабатывает, если после запятой в перечислениях отсутствует пробел или пробелов слишком много.
<p id="descr">Убедитесь, что после каждой запятой в перечислениях установлен ровно один пробел перед следующим литералом.</p>

## Примеры

Некорректно

```sql
DECLARE
    @a INT
    ,  @b INT
    ,@c DECIMAL(18,   2);
```

Корректно

```sql
DECLARE
    @a  INT
    , @b INT
    , @c DECIMAL(18, 2);
```
