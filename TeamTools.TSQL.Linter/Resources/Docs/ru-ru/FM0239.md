

# Однострочный комментарий должен начинаться с пробела

|||
|-|-|
| Id | **FM0239**
| Мнемо | SL_COMMENT_LEADING_SPACE
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [SingleSpaceAfterSingleLineCommentBeginRule.cs](../../../Rules/Formatting/SingleSpaceAfterSingleLineCommentBeginRule.cs)

## Причина

Правило срабатывает, если однострочный комментарий начинается не с двух минусов и пробела.
<p id="descr">Убедитесь, что перед текстом однострочного комментария стоят два минуса и один пробел.</p>

## Примеры

Некорректно

```sql
/* текст комментария */
```

Корректно

```sql
-- текст комментария
```
