

# Выравнивание многострочного комментария

|||
|-|-|
| Id | **FM0243**
| Мнемо | ML_COMMENT_CONTENT_ALIGN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [MultilineCommentOpenCloseTagAlignRule.cs](../../../Rules/Formatting/MultilineCommentOpenCloseTagAlignRule.cs)

## Причина

Правило срабатывает, если отступ текста многострочного комментария меньше отступа открывающего (`/*`) и закрывающего (`*/`) тегов.
<p id="descr">Увеличьте отступ у текста комментария.</p>

## Примеры

Некорректно

```sql
    /*
Это многострочный
комментарий
    */

    /* Это многострочный
комментарий */
```

Корректно

```sql
    /*
    Это многострочный
    комментарий
    */

    /* Это многострочный
    комментарий */
```
