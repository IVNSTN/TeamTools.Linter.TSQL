

# Multiline comment not aligned with open-close tags

|||
|-|-|
| Id | **FM0243**
| Мнемо | ML_COMMENT_CONTENT_ALIGN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [MultilineCommentOpenCloseTagAlignRule.cs](../../../Rules/Formatting/MultilineCommentOpenCloseTagAlignRule.cs)

## Причина

<p id="descr">Содержимое многострочного комментария не должно "выпадать" из него. Отступ слева для текста комментария должен быть &gt;= отступу открывающего и закрывающего тегов.</p>
