

# Выравнивание нескольких однострочных комментариев

|||
|-|-|
| Id | **FM0246**
| Мнемо | MANY_SL_COMMENTS_ALIGNED
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [MultipleSingleLineCommentsAlignRule.cs](../../../Rules/Formatting/MultipleSingleLineCommentsAlignRule.cs)

## Причина

Правило срабатывает, если несколько однострочных комментариев идущих подряд имеют разный отступ.
<p id="descr">Выровняйте отступ у однострочных комментариев.</p>

## Примеры

Некорректно

```sql
-- Комментарий
    -- Ещё один комментарий
```

Корректно

```sql
-- Комментарий
-- Ещё один комментарий
```
