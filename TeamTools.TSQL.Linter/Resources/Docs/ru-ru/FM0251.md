

# Лишняя точка с запятой

|||
|-|-|
| Id | **FM0251**
| Мнемо | DANGLING_SEMICOLON
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [DanglingSemicolonRule.cs](../../../Rules/Formatting/DanglingSemicolonRule.cs)

## Причина

Правило срабатывает, если в строке находится одиночная точка с запятой.
<p id="descr">Удалите одиночную точку с запятой или всю строку.</p>

## Примеры

Некорректно

```sql
SELECT Id FROM SomeTable;
;
```

Корректно

```sql
SELECT Id FROM SomeTable;
```

## Подсказки

💡 Такое может произойти в результате применения автоматического форматирования, если точка с запятой перед этим была неудачно размещена.
