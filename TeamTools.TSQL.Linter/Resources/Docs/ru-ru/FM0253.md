

# Выравнивание открывающей и закрывающей скобок

|||
|-|-|
| Id | **FM0253**
| Мнемо | OPEN_CLOSE_PARENTHESIS_ALIGNED
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [OpenCloseParenthesisAlignRule.cs](../../../Rules/Formatting/OpenCloseParenthesisAlignRule.cs)

## Причина

Правило срабатывает, если открывающая и закрывающая скобки находятся на отдельных строках и имеют разный отступ.
<p id="descr">Выровняйте открывающую и закрывающую скобки.</p>

## Примеры

Некорректно

```sql
SELECT Id FROM (
    SELECT Id FROM SomeTable
) AS tbl;
```

Корректно

```sql
SELECT Id FROM
(
    SELECT Id FROM SomeTable
) AS tbl;
```

## Подсказки

💡 Если скобки находятся в строках с кодом, то правило не применяется. Например:

```sql
IIF(asdf=asdf,
    asdf, adsf * asdf / adsf) -- OK
```
