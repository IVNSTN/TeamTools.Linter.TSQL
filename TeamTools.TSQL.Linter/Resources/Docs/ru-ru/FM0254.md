

# Выравнивание вложенного BEGIN-END

|||
|-|-|
| Id | **FM0254**
| Мнемо | NESTED_BEGIN_END_LEFT_MARGIN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [NestedBeginEndAlignLeftMarginRule.cs](../../../Rules/Formatting/NestedBeginEndAlignLeftMarginRule.cs)

## Причина

Правило срабатывает, если у вложенных блоков BEGIN-END отступ меньше либо равен отступу внешних блоков.
<p id="descr">Увеличьте отступ вложенных блоков BEGIN-END.</p>

## Примеры

Некорректно

```sql
BEGIN -- родительский блок
BEGIN -- вложенный блок
END;
END;
```

Корректно

```sql
BEGIN -- родительский блок
    BEGIN -- вложенный блок
    END;
END;
```
