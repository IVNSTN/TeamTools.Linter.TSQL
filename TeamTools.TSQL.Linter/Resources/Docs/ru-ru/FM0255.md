

# Выравнивание вложенных скобок

|||
|-|-|
| Id | **FM0255**
| Мнемо | NESTED_PARENTHESIS_LEFT_MARGIN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [NestedParenthesisAlignLeftMarginRule.cs](../../../Rules/Formatting/NestedParenthesisAlignLeftMarginRule.cs)

## Причина

Правило срабатывает, если у вложенных скобок отступ меньше либо равен отступу внешних скобок.
<p id="descr">Увеличьте отступ у вложенных скобок.</p>

## Примеры

Некорректно

```sql
SELECT Field
FROM SomeTable
WHERE
(
    Field1 = 'value1' AND
(
    Field2 = 'value2' OR Field3 = 'value2'
)
);
```

Корректно

```sql
SELECT Field
FROM SomeTable
WHERE
(
    Field1 = 'value1' AND
    (
        Field2 = 'value2' OR Field3 = 'value2'
    )
);
```
