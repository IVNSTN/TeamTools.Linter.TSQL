

# Выравнивание открывающей и закрывающей скобок в JOIN

|||
|-|-|
| Id | **FM0260**
| Мнемо | JOIN_PARENTHESIS_ALIGN
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [JoinParenthesisAlignedToFromRule.cs](../../../Rules/Formatting/JoinParenthesisAlignedToFromRule.cs)

## Причина

Правило срабатывает, если открывающая и закрывающая скобки в подзапросе `JOIN` имеют разный отступ.
<p id="descr">Выровняйте открывающую и закрывающую скобки подзапроса JOIN.</p>

## Примеры

Некорректно

```sql
SELECT
    tbl.Id
FROM SomeTable AS tbl
INNER JOIN (
    SELECT Id FROM SomeTable2
) AS joinTbl ON tbl.Id = joinTbl.Id;
```

Корректно

```sql
SELECT
    tbl.Id
FROM SomeTable AS tbl
INNER JOIN
(
    SELECT Id FROM SomeTable2
) AS joinTbl ON tbl.Id = joinTbl.Id;
```
