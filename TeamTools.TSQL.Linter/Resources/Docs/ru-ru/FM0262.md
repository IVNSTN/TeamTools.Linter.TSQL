

# Открывающий и закрывающий теги комментария отделены пробелом

|||
|-|-|
| Id | **FM0262**
| Мнемо | ML_COMMENT_OPEN_CLOSE_WHITESPACE
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [MultilineCommentOpenCloseWhitespaceRule.cs](../../../Rules/Formatting/MultilineCommentOpenCloseWhitespaceRule.cs)

## Причина

Правило срабатывает, если при оформлении многострочного комментария в строках с текстом после открывающего или перед закрывающим тегами больше одного пробела.
<p id="descr">Уберите лишние пробелы вокруг тегов многострочного комментария.</p>

## Примеры

Некорректно

```sql
/*   Это многострочный
комментарий   */
```

Корректно

```sql
/* Это многострочный
комментарий */
```
