
# Выравнивание типов столбцов

|||
|-|-|
| Id | **FM0269**
| Мнемо | COLUMN_TYPE_ALIGNED
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [ColumnTypeAlignedRule.cs](../../../Rules/Formatting/ColumnTypeAlignedRule.cs)

## Причина

Правило срабатывает, если типы столбцов в определении таблиц или табличных переменных не выровнены.
<p id="descr">Выровняйте типы столбцов.</p>

## Примеры

Некорректно

```sql
DECLARE @Table TABLE
(
    Id INT NOT NULL
    , Name VARHCAR(50) NOT NULL
);
```

Корректно

```sql
DECLARE @Table TABLE
(
    Id      INT NOT NULL
    , Name  VARHCAR(50) NOT NULL
);
```

## Подсказки

💡 Однострочные выражения игнорируются.
