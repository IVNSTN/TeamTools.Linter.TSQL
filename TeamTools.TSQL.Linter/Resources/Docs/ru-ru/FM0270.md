

# Выравнивание типов переменных

|||
|-|-|
| Id | **FM0270**
| Мнемо | VAR_TYPE_ALIGNED
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [VariableTypeAlignedRule.cs](../../../Rules/Formatting/VariableTypeAlignedRule.cs)

## Причина

Правило срабатывает, если типы переменных или параметров не выровнены.
<p id="descr">Выровняйте типы.</p>

## Примеры

Некорректно

```sql
DECLARE
    @Id INT
    , @Name VARHCAR(50);

CREATE PROCEDURE dbo.procName
    @Id INT
    , @Name VARHCAR(50)
AS
```

Корректно

```sql
DECLARE
    @Id     INT
    , @Name VARHCAR(50);

CREATE PROCEDURE dbo.procName
    @Id     INT
    , @Name VARHCAR(50)
AS
```

## Подсказки

💡 Однострочные выражения игнорируются.
