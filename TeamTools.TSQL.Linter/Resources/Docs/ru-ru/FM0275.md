
# Одиночный пробел вокруг AS

|||
|-|-|
| Id | **FM0275**
| Мнемо | AS_WITH_SPACES
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [SingleSpaceAroundAsRule.cs](../../../Rules/Formatting/SingleSpaceAroundAsRule.cs)

## Причина

Правило срабатывает, если при указании алиаса таблицы или столбца с ключевым словом `AS` вокруг `AS` больше одного пробела.
<p id="descr">Удалите лишние пробелы вокруг AS.</p>

## Примеры

Некорректно

```sql
SELECT Id FROM SomeTable   AS   tbl
```

Корректно

```sql
SELECT Id FROM SomeTable AS tbl
```
