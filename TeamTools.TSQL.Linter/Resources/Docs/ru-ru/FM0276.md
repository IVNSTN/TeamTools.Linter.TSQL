

# Одиночный пробел между словами

|||
|-|-|
| Id | **FM0276**
| Мнемо | TWO_WORD_WITH_SINGLE_SPACE
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [SingleSpaceInTwoWordInstructionRule.cs](../../../Rules/Formatting/SingleSpaceInTwoWordInstructionRule.cs)

## Причина

Правило срабатывает, если в многословных инструкциях TSQL между словами больше одного пробела.
<p id="descr">Удалите лишние пробелы.</p>

## Примеры

Некорректно

```sql
INNER  JOIN

GROUP   BY

BEGIN     TRAN
```

Корректно

```sql
INNER JOIN

GROUP BY

BEGIN TRAN
```
