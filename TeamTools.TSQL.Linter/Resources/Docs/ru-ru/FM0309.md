

# Использование символа табуляции

|||
|-|-|
| Id | **FM0309**
| Мнемо | TAB_SYMBOL
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [TabSymbolRule.cs](../../../Rules/Formatting/TabSymbolRule.cs)

## Причина

Правило срабатывает, если в файле есть символ табуляции.
<p id="descr">Используйте пробелы.</p>

## Примеры

Некорректно
<!-- markdownlint-disable no-hard-tabs -->
```sql
	SELECT Id FROM SomeTable; -- В строке есть символ табуляции

    --	В комментарии есть	символы	табуляции
```
<!-- markdownlint-enable no-hard-tabs -->

Корректно

```sql
    SELECT Id FROM SomeTable; -- В строке только пробелы

    -- В комментарии только пробелы
```

## Подсказки

💡 Если Вам нужно сформировать строку/текст, в которой необходимо использовать символ табуляции, то заведите переменную, которой задайте значение `CHAR(9)` и воспользуйтесь этой переменной в коде формирования нужной строки/текста.
