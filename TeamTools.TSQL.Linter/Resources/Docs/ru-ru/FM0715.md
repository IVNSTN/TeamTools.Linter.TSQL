

# Лишние символы в конце выражения

|||
|-|-|
| Id | **FM0715**
| Мнемо | SEMICOLON_IS_THE_END
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [GarbageBeforeSemicolonRule.cs](../../../Rules/Formatting/GarbageBeforeSemicolonRule.cs)

## Причина

Правило срабатывает, если перед финальной точкой с запятой в конце выражения есть посторонние символы.
<p id="descr">Удалите посторонние символы перед точкой с запятой.</p>

## Примеры

Некорректно

```sql
SELECT Id FROM SomeTable ;

SELECT Id FROM SomeTable /* комментарий */;
```

Корректно

```sql
SELECT Id FROM SomeTable;
```
