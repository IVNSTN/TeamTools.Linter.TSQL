

# Составной идентификатор написан не слитно

|||
|-|-|
| Id | **FM0809**
| Мнемо | SPACE_IN_NAME_REF
| Серьёзность | ⚠ Предупреждение
| Категория | [Форматирование](./Категория_Форматирование.md)
| Исходный код | [SpaceInMultipartReferenceRule.cs](../../../Rules/Formatting/SpaceInMultipartReferenceRule.cs)

## Причина

Правило срабатывает, если составной идентификатор написано не слитно.
<p id="descr">Напишите идентификатор слитно.</p>

## Примеры

Некорректно

```sql
SELECT t.    col FROM dbo    .tbl AS t;
```

Корректно

```sql
SELECT t.col FROM dbo.tbl AS t;
```
