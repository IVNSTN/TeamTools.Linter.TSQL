

# Обращение в другую БД не через синоним

|||
|-|-|
| Id | **HD0122**
| Мнемо | CROSS_DB_REF_HARDCODED
| Серьёзность | ⚠ Предупреждение
| Категория | [Хардкод](./Категория_Хардкод.md)
| Исходный код | [CrossDatabaseRefsHardcodedRule.cs](../../../Rules/Hardcode/CrossDatabaseRefsHardcodedRule.cs)

## Причина

Правило срабатывает, если обнаружено обращения к объектам других баз без использования синонимов.
<p id="descr">Для обращения к объектам других баз используйте синонимы.</p>

## Примеры

Некорректно

```sql
SELECT id FROM other_db.dbo.some_table;
```

Корректно

```sql
CREATE SYNONYM other_db_synonym.some_table
FOR [$(other_db)].dbo.some_table;

SELECT id FROM other_db_synonym.some_table;
```
