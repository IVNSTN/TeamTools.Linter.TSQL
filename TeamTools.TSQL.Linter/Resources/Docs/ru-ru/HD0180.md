

# Указание файловой группы для временной таблицы или её индекса

|||
|-|-|
| Id | **HD0180**
| Мнемо | TEMP_TABLE_FILEGROUP
| Серьёзность | ⚠ Предупреждение
| Категория | [Хардкод](./Категория_Хардкод.md)
| Исходный код | [TempTableFileGroupRule.cs](../../../Rules/Hardcode/TempTableFileGroupRule.cs)

## Причина

Правило срабатывает, если для временой таблицы или её индекса указана конкретная файловая группа.
<p id="descr">Удалите определение файловой группы.</p>

## Примеры

Некорректно

```sql
CREATE TABLE #far (id int) ON [PRIMARY]
CREATE INDEX #idx ON #far (id) ON [PRIMARY]
```

Корректно

```sql
CREATE TABLE #far (id int)
CREATE INDEX #idx ON #far (id)
```
