

# Персональные данные (номер телефона)

|||
|-|-|
| Id | **HD0258**
| Мнемо | PERSONAL_DATA_PHONE
| Серьёзность | ⚠ Предупреждение
| Категория | [Хардкод](./Категория_Хардкод.md)
| Исходный код | [PersonalDataPhoneRule.cs](../../../Rules/Hardcode/PersonalDataPhoneRule.cs)

## Причина

Правило срабатывает, если в коде обнаруживается хардкод с номером телефона.
<p id="descr">Удалите хардкод с номером телефона.</p>

## Примеры

Некорректно

```sql
-- +9(999)999-999-99

SELECT '+7(123)123-234-45' AS phone
```

Корректно

```sql
-- 9999999999

SELECT '495 adsf 926 --- 00011' AS phone
```

## Подсказки

💡Внутренние рассылки оформляйте через admin_role_members и мнемо-код. Внешние - выносите в оформление в SSRS, движки шаблонов, хотя бы в dbo.environment

💡Похожими на телефоны могут оказаться закомментированные примеры с длинными цифровыми идентификаторами - чтобы правило перестало срабатывать, попробуйте выбрать менее похожий на телефон идентификатор.
