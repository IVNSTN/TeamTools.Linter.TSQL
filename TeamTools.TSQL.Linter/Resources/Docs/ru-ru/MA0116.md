
# Заглушенное исключение

|||
|-|-|
| Id | **MA0116**
| Мнемо | ERROR_SWALLOW
| Серьёзность | ⚠ Предупреждение
| Категория | [Обслуживаемость](./Категория_Обслуживаемость.md)
| Исходный код | [ErrorSwallowingRule.cs](../../../Rules/Maintainability/ErrorSwallowingRule.cs)

## Причина

Правило срабатывает, если в конструкции `TRY-CATCH` используется пустой блок `CATCH`.
<p id="descr">Обработайте исключения в блоке CATCH.</p>

## Примеры

Некорректно

```sql
BEGIN TRY
    SELECT 1 / 0;
END TRY
BEGIN CATCH
--PRINT ""error"";
END CATCH;
```

Корректно

```sql
BEGIN TRY
    SELECT 1 / 0;
END TRY
BEGIN CATCH
    PRINT 'error';
END CATCH;
```
