

# Вычисляемой колонке не присвоен псевдоним

|||
|-|-|
| Id | **MA0167**
| Мнемо | COMPUTED_OUTPUT_NO_ALIAS
| Серьёзность | ⚠ Предупреждение
| Категория | [Обслуживаемость](./Категория_Обслуживаемость.md)
| Исходный код | [ComputedColumnOutputAliasRule.cs](../../../Rules/Maintainability/ComputedColumnOutputAliasRule.cs)

## Причина

<p id="descr">Правило срабатывает, если вычисляемой колонке не присвоен псевдоним.</p>

## Примеры

Некорректно

```sql
SELECT
    (SELECT a FROM bar)
    , foo.b
FROM foo
```

Корректно

```sql
SELECT
    (SELECT a FROM bar) AS a
    , foo.c
FROM foo
```
