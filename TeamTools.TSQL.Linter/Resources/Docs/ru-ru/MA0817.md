

# Отсутствие аргументов у функции или хранимой процедуры

|||
|-|-|
| Id | **MA0817**
| Мнемо | ZERO_ARGS
| Серьёзность | ℹ Подсказка
| Категория | [Обслуживаемость](./Категория_Обслуживаемость.md)
| Исходный код | [ProgrammabilityHasZeroArgsRule.cs](../../../Rules/Maintainability/ProgrammabilityHasZeroArgsRule.cs)

## Причина

Правило срабатывает, если у функции или хранимой процедуры нет аргументов.
<p id="descr">Добавьте аргументы.</p>

## Примеры

Некорректно

```sql
CREATE PROC dbo.foo
AS
BEGIN
    PRINT 'no ok'
END
```

Корректно

```sql
CREATE PROC dbo.foo
    @message VARCHAR(100)
AS
BEGIN
    PRINT message
END
```
