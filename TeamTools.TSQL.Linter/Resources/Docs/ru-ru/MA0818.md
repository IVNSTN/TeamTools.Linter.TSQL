

# Слишком много аргументов у функции или хранимой процедуры

|||
|-|-|
| Id | **MA0818**
| Мнемо | TOO_MANY_ARGS
| Серьёзность | ℹ Подсказка
| Категория | [Обслуживаемость](./Категория_Обслуживаемость.md)
| Исходный код | [ProgrammabilityHasManyArgsRule.cs](../../../Rules/Maintainability/ProgrammabilityHasManyArgsRule.cs)

## Причина

Правило срабатывает, если у функции или хранимой процедуры более 12 аргументов.
<p id="descr">Используйте табличные типы, или форматы обмена данными (JSON, XML).</p>

## Примеры

Некорректно

```sql
CREATE PROC dbo.foo
    @a INT
    , @b INT
    , @c INT
    , @d INT
    , @e INT
    , @f INT
    , @g INT
    , @h INT
    , @i INT
    , @j INT
    , @k INT
    , @l INT
    , @m INT
AS;
```

Корректно

```sql
CREATE PROC dbo.foo
    @json_request NVARCHAR(MAX)
AS;
```
