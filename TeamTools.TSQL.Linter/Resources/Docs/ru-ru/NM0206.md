

# System-like наименование объекта

|||
|-|-|
| Id | **NM0206**
| Мнемо | SYS_LIKE_NAME
| Серьёзность | ⚠ Предупреждение
| Категория | [Именование](./Категория_Именование.md)
| Исходный код | [SystemLikeObjectNamesRule.cs](../../../Rules/Naming/SystemLikeObjectNamesRule.cs)

## Причина

Правило срабатывает, если в именах объектов встречаются префиксы используемые системными объектами (sp_, sys_ и т.д.).
<p id="descr">Удалите префиксы используемые системными объектами.</p>

## Примеры

Некорректно

```sql
CREATE TABLE sys_test (a INT);
CREATE VIEW xp_foo AS SELECT * FROM bar;
```

Корректно

```sql
CREATE TABLE test (a INT);
CREATE VIEW foo AS SELECT * FROM bar;
```
