

# Односимвольный табличный алиас

|||
|-|-|
| Id | **NM0207**
| Мнемо | ALIAS_LENGTH
| Серьёзность | ℹ Подсказка
| Категория | [Именование](./Категория_Именование.md)
| Исходный код | [TableAliasLengthRule.cs](../../../Rules/Naming/TableAliasLengthRule.cs)

## Причина

Правило срабатывает, если для таблиц используются односимвольные алиасы.
<p id="descr">Используете в алиасах более одного символа.</p>

## Примеры

Некорректно

```sql
SELECT * FROM dbo.test AS t
```

Корректно

```sql
SELECT * FROM dbo.test AS tst
```

## Подсказки

💡 Для системных таблиц INSERTED и DELETED допустимо использовать алиасы, соответственно, i и d.

💡 На источник выражения FOR XML AUTO не реагирует, т.к. названия алиасов напрямую влияют на итоговую структуру XML.
