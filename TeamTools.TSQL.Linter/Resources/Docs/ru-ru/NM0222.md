
# Суффикс/префикс ID применён к нецелочисленному столбцу

|||
|-|-|
| Id | **NM0222**
| Мнемо | ID_FOR_INT
| Серьёзность | ℹ Подсказка
| Категория | [Именование](./Категория_Именование.md)
| Исходный код | [IdSuffixForIntegerTypesOnlyRule.cs](../../../Rules/Naming/IdSuffixForIntegerTypesOnlyRule.cs)

## Причина

Правило срабатывает, если префикс/суффикс Id/_id используется для нецелочисленного столбца.
<p id="descr">Используйте префикс/суффикс Id/_id только для целочисленных столбцов.</p>

## Примеры

Некорректно

```sql
act_code    INT
place_id    VARCHAR
```

Корректно

```sql
act_id        INT
place_code    VARCHAR
```
