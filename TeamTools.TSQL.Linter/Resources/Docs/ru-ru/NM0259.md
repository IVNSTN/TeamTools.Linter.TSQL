

# Смешение языков в наименование объекта

|||
|-|-|
| Id | **NM0259**
| Мнемо | ALPHABET_MIX_IDENTIFIER
| Серьёзность | ⚠ Предупреждение
| Категория | [Именование](./Категория_Именование.md)
| Исходный код | [AlphabetMixInIdentifierRule.cs](../../../Rules/Naming/AlphabetMixInIdentifierRule.cs)

## Причина

Правило срабатывает, если в наименование объекта используются разные языки.
<p id="descr">Используйте один язык при именовании объектов.</p>

## Примеры

Некорректно

```sql
DECLARE @var_число INT
```

Корректно

```sql
DECLARE @var_number INT
```

## Подсказки

💡 Частая проблема это кириллическая С в латинском идентификаторе.
