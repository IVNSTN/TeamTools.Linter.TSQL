

# Зарезервированные ключевые слова в наименовании объектов

|||
|-|-|
| Id | **NM0263**
| Мнемо | KEYWORD_IDENTIFIER
| Серьёзность | ℹ Подсказка
| Категория | [Именование](./Категория_Именование.md)
| Исходный код | [KeywordIdentifierRule.cs](../../../Rules/Naming/KeywordIdentifierRule.cs)

## Причина

Правило срабатывает, если в наименовании объекта используются зарезервированные ключевые слова.
<p id="descr">Не используйте зарезервированные ключевые слова в наименовании объектов.</p>

## Примеры

Некорректно

```sql
DECLARE @[return] INT
```

Корректно

```sql
DECLARE @start INT
```
