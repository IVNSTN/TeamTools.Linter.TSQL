

# Запрещены наименования @@, ## или подобные

|||
|-|-|
| Id | **NM0271**
| Мнемо | MAGIC_@@_##_NAME
| Серьёзность | ⚠ Предупреждение
| Категория | [Именование](./Категория_Именование.md)
| Исходный код | [MagicNameRule.cs](../../../Rules/Naming/MagicNameRule.cs)

## Причина

Правило срабатывает, если наименование объекта нечитабельно (@@, ## или подобные).
<p id="descr">Используйте читабельные наименования объектов.</p>

## Примеры

Некорректно

```sql
DECLARE @@ INT
```

Корректно

```sql
DECLARE @var INT
```
