

# Объект данного типа не может быть временным

|||
|-|-|
| Id | **NM0712**
| Мнемо | NON_TEMP_OBJECT_LIKE_TEMP
| Серьёзность | ⚠ Предупреждение
| Категория | [Именование](./Категория_Именование.md)
| Исходный код | [NonTempObjectLikeTempRule.cs](../../../Rules/Naming/NonTempObjectLikeTempRule.cs)

## Причина

Правило срабатывает, если в наименовании объекта, не являющегося таблицей или хранимой процедурой, встречается префикс #.
<p id="descr">Удалите префикс #.</p>

## Примеры

Некорректно

```sql
CREATE SCHEMA #tmp
```

Корректно

```sql
CREATE SCHEMA tmp
```
