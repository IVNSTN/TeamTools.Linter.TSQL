

# Ключевое слово использовано для алиаса

|||
|-|-|
| Id | **NM0714**
| Мнемо | ALIAS_IS_KEYWORD
| Серьёзность | ℹ Подсказка
| Категория | [Именование](./Категория_Именование.md)
| Исходный код | [KeywordAliasRule.cs](../../../Rules/Naming/KeywordAliasRule.cs)

## Причина

Правило срабатывает, если в наименовании алиаса используется ключевое слово.
<p id="descr">Замените ключевое слово в алиасе.</p>

## Примеры

Некорректно

```sql
SELECT * FROM dbo.temp AS INT
```

Корректно

```sql
SELECT * FROM dbo.temp AS tmp
```
