

# Наименование индекса нарушает установленный шаблон

|||
|-|-|
| Id | **NM0961**
| Мнемо | INDEX_NAME_PATTERN
| Серьёзность | ⚠ Предупреждение
| Категория | [Именование](./Категория_Именование.md), [Индексы](./Группа_Индексы.md)
| Исходный код | [IndexNamePatternRule.cs](../../../Rules/Naming/IndexNamePatternRule.cs)

## Причина

Правило срабатывает, если наименование индекса не соответствует установленному шаблону.
<p id="descr">Исправьте наименование индекса согласно шаблону.</p>

## Примеры

Некорректно

```sql
CREATE INDEX IXI_zoo_foo_open_date ON zoo.foo (open_date);
CREATE UNIQUE INDEX IX_zoo_foo_title_open_date ON zoo.foo (title, open_date);
```

Корректно

```sql
CREATE INDEX IX_zoo_foo_open_date ON zoo.foo (open_date);
CREATE UNIQUE INDEX IU_zoo_foo_title_open_date ON zoo.foo (title, open_date);
```
