

# Наименование триггера нарушает установленный шаблон

|||
|-|-|
| Id | **NM0962**
| Мнемо | TRIGGER_NAME_PATTERN
| Серьёзность | ⚠ Предупреждение
| Категория | [Именование](./Категория_Именование.md), [Триггеры](./Группа_Триггеры.md)
| Исходный код | [TriggerNamePatternRule.cs](../../../Rules/Naming/TriggerNamePatternRule.cs)

## Причина

Правило срабатывает, если наименование триггера не соответствует установленному шаблону.
<p id="descr">Исправьте наименование триггера согласно шаблону.</p>

## Примеры

Некорректно

```sql
CREATE TRIGGER zoo.instead_upd
ON zoo.foo
INSTEAD OF INSERT, UPDATE
AS
    RETURN;
```

Корректно

```sql
CREATE TRIGGER zoo.foo_instead_upd
ON zoo.foo
INSTEAD OF INSERT, UPDATE
AS
    RETURN;
```
