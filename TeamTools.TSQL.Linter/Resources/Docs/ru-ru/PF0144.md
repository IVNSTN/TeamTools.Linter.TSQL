
# В хранимой процедуре не установлена опция NOCOUNT

|||
|-|-|
| Id | **PF0144**
| Мнемо | PROC_SET_NOCOUNT
| Серьёзность | ⚠ Предупреждение
| Категория | [Производительность](./Категория_Производительность.md)
| Исходный код | [NocountOptionInProcRule.cs](../../../Rules/Performance/NocountOptionInProcRule.cs)

## Причина

Правило срабатывает, если в хранимой процедуре не установлена опция `NOCOUNT`.
<p id="descr">Установите опцию NOCOUNT.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.foo
AS
BEGIN
    SET XACT_ABORT ON;

    SELECT 1 AS Alias;
```

Корректно

```sql
CREATE PROCEDURE dbo.foo
AS
BEGIN
    SET NOCOUNT, XACT_ABORT ON;

    SELECT 1 AS Alias;
```

## Подсказки

💡 В общем случае нужен `ON`. Если зачем-то нужен `OFF` - ставьте, правило само значение опции не проверяет.
