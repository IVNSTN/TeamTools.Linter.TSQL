
# Указано двойное требование рекомпиляции

|||
|-|-|
| Id | **PF0823**
| Мнемо | RECOMPILE_RECOMPILE
| Серьёзность | ℹ Подсказка
| Категория | [Производительность](./Категория_Производительность.md)
| Исходный код | [DoubleRecompileRule.cs](../../../Rules/Performance/DoubleRecompileRule.cs)

## Причина

Правило срабатывает, если опция `RECOMPILE` указана для модуля, запросы которого уже содержат `RECOMPILE`.
<p id="descr">Удалите опцию RECOMPILE для модуля.</p>

## Примеры

Некорректно

```sql
CREATE PROC dbo.foo
    @category_id INT
WITH RECOMPILE
AS
BEGIN
    SELECT *
    FROM dbo.bar
    WHERE category_id = @category_id
    OPTION (RECOMPILE)
END
```

Корректно

```sql
CREATE PROC dbo.foo
    @category_id INT
AS
BEGIN
    SELECT *
    FROM dbo.bar
    WHERE category_id = @category_id
    OPTION (RECOMPILE)
END
```
