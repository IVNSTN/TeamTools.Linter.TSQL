

# Non-SARGable ISNULL вместо OR

|||
|-|-|
| Id | **PF0877**
| Мнемо | EXPAND_ISNULL_FOR_OPTIONAL_ARG
| Серьёзность | ⚠ Предупреждение
| Категория | [Производительность](./Категория_Производительность.md)
| Исходный код | [OptionalParameterIsNullPredicateRule.cs](../../../Rules/Performance/OptionalParameterIsNullPredicateRule.cs)

## Причина

Правило срабатывает, если в предикате используется ISNULL вместо OR для обработки логики необязательного параметра.
<p id="descr">Используйте OR вместо ISNULL.</p>

## Примеры

Некорректно

```sql
SELECT ord.id
FROM dbo.orders AS ord
WHERE o.client_id = ISNULL(@client_id, ord.client_id);
```

Корректно

```sql
SELECT ord.id
FROM dbo.orders AS ord
WHERE ord.client_id = @client_id OR @client_id IS NULL;
```
