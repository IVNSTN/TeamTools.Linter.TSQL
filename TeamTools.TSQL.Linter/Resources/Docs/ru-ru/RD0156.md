

# Излишнее переключение опций

|||
|-|-|
| Id | **RD0156**
| Мнемо | REDUNDANT_OPTION_SWITCH
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantOptionSwitchRule.cs](../../../Rules/Redundancy/RedundantOptionSwitchRule.cs)

## Причина

<p id="descr">Лишнее переключение опций без выражений между ними нужно удалить.</p>

## Примеры

Некорректно

```sql
SET XACT_ABORT ON;

SET XACT_ABORT OFF;

SELECT id FROM dbo.employee
```

Корректно

```sql
SET XACT_ABORT OFF;

SELECT id FROM dbo.employee
```
