
# Излишнее переключение опций

|||
|-|-|
| Id | **RD0156**
| Мнемо | REDUNDANT_OPTION_SWITCH
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantOptionSwitchRule.cs](../../../Rules/Redundancy/RedundantOptionSwitchRule.cs)

## Причина

Правило срабатывает, если обнаружены последовательные установки одной и той же опции, без выражений между ними.
<p id="descr">Удалите лишние установки опции.</p>

## Примеры

Некорректно

```sql
SET XACT_ABORT ON;

SET XACT_ABORT OFF;

SELECT id FROM dbo.employee
```

Корректно

```sql
SET XACT_ABORT OFF;

SELECT id FROM dbo.employee
```
