
# Избыточное использование предиката в HAVING, возможный кандидат на предикат в WHERE

|||
|-|-|
| Id | **RD0193**
| Мнемо | REDUNDANT_HAVING
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantHavingUsageRule.cs](../../../Rules/Redundancy/RedundantHavingUsageRule.cs)

## Причина

Правило срабатывает, если в выражении `HAVING` используется столбец, который уже присутствует в `GROUP BY`.
<p id="descr">Переместите предикат из HAVING в WHERE.</p>

## Примеры

Некорректно

```sql
SELECT salary, COUNT(1) as cnt
FROM dbo.employee
GROUP BY salary
HAVING salary >= 100
```

Корректно

```sql
SELECT salary, COUNT(1) as cnt
FROM dbo.employee
WHERE salary >= 100
GROUP BY salary
```
