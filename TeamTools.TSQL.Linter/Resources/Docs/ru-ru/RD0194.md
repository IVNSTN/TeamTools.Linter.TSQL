

# Избыточный аргумент в COALESCE

|||
|-|-|
| Id | **RD0194**
| Мнемо | REDUNDANT_COALESCE_ARGUMENT
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantCoalesceArgumentRule.cs](../../../Rules/Redundancy/RedundantCoalesceArgumentRule.cs)

## Причина

<p id="descr">Ненужный аргумент в COALESCE: явный NULL просто не нужен и ни на что не влияет, как и аргумент после явного не NULL (литерала) - на литерале всё закончится.</p>

## Примеры

Некорректно

```sql
SELECT COALESCE(@var1, @var2, NULL)
```

Корректно

```sql
SELECT COALESCE(@var1, @var2)
```
