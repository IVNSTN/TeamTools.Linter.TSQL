
# Избыточный вложенный блок BEGIN-END

|||
|-|-|
| Id | **RD0221**
| Мнемо | BEGIN_BEGIN_END_END
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [BeginBeginEndEndRule.cs](../../../Rules/Redundancy/BeginBeginEndEndRule.cs)

## Причина

Правило срабатывает, если имеются вложенные друг в друга конструкции `BEGIN-END`, между которыми отсутствуют какие-либо выражения.
<p id="descr">Удалите лишние вложенные BEGIN-END блоки.</p>

## Примеры

Некорректно

```sql
BEGIN
    BEGIN
        SELECT TOP(10) salary FROM dbo.employee ORDER BY salary
    END
END
```

Корректно

```sql
BEGIN
    SELECT TOP(10) salary FROM dbo.employee ORDER BY salary
END
```

## Подсказки

💡 Могут возникать, например, при удалении более не актуальных IF
