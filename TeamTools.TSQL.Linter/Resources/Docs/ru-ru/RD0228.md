

# Избыточные множественные GO один за другим

|||
|-|-|
| Id | **RD0228**
| Мнемо | GO_GO
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [GoGoRule.cs](../../../Rules/Redundancy/GoGoRule.cs)

## Причина

<p id="descr">Несколько не имеющих смысла GO подряд в коде не нужны.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END
GO
GO
GO
```

Корректно

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END
GO
```
