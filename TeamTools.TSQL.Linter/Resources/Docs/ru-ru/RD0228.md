
# Избыточные множественные GO один за другим

|||
|-|-|
| Id | **RD0228**
| Мнемо | GO_GO
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [GoGoRule.cs](../../../Rules/Redundancy/GoGoRule.cs)

## Причина

Правило срабатывает, если в коде встречаются несколько последовательных операторов `GO`, не имеющих функционального смысла.
<p id="descr">Удалите лишние операторы GO.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END
GO
GO
GO
```

Корректно

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END
GO
```
