
# Избыточные пробелы после ( и перед )

|||
|-|-|
| Id | **RD0235**
| Мнемо | WHITESPACE_IN_PARENTHESIS
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantSpaceWithinParenthesisRule.cs](../../../Rules/Redundancy/RedundantSpaceWithinParenthesisRule.cs)

## Причина

Правило срабатывает, если после открывающей скобки `(` или перед закрывающей скобкой `)` есть пробел.
<p id="descr">Удалите пробелы после открывающей и перед закрывающей скобками.</p>

## Примеры

Некорректно

```sql
IF EXISTS (   SELECT 1 FROM dbo.accounts    )...
```

Корректно

```sql
IF EXISTS (SELECT 1 FROM dbo.accounts)...
```

## Подсказки

💡 Убедитесь, что применили форматтер и он корректно настроен.

💡 Многострочные скобки, оформленные как `BEGIN-END`, игнорируются - там перед `)` всегда много пробелов. Но после `(` все равно пробел не нужен.
