

# Избыточная пустая строка

|||
|-|-|
| Id | **RD0236**
| Мнемо | REDUNDANT_NEWLINE
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantNewLineBetweenStatementsRule.cs](../../../Rules/Redundancy/RedundantNewLineBetweenStatementsRule.cs)

## Причина

<p id="descr">Между выражениями ожидается не более двух пустых строк.</p>

## Примеры

Некорректно

```sql
SELECT TOP (10) id from dbo.employee ORDER BY salary



UPDATE dbo.employee SET salary += 1000
```

Корректно

```sql
SELECT TOP (10) id from dbo.employee ORDER BY salary

UPDATE dbo.employee SET salary += 1000
```

## Подсказки

💡 Убедитесь что применили форматтер и он корректно настроен
