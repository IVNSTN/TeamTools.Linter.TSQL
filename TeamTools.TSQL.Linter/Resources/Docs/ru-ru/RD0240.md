

# Для начала однострочного комментария ожидаются два тире

|||
|-|-|
| Id | **RD0240**
| Мнемо | SL_COMMENT_LEADING_DASHES
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantDashBeforeSingleLineCommentRule.cs](../../../Rules/Redundancy/RedundantDashBeforeSingleLineCommentRule.cs)

## Причина

<p id="descr">Для начала однострочного комментария достаточно двух минусов. Больше двух не надо.</p>

## Примеры

Некорректно

```sql
------- однострочный комментарий
```

Корректно

```sql
-- однострочный комментарий
```
