

# Для однострочного комментария не требуется тире в конце строки

|||
|-|-|
| Id | **RD0241**
| Мнемо | SL_COMMENT_TRAILING_DASHES
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantDashAfterSingleLineCommentRule.cs](../../../Rules/Redundancy/RedundantDashAfterSingleLineCommentRule.cs)

## Причина

<p id="descr">Завершать однострочный комментарий минусами "для красоты" не нужно.</p>

## Примеры

Некорректно

```sql
-- однострочный комментарий --
```

Корректно

```sql
-- однострочный комментарий
```
