
# Для многострочных тегов комментариев ожидается наличие одиночной звездочки

|||
|-|-|
| Id | **RD0242**
| Мнемо | ML_COMMENT_ASTERISK_COUNT
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantAsteriskInMultilineCommentBlockRule.cs](../../../Rules/Redundancy/RedundantAsteriskInMultilineCommentBlockRule.cs)

## Причина

Правило срабатывает, если в многострочном комментарии используются несколько звездочек после открывающего или перед закрывающим тегами.
<p id="descr">Удалите лишние звездочки после открывающего и перед закрывающим тегами.</p>

## Примеры

Некорректно

```sql
/********
 комментарий
 на несколько
 строк
*********/
```

Корректно

```sql
/*
 комментарий
 на несколько
 строк
*/
```
