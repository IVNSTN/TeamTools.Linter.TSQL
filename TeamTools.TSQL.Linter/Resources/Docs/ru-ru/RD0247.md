
# Избыточные скобки

|||
|-|-|
| Id | **RD0247**
| Мнемо | REDUNDANT_PARENTHESIS
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantParenthesisRule.cs](../../../Rules/Redundancy/RedundantParenthesisRule.cs)

## Причина

Правило срабатывает, если в выражении используются лишние скобки, которые ни на что не влияют.
<p id="descr">Удалите лишние скобки.</p>

## Примеры

Некорректно

```sql
CREATE TABLE dbo.employee(id INT NOT NULL, salary INT NOT NULL CONSTRAINT DF_employee_salary DEFAULT (((5))))
```

Корректно

```sql
CREATE TABLE dbo.employee(id INT NOT NULL, salary INT NOT NULL CONSTRAINT DF_employee_salary DEFAULT 5)
```
