
# Избыточная точка с запятой

|||
|-|-|
| Id | **RD0248**
| Мнемо | REDUNDANT_SEMICOLON
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantSemicolonRule.cs](../../../Rules/Redundancy/RedundantSemicolonRule.cs)

## Причина

Правило срабатывает, если в конце выражения используются несколько точек с запятой подряд.
<p id="descr">Удалите лишние точки с запятой в конце выражения.</p>

## Примеры

Некорректно

```sql
SELECT id FROM dbo.employee;;;
```

Корректно

```sql
SELECT id FROM dbo.employee;
```
