
# Избыточные квадратные скобки

|||
|-|-|
| Id | **RD0249**
| Мнемо | REDUNDANT_BRACKETS
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantSquareBracketsRule.cs](../../../Rules/Redundancy/RedundantSquareBracketsRule.cs)

## Причина

Правило срабатывает, если валидные идентификаторы заключены в квадратные скобки (квотированы).
<p id="descr">Удалите квадратные скобки.</p>

## Примеры

Некорректно

```sql
SELECT id FROM [dbo].[employee];
```

Корректно

```sql
SELECT id FROM dbo.employee;
```
