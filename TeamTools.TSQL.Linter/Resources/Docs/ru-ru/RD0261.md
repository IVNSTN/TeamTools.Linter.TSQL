

# Избыточная новая строка в многострочном комментарии

|||
|-|-|
| Id | **RD0261**
| Мнемо | ML_COMMENT_REDUNDANT_NEWLINE
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantNewLineInMultilineCommentRule.cs](../../../Rules/Redundancy/RedundantNewLineInMultilineCommentRule.cs)

## Причина

<p id="descr">Для разделения текста в многострочном комментарии больше двух пустых строк использовать не нужно.</p>

## Примеры

Некорректно

```sql
/*
многострочный



комментарий
*/
```

Корректно

```sql
/*
многострочный

комментарий
*/
```
