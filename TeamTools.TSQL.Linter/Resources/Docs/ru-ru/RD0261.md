
# Избыточная новая строка в многострочном комментарии

|||
|-|-|
| Id | **RD0261**
| Мнемо | ML_COMMENT_REDUNDANT_NEWLINE
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantNewLineInMultilineCommentRule.cs](../../../Rules/Redundancy/RedundantNewLineInMultilineCommentRule.cs)

## Причина

Правило срабатывает, если в многострочном комментарии используются более двух пустых строк для разделения текста.
<p id="descr">Удалите лишние пустые строки.</p>

## Примеры

Некорректно

```sql
/*
многострочный



комментарий
*/
```

Корректно

```sql
/*
многострочный

комментарий
*/
```
