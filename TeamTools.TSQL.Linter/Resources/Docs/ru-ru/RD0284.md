
# Избыточный вызов функции

|||
|-|-|
| Id | **RD0284**
| Мнемо | REDUNDANT_FUNCTION_CALL
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantFunctionCallRule.cs](../../../Rules/Redundancy/RedundantFunctionCallRule.cs)

## Причина

Правило срабатывает, если используется вызов функции, который не имеет смысла и ни к чему не приводит (не влияет на результат).
<p id="descr">Удалите вызов функции, не влияющий на результат.</p>

## Примеры

Некорректно

```sql
SELECT DATEADD(YEAR, DATEDIFF(YEAR, 0, GETDATE()), 0);
```

Корректно

```sql
SELECT DATEFROMPARTS(YEAR(GETDATE()), 1, 1);
```
