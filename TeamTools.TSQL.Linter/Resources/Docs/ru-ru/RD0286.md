
# Избыточное отрицание

|||
|-|-|
| Id | **RD0286**
| Мнемо | REDUNDANT_NEGATION
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantNegationRule.cs](../../../Rules/Redundancy/RedundantNegationRule.cs)

## Причина

Правило срабатывает, если в коде обнаружено отрицание операторов сравнения.
<p id="descr">Замените отрицание сравнения на противоположный оператор.</p>

## Примеры

Некорректно

```sql
IF NOT (@a = @b)
BEGIN
...
END;
```

Корректно

```sql
IF @a <> @b
BEGIN
...
END;
```

## Подсказки

💡 При включенных ANSI_NULLS любой вариант сравнения чего-то с NULL не через IS дает unknown, отрицание unknown или переворачивание сравнения дает такой же unknown.
