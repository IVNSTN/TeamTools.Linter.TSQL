
# Избыточная вложенность CASE

|||
|-|-|
| Id | **RD0287**
| Мнемо | REDUNDANT_NESTED_CASE
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantNestedCaseRule.cs](../../../Rules/Redundancy/RedundantNestedCaseRule.cs)

## Причина

Правило срабатывает, если в конструкции `CASE` следующая ветка может быть продолжена без дополнительной вложенности.
<p id="descr">Уберите лишнюю вложенность и продолжите CASE на том же уровне.</p>

## Примеры

Некорректно

```sql
CASE
    WHEN tbl.col = 1 THEN 'a'
    ELSE
        CASE WHEN tbl.col = 2 THEN 'b' ELSE 'x' END
END
```

Корректно

```sql
CASE
    WHEN tbl.col = 1 THEN 'a'
    WHEN tbl.col = 2 THEN 'b'
    ELSE 'x'
END
```
