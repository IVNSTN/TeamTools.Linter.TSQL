
# Избыточный выбор скаляра

|||
|-|-|
| Id | **RD0288**
| Мнемо | REDUNDANT_SELECT_SCALAR
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantSelectScalarRule.cs](../../../Rules/Redundancy/RedundantSelectScalarRule.cs)

## Причина

Правило срабатывает, если скалярные переменные или литералы заворачиваются в `SELECT` без необходимости.
<p id="descr">Перепишите без использования SELECT для скалярных значений.</p>

## Примеры

Некорректно

```sql
INSERT dbo.foo (title)
SELECT @title
WHERE @title IS NOT NULL
```

Корректно

```sql
IF @title IS NOT NULL
BEGIN
    INSERT dbo.foo(title)
    VALUES (@title)
END
```

## Подсказки

💡 Правило не срабатывает на `ORDER BY (SELECT NULL)`. Хоть это и не самая желанная конструкция в T-SQL.

💡 Правило не срабатывает на `CROSS`/`OUTER APPLY`, используемые для вычислений.
