

# Попытка избыточного упорядочивания по константе

|||
|-|-|
| Id | **RD0296**
| Мнемо | REDUNDANT_ORDER_BY_CONST
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantOrderByConstantRule.cs](../../../Rules/Redundancy/RedundantOrderByConstantRule.cs)

## Причина

<p id="descr">Присутствие литерала или скалярной переменной в ORDER BY ни на что не влияет.</p>

## Примеры

Некорректно

```sql
SELECT treaty FROM dbo.accounts ORDER BY treaty, @a
```

Корректно

```sql
SELECT treaty FROM dbo.accounts ORDER BY treaty
```
