
# Избыточная пустая строка в конце файла

|||
|-|-|
| Id | **RD0307**
| Мнемо | EOF_REDUNDANT_NEWLINE
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantNewLineAtEndOfFileRule.cs](../../../Rules/Redundancy/RedundantNewLineAtEndOfFileRule.cs)

## Причина

Правило срабатывает, если файл заканчивается более чем одной пустой строкой.
<p id="descr">Удалите лишние пустые строки. Оставьте только одну.</p>

## Примеры

Некорректно

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END;
GO




```

Корректно

```sql
CREATE PROCEDURE dbo.accounts_get
AS
BEGIN
...
END;
GO

```
