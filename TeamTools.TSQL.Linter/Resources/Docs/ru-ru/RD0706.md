
# Избыточное преобразование типов

|||
|-|-|
| Id | **RD0706**
| Мнемо | REDUNDANT_TYPE_CONVERSION
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantTypeConversionRule.cs](../../../Rules/Redundancy/RedundantTypeConversionRule.cs)

## Причина

Правило срабатывает, если переменная явно приводится к тому же типу, с которым она была объявлена.
<p id="descr">Уберите избыточное преобразование типа.</p>

## Примеры

Некорректно

```sql
DECLARE @b VARCHAR(1) = 'B'

PRINT CAST('A' + @b AS VARCHAR(2))
```

Корректно

```sql
DECLARE @b VARCHAR(1) = 'B'

PRINT 'A' + @b
```

## Подсказки

💡 Литералы правило игнорирует, поскольку явное указание, какой тип должен иметь вычисляемый столбец в запросе, это скорее хороший стиль.
