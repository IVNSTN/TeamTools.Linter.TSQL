

# Избыточный аргумент

|||
|-|-|
| Id | **RD0707**
| Мнемо | REDUNDANT_ARGUMENT
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantFunctionArgumentRule.cs](../../../Rules/Redundancy/RedundantFunctionArgumentRule.cs)

## Причина

<p id="descr">В функцию передаётся аргумент, который не является обязательным и содержит значение, которое никак не влияет на результат.</p>

## Примеры

Некорректно

```sql
SELECT CONCAT(@a, @b, '')
```

Корректно

```sql
SELECT CONCAT(@a, @b)
```
