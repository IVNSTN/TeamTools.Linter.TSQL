
# Избыточный аргумент

|||
|-|-|
| Id | **RD0707**
| Мнемо | REDUNDANT_ARGUMENT
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantFunctionArgumentRule.cs](../../../Rules/Redundancy/RedundantFunctionArgumentRule.cs)

## Причина

Правило срабатывает, если в функцию передаётся необязательный аргумент, значение которого никак не влияет на результат выполнения.
<p id="descr">Удалите лишний бессмысленный аргумент.</p>

## Примеры

Некорректно

```sql
SELECT CONCAT(@a, @b, '')
```

Корректно

```sql
SELECT CONCAT(@a, @b)
```
