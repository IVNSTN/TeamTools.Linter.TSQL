
# Псевдоним равен имени столбца и может быть опущен

|||
|-|-|
| Id | **RD0713**
| Мнемо | COLUMN_ALIAS_IS_THE_SAME
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [ColumnAliasSameAsSourceRule.cs](../../../Rules/Redundancy/ColumnAliasSameAsSourceRule.cs)

## Причина

Правило срабатывает, если псевдоним столбца совпадает с его именем.
<p id="descr">Удалите избыточный псевдоним столбца.</p>

## Примеры

Некорректно

```sql
SELECT treaty AS treaty FROM dbo.accounts
```

Корректно

```sql
SELECT treaty FROM dbo.accounts
```
