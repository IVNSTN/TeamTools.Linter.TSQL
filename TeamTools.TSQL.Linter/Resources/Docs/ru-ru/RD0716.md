

# Избыточный BEGIN-END после ELSE

|||
|-|-|
| Id | **RD0716**
| Мнемо | IF_ELSE_REDUNDANT_BEGIN_END
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantBeginEndForElseRule.cs](../../../Rules/Redundancy/RedundantBeginEndForElseRule.cs)

## Причина

<p id="descr">После ELSE не нужно городить BEGIN-END с единственным блоком IF внутри - достаточно так и написать ELSE IF...</p>

## Примеры

Некорректно

```sql
...
ELSE
BEGIN
    IF @a = @b
    BEGIN
        PRINT 1
    END;
END;
```

Корректно

```sql
...
ELSE IF @a = @b
    BEGIN
        PRINT 1
    END;
```
