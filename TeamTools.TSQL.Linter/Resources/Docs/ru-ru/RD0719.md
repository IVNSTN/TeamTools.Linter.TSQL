

# Избыточное использование EXISTS

|||
|-|-|
| Id | **RD0719**
| Мнемо | SCALAR_PREDICATE_AS_EXISTS
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [ScalarPredicateTurnedIntoExistsRule.cs](../../../Rules/Redundancy/ScalarPredicateTurnedIntoExistsRule.cs)

## Причина

<p id="descr">EXISTS без FROM - это обычный скалярный предикат. Заворачивать его в фиктивный "селект" совершенно незачем. Вынесите содержимое WHERE во внешнее выражение.</p>

## Примеры

Некорректно

```sql
IF EXISTS(SELECT 1 WHERE @a > @b)
BEGIN
    ...
END
```

Корректно

```sql
IF @a > @b
BEGIN
    ...
END
```
