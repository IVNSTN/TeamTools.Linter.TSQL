
# Избыточное использование EXISTS

|||
|-|-|
| Id | **RD0719**
| Мнемо | SCALAR_PREDICATE_AS_EXISTS
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [ScalarPredicateTurnedIntoExistsRule.cs](../../../Rules/Redundancy/ScalarPredicateTurnedIntoExistsRule.cs)

## Причина

Правило срабатывает, если предикат `EXISTS` используется без конструкции `FROM`, оборачивая логическое выражение в лишний подзапрос.
<p id="descr">Уберите EXISTS, перенеся условие из его внутреннего WHERE во внешнее выражение.</p>

## Примеры

Некорректно

```sql
IF EXISTS(SELECT 1 WHERE @a > @b)
BEGIN
    ...
END
```

Корректно

```sql
IF @a > @b
BEGIN
    ...
END
```
