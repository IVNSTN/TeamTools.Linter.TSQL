

# Избыточная обработка NULL в функциях конкатенации

|||
|-|-|
| Id | **RD0721**
| Мнемо | NULL_HANDLING_FOR_CONCAT
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [NullHandlingForConcatRule.cs](../../../Rules/Redundancy/NullHandlingForConcatRule.cs)

## Причина

Правило срабатывает, если обнаруживает в аргументах функции CONCAT обработку NULL-значений с преобразованием их в пустые строки.
<p id="descr">Уберите вызов ISNULL с преобразованием с пустую строку.</p>

## Примеры

Некорректно

```sql
SELECT CONCAT('asdf', @var, ISNULL(@bar, ''))
```

Корректно

```sql
SELECT CONCAT('asdf', @var, @bar)
```
