

# Избыточная команда CONTINUE в конце тела цикла

|||
|-|-|
| Id | **RD0730**
| Мнемо | REDUNDANT_CONTINUE
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantContinueRule.cs](../../../Rules/Redundancy/RedundantContinueRule.cs)

## Причина

Правило срабатывает, если в завершении тела цикла WHILE обнаружена команда CONTINUE.
<p id="descr">Удалите команду CONTINUE.</p>

## Примеры

Некорректно

```sql
WHILE @@FETCH_STATUS = 0
BEGIN
    PRINT @a
    FETCH NEXT FROM cr INTO @a
    CONTINUE
END

WHILE @@FETCH_STATUS = 0
BEGIN
    PRINT @a
    FETCH NEXT FROM cr INTO @a
    IF @a > 10
    BEGIN
        CONTINUE
    END;
END
```

Корректно

```sql
WHILE @@FETCH_STATUS = 0
BEGIN
    PRINT @a
    IF @a > 10
        BREAK
    FETCH NEXT FROM cr INTO @a
END

WHILE @@FETCH_STATUS = 0
BEGIN
    PRINT @a
    IF @a > 10
        CONTINUE
    FETCH NEXT FROM cr INTO @a
END
```
