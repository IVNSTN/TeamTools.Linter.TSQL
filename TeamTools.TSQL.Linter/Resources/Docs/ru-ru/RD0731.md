

# Избыточный знак у нуля

|||
|-|-|
| Id | **RD0731**
| Мнемо | SIGNED_ZERO
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [SignedZeroRule.cs](../../../Rules/Redundancy/SignedZeroRule.cs)

## Причина

Правило срабатывает, если в коде обнаружена попытка присвоения знака плюс (+) или минус (-) нулю (0) или значению NULL.
<p id="descr">Удалите знак присвоенный нулю или значению NULL.</p>

## Примеры

Некорректно

```sql
SET @foo = +NULL
SET @foo = +0
SELECT 3 * (-(0.0)) AS result
```

Корректно

```sql
SET @foo = 1 + 0
SELECT -3 * 0 AS result
```
