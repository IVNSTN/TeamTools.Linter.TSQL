

# Лишние скобки вокруг условий объединения

|||
|-|-|
| Id | **RD0780**
| Мнемо | JOIN_PREDICATE_PARENTHETHIS
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [JoinWithParenthethisRule.cs](../../../Rules/Redundancy/JoinWithParenthethisRule.cs)

## Причина

Правило срабатывает, если обнаружены условия объединения заключенные в общие скобки.
<p id="descr">Удалите лишние скобки вокруг условий объединения.</p>

## Примеры

Некорректно

```sql
SELECT *
FROM dbo.foo f
INNER JOIN dbo.bar b
    ON (f.id = b.id)

SELECT *
FROM dbo.foo f
LEFT JOIN (SELECT id FROM dbo.far) a
INNER JOIN dbo.bar b
    ON a.id = b.parent_id
    ON (f.id = b.id)
```

Корректно

```sql
SELECT *
FROM dbo.foo f
INNER JOIN dbo.bar b
    ON f.id = b.id

SELECT *
FROM dbo.foo f
LEFT JOIN (SELECT id FROM dbo.far) a
INNER JOIN dbo.bar b
    ON a.id = b.parent_id
    ON f.id = b.id
```
