

# Избыточная вложенная проверка

|||
|-|-|
| Id | **RD0782**
| Мнемо | REDUNDANT_NESTED_CONDITION
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantNestedConditionRule.cs](../../../Rules/Redundancy/RedundantNestedConditionRule.cs)

## Причина

Правило срабатывает, если внутри IF или CASE обнаружена повторная проверка какого-то из условий.
<p id="descr">Удалите избыточную проверку.</p>

## Примеры

Некорректно

```sql
IF @a > 0
    IF @a > 0
        SET @a = 0

IF @a > 0
BEGIN
    IF @a =
        CASE
            WHEN ((@a > 0)) THEN 1
            ELSE 0
        END
    BEGIN
        PRINT '1'
    END
END
```

Корректно

```sql
IF @a > 0
    IF @a < 100
        SET @a = 0

IF @a = 1
BEGIN
    PRINT '1'
END
```
