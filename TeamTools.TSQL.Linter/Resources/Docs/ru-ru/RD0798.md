

# Избыточная инициализация переменной значением NULL

|||
|-|-|
| Id | **RD0798**
| Мнемо | REDUNDANT_INIT_NULL
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantInitWithNullRule.cs](../../../Rules/Redundancy/RedundantInitWithNullRule.cs)

## Причина

Правило срабатывает, если переменная иницализируется значением NULL.
<p id="descr">Удалите инициализацию значением NULL.</p>

## Примеры

Некорректно

```sql
DECLARE @percent INT = NULL;
```

Корректно

```sql
DECLARE @percent INT;
```
