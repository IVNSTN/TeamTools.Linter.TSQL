

# Избыточное указание EXECUTE AS CALLER

|||
|-|-|
| Id | **RD0811**
| Мнемо | EXECUTE_AS_CALLER
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [ExecuteAsCallerRule.cs](../../../Rules/Redundancy/ExecuteAsCallerRule.cs)

## Причина

Правило срабатывает, если для модуля добавлено указание EXECUTE AS CALLER.
<p id="descr">Удалите EXECUTE AS CALLER.</p>

## Примеры

Некорректно

```sql
CREATE PROC dbo.bar
WITH EXECUTE AS CALLER
AS;
```

Корректно

```sql
CREATE PROC dbo.bar
AS;
```
