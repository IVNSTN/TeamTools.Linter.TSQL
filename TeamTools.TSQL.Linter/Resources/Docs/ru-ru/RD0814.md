

# Переменная указана более одного раза в предикате IN

|||
|-|-|
| Id | **RD0814**
| Мнемо | IN_DUP_VAR
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [DuplicateInVariableRule.cs](../../../Rules/Redundancy/DuplicateInVariableRule.cs)

## Причина

Правило срабатывает, если переменная указана более одного раза в предикате IN.
<p id="descr">Удалите дубли.</p>

## Примеры

Некорректно

```sql
SELECT name
FROM dbo.foo
WHERE name IN (@first_name, @second_name, @first_name)
```

Корректно

```sql
SELECT name
FROM dbo.foo
WHERE name IN (@first_name, @second_name)
```
