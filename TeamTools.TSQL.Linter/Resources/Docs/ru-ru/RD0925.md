

# Использование LIKE без спецсимволов

|||
|-|-|
| Id | **RD0925**
| Мнемо | REDUNDANT_LIKE
| Серьёзность | ⚠ Предупреждение
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [RedundantLikePredicateRule.cs](../../../Rules/Redundancy/RedundantLikePredicateRule.cs)

## Причина

Правило срабатывает, если не обнаруживает в операторе LIKE спецсимволов (%, _, [], ?).
<p id="descr">Замените LIKE на сравнение или добавьте спецсимволы.</p>

## Примеры

Некорректно

```sql
SELECT name
FROM dbo.foo
WHERE name LIKE 'asdf'
```

Корректно

```sql
SELECT *
FROM dbo.foo
WHERE name LIKE 'a%sdf'

SELECT name
FROM dbo.foo
WHERE name = 'asdf'
```
