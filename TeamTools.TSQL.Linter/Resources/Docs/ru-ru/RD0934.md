

# CTE объявлено, но не используется

|||
|-|-|
| Id | **RD0934**
| Мнемо | CTE_UNUSED
| Серьёзность | ℹ Подсказка
| Категория | [Избыточность](./Категория_Избыточность.md)
| Исходный код | [UnusedCteRule.cs](../../../Rules/Redundancy/UnusedCteRule.cs)

## Причина

Правило срабатывает, если обнаружено объявленное, но не используемое далее по коду CTE.
<p id="descr">Удалите неиспользуемое CTE.</p>

## Примеры

Некорректно

```sql
WITH cte AS (SELECT id FROM dbo.bar)
DELETE f
FROM dbo.foo AS f;
```

Корректно

```sql
WITH cte AS (SELECT id FROM dbo.bar)
SELECT d, b
FROM dbo.foo AS f
WHERE EXISTS(SELECT 1 FROM cte)
```
