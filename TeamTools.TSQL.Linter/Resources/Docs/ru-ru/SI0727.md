

# Отрицание простого сравнения можно упростить

|||
|-|-|
| Id | **SI0727**
| Мнемо | NOT_FOR_SIMPLE_PREDICATE
| Серьёзность | ⚠ Предупреждение
| Категория | [Упрощение](./Категория_Упрощение.md)
| Исходный код | [SimplePredicateNegationRule.cs](../../../Rules/Simplification/SimplePredicateNegationRule.cs)

## Причина

Правило срабатывает, если обнаружено отрицание простого условия сравнения.
<p id="descr">Избавьтесь от отрицания, перевернув условие сравнения.</p>

## Примеры

Некорректно

```sql
IF NOT (@a = @b)
    SELECT CASE WHEN (NOT (@c > @d)) THEN 1 ELSE 0 END AS res
```

Корректно

```sql
IF @a <> @b
    SELECT CASE WHEN @c <= @d THEN 1 ELSE 0 END AS res
```
