

# Значение переменной можно указать в DECLARE

|||
|-|-|
| Id | **SI0735**
| Мнемо | SET_TO_DECLARE
| Серьёзность | ℹ Подсказка
| Категория | [Упрощение](./Категория_Упрощение.md)
| Исходный код | [SetToDeclareRule.cs](../../../Rules/Simplification/SetToDeclareRule.cs)

## Причина

Правило срабатывает, если обнаруживает присвоение начального значения переменной, которое может быть задано при объявлении переменной в выражении DECLARE.
<p id="descr">Присвойте значение в выражении DECLARE.</p>

## Примеры

Некорректно

```sql
DECLARE @a   INT
        , @b DATETIME

SET @a = 2
SELECT @b = GETDATE()
```

Корректно

```sql
DECLARE @a   INT      = 2
        , @b DATETIME = GETDATE()
```
