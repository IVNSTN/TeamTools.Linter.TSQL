

# Несколько выражений DROP можно объединить в одно

|||
|-|-|
| Id | **SI0753**
| Мнемо | DROP_STATEMENTS_INTO_ONE
| Серьёзность | ℹ Подсказка
| Категория | [Упрощение](./Категория_Упрощение.md)

## Причина

Правило срабатывает, если обнаружено несколько выражений удаления однотипных объектов, которые возможно оформить в виде одного.
<p id="descr">Перечислите объекты через запятую в одном выражении DROP.</p>

## Примеры

Некорректно

```sql
DROP TABLE dbo.foo
DROP TABLE dbo.bar

DROP PROC dbo.far
DROP PROC dbo.jar
```

Корректно

```sql
DROP TABLE dbo.foo, dbo.bar

DROP PROC dbo.far, dbo.jar
```
