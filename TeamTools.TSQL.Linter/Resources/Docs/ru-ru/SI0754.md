

# Несколько выражений ALTER можно объединить в одно

|||
|-|-|

| Мнемо | ALTER_STATEMENTS_INTO_ONE
| Серьёзность | ℹ Подсказка
| Категория | [Упрощение](./Категория_Упрощение.md)

## Причина

Правило срабатывает, если обнаружено несколько выражений модификации однотипных объектов, связанных с таблицами (колонок или ограничений), которые можно объединить в одно выражение.
<p id="descr">Перечислите объекты через запятую в одном выражении ALTER TABLE.</p>

## Примеры

Некорректно

```sql
ALTER TABLE dbo.foo ADD bar INT
ALTER TABLE dbo.foo ADD far BIT
ALTER TABLE dbo.foo ADD jar VARCHAR(10);

ALTER TABLE dbo.foo DROP CONSTRAINT zar
ALTER TABLE dbo.foo DROP CONSTRAINT car
ALTER TABLE dbo.foo DROP CONSTRAINT tar
```

Корректно

```sql
ALTER TABLE dbo.foo ADD bar INT, far BIT, jar VARCHAR(10);

ALTER TABLE dbo.foo DROP COLUMN zar, car, tar
```
