
# Проверки на неравенство могут быть объединены в предикат NOT IN

## Причина

Правило срабатывает, если проверки на неравенство могут быть объединены в предикат `NOT IN`.
<p id="descr">Объедините проверки в один NOT IN предикат.</p>

## Примеры

Некорректно

```sql
SELECT *
FROM dbo.foo
WHERE category_id <> 1 
    AND category_id <> 2;
```

Корректно

```sql
SELECT *
FROM dbo.foo
WHERE category_id NOT IN (1, 2);
```
