
# Схожие предикаты IN могут быть объединены

|||
|-|-|

| Мнемо | MULTIPLE_IN_TO_SINGLE
| Серьёзность | ℹ Подсказка

## Причина

Правило срабатывает, если схожие предикаты `IN` могут быть объединены.
<p id="descr">Объедините предикаты IN.</p>

## Примеры

Некорректно

```sql
SELECT *
FROM dbo.foo
WHERE category_id IN (1, 2, 3) 
    OR category_id IN (4, 5);
```

Корректно

```sql
SELECT *
FROM dbo.foo
WHERE category_id IN (1, 2, 3, 4, 5);
```
