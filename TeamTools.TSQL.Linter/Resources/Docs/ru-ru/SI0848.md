
# Схожие предикаты NOT IN могут быть объединены

|||
|-|-|

## Причина

Правило срабатывает, если схожие предикаты `NOT IN` могут быть объединены.
<p id="descr">Объедините предикаты NOT IN.</p>

## Примеры

Некорректно

```sql
SELECT *
FROM dbo.foo
WHERE category_id NOT IN (1, 2, 3) 
    AND category_id NOT IN (4, 5);
```

Корректно

```sql
SELECT *
FROM dbo.foo
WHERE category_id NOT IN (1, 2, 3, 4, 5);
```
