

# Неожидаемое изменение контекста выполнения

|||
|-|-|
| Id | **VU0503**
| Мнемо | DATABASE_CONTEXT_CHANGE
| Серьёзность | ⚠ Предупреждение
| Категория | [Уязвимость](./Категория_Уязвимость.md)
| Исходный код | [DatabaseContextChangeRule.cs](../../../Rules/Vulnerability/DatabaseContextChangeRule.cs)

## Причина

Правило срабатывает при обнаружении команды USE.
<p id="descr">Удалите команду изменения контекста базы данных.</p>

## Примеры

Некорректно

```sql
USE master;
```
